# Document Name: [Coding/Determinants] Cn

## Aliases / Search Terms

- Cn
- color naming
- named color

## Core Definition

`Cn` is used when the person names a color, but does not use color as a true determinant of the response.
That is why `Cn` does not belong to the full chromatic family `C / CF / FC` as a full color code.

## Application Conditions

- The person explicitly mentions a color.
- That mention does not support the main perceptual construction of the response.
- The object would remain the same even if the color reference were removed.
- If color truly participates in the response, reconsider [`FC`](ref://scoring-input/determinants/FC), [`CF`](ref://scoring-input/determinants/CF), or [`C`](ref://scoring-input/determinants/C).

## Cautions / Distinctions

- `Cn` is not the same as `C`. In `C`, color is a true determinant; in `Cn`, color is only named.
- A casual color mention should not be inflated into `FC` or `CF` if the response does not depend on it.
- `Cn` should not be counted interpretively as if it were full chromatic color.
- When in doubt, ask whether color makes it look like that, or whether the person is only describing an accessory feature.
