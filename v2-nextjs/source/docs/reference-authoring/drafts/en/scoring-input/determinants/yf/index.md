# Document Name: [Coding/Determinants] YF

## Aliases / Search Terms

- YF
- shading-form
- shading dominant

## Core Definition

`YF` is coded when diffuse shading dominates the response and form stays secondary.  
There is still some form support, but the vague shaded quality carries more weight than the percept's structure.

## Application Conditions

- The response begins from shading, darkness, or diffuse vagueness.
- Form participates, but only secondarily.
- If form ends up organizing the response, reconsider [`FY`](ref://scoring-input/determinants/FY).
- If there is barely any form support, reconsider [`Y`](ref://scoring-input/determinants/Y).

## Cautions / Distinctions

- `YF` should not be confused with [`VF`](ref://scoring-input/determinants/VF); one points to diffuse shading and the other to depth.
- `YF` should also not be confused with [`TF`](ref://scoring-input/determinants/TF); one points to shading and the other to texture.
- If the response depends more on relief or distance, review `VF`.
- The interpretive use of this family returns later in [`SumY`](ref://result-interpretation/lower-section/core/SumY).
