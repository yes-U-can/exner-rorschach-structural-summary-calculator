# Document Name: [Coding/Determinants] V

## Aliases / Search Terms

- V
- pure vista
- vista only

## Core Definition

`V` is coded when the response depends on depth or relief without enough form support.  
The dimensional experience dominates on its own and form does not truly organize the percept.

## Application Conditions

- The response grows out of distance, depth, or relief.
- There is not enough form support for `FV` or `VF`.
- Inquiry must show that the dimensional quality is really part of the person's response.
- If secondary form support appears, reconsider `VF`.

## Cautions / Distinctions

- `V` is not the same as Card `V`; the symbol is shared, but the context is different.
- If the response depends more on texture, review [`T`](ref://scoring-input/determinants/T).
- If the response depends more on diffuse shading or vague darkness, review [`Y`](ref://scoring-input/determinants/Y).
- The interpretive use of this family returns later in [`SumV`](ref://result-interpretation/lower-section/core/SumV).
