# Document Name: [Coding/Determinants] FC

## Aliases / Search Terms

- FC
- form-color
- color secondary to form

## Core Definition

`FC` is used when the response is based mainly on form and chromatic color remains secondary.
Color is present, but the person organizes the response chiefly through formal features.

## Application Conditions

- Form guides the main identification of the object.
- Chromatic color participates in the response, but with secondary importance.
- If color appears more dominant than form, reconsider [`CF`](ref://scoring-input/determinants/CF).
- If form is not really used, [`C`](ref://scoring-input/determinants/C) may fit better.

## Cautions / Distinctions

- `FC` does not mean merely that color is present in the card; color has to take part in a form-dominated response.
- The key distinction from `CF` is which aspect carries the main weight.
- Some responses first appear to be `CF`, but later clarification shows that form dominates and `FC` is correct.
- The interpretive use of this family is reviewed later in [`FC:CF+C`](ref://result-interpretation/lower-section/affect/FC_CF_C).
