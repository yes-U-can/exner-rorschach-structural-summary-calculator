# Document Name: [Coding/Determinants] FC'

## Aliases / Search Terms

- FC'
- achromatic form-color
- form with achromatic color

## Core Definition

`FC'` is used when form organizes the response and achromatic color remains secondary.
Black, white, or gray features must function clearly as color, not merely as shading or area description.

## Application Conditions

- Form supports the main identification of the object.
- Achromatic features participate as color, but with secondary weight.
- If achromatic color dominates the response, reconsider [`C'F`](ref://scoring-input/determinants/C%27F).
- If there is little or no real form support, [`C'`](ref://scoring-input/determinants/C%27) may fit better.

## Cautions / Distinctions

- `FC'` is not the same as [`FC`](ref://scoring-input/determinants/FC); the first uses achromatic color, the second chromatic color.
- Words like black, white, gray, light, or dark are not enough by themselves; they must function as color.
- If the response points more to depth or distance, review [`V`](ref://scoring-input/determinants/V).
- If the response points more to vague atmosphere, darkness, or diffuse distress, review [`Y`](ref://scoring-input/determinants/Y).
