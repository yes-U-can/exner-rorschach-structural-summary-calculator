# Document Name: [Coding/Determinants] Y

## Aliases / Search Terms

- Y
- pure shading
- diffuse shading only

## Core Definition

`Y` is coded when the response depends on diffuse shading without enough form support.  
The dominant experience is vague, shaded, or cloudy, and form does not truly organize the percept.

## Application Conditions

- The response grows out of shading, darkness, or diffuse uncertainty.
- There is not enough form support for `FY` or `YF`.
- Inquiry must confirm that shading is the real basis of the response.
- If some secondary form support appears, reconsider `YF`.

## Cautions / Distinctions

- `Y` is not the same as [`V`](ref://scoring-input/determinants/V); one points to diffuse shading and the other to depth.
- `Y` is also not the same as [`T`](ref://scoring-input/determinants/T); one points to shading and the other to texture.
- If the response uses real secondary form support, pure `Y` no longer fits.
- The interpretive use of this family returns later in [`SumY`](ref://result-interpretation/lower-section/core/SumY).
