# Document Name: [Coding/Determinants] CF

## Aliases / Search Terms

- CF
- color-form
- form secondary to color

## Core Definition

`CF` is used when chromatic color is more important than form, but form still contributes secondarily.
The response is color-driven, yet not completely free of form.

## Application Conditions

- Chromatic color carries the main weight of the response.
- Form still contributes enough to keep the response from being pure color.
- If form clearly dominates, use [`FC`](ref://scoring-input/determinants/FC).
- If form does not genuinely participate, use [`C`](ref://scoring-input/determinants/C).

## Cautions / Distinctions

- `CF` is not assigned just because the person mentions color. Color has to lead the percept.
- The distinction from `FC` depends on which aspect the person relies on more heavily.
- `CF` still differs from `C` because form remains present at a secondary level.
