# Document Name: [Coding/Determinants] FT

## Aliases / Search Terms

- FT
- form-texture
- texture secondary

## Core Definition

`FT` is coded when form organizes the response and texture stays secondary.  
The tactile or surface quality is present, but the response still rests first on form.

## Application Conditions

- Form identifies the main object.
- The person then adds a tactile, surface, or texture quality.
- The texture cue has to be genuinely stated in the response or inquiry.
- If texture carries more weight than form, reconsider [`TF`](ref://scoring-input/determinants/TF).

## Cautions / Distinctions

- `FT` is not the same as [`TF`](ref://scoring-input/determinants/TF); the difference is which feature dominates.
- If there is not a real tactile component, the response may remain [`F`](ref://scoring-input/determinants/F).
- If the dominant experience is depth or relief, review [`V`](ref://scoring-input/determinants/V).
- If the dominant experience is vague shaded atmosphere, review [`Y`](ref://scoring-input/determinants/Y).
