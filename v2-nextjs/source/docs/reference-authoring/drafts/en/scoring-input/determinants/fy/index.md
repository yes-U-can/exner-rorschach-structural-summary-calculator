# Document Name: [Coding/Determinants] FY

## Aliases / Search Terms

- FY
- form-shading
- shading secondary

## Core Definition

`FY` is coded when form organizes the response and diffuse shading stays secondary.  
The impression of shadow, cloudiness, or diffuse uncertainty is present, but the response still rests first on form.

## Application Conditions

- Form identifies the main object.
- The person then adds a diffuse shading quality.
- That quality must be genuinely present in the response or inquiry.
- If shading carries more weight than form, reconsider [`YF`](ref://scoring-input/determinants/YF).

## Cautions / Distinctions

- `FY` is not the same as [`FV`](ref://scoring-input/determinants/FV); one points to diffuse shading and the other to depth.
- `FY` is also not the same as [`FT`](ref://scoring-input/determinants/FT); one points to shading and the other to texture.
- If form does not really organize the response, pure [`Y`](ref://scoring-input/determinants/Y) may fit better.
- The interpretive use of this family returns later in [`SumY`](ref://result-interpretation/lower-section/core/SumY).
