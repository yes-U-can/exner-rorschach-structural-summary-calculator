# Document Name: [Coding/Determinants] C'F

## Aliases / Search Terms

- C'F
- achromatic color-form
- achromatic color with secondary form

## Core Definition

`C'F` is used when achromatic color dominates the response and form remains secondary.
The response depends more on black, white, or gray used as color than on formal organization.

## Application Conditions

- The person uses achromatic features as the primary basis of the response.
- Some formal support is present, but clearly secondary.
- If form carries the main weight, reconsider [`FC'`](ref://scoring-input/determinants/FC%27).
- If no real form support is present, [`C'`](ref://scoring-input/determinants/C%27) may fit better.

## Cautions / Distinctions

- `C'F` is not the same as [`CF`](ref://scoring-input/determinants/CF); both the type of color and the perceptual basis differ.
- Saying black, white, or gray does not by itself require `C'F`.
- If the response depends more on distance or relief, review [`V`](ref://scoring-input/determinants/V).
- If the response is mainly indefinite, murky, or diffuse shading, review [`Y`](ref://scoring-input/determinants/Y).
