# Document Name: [Coding/Determinants] F

## Aliases / Search Terms

- F
- pure form
- form only

## Core Definition

`F` is coded when the response is supported only by form.  
There is no operative use of movement, color, reflection, texture, vista, or additional dimension.

## Application Conditions

- Form must be the only explicit basis of the response.
- During inquiry, the person must still rely on contour, outline, or formal configuration.
- If another genuine determinant appears, `F` by itself is no longer enough.
- Final coding must follow what the person said, not free examiner inference.

## Cautions / Distinctions

- `F` is not the same as [`FQ`](ref://scoring-input/fq); `F` is a determinant and `FQ` is form quality.
- If the response includes depth created by contour or relative size, review [`FD`](ref://scoring-input/determinants/FD).
- If the response adds texture, vista, or diffuse shading, review [`FT`](ref://scoring-input/determinants/FT), [`V`](ref://scoring-input/determinants/V), or [`Y`](ref://scoring-input/determinants/Y).
- A response can include `F` inside a more complex combination, but that does not remove the need to decide whether another determinant actually dominates.
