# Document Name: [Coding/Determinants] C'

## Aliases / Search Terms

- C'
- pure achromatic color
- achromatic color without form

## Core Definition

`C'` is used when the response depends exclusively on achromatic color in the blot.
There is not enough formal support, and black, white, or gray clearly function as color.

## Application Conditions

- The response is based on black, white, or gray used as color.
- There is not enough formal organization for `FC'` or `C'F`.
- If inquiry shows real secondary form, reconsider `C'F`.
- The decision requires clear and unmistakable achromatic color use.

## Cautions / Distinctions

- `C'` is not the same as [`C`](ref://scoring-input/determinants/C); both are pure color, but one is achromatic and the other chromatic.
- Many achromatic cue words also occur in shading. Confirm that they function as color.
- If the response points to depth or distance, review [`V`](ref://scoring-input/determinants/V).
- If the response points to diffuse shading, darkness, or atmospheric tone, review [`Y`](ref://scoring-input/determinants/Y).
