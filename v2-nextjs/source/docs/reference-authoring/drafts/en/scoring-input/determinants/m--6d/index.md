# Document Name: [Coding/Determinants] m

## Aliases / Search Terms

- m
- inanimate movement
- passive movement of object

## Core Definition

`m` is used when the response implies movement of an inanimate object or passive movement not coded as human or animal movement.
It reflects mechanical, driven, or object-based movement.

## Application Conditions

- The perceived action is movement.
- The mover is inanimate, mechanical, or object-based.
- The movement does not qualify as human or ordinary animal movement.

## Cautions / Distinctions

- If the movement is human or anthropomorphic, use [`M`](ref://scoring-input/determinants/M).
- If it is ordinary animal movement, use [`FM`](ref://scoring-input/determinants/FM).
- Do not assign `m` merely because the response feels tense or unstable; there has to be perceived movement.
