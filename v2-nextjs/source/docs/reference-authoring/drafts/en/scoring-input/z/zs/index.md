# Document Name: [Coding/Z] ZS

## Aliases / Search Terms

- ZS
- white-space Z
- space-based organizational activity

## Core Definition

`ZS` is used when organizational activity depends on meaningful integration of white space.
The response has to organize white space as part of the percept, not merely notice it.

## Application Conditions

- The response uses form.
- White space is meaningfully integrated into the response.
- The organization depends on that white-space use.
- The response fits white-space organization better than `ZW`, `ZA`, or `ZD`.

## Cautions / Distinctions

- Visible white space alone is not enough for `ZS`; the white space has to function as part of the organized percept.
- If white space is used without Z-level organizational activity, code location with `S` but do not assume `ZS`.
- Keep the distinction clear between location coding with `S` and organizational activity coding with `ZS`.
