# Document Name: [Coding/Z] ZA

## Aliases / Search Terms

- ZA
- adjacent detail Z
- adjacent organizational activity

## Core Definition

`ZA` is used when organizational activity is based on bringing adjacent blot areas into a meaningful relation.
The response shows organization across nearby parts of the blot.

## Application Conditions

- The response uses form.
- The response links adjacent blot areas in a meaningful way.
- The organization is more than simple naming of nearby parts.
- `ZA` has to reflect actual integration, not just location change.

## Cautions / Distinctions

- Adjacent areas alone do not guarantee `ZA`; there must be organizational activity.
- If the organization is based on distant areas, consider [`ZD`](ref://scoring-input/z/ZD).
- If white space is the core organizing feature, consider [`ZS`](ref://scoring-input/z/ZS).
