# Document Name: [Coding/Z] Z

## Aliases / Search Terms

- Z
- organizational activity
- Z score coding

## Core Definition

`Z` codes organizational activity in the response.
It is assigned only when the response uses form and fits one of four Z patterns: `ZW`, `ZA`, `ZD`, or `ZS`.

## Application Conditions

- The response must include form use.
- Consider `Z` when the response shows organization among blot parts or between blot area and white space.
- If more than one Z pattern could apply in the same response, retain the type with the higher card value.
- `Wv` does not receive `Z`.
- Purely vague responses without enough formal support do not receive `Z`.

## Cautions / Distinctions

- `Z` is not a location code. It uses information from [`W`](ref://scoring-input/location/W), [`D`](ref://scoring-input/location/D), [`Dd`](ref://scoring-input/location/Dd), and [`S`](ref://scoring-input/location/S), but it codes something different.
- Listing parts of the blot is not enough. There has to be meaningful integration or relationship.
- In global responses, `ZW` also requires compatible DQ.
- `ZS` requires real white-space integration, not just visible white area inside the response area.
