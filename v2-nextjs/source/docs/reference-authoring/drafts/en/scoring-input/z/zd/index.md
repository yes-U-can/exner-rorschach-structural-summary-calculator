# Document Name: [Coding/Z] ZD

## Aliases / Search Terms

- ZD
- distant detail Z
- distant organizational activity

## Core Definition

`ZD` is used when organizational activity is based on linking distant blot areas into one response.
The response organizes non-adjacent areas into a meaningful whole.

## Application Conditions

- The response uses form.
- The response brings distant, non-adjacent blot areas into relation.
- The connection is meaningful and not just a list of separated parts.
- The response fits distant-area organization better than other Z subtypes.

## Cautions / Distinctions

- `ZD` is not assigned just because two areas are far apart. The response must organize them.
- If the organization is based on adjacent areas, consider [`ZA`](ref://scoring-input/z/ZA).
- `ZD` coding contributes later to the interpreted `Zd` pattern and should therefore be coded carefully.
