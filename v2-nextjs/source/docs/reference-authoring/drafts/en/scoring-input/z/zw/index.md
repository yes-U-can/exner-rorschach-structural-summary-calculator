# Document Name: [Coding/Z] ZW

## Aliases / Search Terms

- ZW
- global Z
- global organizational activity

## Core Definition

`ZW` is used when the response is global and also meets Z requirements.
The basis is a `W` response with compatible developmental quality.

## Application Conditions

- The response uses the whole blot.
- The response includes form use.
- The response has DQ of `+`, `v/+`, or `o`.
- The response is not `Wv`.

## Cautions / Distinctions

- A global response does not automatically receive `ZW`. If it is `Wv`, it does not receive Z.
- `ZW` depends on the combination of whole-blot location and compatible DQ.
- If the main organizational activity is based on adjacent details, distant details, or white space, another Z subtype may fit better.
