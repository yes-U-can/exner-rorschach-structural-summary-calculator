# Document Name: [Coding/Content] H

## Aliases / Search Terms

- H
- human
- whole human figure

## Core Definition

`H` is coded when the percept is a whole, realistic human figure.  
The response points to an entire person, not a body part and not a mythic or fantastic figure.

## Application Conditions

- The main object has to be a whole human figure.
- The figure must be presented as realistically human, not fantasy or mythology.
- If the figure is historically or culturally specific, [`Ay`](ref://scoring-input/contents/Ay) may be added as a secondary content code.
- If only a body part is described, reconsider [`Hd`](ref://scoring-input/contents/Hd).

## Cautions / Distinctions

- `H` is not the same as [`(H)`](ref://scoring-input/contents/%28H%29); `(H)` is reserved for fictional or mythic whole human figures.
- `H` is not the same as [`Hd`](ref://scoring-input/contents/Hd); `Hd` is used for an incomplete human detail.
- `H` is not the same as [`Hx`](ref://scoring-input/contents/Hx); `Hx` codes human experience rather than a person.
- Human content alone does not settle [`GHR`](ref://scoring-input/gphr/GHR) or [`PHR`](ref://scoring-input/gphr/PHR).
