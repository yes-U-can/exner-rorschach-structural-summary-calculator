# Document Name: [Coding/Content] Art

## Aliases / Search Terms

- Art
- art
- art object
- decorative object

## Core Definition

`Art` is coded when the main content is perceived as an artistic work, decorative object, or aesthetic product.  
The key point is not just seeing an object, but understanding it as something artistic or ornamental.

## Application Conditions

- Use it for paintings, statues, drawings, sculptures, emblems, or decorative pieces.
- It may be added together with another content code if the response names both the object and its artistic character.
- If the response is mainly anatomy or biological structure, first reconsider [`An`](ref://scoring-input/contents/An).
- If the main emphasis is historical, ethnic, or anthropological meaning rather than art object status, first reconsider [`Ay`](ref://scoring-input/contents/Ay).

## Cautions / Distinctions

- `Art` and [`Ay`](ref://scoring-input/contents/Ay) can co-occur, but they do not mean the same thing.
- `Art` centers on artwork or decoration; `Ay` centers on cultural or historical reference.
- A biological image presented like a mounted specimen or stylized illustration may take [`An`](ref://scoring-input/contents/An) as the main content and `Art` as a secondary code.
