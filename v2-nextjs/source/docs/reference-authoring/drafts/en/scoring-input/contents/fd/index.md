# Document Name: [Coding/Content] Fd

## Aliases / Search Terms

- Fd
- food
- edible object

## Core Definition

`Fd` is coded when the main content is food or something understood as edible.  
It is a content code, not a determinant.

## Application Conditions

- Use it for prepared food, a meal, meat, cake, served fruit, or another clearly edible item.
- It may also be used when the scene is organized around eating or a clearly edible object.
- If the main focus is the animal itself rather than food, first reconsider [`A`](ref://scoring-input/contents/A) or another animal content code.
- Do not confuse it with [`FD`](ref://scoring-input/determinants/FD), which belongs to determinants.

## Cautions / Distinctions

- `Fd` as content and [`FD`](ref://scoring-input/determinants/FD) as determinant belong to different coding axes.
- `Fd` and botanical content can seem close when vegetables are named, but `Fd` requires that they be organized as food.
- The later interpersonal meaning of [`Food`](ref://result-interpretation/lower-section/interpersonal/Food) does not change the coding decision itself.
