# Document Name: [Coding/Content] Hh

## Aliases / Search Terms

- Hh
- household
- domestic object

## Core Definition

`Hh` is coded when the main content is a household utensil or domestic object.  
The key point is that the form is understood as something used in the home, not as clothing and not as artwork by itself.

## Application Conditions

- Use it for cup, pot, lamp, plate, utensil, domestic tool, or another household-use object.
- The domestic object has to be the main content focus.
- If the object is mainly understood as decorative or artistic, it may also require [`Art`](ref://scoring-input/contents/Art).
- If the person stresses that the object is made from an animal part, it may coexist with [`Ad`](ref://scoring-input/contents/Ad).

## Cautions / Distinctions

- `Hh` does not cover clothing; for that, reconsider [`Cg`](ref://scoring-input/contents/Cg).
- `Hh` and [`Art`](ref://scoring-input/contents/Art) may come close, but they are not equivalent.
- `Hh` and [`Ad`](ref://scoring-input/contents/Ad) only overlap when the same household object is described as being made from an animal part.
