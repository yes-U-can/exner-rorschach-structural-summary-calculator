# Document Name: [Coding/Content] Cl

## Aliases / Search Terms

- Cl
- cloud
- clouds

## Core Definition

`Cl` is coded when the main content is a cloud.  
It is used for the cloud itself, not for every similar atmospheric form.

## Application Conditions

- Use it when the person names a cloud, cloud layer, or form clearly organized as a cloud.
- The cloud has to be the main content and not just a vague background.
- If the image is better understood as mist, vapor, or haze, first reconsider [`Na`](ref://scoring-input/contents/Na).
- If the form is seen as smoke or fire, first reconsider [`Fi`](ref://scoring-input/contents/Fi).

## Cautions / Distinctions

- `Cl` and [`Na`](ref://scoring-input/contents/Na) split cloud from mist or vapor.
- `Cl` and [`Fi`](ref://scoring-input/contents/Fi) can look visually close, but `Fi` requires smoke or fire content.
- It is not enough that something looks diffuse. The person has to organize it as a cloud.
