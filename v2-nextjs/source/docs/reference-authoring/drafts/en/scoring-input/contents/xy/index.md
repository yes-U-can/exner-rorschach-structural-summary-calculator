# Document Name: [Coding/Content] Xy

## Aliases / Search Terms

- Xy
- x-ray
- radiograph

## Core Definition

`Xy` is coded when the content is perceived as an X-ray or radiographic image.  
If the response is coded `Xy`, the focus is on the radiographic way of seeing the object.

## Application Conditions

- Use it for an X-ray, radiographic plate, internal image seen explicitly as an X-ray, or another radiographic study.
- The person has to indicate that it is being seen as an X-ray or radiographic image.
- When `Xy` is coded, do not add [`An`](ref://scoring-input/contents/An) as a secondary content code.
- This code later connects to the interpretation route [`An + Xy`](ref://result-interpretation/lower-section/selfPerception/An_Xy).

## Cautions / Distinctions

- `Xy` is not the same as [`An`](ref://scoring-input/contents/An); `An` points to anatomy and `Xy` points to radiographic format.
- If the person only names bone, organ, or internal body structure without saying it is an X-ray, first reconsider `An`.
- The important rule is that `Xy` excludes `An` as a companion code in the same response.
