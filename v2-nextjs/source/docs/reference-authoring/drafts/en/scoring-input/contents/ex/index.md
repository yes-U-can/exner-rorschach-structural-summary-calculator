# Document Name: [Coding/Content] Ex

## Aliases / Search Terms

- Ex
- explosion
- blast

## Core Definition

`Ex` is coded when the main content is an explosion or an explosive scene.  
The key point is the bursting event itself, not just smoke or brightness.

## Application Conditions

- Use it for explosion, blast, detonation, or a clearly named explosive scene.
- It may include sparks or debris as long as the person keeps the central idea of explosion.
- If the main focus is fire or smoke rather than explosion, first reconsider [`Fi`](ref://scoring-input/contents/Fi).

## Cautions / Distinctions

- `Ex` and [`Fi`](ref://scoring-input/contents/Fi) may coexist visually, but they are not the same code.
- `Ex` requires an explosive event; `Fi` requires fire or smoke as the main content.
- Do not use `Ex` for a mere sense of energy or motion.
