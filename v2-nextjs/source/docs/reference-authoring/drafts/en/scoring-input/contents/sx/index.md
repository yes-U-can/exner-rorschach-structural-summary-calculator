# Document Name: [Coding/Content] Sx

## Aliases / Search Terms

- Sx
- sex
- sexual content

## Core Definition

`Sx` is coded when the content includes a sexual organ, sexual activity, or explicit sexual meaning.  
It often works as a secondary content code alongside a human or anatomy code.

## Application Conditions

- Use it for genitals, sexualized breasts, intercourse, sexual act, or another direct sexual reference.
- The sexual content has to be actually present in the response and not inferred only by the examiner.
- It is often added together with [`H`](ref://scoring-input/contents/H), [`Hd`](ref://scoring-input/contents/Hd), or [`An`](ref://scoring-input/contents/An).
- If the response only indicates nudity or body form without clear sexual meaning, do not code `Sx` automatically.

## Cautions / Distinctions

- `Sx` has to be separated from [`An`](ref://scoring-input/contents/An), which centers on anatomy rather than sexuality.
- `Sx` does not by itself equal whole human content; it may need [`H`](ref://scoring-input/contents/H) or [`Hd`](ref://scoring-input/contents/Hd).
- Coding requires explicit sexual content, not an examiner's unsupported insinuation.
