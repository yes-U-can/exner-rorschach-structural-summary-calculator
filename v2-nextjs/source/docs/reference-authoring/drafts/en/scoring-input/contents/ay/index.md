# Document Name: [Coding/Content] Ay

## Aliases / Search Terms

- Ay
- anthropology
- cultural content
- historical content

## Core Definition

`Ay` is coded when the response includes a specific historical, ethnic, cultural, or anthropological reference.  
It is not enough to see a person or object; the response has to carry a recognizable cultural frame.

## Application Conditions

- Use it for figures, clothing, weapons, symbols, or scenes tied to a culture, era, or civilization.
- It may be combined with [`H`](ref://scoring-input/contents/H), [`(H)`](ref://scoring-input/contents/%28H%29), or [`A`](ref://scoring-input/contents/A) when the main object is still a person or animal.
- The key criterion is historical or cultural marking, not mere appearance.
- If the response sounds more like an artwork or decorative piece than a cultural reference, first reconsider [`Art`](ref://scoring-input/contents/Art).

## Cautions / Distinctions

- `Ay` and [`Art`](ref://scoring-input/contents/Art) may look close, but `Ay` depends on cultural or historical context.
- Do not code `Ay` only because a response sounds exotic; the cultural reference has to be identifiable.
- `Ay` often works as an additional content code rather than the only descriptive axis.
