# Document Name: [Coding/Content] An

## Aliases / Search Terms

- An
- anatomy
- anatomical content

## Core Definition

`An` is coded when the response centers on anatomy, body structure, or internal parts.  
It is not enough that the response seems like an animal fragment; the main idea has to be anatomical.

## Application Conditions

- Use it for bones, organs, musculature, X-ray views, or internal bodily structure.
- The main criterion is that the response points to anatomy rather than only to an external animal part.
- If the figure is only an external animal part, first consider [`Ad`](ref://scoring-input/contents/Ad).
- If the response turns into a diagrammatic or illustrative anatomy image, [`Art`](ref://scoring-input/contents/Art) may also be added when appropriate.

## Cautions / Distinctions

- `An` and [`Ad`](ref://scoring-input/contents/Ad) can look close, but `An` requires an anatomical focus.
- `An` is not assigned just because the person mentions a body part; the response should sound internal, structural, or biological.
- This content often matters later for routes such as [`PHR`](ref://scoring-input/gphr/PHR) or [`An + Xy`](ref://result-interpretation/lower-section/selfPerception/An_Xy), but it does not determine them by itself.
