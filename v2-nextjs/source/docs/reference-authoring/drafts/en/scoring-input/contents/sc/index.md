# Document Name: [Coding/Content] Sc

## Aliases / Search Terms

- Sc
- science
- scientific object
- technical object

## Core Definition

`Sc` is coded when the content is understood as scientific, technical, mechanical, or technological.  
The important point is that the person sees an apparatus, instrument, machine, or product of applied science.

## Application Conditions

- Use it for microscope, robot, machine, weapon, vehicle, technical device, laboratory instrument, or other scientific apparatus.
- It can also include ordinary technology when the response treats it as a device or mechanism.
- If the object has an artistic or decorative presentation, `Sc` may coexist with [`Art`](ref://scoring-input/contents/Art).
- Before sending an unusual response to [`Id`](ref://scoring-input/contents/Id), first test whether it fits `Sc`.

## Cautions / Distinctions

- `Sc` does not mean merely "strange"; it requires a scientific or technical reading.
- An unusual object such as a test tube, engine, or machine part still belongs under `Sc`, not automatically under [`Id`](ref://scoring-input/contents/Id).
- If the response emphasizes cultural symbol or aesthetic piece, it may require [`Art`](ref://scoring-input/contents/Art) in addition to `Sc`.
