# Document Name: [Coding/Content] Hd

## Aliases / Search Terms

- Hd
- human detail
- partial human

## Core Definition

`Hd` is coded when the response describes only a part of a human figure.  
There is not a whole person, but a bodily detail or human fragment.

## Application Conditions

- The main content has to be a part of a person rather than a whole figure.
- The part must be understood as realistically human.
- Typical examples are an arm, hand, leg, fingers, head, or incomplete torso.
- If the response builds a whole person, go back to [`H`](ref://scoring-input/contents/H).

## Cautions / Distinctions

- `Hd` is not the same as [`H`](ref://scoring-input/contents/H); there is no whole human figure here.
- `Hd` is not the same as [`(Hd)`](ref://scoring-input/contents/%28Hd%29); `(Hd)` is reserved for a fantastic or mythic human detail.
- `Hd` often pushes human representation toward poorer quality, so it is worth reviewing [`PHR`](ref://scoring-input/gphr/PHR).
- `Hd` is also not the same as [`Hx`](ref://scoring-input/contents/Hx); `Hx` codes human experience, not a body fragment.
