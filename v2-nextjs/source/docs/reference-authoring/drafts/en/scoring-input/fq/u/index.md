# Document Name: [Coding/Form Quality] u

## Aliases / Search Terms

- FQu
- unusual form quality
- unusual but plausible fit

## Core Definition

`FQu` is used when the response is uncommon or unusual, but still plausible and supportable by the blot area.
It is unusual fit, not distorted fit.

## Application Conditions

- The response is less common than ordinary good-fit responses.
- The object can still be seen as plausible without major forcing.
- The fit remains supportable even if it is not the most immediate reading.
- The response does not cross into clear distortion.

## Cautions / Distinctions

- `FQu` should not be used just because a response is creative. The fit must still be supportable.
- If the response is ordinary and clearly fitting, use [`FQo`](ref://scoring-input/fq/o).
- If the response depends on forced contours or clear distortion, use [`FQ-`](ref://scoring-input/fq/-).
