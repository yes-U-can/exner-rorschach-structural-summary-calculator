# Document Name: [Coding/Form Quality] +

## Aliases / Search Terms

- FQ+
- plus form quality
- elaborated good fit

## Core Definition

`FQ+` is used when the response shows good form fit and a more detailed or elaborated formal articulation than ordinary good responses.
It is both well fitted and especially well specified.

## Application Conditions

- The response already shows good baseline form fit.
- The subject articulates form features in a richer or more precise way than ordinary good responses.
- The elaboration improves the response without losing formal fit.
- `FQ+` remains less common than [`FQo`](ref://scoring-input/fq/o).

## Cautions / Distinctions

- A response is not `FQ+` just because it sounds complex or imaginative. The improvement must be formal.
- If the response is good but not clearly more elaborated than ordinary good fit, use [`FQo`](ref://scoring-input/fq/o).
- `FQ+` is not the same as [`DQ+`](ref://scoring-input/dq/%2B). One concerns form fit, the other organizational synthesis.
