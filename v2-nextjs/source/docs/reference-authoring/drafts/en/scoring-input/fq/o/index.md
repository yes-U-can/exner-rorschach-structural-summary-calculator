# Document Name: [Coding/Form Quality] o

## Aliases / Search Terms

- FQo
- ordinary form quality
- good ordinary fit

## Core Definition

`FQo` is used when the response shows acceptable and ordinary good fit to the blot.
It is the standard good-fit category and the most common positive FQ code.

## Application Conditions

- The object can be seen with reasonable speed and plausibility.
- The fit is acceptable without major distortion or forcing.
- The response does not show the extra elaboration needed for `FQ+`.
- The response is more clearly fitting than `FQu`.

## Cautions / Distinctions

- If the response is especially well articulated beyond ordinary good fit, use [`FQ+`](ref://scoring-input/fq/%2B).
- If the response is unusual but still plausible, consider [`FQu`](ref://scoring-input/fq/u).
- `FQo` is not the same as [`DQo`](ref://scoring-input/dq/o). One is form fit, the other is developmental quality.
