# Document Name: [Coding/Location] WS

## Aliases / Search Terms

- WS
- whole blot with white space
- global with S

## Core Definition

`WS` is used when the response uses the whole blot and also integrates white space as part of the percept.
It is a `W` response modified by meaningful white-space use.

## Application Conditions

- The whole blot is used.
- White space plays a real role in the percept.
- The white-space use is confirmed in inquiry rather than assumed from wording alone.

## Cautions / Distinctions

- `WS` is not the same as `W` plus a vague mention of background. White space must be structurally involved.
- If only a common detail plus white space is used, code [`DS`](ref://scoring-input/location/DS).
- If only an uncommon detail plus white space is used, code [`DdS`](ref://scoring-input/location/DdS).
