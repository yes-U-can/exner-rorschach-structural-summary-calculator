# Document Name: [Coding/Location] DdS

## Aliases / Search Terms

- DdS
- unusual detail with white space
- Dd with S

## Core Definition

`DdS` is used when an uncommon detail is combined with meaningful white-space use.
It is a `Dd` response modified by white-space integration.

## Application Conditions

- The main response area is an uncommon detail.
- White space plays a real part in the percept.
- The use of both the uncommon detail and white space is confirmed in inquiry.

## Cautions / Distinctions

- `DdS` should not be assigned just because the wording sounds unusual.
- If a common detail plus white space is used, code [`DS`](ref://scoring-input/location/DS).
- If the whole blot plus white space is used, code [`WS`](ref://scoring-input/location/WS).
