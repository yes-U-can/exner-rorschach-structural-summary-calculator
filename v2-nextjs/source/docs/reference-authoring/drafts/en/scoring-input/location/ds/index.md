# Document Name: [Coding/Location] DS

## Aliases / Search Terms

- DS
- common detail with white space
- D with S

## Core Definition

`DS` is used when a common detail is used together with meaningful white-space integration.
It is a `D` response modified by white-space use.

## Application Conditions

- The main response area is a common detail.
- White space plays a real part in the percept.
- The combined use is confirmed in inquiry.

## Cautions / Distinctions

- `DS` should not be assigned just because white space is mentioned casually.
- If the whole blot plus white space is used, code [`WS`](ref://scoring-input/location/WS).
- If an uncommon detail plus white space is used, code [`DdS`](ref://scoring-input/location/DdS).
