# Document Name: [Coding/Location] D

## Aliases / Search Terms

- D
- common detail
- usual detail location

## Core Definition

`D` is used when the response is based on a commonly used blot detail.
The code is about frequency of area use, not simply about the physical size of the area.

## Application Conditions

- The response is based on a blot area commonly used by respondents.
- The area is confirmed on the location sheet or in inquiry.
- The response does not require whole-blot use or uncommon-detail coding.

## Cautions / Distinctions

- `D` and `Dd` are separated by common versus uncommon use, not by “big” versus “small.”
- If white space is meaningfully integrated with the common detail, use [`DS`](ref://scoring-input/location/DS).
- `D` does not imply simplicity or normality by itself.
