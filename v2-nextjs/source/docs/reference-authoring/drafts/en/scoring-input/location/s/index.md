# Document Name: [Coding/Location] S

## Aliases / Search Terms

- S
- white space use
- space integration

## Core Definition

`S` indicates meaningful use of white space in the response.
It is not a stand-alone basic location code; it modifies another location decision.

## Application Conditions

- White space is actively used as part of the percept.
- The role of white space is confirmed in inquiry.
- Attach `S` to the relevant basic location code: `WS`, `DS`, or `DdS`.

## Cautions / Distinctions

- Mentioning background alone is not enough. White space has to function as part of the percept.
- `S` should not be coded by itself as if it were a complete location decision.
- Interpretation of `S` belongs to the Structural Summary pattern, not to location coding alone.
