# Document Name: [Coding/Location] W

## Aliases / Search Terms

- W
- whole blot response
- global location

## Core Definition

`W` is used when the response uses the whole blot.
It is not enough for the response to sound global; the whole blot must actually be used.

## Application Conditions

- The response includes all parts of the blot.
- No relevant blot sections are left out in constructing the object.
- The location is confirmed in inquiry or on the location sheet.

## Cautions / Distinctions

- A response commonly given to the whole blot should not receive `W` if the person leaves out important blot areas.
- If the whole blot is used together with white space, code [`WS`](ref://scoring-input/location/WS).
- `W` does not by itself mean good organization or good form quality.
