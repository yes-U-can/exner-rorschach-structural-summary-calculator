# Document Name: [Coding/Developmental Quality] DQ

## Aliases / Search Terms

- DQ
- Developmental Quality
- developmental quality coding

## Core Definition

Developmental Quality (`DQ`) codes how organized or elaborated the response is as a percept.
It focuses on the structural level of the response, not on whether the form fit is good or poor.

## Application Conditions

- Choose one DQ code for each scorable response.
- Decide whether the response shows meaningful synthesis.
- Decide whether at least one involved object requires specific form demand.
- Use the four-code set: `+`, `v/+`, `o`, `v`.

## Cautions / Distinctions

- `DQ` is not the same as [`FQ`](ref://scoring-input/fq). `DQ` is about organization; `FQ` is about form quality.
- The same symbol can mean something different in another family. For example, [`DQ+`](ref://scoring-input/dq/%2B) is not the same construct as [`FQ+`](ref://scoring-input/fq/%2B).
- DQ should be assigned from the response structure itself, not from later interpretation.
