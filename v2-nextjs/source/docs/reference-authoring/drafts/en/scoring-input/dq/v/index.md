# Document Name: [Coding/Developmental Quality] v

## Aliases / Search Terms

- DQv
- vague developmental quality
- vague non-synthesized response

## Core Definition

`DQv` is used when the response remains vague and does not show meaningful synthesis.
It is the least structurally developed DQ code in the standard four-code set.

## Application Conditions

- No meaningful synthesis is present.
- The percept does not require specific form demand.
- The response remains comparatively diffuse or impressionistic.
- The response does not fit `DQo`, `DQv/+`, or `DQ+`.

## Cautions / Distinctions

- If the object requires specific form demand but synthesis is absent, use [`DQo`](ref://scoring-input/dq/o).
- If synthesis is present without specific form demand, use [`DQv/+`](ref://scoring-input/dq/v/%2B).
- If synthesis is present with specific form demand, use [`DQ+`](ref://scoring-input/dq/%2B).
