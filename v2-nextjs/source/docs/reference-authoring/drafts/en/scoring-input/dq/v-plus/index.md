# Document Name: [Coding/Developmental Quality] v/+

## Aliases / Search Terms

- DQv/+
- v/+
- synthesis without formal demand

## Core Definition

`DQv/+` is assigned when the response shows synthesis, but the involved objects do not require specific form demand.
The response is organized, yet the objects remain comparatively vague.

## Application Conditions

- More than one object or part is combined into one organized response.
- The objects are meaningfully connected.
- No involved object clearly requires specific form demand.
- The response is more than simple listing or repetition.

## Cautions / Distinctions

- If at least one involved object requires specific form demand, use [`DQ+`](ref://scoring-input/dq/%2B) instead.
- If there is no meaningful synthesis and the object still requires specific form demand, use [`DQo`](ref://scoring-input/dq/o).
- If there is no meaningful synthesis and the object remains vague, use [`DQv`](ref://scoring-input/dq/v).
