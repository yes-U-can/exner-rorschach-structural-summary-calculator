# Document Name: [Coding/Cards] X

## Aliases / Search Terms

- Card X
- card X
- card 10
- plate X

## Core Definition

`Card X` combines many areas and many colors, so it often requires good delimitation of the response.  
This page summarizes the coding focal points that are most useful for that complexity.

## Application Conditions

- Review `Popular` when a clearly organized typical response appears.
- If color participates in the perception, review `FC`, `CF`, or `C`.
- If the response relies on multiple fragments and form becomes doubtful, review `FQ-`.
- The fact that the card is complex does not replace the need to describe the actual response clearly.

## Cautions / Distinctions

- A lot of color does not automatically mean a chromatic determinant.
- A lot of visual complexity does not automatically mean a Special Score.
- Coding still has to follow the actual response, not the apparent complexity of the plate.
