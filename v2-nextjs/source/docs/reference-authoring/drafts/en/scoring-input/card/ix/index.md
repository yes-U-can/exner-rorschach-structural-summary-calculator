# Document Name: [Coding/Cards] IX

## Aliases / Search Terms

- Card IX
- card IX
- card 9
- plate IX

## Core Definition

`Card IX` often requires more organization in order to maintain good form.  
This page summarizes what to watch in human content, movement, and formal quality.

## Application Conditions

- Some responses on this card approach human or humanoid figures, so it is useful to review `H` and `(H)`.
- If the response includes clear human action, review `M`.
- Because form can be less stable here, review `FQ` carefully.
- If a popular response appears, it still has to be confirmed by formal criteria rather than familiarity alone.

## Cautions / Distinctions

- A figure that merely seems human does not automatically mean `H`; it may fit `(H)` better.
- The organizational difficulty of the card does not by itself make the response poor form.
- This page does not interpret the card; it only orients coding.
