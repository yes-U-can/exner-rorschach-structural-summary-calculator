# Document Name: [Coding/Cards] VI

## Aliases / Search Terms

- Card VI
- card VI
- card 6
- plate VI

## Core Definition

`Card VI` often facilitates responses involving skin, texture, and form.  
This page summarizes what is useful to review in coding when those responses appear.

## Application Conditions

- Responses such as animal skin, leather, or hide rug are common.
- If the response uses touch or textural quality, review `T`, `TF`, or `FT` depending on the weight of form.
- If silhouette or form dominates without genuine tactile experience, review `F`.
- When the content is partial animal content, review `Ad`.
- If damage, stiffness, or deterioration is attributed to the skin or surface, also review `MOR`.

## Cautions / Distinctions

- Mentioning skin does not automatically imply a texture determinant.
- If tactile quality does not actually participate in the perception, do not add `T`.
- `Card VI` may facilitate certain popular responses, but `Popular` still has to be confirmed by formal criteria.
