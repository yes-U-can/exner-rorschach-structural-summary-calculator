# Document Name: [Coding/Cards] VIII

## Aliases / Search Terms

- Card VIII
- card VIII
- card 8
- plate VIII

## Core Definition

`Card VIII` often facilitates animal responses and can make the participation of color more noticeable.  
This page summarizes those coding focal points.

## Application Conditions

- Animal responses and popular responses tied to relatively clear zones are common.
- If color participates in building the response, review `FC`, `CF`, or `C`.
- If several areas are organized into one response, also review `Z`.
- When the response remains basically animal content, review `A` before complicating the coding.

## Cautions / Distinctions

- Visible color on the card is not enough by itself for a chromatic determinant.
- Not every animal response on this card is `Popular`.
- If form still dominates, the weight of color has to be evaluated carefully.
