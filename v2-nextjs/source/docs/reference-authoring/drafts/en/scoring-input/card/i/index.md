# Document Name: [Coding/Cards] I

## Aliases / Search Terms

- Card I
- card I
- card 1
- plate I

## Core Definition

`Card I` is the first plate in the protocol.  
This page summarizes the kinds of responses that often appear when coding begins; it is not a symbolic interpretation of the card.

## Application Conditions

- Global `W` responses are common on `Card I`.
- Simple form responses such as bat, butterfly, or other broad animal figures often appear.
- When the response uses the whole blot with clear organization, review `F`, `FQ`, and `Popular`.
- If the person uses only a small part of the blot, the card by itself does not justify `Popular`.

## Cautions / Distinctions

- Do not code `Popular` just because the response sounds familiar; it has to meet the formal standard for that card.
- `Card I` and `Card V` share bat or butterfly-type responses, but they are not treated as the same plate.
- Mentioning dark color verbally is not enough to add another determinant.
