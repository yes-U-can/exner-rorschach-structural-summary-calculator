# Document Name: [Coding/Special Scores] PSV

## Aliases / Search Terms

- PSV
- perseveration
- perseverative responding

## Core Definition

`PSV` is coded when the subject perseverates and repeats the same response or same kind of response in a rigid way.  
It does not refer to simple consistency; it points to stereotyped or mechanical repetition.

## Application Conditions

- Consider it when the same response or a very close pattern is repeated perseveratively.
- It may appear within the same card or across multiple cards.
- The important point is not simple similarity alone, but the quality of rigid repetition.
- The later interpretive meaning connects with [`PSV`](ref://result-interpretation/lower-section/processing/PSV).

## Cautions / Distinctions

- A similar response based on economy or preference is not enough by itself for `PSV`.
- Coding requires insistent or mechanical repetition, not merely a stable style.
- `PSV` marks the presence of perseveration; it does not by itself explain its cause.
