# Document Name: [Coding/Special Scores] DV1

## Aliases / Search Terms

- DV1
- deviant verbalization level 1
- mild deviant verbalization

## Core Definition

`DV1` is coded when the verbal expression of the response is odd or deviant, but only at a mild level.  
The core idea is that the wording sounds strange or improper without reaching a severe level of disturbance.

## Application Conditions

- Consider it when the response shows deviant wording, improper phrasing, or a strange verbal combination at a relatively mild level.
- The main problem lies in the expression itself, not in a broad breakdown of the whole response.
- If the verbal oddity is more marked, review [`DV2`](ref://scoring-input/special-score/DV2) first.
- If the disturbance seems to affect the structure of the whole response, also review [`DR1`](ref://scoring-input/special-score/DR1) and [`DR2`](ref://scoring-input/special-score/DR2).

## Cautions / Distinctions

- A long or awkward response is not enough by itself for `DV1`.
- If the difference between level 1 and level 2 is not clearly resolved, keep the more conservative level.
- `DV1` later contributes to measures such as [`Sum6`](ref://result-interpretation/lower-section/ideation/Sum6) and [`WSum6`](ref://result-interpretation/lower-section/ideation/WSum6), but the coding decision remains local to the response.
