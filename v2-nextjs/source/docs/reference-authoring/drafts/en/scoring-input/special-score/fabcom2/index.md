# Document Name: [Coding/Special Scores] FABCOM2

## Aliases / Search Terms

- FABCOM2
- fabulized combination level 2
- severe fabulized combination

## Core Definition

`FABCOM2` is a level 2 Special Score for fabulized relations that are clearly more severe and more implausible than `FABCOM1`.  
It is used when the organization of the response is distorted by a very improbable relation among its elements.

## Application Conditions

- Use `FABCOM2` when the relation among elements is markedly illogical, highly forced, or difficult to sustain as only a mild deviation.
- The main problem should lie in the fabulized connection of the response, not merely in a strange word or an isolated incongruous mixture.
- `FABCOM2` is reserved for the more severe version of this relational distortion.
- Its presence often foreshadows a later link with `Lv2` and the weight of `WSum6`.

## Cautions / Distinctions

- If the main oddity lies inside a single perceptual combination, compare it with `INCOM2` first.
- If the response fuses incompatible percepts into one contaminated perception, review `CONTAM`.
- If the distortion is present but does not reach this level of severity, use `FABCOM1`.
- `FABCOM2` is not assigned just for broad fantasy or poetic style; it requires a clearly fabulized relation.
