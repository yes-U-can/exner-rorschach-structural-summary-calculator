# Document Name: [Coding/Special Scores] PER

## Aliases / Search Terms

- PER
- personalized answers
- personalized justification

## Core Definition

`PER` is coded when the subject introduces personal knowledge, personal experience, or autobiographical reference in order to justify or support the response.  
The important point is that the personal information functions as direct support for the coding.

## Application Conditions

- Consider `PER` when the subject says the response is known to be true because of personal experience, prior learning, or personal knowledge.
- The personal information has to be used to justify or support the response itself.
- Personal pronouns alone are not enough for `PER`.
- This code later connects with [`PER`](ref://result-interpretation/lower-section/interpersonal/PER) and may also be reviewed with [`Xu%`](ref://result-interpretation/lower-section/mediation/Xu_percent).

## Cautions / Distinctions

- Saying "I see" or "it seems to me" is not enough by itself for `PER`.
- `PER` is not a content code; it marks a personalized way of justifying the response.
- A personal remark that is not tied to the specific response should not receive `PER`.
