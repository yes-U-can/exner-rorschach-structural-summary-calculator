# Document Name: [Coding/Special Scores] INCOM2

## Aliases / Search Terms

- INCOM2
- incongruous combination level 2
- severe incongruous combination

## Core Definition

`INCOM2` is a level 2 Special Score for incongruous combinations that are clearly more severe and stranger than `INCOM1`.  
It is used when the union of parts or qualities is markedly implausible or illogical within the response.

## Application Conditions

- Use `INCOM2` when the incongruous combination is intense, striking, and difficult to support as only a mild variation.
- The response should show a mixture of elements that is not just improper but clearly strange or severely mismatched.
- If the real question is whether the level is 1 or 2, reserve `INCOM2` for the more severe case.
- The presence of `INCOM2` often anticipates a link with `Lv2` and the weight of `WSum6`, although the code is still decided first at the response level.

## Cautions / Distinctions

- `INCOM2` is not used just because a response is long, odd, or highly imaginative.
- If the main disturbance is a distorted organization among ideas or actions inside the response, review `FABCOM2`.
- If two incompatible percepts are fused into one contaminated perception, review `CONTAM`.
- If the oddity is more a matter of strange wording than an incongruous perceptual combination, compare it with `DV2`.
