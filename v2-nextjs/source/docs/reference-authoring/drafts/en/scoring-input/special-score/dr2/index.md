# Document Name: [Coding/Special Scores] DR2

## Aliases / Search Terms

- DR2
- deviant response level 2
- severe deviant response

## Core Definition

`DR2` is coded when the full response shows a strong deviation that is clearly more severe than `DR1`.  
The overall organization appears seriously disturbed or far removed from a comprehensible course.

## Application Conditions

- Consider it when the deviation affects the response broadly and in a marked way.
- The disturbance is more severe than [`DR1`](ref://scoring-input/special-score/DR1).
- If the problem seems limited to isolated words or phrases, review [`DV1`](ref://scoring-input/special-score/DV1) and [`DV2`](ref://scoring-input/special-score/DV2) first.
- `DR2` later connects with [`Sum6`](ref://result-interpretation/lower-section/ideation/Sum6), [`Lv2`](ref://result-interpretation/lower-section/ideation/Lv2), and [`WSum6`](ref://result-interpretation/lower-section/ideation/WSum6).

## Cautions / Distinctions

- Do not use `DR2` just because a response is very long or very complex.
- `DR2` requires clear disruption of the overall response.
- If level 2 severity is not clear, do not force `DR2`.
