# Document Name: [Coding/Special Scores] ALOG

## Aliases / Search Terms

- ALOG
- inappropriate logic
- faulty logic

## Core Definition

`ALOG` is coded when the person supports the response with logic that is improper or clearly insufficient.  
The focus is on an unjustified inference, not on a strange word or a perceptual mixture.

## Application Conditions

- Use `ALOG` when the explanation or justification of the response shows an inappropriate logical leap.
- The conclusion being offered should sound poorly supported even within the subject's own framework.
- The problem lies in the logic used to support the idea, not only in verbal form.
- `ALOG` contributes to `Sum6` and may later influence `WSum6`.

## Cautions / Distinctions

- Do not assign `ALOG` just for unusual style, dramatized tone, or a response that is hard to follow.
- If the main oddity lies in a deviant word or phrase, compare it with `DV1` first.
- If the deviation affects the global organization of the response, also compare it with `DR1`.
- Responses induced by follow-up questioning should not automatically be turned into `ALOG`.
