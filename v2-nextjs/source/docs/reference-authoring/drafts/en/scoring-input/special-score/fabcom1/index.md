# Document Name: [Coding/Special Scores] FABCOM1

## Aliases / Search Terms

- FABCOM1
- fabulized combination level 1
- mild fabulized combination

## Core Definition

`FABCOM1` is a level 1 Special Score for fabulized or implausible relations between elements of a response.  
It does not describe a single incongruous combination inside one object, but a distorted relation among ideas, actions, or parts of the scene.

## Application Conditions

- Use `FABCOM1` when the response organizes an improbable or forced relation between elements that should connect in a different way.
- The central problem is the fabulized logic of the whole arrangement, not merely an incongruous mixture inside one object.
- The oddity has to be observable, but not yet so severe that it requires `FABCOM2`.
- If there is doubt between level 1 and level 2, reserve `FABCOM2` for the more extreme distortion.

## Cautions / Distinctions

- If the main oddity lies in an incongruous combination within a single perceptual unit, review `INCOM1` first.
- If two incompatible percepts are fused into one contaminated perception, review `CONTAM`.
- `FABCOM1` is not used just because a response seems fantastic or imaginative.
- The key point is that the relation between elements becomes forced or implausible within the response.
