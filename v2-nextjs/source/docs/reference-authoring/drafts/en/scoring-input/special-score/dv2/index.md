# Document Name: [Coding/Special Scores] DV2

## Aliases / Search Terms

- DV2
- deviant verbalization level 2
- severe deviant verbalization

## Core Definition

`DV2` is coded when the verbal deviation is clearly more severe and marked than in `DV1`.  
The wording appears notably distorted, improper, or strange at a high level.

## Application Conditions

- Consider it when the verbal oddity is intense and not limited to a minor impropriety.
- The main problem is still verbal, but at a more severe level than [`DV1`](ref://scoring-input/special-score/DV1).
- If the whole response seems broadly disorganized, also review [`DR2`](ref://scoring-input/special-score/DR2).
- `DV2` later feeds measures such as [`Sum6`](ref://result-interpretation/lower-section/ideation/Sum6), [`Lv2`](ref://result-interpretation/lower-section/ideation/Lv2), and [`WSum6`](ref://result-interpretation/lower-section/ideation/WSum6).

## Cautions / Distinctions

- If the severity is not clearly level 2, do not force `DV2`.
- `DV2` is not decided by verbal length or complexity alone.
- The difference from `DR2` is that `DV2` is still centered on verbal deviation, whereas `DR2` points to broader disruption in the full response.
