# Document Name: [Coding/Special Scores] INCOM1

## Aliases / Search Terms

- INCOM1
- incongruous combination level 1
- mild incongruous combination

## Core Definition

`INCOM1` is a level 1 Special Score for incongruous combinations within a single response.  
It is used when joined elements do not fit well together, but the oddity does not yet reach the more severe level of `INCOM2`.

## Application Conditions

- Consider `INCOM1` when two parts, qualities, or functions are combined incongruously within the same object or scene.
- The combination has to sound improper or poorly matched, not just uncommon.
- If the response is odd but can still be followed with some logic, `INCOM1` is often more appropriate than `INCOM2`.
- When the incongruity is limited to an isolated word or an odd verbal phrase, first review whether `DV1` or `DV2` fits better.

## Cautions / Distinctions

- `INCOM1` is not assigned just because a response is creative, fantastic, or unusual.
- If the main problem is an impossible relation between parts of the response, but with a more distorted organization, review `FABCOM1`.
- If two incompatible realities are fused into one contaminated percept, review `CONTAM`.
- If there is doubt between level 1 and level 2, use `INCOM1` conservatively and reserve `INCOM2` for the more severe incongruity.
