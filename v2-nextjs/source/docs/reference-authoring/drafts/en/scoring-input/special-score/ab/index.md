# Document Name: [Coding/Special Scores] AB

## Aliases / Search Terms

- AB
- abstractions
- abstract response

## Core Definition

`AB` is coded when the response moves away from a concrete object and shifts into an abstract, generalized, or loosely conceptual formulation.  
The key point is that the subject names an abstraction or broad quality rather than a clearly bounded object.

## Application Conditions

- Consider `AB` when the response names something like an emotion, an idea, a force, a sensation, or another abstract formulation.
- It may appear near [`Hx`](ref://scoring-input/contents/Hx) when the response points to generalized human experience rather than a specific person.
- It may also coexist with [`Art`](ref://scoring-input/contents/Art) or [`Ay`](ref://scoring-input/contents/Ay) when the subject turns the content into a highly abstract category.
- This is added as a Special Score and does not replace the main content code.

## Cautions / Distinctions

- Not every symbolic or artistic response receives `AB`.
- `Hx` by itself does not require `AB`; the wording has to be genuinely abstract.
- A response that is unusual but still concrete should not receive `AB` just because it sounds sophisticated.
