# Document Name: [Coding/Special Scores] DR1

## Aliases / Search Terms

- DR1
- deviant response level 1
- mild deviant response

## Core Definition

`DR1` is coded when the response as a whole departs from the expected course in a mild way.  
The deviation is not just in a single word, but in the overall organization or development of the response.

## Application Conditions

- Consider it when the response includes a mild digression, derailment, or improper development.
- The disturbance affects the whole response more than a single verbal expression.
- If the problem seems centered only in wording, review [`DV1`](ref://scoring-input/special-score/DV1) and [`DV2`](ref://scoring-input/special-score/DV2) first.
- If the global disruption is clearly more severe, also review [`DR2`](ref://scoring-input/special-score/DR2).

## Cautions / Distinctions

- A long or complicated response is not enough by itself for `DR1`.
- `DR1` differs from `DV` because it addresses deviation in the full response, not only in word choice.
- If there is doubt between level 1 and level 2, keep the more conservative level.
