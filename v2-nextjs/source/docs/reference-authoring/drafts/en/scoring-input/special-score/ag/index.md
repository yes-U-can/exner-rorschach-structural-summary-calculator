# Document Name: [Coding/Special Scores] AG

## Aliases / Search Terms

- AG
- aggressive movement
- aggressive action

## Core Definition

`AG` is coded when a movement response shows clearly aggressive action.  
What matters is aggression as an ongoing act, not just a vague hostile quality of the object.

## Application Conditions

- The response must include movement, usually [`M`](ref://scoring-input/determinants/M), [`FM`](ref://scoring-input/determinants/FM), or [`m`](ref://scoring-input/determinants/m).
- The action has to be observably aggressive: striking, attacking, tearing, violently pushing, biting, or injuring.
- The code is used for aggression as movement, not merely for damage that has already happened.
- If the aggressive action is carried out jointly by more than one figure, `AG` may coexist with [`COP`](ref://scoring-input/special-score/COP).

## Cautions / Distinctions

- A hostile adjective without clear action does not justify `AG`.
- If the main focus is that the object is already broken, dead, or deteriorated, first reconsider [`MOR`](ref://scoring-input/special-score/MOR).
- `AG` does not automatically decide [`PHR`](ref://scoring-input/gphr/PHR); that decision requires an additional coding step.
