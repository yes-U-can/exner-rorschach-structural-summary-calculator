# Document Name: [Interpretation/Special Indices] PTI
## Aliases / Search Terms
- PTI
- Perceptual-Thinking Index
- perceptual-thinking index

## Core Definition
`PTI` is a composite index that groups signs of perceptual and thinking disorganization.
Its function is to alert the reader to review possible distortion of reality, ideational slippage, or loss of perceptual precision.
## Interpretation Points
- If `PTI` is positive, mediation and ideation should be reviewed first.
- The index usually becomes more meaningful when problems converge in `WSum6`, `Lv2`, `M-`, `X-%`, and `XA%`.
- Its interpretation points to possible thought disturbance, not an automatic diagnosis.
- The index gains weight when several different parts of the protocol point in the same direction.
## Cautions / Distinctions
- A positive `PTI` does not by itself mean psychosis.
- A negative `PTI` does not remove all possibility of disturbed thinking.
- Extreme stress, intoxication, situational disorganization, or very low response volume can complicate the reading.
## Limits of Isolated Interpretation
- `PTI` is an actuarial signal and requires returning to the base variables.
- `WSum6`, `Lv2`, `M-`, `X-%`, and `XA%` should be reviewed before drawing strong conclusions.
