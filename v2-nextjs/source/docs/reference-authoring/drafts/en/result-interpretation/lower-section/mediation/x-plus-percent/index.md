# Document Name: [Interpretation/Mediation] X+%

## Aliases / Search Terms

- X+%
- Conventional Form

## Core Definition

`X+%` indicates the proportion of responses whose form is conventionally and clearly shareable.
It summarizes how strongly mediation stays aligned with common readings of the blot.

## Interpretation Points

- If `X+%` is high, perception usually stays closer to stable and consensual translations.
- If it drops, there may be less reliance on conventional readings and more weight given to unusual or less precise translations.
- Reading it with `P` helps show whether socially shared reference points are still available.

## Cautions / Distinctions

- `X+%` is not the same as `XA%`; `XA%` includes a broader band of adequacy.
- A low `X+%` does not automatically equal severe distortion; `Xu%` and `X-%` also need to be reviewed.
- In short protocols, the ratio can shift with only a few responses.
