# Document Name: [Interpretation/Core] EB

## Aliases / Search Terms

- EB
- Erlebnistypus
- Experience Balance
- experiential balance

## Core Definition

`EB` is the ratio `M : WSumC`.
It shows whether the person tends to rely more on ideational, reflective elaboration or more on affective modulation and emotional response when dealing with problems.

## Interpretation Points

- If `M` predominates, the style is usually described as more `introversive`.
- If `WSumC` predominates, the style is usually described as more `extratensive`.
- If there is a clear but not extreme direction, there may be a preferred style with some flexibility.
- If no clear direction appears, the style is usually described as `ambitent`, suggesting less consistency in coping direction.

## Cautions / Distinctions

- A marked direction does not automatically equal health or pathology.
- If the difference is very large, `EBPer` should also be checked.
- In short protocols, protocols with high `Lambda`, or protocols with low `EA`, `EB` can look more rigid or emptier than it actually is.
