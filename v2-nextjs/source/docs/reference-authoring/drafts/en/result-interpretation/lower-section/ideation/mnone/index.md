# Document Name: [Interpretation/Ideation] Mnone

## Aliases / Search Terms

- Mnone
- formless human movement
- human movement without form

## Core Definition

`Mnone` indicates human movement responses that do not have enough form to support a regular `M` interpretation.
In interpretation, it functions as a sign that human fantasy or imagined human activity is present but weakly structured.

## Interpretation Points

- If `Mnone` appears, human representations may be vague, weakly organized, or only loosely anchored in form.
- It can point to human imagination that is present but not clearly shaped enough to carry the same weight as well-formed `M`.
- If it appears with `M-`, `WSum6`, or `SCON`, concern about ideational vulnerability rises.

## Cautions / Distinctions

- `Mnone` is not the same as `M-`; `Mnone` lacks sufficient form, while `M-` has minus form quality.
- A single `Mnone` should not be treated as proof of major ideational disturbance.
- The `M` coding should be checked first so that the response is not misclassified.
