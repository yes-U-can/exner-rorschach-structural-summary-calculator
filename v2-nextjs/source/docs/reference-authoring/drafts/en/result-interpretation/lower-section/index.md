## Aliases / Search Terms

- lower-section
- lower section

## Core Definition

`Lower Section` is the set of clusters where the structural summary is read as a functional pattern of resources, stress, ideation, affect, mediation, processing, interpersonal style, and self-perception.
If `Upper Section` works as the first screen, `Lower Section` is usually where clinical formulation becomes denser.

## Interpretation Points

- The reading flow usually follows `core -> ideation -> affect -> mediation -> processing -> interpersonal -> selfPerception`.
- A single number matters less than the way several indicators support or correct one another.
- This is where early signals from `Upper Section` are confirmed, qualified, or changed in meaning.

## Cautions / Distinctions

- `Lower Section` clusters influence one another; isolated reading increases the risk of overgeneralizing.
- Protocol length, defensiveness, cultural context, and situational stress can all change the weight of a variable.
