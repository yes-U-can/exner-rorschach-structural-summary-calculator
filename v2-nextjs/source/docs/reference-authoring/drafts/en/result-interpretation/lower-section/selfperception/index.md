# Document Name: [Interpretation/Lower Section/Self Perception] Self Perception

## Aliases / Search Terms

- self perception
- self-image
- self-regard

## Core Definition

`Self Perception` gathers the variables that help read how the person views the self, how much value is assigned to the self, how much self-observation is present, and what tone the self-image takes.
In this cluster, the main review points are `3r+(2)/R`, `Fr+rF`, `SumV`, `FD`, `An+Xy`, `MOR`, and `H ratio`.

## Interpretation Points

- `3r+(2)/R` helps estimate how much weight the self occupies in the protocol.
- `Fr+rF` helps show whether the self-image needs special shine, enhancement, or confirmation.
- `SumV` and `FD` help distinguish self-criticism, self-observation, and concern about personal limitations.
- `An+Xy`, `MOR`, and `H ratio` help show whether the self-image is experienced as fragile, damaged, partial, or less integrated.

## Cautions / Distinctions

- This cluster does not answer only whether the person "likes" the self or not.
- Self-observation, egocentricity, self-criticism, and bodily vulnerability can rise for different reasons.
- No variable in this family should be read apart from interpersonal and affective context.

## Limits of Isolated Interpretation

- Self-image shifts with affective state, stress, and recent history.
- `Self Perception` should be integrated with `Interpersonal`, `Affect`, and `Core`.
