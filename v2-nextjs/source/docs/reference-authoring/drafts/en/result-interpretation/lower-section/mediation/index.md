# Document Name: [Interpretation/Lower Section/Mediation] Mediation

## Aliases / Search Terms

- mediation
- cognitive mediation

## Core Definition

`Mediation` gathers the variables used to examine how the person translates the blot into a shareable perception.
This section focuses on adequacy, unusualness, and the degree to which perception departs from shared reality.

## Interpretation Points

- `XA%`, `WDA%`, `X+%`, and `P` help estimate closeness to more conventional forms of perception.
- `Xu%` shows less common but still possible perceptions.
- `X-%` and `S-%` help estimate how far mediation departs from shared reality.
- This family should be read together with `Processing` and `Ideation`, not by itself.

## Cautions / Distinctions

- A single variable is not enough to summarize all perceptual quality.
- If `R` is low, several ratios become less stable.
- `XA%` and `WDA%` are related but not identical: `WDA%` looks only at more common areas, while `XA%` covers the broader acceptable range.
