# Document Name: [Interpretation/Affect] CP

## Aliases / Search Terms

- CP
- color projection

## Core Definition

`CP` in affect is used to read attribution of color or emotional tone to a field that would not ordinarily contain it.
It is reviewed as a sign that affect is imposing itself on perception in an unusual way.

## Interpretation Points

- If `CP` appears, there may be a tendency to load more neutral contexts with emotion.
- Because it is rare, even a single case deserves clinical attention.
- It is often useful to see whether it appears together with shading determinants or other markers of tension.

## Cautions / Distinctions

- `CP` is not the same as `Pure C`, `CF`, or `C`.
- The special-score coding should be confirmed before interpretation.
- Its rarity gives it clinical weight, but it is not enough by itself for a broad conclusion.
