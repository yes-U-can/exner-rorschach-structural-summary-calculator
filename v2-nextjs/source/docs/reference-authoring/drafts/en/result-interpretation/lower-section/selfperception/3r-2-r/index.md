# Document Name: [Interpretation/Self Perception] Egocentricity Index

## Aliases / Search Terms

- 3r+(2)/R
- Egocentricity Index
- egocentricity ratio

## Core Definition

`3r+(2)/R` is an egocentricity index calculated as `[3 x (Fr + rF) + Sum(2)] / R`. Each reflection determinant is therefore weighted as three pair responses; the multiplier is not two despite the `(2)` pair notation in the label.
In interpretation, it helps estimate how much weight the self, personal value, or personal importance occupies in the perceptual field.

## Interpretation Points

- If the index is high, there may be greater self-focus, need for self-confirmation, or sensitivity to personal value.
- If it is very high, not only self-interest but also the need to validate or protect self-esteem may increase.
- If it is low, there may be less self-assertion, less investment in personal importance, or diminished self-valuation.
- The index reflects self-concern, not healthy self-esteem by itself.

## Cautions / Distinctions

- In adults, values below `.33` or above `.44` often deserve more careful review, but the index does not decide by itself.
- If `R` is low, the ratio can look more extreme than it really is.
- A high value does not automatically equal clinical narcissism.

## Limits of Isolated Interpretation

- This index should be read with `Fr+rF`, `FD`, `SumV`, `MOR`, and `H ratio`.
- Its meaning changes if there are very few pair responses or very few total responses.
