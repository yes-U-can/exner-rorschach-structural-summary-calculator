# Document Name: [Interpretation/Interpersonal] SumT

## Aliases / Search Terms

- SumT
- interpersonal SumT
- texture total
- need for closeness

## Core Definition

`SumT_inter` reads `SumT` from the interpersonal level.
In interpretation, it helps estimate how much need for closeness, contact, and supportive holding appears in relatedness.

## Interpretation Points

- If `SumT_inter` is high, there may be greater need for emotional proximity and protective contact.
- The value says more about need for closeness than about relational skill.
- If it appears together with `COP`, the search for closeness may be represented inside more reciprocal relationships.
- If it combines with `ISO_Index`, there may be need for closeness alongside distance or isolation.
- With `Food`, the relational need may take on a more dependent or longing quality.

## Cautions / Distinctions

- A high value does not automatically mean good relationships.
- A zero value does not eliminate all need for closeness; it also depends on overall use of shading.
- `SumT` in `Core` should be distinguished from `SumT_inter` in its relational reading.

## Limits of Isolated Interpretation

- This variable needs interpersonal context to become clear.
- It should be read with `Food`, `COP`, `HumanCont`, and `ISO_Index`.
