# Document Name: [Interpretation/Ideation] Lv2

## Aliases / Search Terms

- Lv2
- Level 2 special scores
- level 2 markers

## Core Definition

`Lv2` shows how many level-2 special scores appear in the protocol.
In interpretation, it indicates how much weight the more severe cognitive peculiarities carry within thought.

## Interpretation Points

- If `Lv2 = 0`, no level-2 markers appear in the special-score set.
- If there is only one occurrence, the context should be checked carefully before overstating it.
- If several level-2 occurrences appear, the likelihood of more marked ideational disorganization or distortion increases.

## Cautions / Distinctions

- `Lv2` carries more weight because of severity than because of simple frequency.
- It is not enough by itself to conclude a thought disorder; it should be read with `WSum6`, `M-`, `PTI`, and mediation variables.
- `CONTAM` also counts as level 2, but its structure is different from `DV2`, `DR2`, `INCOM2`, or `FABCOM2`.
