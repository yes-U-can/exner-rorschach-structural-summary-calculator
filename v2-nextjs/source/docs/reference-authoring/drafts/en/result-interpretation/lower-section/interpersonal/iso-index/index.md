# Document Name: [Interpretation/Interpersonal] Isolation Index

## Aliases / Search Terms

- ISO Index
- Isolation Index
- Isolate/R
- isolation ratio

## Core Definition

`ISO_Index` is the ratio `(Bt + 2Cl + Ge + Ls + 2Na) / R`.
In interpretation, it helps estimate how much the record is organized around distance, social withdrawal, or limited interpersonal participation.

## Interpretation Points

- If the index rises above `.24` or `.25`, it is worth reviewing hypotheses about social distance or interpersonal withdrawal more carefully.
- A value near `1/3` or higher increases concern about meaningful isolation.
- When `ISO_Index` is high and `COP` is low, relational participation may look limited or weakly reciprocal.
- If `ISO_Index` is high while `SumT_inter` or `Food` also rise, need for closeness may coexist with a tendency toward withdrawal.

## Cautions / Distinctions

- Because it is a ratio, it can be inflated when `R` is low.
- It should not be read as a stand-alone proof of chronic isolation.
- The variable needs context from `COP`, `HumanCont`, `CDI`, and `DEPI`.

## Limits of Isolated Interpretation

- `ISO_Index` alone does not explain whether isolation is defensive, depressive, or situational.
- It should be read together with response volume and other interpersonal variables.
