# Document Name: [Interpretation/Core] D

## Aliases / Search Terms

- D
- D score
- current stress tolerance

## Core Definition

`D` is not the raw difference itself. First calculate `EA - es`, then convert that raw difference to the scaled D score. A raw difference from `-2.5` through `+2.5` yields `D = 0`; beyond that interval, D changes by one unit for each additional `2.5` raw-score points, preserving the sign. The same conversion rule continues beyond the usual printed `-5` to `+5` table when an extreme protocol requires it.

It summarizes whether available resources are enough to manage the current level of demand without clear overload.

## Interpretation Points

- `D = 0` usually suggests approximate balance between resources and current demand.
- `D > 0` suggests adaptive margin and less immediate risk of becoming overwhelmed by the current situation.
- `D < 0` suggests that current demand exceeds available resources and may be associated with tension, irritability, or lower frustration tolerance.
- `D` is especially sensitive to current conditions, so it can move with recent situational stress.

## Cautions / Distinctions

- `D = 0` does not automatically mean sound adjustment; a simplified or defensive protocol can look balanced.
- `D > 0` does not by itself prove stable overall strength.
- `D < 0` points to current overload, but the comparison with `AdjD` is needed to judge whether the strain is more situational or more persistent.
