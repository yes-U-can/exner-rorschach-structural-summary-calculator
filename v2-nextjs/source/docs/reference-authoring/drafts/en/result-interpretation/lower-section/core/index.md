# Document Name: [Interpretation/Lower Section/Core] Core

## Aliases / Search Terms

- core
- basic resources and stress
- core cluster

## Core Definition

`Core` is the cluster where the protocol's basic resources, current level of demand, coping balance, and tension load are reviewed together.
This is where `R`, `Lambda`, `EB`, `EA`, `eb`, `es`, `D`, `AdjD`, and the tension variables such as `m`, `SumCprime`, `SumT`, `SumV`, and `SumY` are read as a set.

## Interpretation Points

- `R` helps determine whether the protocol offers enough material for stable interpretation.
- `Lambda` shows the degree of simplification or economy in processing.
- `EB`, `EA`, `eb`, `es`, `D`, and `AdjD` help estimate resources, demand, and coping balance.
- `FM`, `m`, `SumCprime`, `SumT`, `SumV`, and `SumY` qualify the nature of tension and internal burden.

## Cautions / Distinctions

- `Core` variables influence one another; a single number can be misleading.
- In short protocols or protocols with very high `Lambda`, several values in the cluster can look reduced or distorted.
