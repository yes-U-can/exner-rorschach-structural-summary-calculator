# Document Name: [Interpretation/Processing] Zd

## Aliases / Search Terms

- Zd
- processing Zd
- processing efficiency

## Core Definition

`Zd` compares the actual level of organizational scanning with the expected level.
It is used to estimate whether processing tends to be balanced, excessive, or insufficient.

## Interpretation Points

- Values near the expected range suggest a more proportionate scanning and organizing style.
- A strongly positive `Zd` can point to overincorporation or a tendency to take in more than necessary.
- A strongly negative `Zd` can suggest underincorporation or a tendency to move through the field too quickly.
- Reading it with `PSV` helps distinguish inefficiency driven by rigidity from simple perceptual economy.

## Cautions / Distinctions

- A positive `Zd` is not automatically better.
- A negative `Zd` does not by itself imply severe impairment.
- The value needs context from `Zf`, `PSV`, and `W:D:Dd`.
