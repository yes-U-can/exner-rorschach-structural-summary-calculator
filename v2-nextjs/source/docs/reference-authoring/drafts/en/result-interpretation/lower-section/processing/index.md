# Document Name: [Interpretation/Lower Section/Processing] Processing

## Aliases / Search Terms

- processing
- information processing

## Core Definition

`Processing` gathers the variables that describe how the person scans, organizes, and segments the perceptual field.
This section is usually read through `Zf`, `Zd`, `W:D:Dd`, `W:M`, `PSV`, `DQ+`, and `DQv`.

## Interpretation Points

- This group helps show how much material is examined, how much complexity is taken in, and how flexibly the field is organized.
- `Zf` and `Zd` help estimate scope and efficiency of scanning.
- `W:D:Dd` and `W:M` show segmentation style and level of organization.
- `PSV`, `DQ+`, and `DQv` add information about repetition, organizational quality, and oversimplification.

## Cautions / Distinctions

- Processing is not the same as mediation; a person can scan widely and still interpret inaccurately.
- Processing is also not the same as ideation; perceptual segmentation and conceptual elaboration are different levels.
- In short protocols, several ratios in this family lose stability.
