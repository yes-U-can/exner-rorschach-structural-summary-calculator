# Document Name: [Interpretation/Affect] Pure C

## Aliases / Search Terms

- Pure C
- pure color

## Core Definition

`Pure C` is the frequency of color responses that do not have formal support.
In interpretation, it is used as a sign of more direct affective expression that is less shaped by structure or formal control.

## Interpretation Points

- If `Pure C` appears, there may be moments when feeling is expressed more immediately.
- Repeated `Pure C` suggests that affect can override formal organization in the response.
- In adult protocols, even a small amount deserves attention because the base rate is usually low.

## Cautions / Distinctions

- `Pure C` does not by itself equal pathological impulsivity.
- It should be distinguished from `CF`, `FC`, and `CP`.
- If there were few opportunities for chromatic responses, absence of `Pure C` does not mean absence of affect.
