# Nombre del documento: [Interpretacion/Upper Section] DQ+

## Alias y busqueda

- DQ+
- synthesized responses
- numero de respuestas sintetizadas

## Definicion central

`DQ+` es la frecuencia de respuestas con calidad evolutiva bien integrada.
En interpretacion indica con cuanta regularidad aparece una organizacion compleja y bien articulada del campo.

## Puntos de interpretacion

- La presencia de `DQ+` sugiere capacidad para integrar partes del estimulo en configuraciones mas complejas.
- Un `DQ+` relativamente alto puede reflejar mayor complejidad organizativa disponible dentro del protocolo.
- Un `DQ+` bajo puede indicar predominio de respuestas mas simples o pocos intentos de integracion.

## Precauciones y distinciones

- Un `DQ+` alto no se interpreta automaticamente como superioridad cognitiva.
- Si `DQ+` es alto pero `FQ` o `Zd` son debiles, la complejidad puede no ser eficiente.
- Un `DQ+` bajo tambien puede aparecer en protocolos breves o muy defensivos.

## Limites de interpretacion aislada

- `DQ+` debe leerse dentro de toda la distribucion de `DQ`.
- Conviene distinguir organizacion compleja de simple extension verbal.
