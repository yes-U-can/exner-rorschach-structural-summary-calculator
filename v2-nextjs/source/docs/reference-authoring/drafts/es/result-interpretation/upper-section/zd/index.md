# Nombre del documento: [Interpretación/Upper Section] Zd

## Alias y búsqueda

- Zd
- Z difference
- diferencia Z

## Definición central

`Zd` es la diferencia entre la organización real (`ZSum`) y la organización esperada (`ZEst`).
Es el indicador más directo de eficiencia organizativa dentro de la familia `Z`.

## Puntos de interpretación

- Un `Zd` cercano a cero suele sugerir equilibrio razonable entre frecuencia y carga organizativa.
- Un `Zd` claramente positivo puede apuntar a sobreincorporación, exceso de chequeo o demasiada elaboración del campo.
- Un `Zd` claramente negativo puede sugerir subincorporación, apresuramiento o inspección insuficiente del estímulo.

## Precauciones y distinciones

- Un valor positivo no se confunde automáticamente con meticulosidad adaptativa.
- Un valor negativo no se traduce automáticamente en deterioro.
- El significado mejora cuando se revisa junto con `Lambda`, `R`, `Zf` y `ZSum`.

## Límites de interpretación aislada

- `Zd` es fuerte como señal de estilo de procesamiento, pero sigue siendo una inferencia de patrón.
- Debe integrarse con `processing` antes de cerrar una conclusión.
