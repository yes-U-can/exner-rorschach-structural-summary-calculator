# Nombre del documento: [Interpretacion/Upper Section] Zd

## Alias y busqueda

- Zd
- Z difference
- diferencia Z

## Definicion central

`Zd` es la diferencia entre la organizacion real (`ZSum`) y la organizacion esperada (`ZEst`).
Es el indicador mas directo de eficiencia organizativa dentro de la familia `Z`.

## Puntos de interpretacion

- Un `Zd` cercano a cero suele sugerir equilibrio razonable entre frecuencia y carga organizativa.
- Un `Zd` claramente positivo puede apuntar a sobreincorporacion, exceso de chequeo o demasiada elaboracion del campo.
- Un `Zd` claramente negativo puede sugerir subincorporacion, apresuramiento o inspeccion insuficiente del estimulo.

## Precauciones y distinciones

- Un valor positivo no se confunde automaticamente con meticulosidad adaptativa.
- Un valor negativo no se traduce automaticamente en deterioro.
- El significado mejora cuando se revisa junto con `Lambda`, `R`, `Zf` y `ZSum`.

## Limites de interpretacion aislada

- `Zd` es fuerte como senal de estilo de procesamiento, pero sigue siendo una inferencia de patron.
- Debe integrarse con `processing` antes de cerrar una conclusion.
