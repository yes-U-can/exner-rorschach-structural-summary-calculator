# Nombre del documento: [Interpretación/Índices Especiales] Obsessive Style Index

## Alias y búsqueda

- OBS
- Obsessive Style Index
- índice de estilo obsesivo

## Definición central

`OBS` es una combinación yes/no de variables relacionadas con precisión, detalle y organización.
La evidencia de validez de constructo es insuficiente para inferir rasgos obsesivos o deterioro funcional.

## Puntos de interpretación

- Un resultado positivo significa primero que se cumplió la combinación operacional; confirme sobrecontrol, perfeccionismo y rigidez de manera independiente en entrevista y conducta.
- Ese estilo puede mejorar orden y precisión, pero también volver más rígido el procesamiento.
- La constelación gana peso cuando coinciden `Zf`, `Zd`, `Dd`, `P` y `FQ+`.
- El índice apunta a dirección estilística, no a trastorno obsesivo por sí solo.

## Precauciones y distinciones

- `OBS` es yes/no, no mide grado y tampoco establece por sí solo el estilo propuesto.
- No debe confundirse con un diagnóstico de TOC.
- Contextos de exigencia técnica o cultural pueden aumentar rasgos de control sin patología.

## Límites de interpretación aislada

- Para entender el estilo hace falta volver a las variables base.
- Conviene leerlo junto con `Zf`, `Zd`, `Dd`, `P` y `FQ+`.
