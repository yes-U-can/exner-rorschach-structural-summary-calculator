# Nombre del documento: [Interpretacion/Special Indices] Obsessive Style Index

## Alias y busqueda

- OBS
- Obsessive Style Index
- indice de estilo obsesivo

## Definicion central

`OBS` es una constelacion que resume tendencia a precision excesiva, control, minuciosidad y chequeo rigido.
Se usa para detectar un estilo obsesivo de organizacion mas que un diagnostico formal.

## Puntos de interpretacion

- Si `OBS` se activa, puede haber sobrecontrol, atencion extrema al detalle y necesidad fuerte de exactitud.
- Ese estilo puede mejorar orden y precision, pero tambien volver mas rigido el procesamiento.
- La constelacion gana peso cuando coinciden `Zf`, `Zd`, `Dd`, `P` y `FQ+`.
- El indice apunta a direccion estilistica, no a trastorno obsesivo por si solo.

## Precauciones y distinciones

- `OBS` es si/no; no mide grado fino.
- No debe confundirse con un diagnostico de TOC.
- Contextos de exigencia tecnica o cultural pueden aumentar rasgos de control sin patologia.

## Limites de interpretacion aislada

- Para entender el estilo hace falta volver a las variables base.
- Conviene leerlo junto con `Zf`, `Zd`, `Dd`, `P` y `FQ+`.
