# Nombre del documento: [Interpretacion/Special Indices] Hypervigilance Index

## Alias y busqueda

- HVI
- Hypervigilance Index
- indice de hipervigilancia

## Definicion central

`HVI` es una constelacion que busca un estilo de alerta defensiva, distancia y vigilancia exagerada frente al entorno.
Cuando se activa, sugiere que la persona puede priorizar control y proteccion antes que confianza.

## Puntos de interpretacion

- Si `HVI` se activa, puede haber estilo interpersonal cauteloso, desconfiado o muy pendiente de amenaza.
- El patron suele leerse mas como disposicion defensiva que como diagnostico paranoide automatico.
- La constelacion gana sentido cuando coincide con `SumT_inter`, `Zf`, `Zd`, `S_aff`, `H_ratio` y `HumanCont`.
- Tambien puede influir el contexto de peligro real o experiencias recientes de amenaza.

## Precauciones y distinciones

- `HVI` es una decision si/no; su ausencia no elimina vigilancia elevada.
- Factores culturales y situacionales pueden influir en el estilo de distancia o reserva.
- La regla primaria con `T = 0` y las condiciones secundarias necesitan leerse juntas.

## Limites de interpretacion aislada

- `HVI` muestra un patron de alerta y defensa, no explica por si solo su causa.
- Conviene volver a `S`, `Zf`, `Zd`, `H_ratio` y variables interpersonales.
