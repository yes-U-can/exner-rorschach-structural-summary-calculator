# Nombre del documento: [Interpretación/Índices Especiales] Hypervigilance Index

## Alias y búsqueda

- HVI
- Hypervigilance Index
- índice de hipervigilancia

## Definición central

`HVI` es una combinación yes/no de variables propuesta para representar alerta defensiva y distancia interpersonal.
El respaldo empírico para interpretarla como hipervigilancia o desconfianza es débil e inconsistente, así que un resultado positivo queda como hipótesis de baja confianza.

## Puntos de interpretación

- Si `HVI` se activa, compruebe primero si peligro real, trauma, reserva cultural o vigilancia observada apoyan la hipótesis de forma independiente.
- El patrón suele leerse más como disposición defensiva que como diagnóstico paranoide automático.
- La constelación gana sentido cuando coincide con `SumT_inter`, `Zf`, `Zd`, `S_aff`, `H_ratio` y `HumanCont`.
- También puede influir el contexto de peligro real o experiencias recientes de amenaza.

## Precauciones y distinciones

- La decisión yes/no no confirma vigilancia cuando es positiva, y su ausencia tampoco elimina vigilancia elevada.
- Factores culturales y situacionales pueden influir en el estilo de distancia o reserva.
- La regla primaria con `T = 0` y las condiciones secundarias necesitan leerse juntas.

## Límites de interpretación aislada

- `HVI` muestra un patrón de alerta y defensa, no explica por sí solo su causa.
- Conviene volver a `S`, `Zf`, `Zd`, `H_ratio` y variables interpersonales.
