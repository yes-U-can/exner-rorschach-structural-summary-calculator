# Nombre del documento: [Interpretacion/Lower Section/Ideacion] Ideacion

## Alias y busqueda

- ideacion
- ideation
- pensamiento

## Definicion central

`Ideacion` es el conjunto de variables que ayuda a leer la direccion del pensamiento, el grado de organizacion de las representaciones y la presencia de desajustes o peculiaridades cognitivas.
En esta zona se revisan juntos `a:p`, `Ma:Mp`, `2AB+Art+Ay`, `MOR`, `Sum6`, `Lv2`, `WSum6`, `M-` y `Mnone`.

## Puntos de interpretacion

- `a:p` y `Ma:Mp` ayudan a ver si la actividad mental tiende mas hacia la iniciativa o hacia la pasividad.
- `2AB+Art+Ay` muestra cuanto se desplaza el pensamiento hacia formulaciones mas abstractas, artificiales o alejadas de la experiencia directa.
- `MOR` aporta el peso de deterioro, dano, perdida o fracaso dentro del contenido ideacional.
- `Sum6`, `Lv2` y `WSum6` resumen la magnitud de las peculiaridades cognitivas.
- `M-` y `Mnone` ayudan a ver cuan bien se organiza el pensamiento ligado al movimiento humano.

## Precauciones y distinciones

- Este grupo no debe leerse sin considerar la calidad del protocolo y el contexto de `Mediation` y `Processing`.
- Una variable aislada ligeramente elevada no basta para concluir distorsion ideacional importante.
- Cuando `M` es muy escaso, algunos cocientes o contrastes pierden estabilidad.

## Limites de interpretacion aislada

- `Ideacion` describe como se orienta y se organiza el pensamiento, pero no explica por si sola la fuente exacta del problema.
- Conviene leerla junto con `Mediation`, `Processing`, `Core` y, cuando corresponda, `PTI` o `SCON`.
