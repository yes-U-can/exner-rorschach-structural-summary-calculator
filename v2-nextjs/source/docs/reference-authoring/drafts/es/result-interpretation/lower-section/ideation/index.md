# Nombre del documento: [Interpretación/Lower Section/Ideación] Ideación

## Alias y búsqueda

- ideación
- ideation
- pensamiento

## Definición central

`Ideación` es el conjunto de variables que ayuda a leer la dirección del pensamiento, el grado de organización de las representaciones y la presencia de desajustes o peculiaridades cognitivas.
En esta zona se revisan juntos `a:p`, `Ma:Mp`, `2AB+Art+Ay`, `MOR`, `Sum6`, `Lv2`, `WSum6`, `M-` y `Mnone`.

## Puntos de interpretación

- `a:p` y `Ma:Mp` ayudan a ver si la actividad mental tiende más hacia la iniciativa o hacia la pasividad.
- `2AB+Art+Ay` muestra cuánto se desplaza el pensamiento hacia formulaciones más abstractas, artificiales o alejadas de la experiencia directa.
- `MOR` aporta el peso de deterioro, daño, pérdida o fracaso dentro del contenido ideacional.
- `Sum6`, `Lv2` y `WSum6` resumen la magnitud de las peculiaridades cognitivas.
- `M-` y `Mnone` ayudan a ver cuán bien se organiza el pensamiento ligado al movimiento humano.

## Precauciones y distinciones

- Este grupo no debe leerse sin considerar la calidad del protocolo y el contexto de `Mediation` y `Processing`.
- Una variable aislada ligeramente elevada no basta para concluir distorsión ideacional importante.
- Cuando `M` es muy escaso, algunos cocientes o contrastes pierden estabilidad.

## Limites de interpretación aislada

- `Ideación` describe como se orienta y se organiza el pensamiento, pero no explica por sí sola la fuente exacta del problema.
- Conviene leerla junto con `Mediation`, `Processing`, `Core` y, cuando corresponda, `PTI` o `SCON`.
