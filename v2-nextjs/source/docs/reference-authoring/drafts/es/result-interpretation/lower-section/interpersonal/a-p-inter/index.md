# Nombre del documento: [Interpretacion/Interpersonal] a:p

## Alias y busqueda

- a:p
- active:passive
- proporcion activo:pasivo

## Definicion central

`a:p` muestra la proporcion entre movimiento activo y movimiento pasivo cuando se lee desde el angulo interpersonal.
En esta zona ayuda a estimar si la persona imagina el vinculo desde una posicion mas activa o mas receptiva.

## Puntos de interpretacion

- Si el polo pasivo pesa mas, puede haber mayor expectativa de recibir, esperar o depender del otro.
- Si el polo activo domina, el vinculo puede imaginarse con mayor iniciativa, intervencion o direccion propia.
- La variable describe una direccion relacional, no la calidad total del vinculo.
- Compararla con `COP`, `AG` y `SumT_inter` ayuda a distinguir iniciativa, dependencia y conflicto.

## Precauciones y distinciones

- Una diferencia pequena puede exagerarse si el total de movimientos es bajo.
- Predominio pasivo no equivale automaticamente a pasividad clinica patologica.
- Predominio activo no equivale automaticamente a autonomia sana; con `AG` alto puede volverse mas controlador o agresivo.

## Limites de interpretacion aislada

- `a:p` solo muestra una direccion del rol relacional.
- Conviene leerlo junto con `Ma:Mp`, `COP`, `AG`, `SumT_inter` y `M`.
