# Nombre del documento: [Interpretación/Interpersonal] a:p

## Alias y búsqueda

- a:p
- active:passive
- proporción activo:pasivo

## Definición central

`a:p` muestra la proporción entre movimiento activo y movimiento pasivo cuando se lee desde el ángulo interpersonal.
En esta zona ayuda a estimar si la persona imagina el vínculo desde una posición más activa o más receptiva.

## Puntos de interpretación

- Si el polo pasivo pesa más, puede haber mayor expectativa de recibir, esperar o depender del otro.
- Si el polo activo domina, el vínculo puede imaginarse con mayor iniciativa, intervención o dirección propia.
- La variable describe una dirección relacional, no la calidad total del vínculo.
- Compararla con `COP`, `AG` y `SumT_inter` ayuda a distinguir iniciativa, dependencia y conflicto.

## Precauciones y distinciones

- Una diferencia pequeña puede exagerarse si el total de movimientos es bajo.
- Predominio pasivo no equivale automáticamente a pasividad clínica patológica.
- Predominio activo no equivale automáticamente a autonomía sana; con `AG` alto puede volverse más controlador o agresivo.

## Límites de interpretación aislada

- `a:p` solo muestra una dirección del rol relacional.
- Conviene leerlo junto con `Ma:Mp`, `COP`, `AG`, `SumT_inter` y `M`.
