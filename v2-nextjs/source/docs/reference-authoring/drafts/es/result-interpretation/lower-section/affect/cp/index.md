# Nombre del documento: [Interpretacion/Afecto] CP

## Alias y busqueda

- CP
- color projection
- proyeccion de color

## Definicion central

`CP` se usa en afecto para leer la atribucion de color o tono emocional a un campo que normalmente no lo contiene.
Se mira como una senal de afecto que se impone o colorea la percepcion de un modo poco habitual.

## Puntos de interpretacion

- Si aparece `CP`, puede haber una tendencia a cargar de emocion contextos mas neutros.
- Dado que es una senal rara, incluso un solo caso merece atencion clinica.
- A menudo conviene revisar si se combina con determinantes de sombreado o con otras senales de tension.

## Precauciones y distinciones

- `CP` no es lo mismo que `Pure C`, `CF` o `C`.
- Primero hay que confirmar que en la codificacion realmente se cumple el criterio de special score `CP`.
- Su rareza le da peso clinico, pero no alcanza por si solo para una conclusion amplia.

## Limites de interpretacion aislada

- `CP` muestra una forma particular de intrusion afectiva, pero no define por si solo direccion ni diagnostico.
- Debe leerse junto con `S`, `SumY`, `Pure C` y `DEPI`.
