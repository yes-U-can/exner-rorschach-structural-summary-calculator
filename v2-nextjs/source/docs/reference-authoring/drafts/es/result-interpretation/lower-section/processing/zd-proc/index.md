# Nombre del documento: [Interpretación/Procesamiento] Zd

## Alias y búsqueda

- Zd
- processing Zd
- processing efficiency
- eficiencia de procesamiento

## Definición central

`Zd` compara el nivel real de exploración organizativa con el nivel esperado.
Se usa para estimar si el procesamiento tiende a ser equilibrado, excesivo o insuficiente.

## Puntos de interpretación

- Valores cercanos al rango esperado sugieren un estilo de escaneo y organización más proporcionado.
- Un `Zd` muy positivo puede apuntar a sobreincorporación o a una tendencia a tomar más de lo necesario.
- Un `Zd` muy negativo puede sugerir subincorporación o una tendencia a pasar demasiado rápido por el campo.
- Leerlo con `PSV` ayuda a distinguir entre ineficiencia por rigidez y simple economía perceptiva.

## Precauciones y distinciones

- Un `Zd` positivo no es automáticamente mejor.
- Un `Zd` negativo no implica por sí solo deterioro grave.
- El valor necesita contexto de `Zf`, `PSV` y `W:D:Dd`.

## Limites de interpretación aislada

- `Zd` solo habla de la proporción entre esfuerzo real y esperado.
- Conviene leerlo junto con `Zf`, `PSV`, `W:D:Dd`, `DQ+` y `DQv`.
