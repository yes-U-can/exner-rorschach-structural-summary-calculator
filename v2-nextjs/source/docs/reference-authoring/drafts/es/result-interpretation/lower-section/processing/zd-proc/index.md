# Nombre del documento: [Interpretacion/Procesamiento] Zd

## Alias y busqueda

- Zd
- processing Zd
- processing efficiency
- eficiencia de procesamiento

## Definicion central

`Zd` compara el nivel real de exploracion organizativa con el nivel esperado.
Se usa para estimar si el procesamiento tiende a ser equilibrado, excesivo o insuficiente.

## Puntos de interpretacion

- Valores cercanos al rango esperado sugieren un estilo de escaneo y organizacion mas proporcionado.
- Un `Zd` muy positivo puede apuntar a sobreincorporacion o a una tendencia a tomar mas de lo necesario.
- Un `Zd` muy negativo puede sugerir subincorporacion o una tendencia a pasar demasiado rapido por el campo.
- Leerlo con `PSV` ayuda a distinguir entre ineficiencia por rigidez y simple economia perceptiva.

## Precauciones y distinciones

- Un `Zd` positivo no es automaticamente mejor.
- Un `Zd` negativo no implica por si solo deterioro grave.
- El valor necesita contexto de `Zf`, `PSV` y `W:D:Dd`.

## Limites de interpretacion aislada

- `Zd` solo habla de la proporcion entre esfuerzo real y esperado.
- Conviene leerlo junto con `Zf`, `PSV`, `W:D:Dd`, `DQ+` y `DQv`.
