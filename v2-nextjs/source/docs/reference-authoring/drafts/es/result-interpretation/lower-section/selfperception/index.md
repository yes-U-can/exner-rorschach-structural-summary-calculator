# Nombre del documento: [Interpretacion/Lower Section/Self Perception] Self Perception

## Alias y busqueda

- self perception
- self-image
- self-regard

## Definicion central

`Self Perception` reune las variables que ayudan a leer como la persona se mira a si misma, cuanto valor se asigna, cuanta observacion interna hace y que tono tiene su autoimagen.
En este grupo se revisan `3r+(2)/R`, `Fr+rF`, `SumV`, `FD`, `An+Xy`, `MOR` y `H ratio`.

## Puntos de interpretacion

- `3r+(2)/R` orienta sobre cuanto peso ocupa el yo dentro del protocolo.
- `Fr+rF` ayuda a ver si la autoimagen necesita reafirmacion o brillo especial.
- `SumV` y `FD` ayudan a distinguir autocritica, observacion de si y preocupacion por las propias limitaciones.
- `An+Xy`, `MOR` y `H ratio` muestran si la autoimagen se vive como fragil, danada, parcial o menos integrada.

## Precauciones y distinciones

- Este grupo no responde solo si la persona "se quiere" o "no se quiere".
- Autoobservacion, egocentrismo, autocritica y vulnerabilidad corporal pueden subir por razones distintas.
- Ninguna variable de este grupo debe leerse aislada del contexto interpersonal y afectivo.

## Limites de interpretacion aislada

- La autoimagen cambia con el estado afectivo, el estres y la historia reciente.
- Conviene integrar `Self Perception` con `Interpersonal`, `Affect` y `Core`.
