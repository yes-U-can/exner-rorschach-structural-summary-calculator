# Nombre del documento: [Interpretacion/Interpersonal] AG

## Alias y busqueda

- AG
- aggressive movement
- movimiento agresivo

## Definicion central

`AG` muestra con que frecuencia el protocolo representa escenas interpersonales en clave agresiva, hostil o de confrontacion.
En interpretacion sirve para leer si el vinculo se organiza con mayor facilidad alrededor del conflicto o la amenaza.

## Puntos de interpretacion

- Si `AG` es alto, puede haber una tendencia a imaginar el contacto interpersonal como mas tenso, agresivo o confrontativo.
- La ausencia de `AG` no elimina todo conflicto, pero reduce el peso de escenas abiertamente agresivas.
- `AG` describe una forma de representar el vinculo, no una prueba directa de conducta violenta.
- Leerlo junto con `COP` ayuda a diferenciar protocolos dominantemente cooperativos de protocolos mas ambivalentes o conflictivos.

## Precauciones y distinciones

- `AG` no debe traducirse de inmediato en riesgo conductual.
- El valor cambia de sentido si coexiste con `MOR`, `S_aff` o `PHR`.
- Conviene revisar si la agresion aparece como defensa, expectativa o simple tonalidad imaginada.

## Limites de interpretacion aislada

- `AG` es solo una pieza del estilo interpersonal.
- Conviene leerlo junto con `COP`, `ISO_Index`, `SumT_inter`, `MOR` y `PHR`.
