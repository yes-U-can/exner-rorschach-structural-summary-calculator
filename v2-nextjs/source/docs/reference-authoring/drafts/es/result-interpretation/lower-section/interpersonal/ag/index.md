# Nombre del documento: [Interpretación/Interpersonal] AG

## Alias y búsqueda

- AG
- aggressive movement
- movimiento agresivo

## Definición central

`AG` muestra con qué frecuencia el protocolo representa escenas interpersonales en clave agresiva, hostil o de confrontación.
En interpretación sirve para leer si el vínculo se organiza con mayor facilidad alrededor del conflicto o la amenaza.

## Puntos de interpretación

- Si `AG` es alto, puede haber una tendencia a imaginar el contacto interpersonal como más tenso, agresivo o confrontativo.
- La ausencia de `AG` no elimina todo conflicto, pero reduce el peso de escenas abiertamente agresivas.
- `AG` describe una forma de representar el vínculo, no una prueba directa de conducta violenta.
- Leerlo junto con `COP` ayuda a diferenciar protocolos dominantemente cooperativos de protocolos más ambivalentes o conflictivos.

## Precauciones y distinciones

- `AG` no debe traducirse de inmediato en riesgo conductual.
- El valor cambia de sentido si coexiste con `MOR`, `S_aff` o `PHR`.
- Conviene revisar si la agresión aparece como defensa, expectativa o simple tonalidad imaginada.

## Límites de interpretación aislada

- `AG` es solo una pieza del estilo interpersonal.
- Conviene leerlo junto con `COP`, `ISO_Index`, `SumT_inter`, `MOR` y `PHR`.
