# Nombre del documento: [Interpretacion/Procesamiento] Zf

## Alias y busqueda

- Zf
- processing Zf
- Z frequency
- frecuencia Z

## Definicion central

`Zf` indica cuantas veces la persona intenta escanear y organizar activamente el campo durante el procesamiento.
En esta zona se usa como una medida de frecuencia del esfuerzo organizativo, no como una medida de eficiencia por si sola.

## Puntos de interpretacion

- Si `Zf` es alto, suele haber mas intentos de examinar y organizar el estimulo.
- Si `Zf` es bajo, puede haber menos exploracion activa o menor iniciativa para estructurar el campo.
- El sentido clinico cambia segun se lea junto con `Zd`, porque frecuencia y eficiencia no son lo mismo.

## Precauciones y distinciones

- Un `Zf` alto no implica automaticamente buen procesamiento.
- Un `Zf` bajo puede ser esperable si el protocolo es corto.
- Para interpretar el valor, conviene revisar tambien la distribucion `W:D:Dd`.

## Limites de interpretacion aislada

- `Zf` solo muestra con que frecuencia aparece el esfuerzo organizativo.
- Conviene leerlo junto con `Zd`, `W:D:Dd`, `DQ+` y `DQv`.
