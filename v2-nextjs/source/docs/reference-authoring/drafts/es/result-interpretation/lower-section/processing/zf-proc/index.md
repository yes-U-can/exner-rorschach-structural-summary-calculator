# Nombre del documento: [Interpretación/Procesamiento] Zf

## Alias y búsqueda

- Zf
- processing Zf
- Z frequency
- frecuencia Z

## Definición central

`Zf` indica cuántas veces la persona intenta escanear y organizar activamente el campo durante el procesamiento.
En esta zona se usa como una medida de frecuencia del esfuerzo organizativo, no como una medida de eficiencia por sí sola.

## Puntos de interpretación

- Si `Zf` es alto, suele haber más intentos de examinar y organizar el estímulo.
- Si `Zf` es bajo, puede haber menos exploración activa o menor iniciativa para estructurar el campo.
- El sentido clínico cambia según se lea junto con `Zd`, porque frecuencia y eficiencia no son lo mismo.

## Precauciones y distinciones

- Un `Zf` alto no implica automáticamente buen procesamiento.
- Un `Zf` bajo puede ser esperable si el protocolo es corto.
- Para interpretar el valor, conviene revisar también la distribución `W:D:Dd`.

## Limites de interpretación aislada

- `Zf` solo muestra con que frecuencia aparece el esfuerzo organizativo.
- Conviene leerlo junto con `Zd`, `W:D:Dd`, `DQ+` y `DQv`.
