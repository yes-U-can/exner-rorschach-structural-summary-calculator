# Nombre del documento: [Interpretación/Lower Section] Lower Section

- lower-section
- lower section

- sección inferior del resumen

## Definición central

`Lower Section` es el conjunto de clusters donde el resumen estructural se lee ya como patrón funcional de recursos, estrés, ideación, afecto, mediación, procesamiento, relaciones interpersonales y autopercepción.
Si el `Upper Section` funciona como pantalla inicial, el `Lower Section` suele ser donde la formulación clínica gana densidad.

## Puntos de interpretación

- La lectura suele seguir el flujo `core -> ideation -> affect -> mediation -> processing -> interpersonal -> selfPerception`.
- Importa menos una cifra aislada que la forma en que varios indicadores se sostienen o se corrigen entre si.
- Aquí se confirma si las señales iniciales del `Upper Section` se consolidan, se matizan o cambian de sentido.

## Precauciones y distinciones

- Los clusters del `Lower Section` se influyen entre si; una lectura aislada aumenta el riesgo de sobregeneralizar.
- La longitud del protocolo, la defensividad, el contexto cultural y el estrés situacional pueden cambiar el peso de una variable.

## Limites de interpretación aislada

- Ningún clúster del `Lower Section` reemplaza la integración total del protocolo.
- Variables como `Lambda`, `D`, `AdjD`, `SumY`, `MOR` o `PER` pueden parecer separadas en el árbol, pero en uso clínico suelen leerse juntas.
