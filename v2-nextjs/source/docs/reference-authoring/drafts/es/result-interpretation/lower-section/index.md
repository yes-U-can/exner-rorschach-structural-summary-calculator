# Nombre del documento: [Interpretacion/Lower Section] Lower Section

## Alias y busqueda

- lower-section
- lower section
- seccion inferior
- seccion inferior del resumen

## Definicion central

`Lower Section` es el conjunto de clusters donde el resumen estructural se lee ya como patron funcional de recursos, estres, ideacion, afecto, mediacion, procesamiento, relaciones interpersonales y autopercepcion.
Si el `Upper Section` funciona como pantalla inicial, el `Lower Section` suele ser donde la formulacion clinica gana densidad.

## Puntos de interpretacion

- La lectura suele seguir el flujo `core -> ideation -> affect -> mediation -> processing -> interpersonal -> selfPerception`.
- Importa menos una cifra aislada que la forma en que varios indicadores se sostienen o se corrigen entre si.
- Aqui se confirma si las senales iniciales del `Upper Section` se consolidan, se matizan o cambian de sentido.

## Precauciones y distinciones

- Los clusters del `Lower Section` se influyen entre si; una lectura aislada aumenta el riesgo de sobregeneralizar.
- La longitud del protocolo, la defensividad, el contexto cultural y el estres situacional pueden cambiar el peso de una variable.

## Limites de interpretacion aislada

- Ningun cluster del `Lower Section` reemplaza la integracion total del protocolo.
- Variables como `Lambda`, `D`, `AdjD`, `SumY`, `MOR` o `PER` pueden parecer separadas en el arbol, pero en uso clinico suelen leerse juntas.
