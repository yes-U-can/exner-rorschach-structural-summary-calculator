# Nombre del documento: [Interpretacion/Afecto] FC:CF+C

## Alias y busqueda

- FC:CF+C
- form-color ratio
- proporcion forma-color

## Definicion central

`FC:CF+C` expresa como se relacionan forma y color en las respuestas cromaticas.
En interpretacion, ayuda a ver si la emocion suele pasar por mayor control formal o si tiende a descargarse de modo mas directo.

## Puntos de interpretacion

- Si domina `FC`, la expresion afectiva suele aparecer mas modulada y contenida por control formal.
- Si domina `CF+C`, la emocion puede salir de forma mas inmediata, menos modulada o menos elaborada.
- Un patron equilibrado sugiere combinacion de control y expresion directa.
- Esta proporcion informa mas sobre el estilo de expresion emocional que sobre la cantidad total de afecto.

## Precauciones y distinciones

- Con pocas respuestas de color, la proporcion puede ser inestable.
- No debe confundirse una mayor presencia de `CF+C` con impulsividad patologica por si sola.
- `Pure C` puede mostrar una descarga afectiva mas directa que no queda resumida del todo por esta proporcion.
- El nombre del route es `FC_CF_C`, pero la lectura clinica incluye la posicion de `Cn` en las notas de fundamento.

## Limites de interpretacion aislada

- `FC:CF+C` no basta por si solo para describir la regulacion afectiva completa.
- Conviene leerlo junto con `Afr`, `Pure C`, `SumC / WSumC`, `EB` y `EA`.
