# Nombre del documento: [Interpretacion/Core] AdjEs

## Alias y busqueda

- AdjEs
- Adjusted es
- es ajustado

## Definicion central

`AdjEs` es una version corregida de `es`.
Busca estimar cuanta carga sigue presente cuando se ajusta parte del peso de `m` y `Y`, que suelen amplificar la tension de manera mas reactiva o situacional.

## Puntos de interpretacion

- Un `AdjEs` alto sugiere que la carga no depende solo de factores momentaneos y puede ser mas sostenida.
- Si `es` es alto pero `AdjEs` baja claramente, una parte importante del malestar actual puede estar asociada con estres situacional.
- `AdjEs` sirve sobre todo para calcular y contextualizar `AdjD`.
- Es una variable de correccion, no una etiqueta diagnostica por si misma.

## Precauciones y distinciones

- `AdjEs` no reemplaza a `es`; ambas deben compararse.
- Un valor alto no especifica por si solo la fuente exacta del malestar.
- Conviene revisar si el peso viene de `m`, `Y` u otros componentes de carga.

## Limites de interpretacion aislada

- `AdjEs` no basta por si sola para decidir gravedad clinica.
- Debe leerse junto con `es`, `D`, `AdjD`, `EA`, `m` y `SumY`.
