# Nombre del documento: [Interpretación/Core] AdjEs

## Alias y búsqueda

- AdjEs
- Adjusted es
- es ajustado

## Definición central

`AdjEs` es una versión corregida de `es`.
Busca estimar cuánta carga sigue presente cuando se ajusta parte del peso de `m` y `Y`, que suelen amplificar la tensión de manera más reactiva o situacional.

## Puntos de interpretación

- Un `AdjEs` alto sugiere que la carga no depende solo de factores momentáneos y puede ser más sostenida.
- Si `es` es alto pero `AdjEs` baja claramente, una parte importante del malestar actual puede estar asociada con estrés situacional.
- `AdjEs` sirve sobre todo para calcular y contextualizar `AdjD`.
- Es una variable de corrección, no una etiqueta diagnóstica por sí misma.

## Precauciones y distinciones

- `AdjEs` no reemplaza a `es`; ambas deben compararse.
- Un valor alto no específica por sí solo la fuente exacta del malestar.
- Conviene revisar si el peso viene de `m`, `Y` u otros componentes de carga.

## Límites de interpretación aislada

- `AdjEs` no basta por sí sola para decidir gravedad clínica.
- Debe leerse junto con `es`, `D`, `AdjD`, `EA`, `m` y `SumY`.
