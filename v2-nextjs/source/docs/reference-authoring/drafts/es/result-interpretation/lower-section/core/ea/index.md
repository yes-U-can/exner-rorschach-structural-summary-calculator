# Nombre del documento: [Interpretación/Core] EA

## Alias y búsqueda

- EA
- Experience Actual
- recursos efectivos

## Definición central

`EA` es el índice de recursos disponibles derivado de los componentes de `EB`.
Indica cuánta capacidad efectiva tiene la persona para manejar demandas cotidianas y modular la experiencia actual.

## Puntos de interpretación

- Un `EA` adecuado sugiere que hay recursos básicos suficientes para afrontar exigencias habituales.
- Un `EA` bajo puede indicar recursos limitados para regular tensión y resolver demandas.
- Un `EA` suficiente no garantiza por sí solo adaptación; debe compararse con la carga representada por `es`.
- `EA` suele ser uno de los puntos de apoyo para leer `D` y `AdjD`.

## Precauciones y distinciones

- Si `EA` parece suficiente pero `EBPer` es muy rígido o `es` es muy alto, el funcionamiento real puede seguir siendo inestable.
- En protocolos cortos o con `Lambda` alto, la cifra puede infraestimar o simplificar el cuadro real.
- `EA` habla de recursos potencialmente disponibles, no de su uso perfecto en toda situación.

## Límites de interpretación aislada

- `EA` no basta por sí solo para concluir déficit o fortaleza global.
- El contraste importante es `EA` frente a `es`, y luego `D` y `AdjD`.
