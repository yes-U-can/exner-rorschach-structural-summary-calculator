# Nombre del documento: [Interpretacion/Afecto] Afr

## Alias y busqueda

- Afr
- Affective Ratio
- proporcion afectiva

## Definicion central

`Afr` compara las respuestas dadas a las laminas VIII, IX y X con las respuestas dadas a las laminas I a VII.
En interpretacion, se usa para estimar cuanto se acerca o se retira la persona frente a estimulos con mayor tono afectivo o cromatico.

## Puntos de interpretacion

- Si `Afr` es bajo, puede haber menor disposicion a acercarse a situaciones con carga afectiva.
- Si `Afr` es alto, puede haber mayor apertura o mayor contacto con estimulos emocionalmente salientes.
- Su lectura mejora cuando se integra con `EB`, `Lambda` y el estilo general de expresion afectiva.
- En situaciones de estres reciente, `Afr` puede bajar temporalmente.

## Precauciones y distinciones

- Si el numero total de respuestas es bajo, la proporcion puede oscilar demasiado.
- Un `Afr` alto no implica por si solo buena salud afectiva, ni un `Afr` bajo significa necesariamente evitacion patologica.
- Tambien conviene mirar si hubo variaciones de administracion o de productividad entre laminas.

## Limites de interpretacion aislada

- `Afr` describe receptividad relativa ante estimulos afectivos, pero no explica por si solo la forma de regulacion emocional.
- Debe leerse junto con `FC:CF+C`, `Blends:R`, `SumC / WSumC` y `Lambda`.
