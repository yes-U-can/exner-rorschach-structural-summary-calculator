# Nombre del documento: [Codificación/Láminas] VIII

## Alias y búsqueda

- Card VIII
- card VIII
- card 8
- lámina VIII
- lámina VIII

## Definición central

`Lámina VIII` suele facilitar respuestas animales y puede hacer más visible la participación del color.
La página resume esos focos de codificación.

## Condiciones de aplicación

- Son comunes respuestas animales y respuestas populares ligadas a zonas relativamente claras.
- Si el color participa en la construcción de la respuesta, revisa `FC`, `CF` o `C`.
- Si varias zonas quedan organizadas en una sola respuesta, revisa también `Z`.
- Cuando la respuesta sigue siendo esencialmente animal, conviene revisar `A` antes de complicar la codificación.

## Precauciones y distinciones

- La presencia de color visible en la lámina no basta para un determinante cromático.
- No toda respuesta animal de esta lámina es `Popular`.
- Si la forma sigue dominando, el peso del color debe evaluarse con cuidado.
