# Nombre del documento: [Codificacion/Tarjetas] VIII

## Alias y busqueda

- Card VIII
- card VIII
- card 8
- tarjeta VIII
- lamina VIII

## Definicion central

`Tarjeta VIII` suele facilitar respuestas animales y puede hacer mas visible la participacion del color.
La pagina resume esos focos de codificacion.

## Condiciones de aplicacion

- Son comunes respuestas animales y respuestas populares ligadas a zonas relativamente claras.
- Si el color participa en la construccion de la respuesta, revisa `FC`, `CF` o `C`.
- Si varias zonas quedan organizadas en una sola respuesta, revisa tambien `Z`.
- Cuando la respuesta sigue siendo esencialmente animal, conviene revisar `A` antes de complicar la codificacion.

## Precauciones y distinciones

- La presencia de color visible en la tarjeta no basta para un determinante cromatico.
- No toda respuesta animal de esta tarjeta es `Popular`.
- Si la forma sigue dominando, el peso del color debe evaluarse con cuidado.
