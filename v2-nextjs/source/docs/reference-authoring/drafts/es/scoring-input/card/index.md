# Nombre del documento: [Codificación/Láminas] Card

## Alias y búsqueda

- Card
- cards
- láminas
- láminas

## Definición central

`Card` es la página guía para las 10 láminas del sistema comprensivo.
Esta familia no interpreta el significado clínico global de cada lámina; funciona como nota de apoyo para recordar dónde suelen aparecer ciertos patrones de codificación.

## Condiciones de aplicación

- Las páginas de lámina ayudan a revisar respuestas frecuentes, focos de `Popular`, tendencias de localización y determinantes que suelen aparecer con más facilidad en cada lámina.
- La codificación siempre se decide desde la respuesta concreta del examinado, no solo desde la lámina.
- Una misma respuesta puede cambiar de código si la descripción real no confirma lo que la lámina suele facilitar.
- El esqueleto de rutas es común entre locales, pero el cuerpo y las referencias cruzadas pueden variar por entorno idiomático y cultural.

## Precauciones y distinciones

- Las páginas de lámina no son páginas de interpretación simbólica de la lámina.
- Deben leerse como apoyo de codificación, no como regla automática.
- El número de lámina puede coincidir con letras usadas en otros códigos; por eso la referencia siempre debe incluir el contexto.
  - Por ejemplo, `Card V` no es lo mismo que el determinante `V`.
