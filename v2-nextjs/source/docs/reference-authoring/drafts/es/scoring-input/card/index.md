# Nombre del documento: [Codificacion/Tarjetas] Card

## Alias y busqueda

- Card
- cards
- tarjetas
- laminas

## Definicion central

`Card` es la pagina guia para las 10 tarjetas del sistema comprensivo.
Esta familia no interpreta el significado clinico global de cada tarjeta; funciona como nota de apoyo para recordar donde suelen aparecer ciertos patrones de codificacion.

## Condiciones de aplicacion

- Las paginas de tarjeta ayudan a revisar respuestas frecuentes, focos de `Popular`, tendencias de localizacion y determinantes que suelen aparecer con mas facilidad en cada lamina.
- La codificacion siempre se decide desde la respuesta concreta del examinado, no solo desde la tarjeta.
- Una misma respuesta puede cambiar de codigo si la descripcion real no confirma lo que la tarjeta suele facilitar.
- El esqueleto de rutas es comun entre locales, pero el cuerpo y las referencias cruzadas pueden variar por entorno idiomatico y cultural.

## Precauciones y distinciones

- Las paginas de tarjeta no son paginas de interpretacion simbolica de la lamina.
- Deben leerse como apoyo de codificacion, no como regla automatica.
- El numero de tarjeta puede coincidir con letras usadas en otros codigos; por eso la referencia siempre debe incluir el contexto.
  - Por ejemplo, `Card V` no es lo mismo que el determinante `V`.
