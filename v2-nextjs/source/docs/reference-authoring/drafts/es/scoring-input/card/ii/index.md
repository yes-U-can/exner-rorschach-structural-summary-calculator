# Nombre del documento: [Codificacion/Tarjetas] II

## Alias y busqueda

- Card II
- card II
- card 2
- tarjeta II
- lamina II

## Definicion central

`Tarjeta II` introduce de forma temprana zonas rojas llamativas.
Esta pagina resume como esa presencia cromatica puede influir en la codificacion, sin convertir la tarjeta en una regla automatica.

## Condiciones de aplicacion

- Si el rojo participa realmente en la respuesta, conviene revisar `FC`, `CF` o `C`.
- Si el color solo se menciona como ubicacion y no como cualidad usada para construir la percepcion, no agregas determinante cromatico.
- La organizacion entre zonas oscuras, claras y rojas puede facilitar revision de `Z`.
- Algunas respuestas parecen humanas o animales en interaccion, pero el movimiento debe comprobarse aparte.

## Precauciones y distinciones

- Ver rojo no basta para asignar un codigo de color.
- La tarjeta puede facilitar varias partes perceptivas; por eso conviene confirmar si la respuesta es una unidad o varias respuestas separadas.
- La simple presencia de dos figuras no crea por si sola `COP` ni `AG`.
