# Nombre del documento: [Codificación/Contenido] H

## Alias y búsqueda

- H
- humano
- figura humana completa

## Definición central

`H` se usa cuando el percepto es una figura humana completa y realista.
La respuesta apunta a una persona entera, no a una parte corporal ni a una figura mitológica o fantástica.

## Condiciones de aplicación

- El objeto principal debe ser una figura humana completa.
- La figura tiene que presentarse como humana realista, no como ser fantástico o mitológico.
- Si la figura es histórica o culturalmente específica, puede agregarse [`Ay`](ref://scoring-input/contents/Ay) como contenido secundario.
- Si solo se describe una parte del cuerpo, hay que reconsiderar [`Hd`](ref://scoring-input/contents/Hd).

## Precauciones y distinciones

- `H` no es igual a [`(H)`](ref://scoring-input/contents/%28H%29); `(H)` se reserva para figuras humanas de ficción o mitológicas.
- `H` no es igual a [`Hd`](ref://scoring-input/contents/Hd); `Hd` se usa para detalle humano incompleto.
- `H` no es igual a [`Hx`](ref://scoring-input/contents/Hx); `Hx` codifica experiencia o emoción humana.
- El hecho de que haya contenido humano no fija por sí solo [`GHR`](ref://scoring-input/gphr/GHR) o [`PHR`](ref://scoring-input/gphr/PHR).
