# Nombre del documento: [Codificacion/Contenido] Cl

## Alias y busqueda

- Cl
- nubes
- clouds

## Definicion central

`Cl` se usa cuando el contenido principal es nube.
Se aplica a la nube misma, no a otras formas atmosfericas parecidas.

## Condiciones de aplicacion

- Se aplica cuando el sujeto nombra nube, capa de nubes o formacion claramente entendida como cloud.
- La nube debe ser el foco del contenido y no solo un fondo difuso.
- Si la imagen se entiende mejor como niebla, vapor o bruma, primero se considera [`Na`](ref://scoring-input/contents/Na).
- Si la forma se percibe como humo o fuego, primero se considera [`Fi`](ref://scoring-input/contents/Fi).

## Precauciones y distinciones

- `Cl` y [`Na`](ref://scoring-input/contents/Na) se separan por nube frente a niebla o vapor.
- `Cl` y [`Fi`](ref://scoring-input/contents/Fi) pueden parecer cercanos visualmente, pero `Fi` exige fuego o humo.
- No basta con que algo se vea borroso; el sujeto tiene que organizarlo como nube.
