# Nombre del documento: [Codificación/Contenido] Hd

## Alias y búsqueda

- Hd
- detalle humano
- human detail

## Definición central

`Hd` se usa cuando la respuesta describe solo una parte de una figura humana.
No aparece una persona completa, sino un detalle corporal o un fragmento humano.

## Condiciones de aplicación

- El contenido principal debe ser una parte de persona y no una figura completa.
- La parte debe entenderse como humana realista.
- Ejemplos típicos son brazo, mano, pierna, dedos, cabeza o torso incompleto.
- Si la respuesta llega a construir una persona completa, hay que volver a [`H`](ref://scoring-input/contents/H).

## Precauciones y distinciones

- `Hd` no es igual a [`H`](ref://scoring-input/contents/H); aquí no hay figura humana completa.
- `Hd` no es igual a [`(Hd)`](ref://scoring-input/contents/%28Hd%29); `(Hd)` se reserva para detalle humano fantástico o mitológico.
- `Hd` suele empujar la lectura de representación humana hacia zonas más pobres, por eso conviene revisar [`PHR`](ref://scoring-input/gphr/PHR).
- `Hd` tampoco equivale a [`Hx`](ref://scoring-input/contents/Hx); `Hx` codifica experiencia humana, no fragmento corporal.
