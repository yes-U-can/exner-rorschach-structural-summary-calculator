# Nombre del documento: [Codificacion/Contenido] Bt

## Alias y busqueda

- Bt
- BT
- botanica
- botany

## Definicion central

`Bt` se usa cuando el contenido principal pertenece al mundo vegetal.
Incluye plantas, flores, arboles, hojas o partes vegetales reconocidas como tales.

## Condiciones de aplicacion

- Se aplica a arbol, flor, hoja, rama, cactus u otras formas vegetales.
- Puede usarse tanto para la planta completa como para una parte vegetal clara.
- Si la respuesta describe animal o detalle animal, primero se consideran [`A`](ref://scoring-input/contents/A) o sus variantes.
- Si el foco se desplaza a sangre o lesion biologica, primero se considera [`Bl`](ref://scoring-input/contents/Bl).

## Precauciones y distinciones

- `Bt` es contenido botanico, no un juicio sobre color, textura o forma.
- No se debe usar solo porque algo parezca organico; tiene que percibirse especificamente como planta.
- Algunas fuentes escriben `BT`, pero el codigo canonico del corpus se mantiene como `Bt`.
