# Nombre del documento: [Codificación/Contenido] (H)

## Alias y búsqueda

- (H)
- humano imaginario
- figura humana mitológica

## Definición central

`(H)` se usa cuando el percepto es una figura humana completa de ficción, fantasía o mitología.
Sigue siendo una figura humana completa, pero no se trata de una persona realista ordinaria.

## Condiciones de aplicación

- El objeto principal debe ser una figura humana completa.
- La figura tiene que ser fantástica, mitológica, ficticia o claramente no realista.
- Ejemplos típicos son ángeles, demonios, gigantes, hadas, fantasmas o humanoides de ciencia ficción.
- Si solo aparece una parte de esa figura, hay que reconsiderar [`(Hd)`](ref://scoring-input/contents/%28Hd%29).

## Precauciones y distinciones

- `(H)` no es igual a [`H`](ref://scoring-input/contents/H); `H` se reserva para figura humana completa realista.
- `(H)` no es igual a [`(Hd)`](ref://scoring-input/contents/%28Hd%29); `(Hd)` se usa para detalle humano imaginario.
- Un disfraz, una máscara o una figura artística no cambian automáticamente el código si el sujeto sigue describiendo una persona completa realista.
- La decisión sobre [`GHR`](ref://scoring-input/gphr/GHR) o [`PHR`](ref://scoring-input/gphr/PHR) requiere un paso separado.
