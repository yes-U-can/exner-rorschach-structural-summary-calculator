# Nombre del documento: [Codificacion/Contenido] (H)

## Alias y busqueda

- (H)
- humano imaginario
- figura humana mitologica

## Definicion central

`(H)` se usa cuando el percepto es una figura humana completa de ficcion, fantasia o mitologia.  
Sigue siendo una figura humana completa, pero no se trata de una persona realista ordinaria.

## Condiciones de aplicacion

- El objeto principal debe ser una figura humana completa.
- La figura tiene que ser fantastica, mitologica, ficticia o claramente no realista.
- Ejemplos tipicos son angeles, demonios, gigantes, hadas, fantasmas o humanoides de ciencia ficcion.
- Si solo aparece una parte de esa figura, hay que reconsiderar [`(Hd)`](ref://scoring-input/contents/%28Hd%29).

## Precauciones y distinciones

- `(H)` no es igual a [`H`](ref://scoring-input/contents/H); `H` se reserva para figura humana completa realista.
- `(H)` no es igual a [`(Hd)`](ref://scoring-input/contents/%28Hd%29); `(Hd)` se usa para detalle humano imaginario.
- Un disfraz, una mascara o una figura artistica no cambian automaticamente el codigo si el sujeto sigue describiendo una persona completa realista.
- La decision sobre [`GHR`](ref://scoring-input/gphr/GHR) o [`PHR`](ref://scoring-input/gphr/PHR) requiere un paso separado.
