# Nombre del documento: [Codificación/Contenido] Bl

## Alias y búsqueda

- Bl
- sangre
- blood

## Definición central

`Bl` se usa cuando el contenido principal es sangre.
No se refiere a una estructura anatómica completa, sino al líquido o a la idea de sangrado.

## Condiciones de aplicación

- Se aplica cuando el sujeto nombra sangre, chorro de sangre, mancha de sangre o sangrado.
- Puede aparecer aunque la sangre proceda de humano o animal.
- Si la respuesta se centra en huesos, órganos o estructura corporal, primero se considera [`An`](ref://scoring-input/contents/An).
- Si la sangre aparece como parte de una figura humana o animal, `Bl` puede combinarse con [`H`](ref://scoring-input/contents/H) o [`A`](ref://scoring-input/contents/A) cuando proceda.

## Precauciones y distinciones

- `Bl` no es un código de color; no reemplaza la decisión sobre determinantes cromáticos.
- `Bl` y [`An`](ref://scoring-input/contents/An) pueden parecer biológicos, pero `Bl` se centra en sangre y `An` en anatomía.
- No basta con ver rojo en la lámina; el sujeto tiene que nombrar sangre o algo claramente equivalente.
