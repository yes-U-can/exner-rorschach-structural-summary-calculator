# Nombre del documento: [Codificacion/Contenido] Bl

## Alias y busqueda

- Bl
- sangre
- blood

## Definicion central

`Bl` se usa cuando el contenido principal es sangre.
No se refiere a una estructura anatomica completa, sino al liquido o a la idea de sangrado.

## Condiciones de aplicacion

- Se aplica cuando el sujeto nombra sangre, chorro de sangre, mancha de sangre o sangrado.
- Puede aparecer aunque la sangre proceda de humano o animal.
- Si la respuesta se centra en huesos, organos o estructura corporal, primero se considera [`An`](ref://scoring-input/contents/An).
- Si la sangre aparece como parte de una figura humana o animal, `Bl` puede combinarse con [`H`](ref://scoring-input/contents/H) o [`A`](ref://scoring-input/contents/A) cuando proceda.

## Precauciones y distinciones

- `Bl` no es un codigo de color; no reemplaza la decision sobre determinantes cromaticos.
- `Bl` y [`An`](ref://scoring-input/contents/An) pueden parecer biologicos, pero `Bl` se centra en sangre y `An` en anatomia.
- No basta con ver rojo en la lamina; el sujeto tiene que nombrar sangre o algo claramente equivalente.
