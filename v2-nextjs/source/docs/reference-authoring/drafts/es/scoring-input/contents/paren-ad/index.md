# Nombre del documento: [Codificacion/Contenido] (Ad)

## Alias y busqueda

- (Ad)
- detalle animal imaginario
- fictional animal detail

## Definicion central

`(Ad)` se usa cuando el sujeto describe una parte de un animal fantastico, mitologico o inventado.
Es la version parcial e imaginaria del cluster animal.

## Condiciones de aplicacion

- La respuesta debe apuntar a una parte animal y no al animal entero.
- El referente debe ser fantastico, ficticio o claramente inventado.
- Ejemplos tipicos son ala de dragon, garra de monstruo o cuerno de criatura mitologica.
- Si el animal fantastico aparece entero, primero se considera [`(A)`](ref://scoring-input/contents/%28A%29).
- Si el detalle corresponde a un animal realista, primero se considera [`Ad`](ref://scoring-input/contents/Ad).

## Precauciones y distinciones

- `(Ad)` y [`(A)`](ref://scoring-input/contents/%28A%29) se separan por parcialidad frente a totalidad.
- `(Ad)` y [`Ad`](ref://scoring-input/contents/Ad) se separan por fantasia frente a realidad.
- Aunque sea imaginario, el contenido sigue siendo un detalle animal y no una parte humana; para eso se compara con [`Hd`](ref://scoring-input/contents/Hd).
