# Nombre del documento: [Codificación/Contenido] (Ad)

## Alias y búsqueda

- (Ad)
- detalle animal imaginario
- fictional animal detail

## Definición central

`(Ad)` se usa cuando el sujeto describe una parte de un animal fantástico, mitológico o inventado.
Es la versión parcial e imaginaria del clúster animal.

## Condiciones de aplicación

- La respuesta debe apuntar a una parte animal y no al animal entero.
- El referente debe ser fantástico, ficticio o claramente inventado.
- Ejemplos típicos son ala de dragón, garra de monstruo o cuerno de criatura mitológica.
- Si el animal fantástico aparece entero, primero se considera [`(A)`](ref://scoring-input/contents/%28A%29).
- Si el detalle corresponde a un animal realista, primero se considera [`Ad`](ref://scoring-input/contents/Ad).

## Precauciones y distinciones

- `(Ad)` y [`(A)`](ref://scoring-input/contents/%28A%29) se separan por parcialidad frente a totalidad.
- `(Ad)` y [`Ad`](ref://scoring-input/contents/Ad) se separan por fantasía frente a realidad.
- Aunque sea imaginario, el contenido sigue siendo un detalle animal y no una parte humana; para eso se compara con [`Hd`](ref://scoring-input/contents/Hd).
