# Nombre del documento: [Codificacion/Contenido] Sc

## Alias y busqueda

- Sc
- ciencia
- science

## Definicion central

`Sc` se usa cuando el contenido se entiende como objeto cientifico, tecnico, mecanico o tecnologico.
Lo importante es que el sujeto lo vea como aparato, instrumento, maquina o producto de ciencia aplicada.

## Condiciones de aplicacion

- Se aplica a microscopio, robot, maquinaria, arma, vehiculo, aparato tecnico, instrumento de laboratorio o dispositivo cientifico.
- Tambien puede incluir objetos de tecnologia comun si la respuesta los trata como aparato o mecanismo.
- Si el objeto tiene presentacion artistica o decorativa, `Sc` puede coexistir con [`Art`](ref://scoring-input/contents/Art).
- Antes de enviar una respuesta rara a [`Id`](ref://scoring-input/contents/Id), primero se comprueba si encaja mejor en `Sc`.

## Precauciones y distinciones

- `Sc` no significa simplemente "raro"; exige una lectura cientifica o tecnica.
- Un objeto poco habitual como tubo de ensayo, motor o pieza mecanica sigue perteneciendo a `Sc`, no a [`Id`](ref://scoring-input/contents/Id).
- Si la respuesta enfatiza simbolo cultural o pieza estetica, puede requerir [`Art`](ref://scoring-input/contents/Art) ademas de `Sc`.
