# Nombre del documento: [Codificación/Contenido] Sc

## Alias y búsqueda

- Sc
- ciencia
- science

## Definición central

`Sc` se usa cuando el contenido se entiende como objeto científico, técnico, mecánico o tecnológico.
Lo importante es que el sujeto lo vea como aparato, instrumento, máquina o producto de ciencia aplicada.

## Condiciones de aplicación

- Se aplica a microscopio, robot, maquinaria, arma, vehículo, aparato técnico, instrumento de laboratorio o dispositivo científico.
- También puede incluir objetos de tecnología común si la respuesta los trata como aparato o mecanismo.
- Si el objeto tiene presentación artística o decorativa, `Sc` puede coexistir con [`Art`](ref://scoring-input/contents/Art).
- Antes de enviar una respuesta rara a [`Id`](ref://scoring-input/contents/Id), primero se comprueba si encaja mejor en `Sc`.

## Precauciones y distinciones

- `Sc` no significa simplemente "raro"; exige una lectura científica o técnica.
- Un objeto poco habitual como tubo de ensayo, motor o pieza mecánica sigue perteneciendo a `Sc`, no a [`Id`](ref://scoring-input/contents/Id).
- Si la respuesta enfatiza símbolo cultural o pieza estética, puede requerir [`Art`](ref://scoring-input/contents/Art) además de `Sc`.
