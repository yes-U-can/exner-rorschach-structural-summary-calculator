# Nombre del documento: [Codificación/Contenido] Sx

## Alias y búsqueda

- Sx
- sexo
- sex

## Definición central

`Sx` se usa cuando el contenido implica órgano sexual, actividad sexual o significado sexual explícito.
Con frecuencia funciona como contenido secundario junto a un código humano o anatómico.

## Condiciones de aplicación

- Se aplica a genitales, pechos sexualizados, coito, acto sexual u otra referencia sexual directa.
- Requiere que el contenido sexual este realmente en la respuesta y no solo inferido por el evaluador.
- Muchas veces se agrega junto con [`H`](ref://scoring-input/contents/H), [`Hd`](ref://scoring-input/contents/Hd) o [`An`](ref://scoring-input/contents/An).
- Si la respuesta solo indica desnudez o cuerpo sin significado sexual claro, no se codifica automáticamente `Sx`.

## Precauciones y distinciones

- `Sx` debe distinguirse de [`An`](ref://scoring-input/contents/An), que se centra en anatomía, no en sexualidad.
- `Sx` tampoco equivale por sí solo a contenido humano completo; puede requerir [`H`](ref://scoring-input/contents/H) o [`Hd`](ref://scoring-input/contents/Hd).
- La codificación pide contenido sexual explícito, no una lectura insinuada sin apoyo en la respuesta.
