# Nombre del documento: [Codificacion/Contenido] Sx

## Alias y busqueda

- Sx
- sexo
- sex

## Definicion central

`Sx` se usa cuando el contenido implica organo sexual, actividad sexual o significado sexual explicito.
Con frecuencia funciona como contenido secundario junto a un codigo humano o anatomico.

## Condiciones de aplicacion

- Se aplica a genitales, pechos sexualizados, coito, acto sexual u otra referencia sexual directa.
- Requiere que el contenido sexual este realmente en la respuesta y no solo inferido por el evaluador.
- Muchas veces se agrega junto con [`H`](ref://scoring-input/contents/H), [`Hd`](ref://scoring-input/contents/Hd) o [`An`](ref://scoring-input/contents/An).
- Si la respuesta solo indica desnudez o cuerpo sin significado sexual claro, no se codifica automaticamente `Sx`.

## Precauciones y distinciones

- `Sx` debe distinguirse de [`An`](ref://scoring-input/contents/An), que se centra en anatomia, no en sexualidad.
- `Sx` tampoco equivale por si solo a contenido humano completo; puede requerir [`H`](ref://scoring-input/contents/H) o [`Hd`](ref://scoring-input/contents/Hd).
- La codificacion pide contenido sexual explicito, no una lectura insinuada sin apoyo en la respuesta.
