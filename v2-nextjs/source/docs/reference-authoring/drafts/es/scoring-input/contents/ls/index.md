# Nombre del documento: [Codificacion/Contenido] Ls

## Alias y busqueda

- Ls
- paisaje
- landscape

## Definicion central

`Ls` se usa cuando el contenido principal es un paisaje, una vista geografica o una escena natural amplia.
El foco debe recaer en la escena o el entorno, no solo en una planta u objeto aislado.

## Condiciones de aplicacion

- Se aplica a montes, costa, valle, horizonte, ribera, paisaje rocoso u otra escena entendida como vista natural.
- Requiere que el sujeto vea un escenario o una extension espacial, no solo un detalle botanico.
- Si la respuesta describe un mapa o una representacion cartografica, primero se considera [`Ge`](ref://scoring-input/contents/Ge).
- Si la respuesta apunta a naturaleza general o ambiente natural mas que a un paisaje delimitado, primero se considera [`Na`](ref://scoring-input/contents/Na).

## Precauciones y distinciones

- `Ls` no es lo mismo que [`Ge`](ref://scoring-input/contents/Ge), que se reserva para mapa o referencia cartografica.
- Si aparece un entorno natural amplio con componentes botanicos, `Na` puede tener prioridad sobre `Ls`.
- Si el foco recae en una planta, rama o forma botanica puntual, primero se considera [`Bt`](ref://scoring-input/contents/Bt).
