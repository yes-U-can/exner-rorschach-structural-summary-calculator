# Nombre del documento: [Codificacion/Contenido] Fd

## Alias y busqueda

- Fd
- comida
- food

## Definicion central

`Fd` se usa cuando el contenido principal es alimento o comida.
No es un determinante; es un codigo de contenido para algo entendido como comestible.

## Condiciones de aplicacion

- Se aplica a comida preparada, plato, carne, pastel, fruta servida u otros alimentos reconocidos como tales.
- Tambien puede usarse cuando la escena gira en torno a comer o a un objeto claramente comestible.
- Si el foco principal es el animal mismo y no el alimento, primero se considera [`A`](ref://scoring-input/contents/A) o una variante de contenido animal.
- No debe confundirse con [`FD`](ref://scoring-input/determinants/FD), que pertenece a determinantes.

## Precauciones y distinciones

- `Fd` como contenido y [`FD`](ref://scoring-input/determinants/FD) como determinante son ejes distintos.
- `Fd` y contenido botanico pueden parecer cercanos si el sujeto nombra vegetales, pero `Fd` exige que se vean como comida.
- El significado interpersonal posterior de [`Food`](ref://result-interpretation/lower-section/interpersonal/Food) no cambia la decision de codificacion.
