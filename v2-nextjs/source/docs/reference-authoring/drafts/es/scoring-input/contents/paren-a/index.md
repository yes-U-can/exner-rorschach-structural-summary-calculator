# Nombre del documento: [Codificacion/Contenido] (A)

## Alias y busqueda

- (A)
- animal imaginario completo
- fictional whole animal

## Definicion central

`(A)` se usa cuando el contenido es un animal completo, pero fantastico, mitologico o claramente inventado.
La forma sigue siendo de animal entero, aunque el referente no exista en la realidad ordinaria.

## Condiciones de aplicacion

- La respuesta debe describir un animal entero y no solo una parte.
- El caracter fantastico puede venir de mitologia, cuentos, dibujos o invencion del sujeto.
- Ejemplos tipicos son dragon, unicornio o monstruo con forma animal completa.
- Si el referente es un animal realista y entero, primero se considera [`A`](ref://scoring-input/contents/A).
- Si solo se describe una parte de un animal fantastico, primero se considera [`(Ad)`](ref://scoring-input/contents/%28Ad%29).

## Precauciones y distinciones

- `(A)` y [`A`](ref://scoring-input/contents/A) se distinguen por el grado de realidad del animal, no por la organizacion de totalidad.
- `(A)` y [`(Ad)`](ref://scoring-input/contents/%28Ad%29) se distinguen por si la figura es completa o parcial.
- El contenido no reemplaza la decision sobre determinante, `FQ` o `Popular`.
