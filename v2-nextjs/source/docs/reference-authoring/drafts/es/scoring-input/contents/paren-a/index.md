# Nombre del documento: [Codificación/Contenido] (A)

## Alias y búsqueda

- (A)
- animal imaginario completo
- fictional whole animal

## Definición central

`(A)` se usa cuando el contenido es un animal completo, pero fantástico, mitológico o claramente inventado.
La forma sigue siendo de animal entero, aunque el referente no exista en la realidad ordinaria.

## Condiciones de aplicación

- La respuesta debe describir un animal entero y no solo una parte.
- El carácter fantástico puede venir de mitología, cuentos, dibujos o invención del sujeto.
- Ejemplos típicos son dragón, unicornio o monstruo con forma animal completa.
- Si el referente es un animal realista y entero, primero se considera [`A`](ref://scoring-input/contents/A).
- Si solo se describe una parte de un animal fantástico, primero se considera [`(Ad)`](ref://scoring-input/contents/%28Ad%29).

## Precauciones y distinciones

- `(A)` y [`A`](ref://scoring-input/contents/A) se distinguen por el grado de realidad del animal, no por la organización de totalidad.
- `(A)` y [`(Ad)`](ref://scoring-input/contents/%28Ad%29) se distinguen por si la figura es completa o parcial.
- El contenido no reemplaza la decisión sobre determinante, `FQ` o `Popular`.
