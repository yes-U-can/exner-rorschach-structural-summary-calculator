# Nombre del documento: [Codificación/Contenido] Art

## Alias y búsqueda

- Art
- arte
- art

## Definición central

`Art` se usa cuando el contenido se percibe como obra artística, objeto decorativo o producto estético.
Lo central no es solo que se vea algo, sino que ese algo sea entendido como pieza de arte o de ornamentación.

## Condiciones de aplicación

- Se aplica a cuadros, estatuas, dibujos, esculturas, emblemas o piezas decorativas.
- Puede agregarse junto con otro contenido si la respuesta identifica al mismo tiempo el objeto y su carácter artístico.
- Si la respuesta destaca anatomía o esquema biológico, primero se considera [`An`](ref://scoring-input/contents/An).
- Si el foco está en simbolismo histórico, étnico o antropológico más que en obra artística, primero se considera [`Ay`](ref://scoring-input/contents/Ay).

## Precauciones y distinciones

- `Art` y [`Ay`](ref://scoring-input/contents/Ay) pueden coexistir, pero no significan lo mismo.
- `Art` se centra en objeto artístico o decorativo; `Ay` se centra en referencia histórica o cultural.
- Un material biológico dibujado o montado como laminilla puede requerir [`An`](ref://scoring-input/contents/An) como código principal y `Art` como acompañante.
