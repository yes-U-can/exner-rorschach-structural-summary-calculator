# Nombre del documento: [Codificacion/Contenido] Art

## Alias y busqueda

- Art
- arte
- art

## Definicion central

`Art` se usa cuando el contenido se percibe como obra artistica, objeto decorativo o producto estetico.
Lo central no es solo que se vea algo, sino que ese algo sea entendido como pieza de arte o de ornamentacion.

## Condiciones de aplicacion

- Se aplica a cuadros, estatuas, dibujos, esculturas, emblemas o piezas decorativas.
- Puede agregarse junto con otro contenido si la respuesta identifica al mismo tiempo el objeto y su caracter artistico.
- Si la respuesta destaca anatomia o esquema biologico, primero se considera [`An`](ref://scoring-input/contents/An).
- Si el foco esta en simbolismo historico, etnico o antropologico mas que en obra artistica, primero se considera [`Ay`](ref://scoring-input/contents/Ay).

## Precauciones y distinciones

- `Art` y [`Ay`](ref://scoring-input/contents/Ay) pueden coexistir, pero no significan lo mismo.
- `Art` se centra en objeto artistico o decorativo; `Ay` se centra en referencia historica o cultural.
- Un material biologico dibujado o montado como laminilla puede requerir [`An`](ref://scoring-input/contents/An) como codigo principal y `Art` como acompanante.
