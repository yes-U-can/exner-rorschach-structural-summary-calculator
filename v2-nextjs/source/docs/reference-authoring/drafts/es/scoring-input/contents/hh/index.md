# Nombre del documento: [Codificación/Contenido] Hh

## Alias y búsqueda

- Hh
- hogar
- household

## Definición central

`Hh` se usa cuando el contenido principal es utensilio doméstico u objeto de uso casero.
Lo central es que la forma se entienda como implemento del hogar, no como prenda ni como obra artística.

## Condiciones de aplicación

- Se aplica a taza, olla, lámpara, plato, cubierto, herramienta doméstica u otro objeto de uso en casa.
- El objeto del hogar debe ser el foco principal del contenido.
- Si el objeto se percibe sobre todo como pieza decorativa o artística, puede requerir también [`Art`](ref://scoring-input/contents/Art).
- Si el sujeto enfatiza que el objeto está hecho con parte animal, puede coexistir con [`Ad`](ref://scoring-input/contents/Ad).

## Precauciones y distinciones

- `Hh` no cubre ropa; para eso se considera [`Cg`](ref://scoring-input/contents/Cg).
- `Hh` y [`Art`](ref://scoring-input/contents/Art) pueden acercarse, pero no son equivalentes.
- `Hh` y [`Ad`](ref://scoring-input/contents/Ad) solo coinciden cuando el mismo objeto doméstico se describe como fabricado con parte animal.
