# Nombre del documento: [Codificacion/Contenido] Hh

## Alias y busqueda

- Hh
- hogar
- household

## Definicion central

`Hh` se usa cuando el contenido principal es utensilio domestico u objeto de uso casero.
Lo central es que la forma se entienda como implemento del hogar, no como prenda ni como obra artistica.

## Condiciones de aplicacion

- Se aplica a taza, olla, lampara, plato, cubierto, herramienta domestica u otro objeto de uso en casa.
- El objeto del hogar debe ser el foco principal del contenido.
- Si el objeto se percibe sobre todo como pieza decorativa o artistica, puede requerir tambien [`Art`](ref://scoring-input/contents/Art).
- Si el sujeto enfatiza que el objeto esta hecho con parte animal, puede coexistir con [`Ad`](ref://scoring-input/contents/Ad).

## Precauciones y distinciones

- `Hh` no cubre ropa; para eso se considera [`Cg`](ref://scoring-input/contents/Cg).
- `Hh` y [`Art`](ref://scoring-input/contents/Art) pueden acercarse, pero no son equivalentes.
- `Hh` y [`Ad`](ref://scoring-input/contents/Ad) solo coinciden cuando el mismo objeto domestico se describe como fabricado con parte animal.
