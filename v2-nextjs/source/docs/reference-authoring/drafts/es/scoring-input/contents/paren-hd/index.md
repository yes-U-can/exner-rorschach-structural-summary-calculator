# Nombre del documento: [Codificación/Contenido] (Hd)

## Alias y búsqueda

- (Hd)
- detalle humano imaginario
- imaginary human detail

## Definición central

`(Hd)` se usa cuando la respuesta describe solo una parte de una figura humana fantástica, mitológica o ficticia.
Es el equivalente imaginario de `Hd`.

## Condiciones de aplicación

- El contenido principal debe ser un detalle humano incompleto.
- Ese detalle tiene que pertenecer a una figura no realista, fantástica o mitológica.
- Ejemplos típicos son cabeza de demonio, brazo de bruja, ojos de ángel o partes humanoides de criatura fantástica.
- Si la respuesta describe una figura fantástica completa, hay que volver a [`(H)`](ref://scoring-input/contents/%28H%29).

## Precauciones y distinciones

- `(Hd)` no es igual a [`Hd`](ref://scoring-input/contents/Hd); aquí la clave es el carácter imaginario o mitológico.
- `(Hd)` no es igual a [`(H)`](ref://scoring-input/contents/%28H%29); `(H)` exige figura completa.
- Muchas respuestas con `(Hd)` también exigen revisar [`PHR`](ref://scoring-input/gphr/PHR), pero esa decisión sigue siendo un paso aparte.
- No usar `(Hd)` solo porque la respuesta parezca rara; el sujeto debe describir una parte humana no realista.
