# Nombre del documento: [Codificacion/Contenido] (Hd)

## Alias y busqueda

- (Hd)
- detalle humano imaginario
- imaginary human detail

## Definicion central

`(Hd)` se usa cuando la respuesta describe solo una parte de una figura humana fantastica, mitologica o ficticia.  
Es el equivalente imaginario de `Hd`.

## Condiciones de aplicacion

- El contenido principal debe ser un detalle humano incompleto.
- Ese detalle tiene que pertenecer a una figura no realista, fantastica o mitologica.
- Ejemplos tipicos son cabeza de demonio, brazo de bruja, ojos de angel o partes humanoides de criatura fantastica.
- Si la respuesta describe una figura fantastica completa, hay que volver a [`(H)`](ref://scoring-input/contents/%28H%29).

## Precauciones y distinciones

- `(Hd)` no es igual a [`Hd`](ref://scoring-input/contents/Hd); aqui la clave es el caracter imaginario o mitologico.
- `(Hd)` no es igual a [`(H)`](ref://scoring-input/contents/%28H%29); `(H)` exige figura completa.
- Muchas respuestas con `(Hd)` tambien exigen revisar [`PHR`](ref://scoring-input/gphr/PHR), pero esa decision sigue siendo un paso aparte.
- No usar `(Hd)` solo porque la respuesta parezca rara; el sujeto debe describir una parte humana no realista.
