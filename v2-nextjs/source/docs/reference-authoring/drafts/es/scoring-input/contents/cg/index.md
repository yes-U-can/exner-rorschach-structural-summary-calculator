# Nombre del documento: [Codificación/Contenido] Cg

## Alias y búsqueda

- Cg
- ropa
- clothing

## Definición central

`Cg` se usa cuando el contenido principal es ropa, prenda o accesorio de vestir.
Lo decisivo es que la forma se entienda como algo que se lleva puesto.

## Condiciones de aplicación

- Se aplica a sombrero, bota, guante, capa, vestido, uniforme u otra prenda reconocible.
- También puede usarse con accesorios de uso corporal cuando se entienden como parte del vestir.
- Si la prenda tiene una referencia histórica o cultural dominante, puede requerir también [`Ay`](ref://scoring-input/contents/Ay).
- Si el énfasis cae en valor decorativo o artístico, puede requerir también [`Art`](ref://scoring-input/contents/Art).

## Precauciones y distinciones

- `Cg` no se usa para cualquier objeto; hace falta que sea una prenda o accesorio de vestir.
- `Cg` y [`Ay`](ref://scoring-input/contents/Ay) pueden coexistir, pero `Ay` exige marco histórico o cultural.
- `Cg` y [`Art`](ref://scoring-input/contents/Art) pueden coexistir, pero `Art` exige que la pieza se perciba como objeto artístico o decorativo.
