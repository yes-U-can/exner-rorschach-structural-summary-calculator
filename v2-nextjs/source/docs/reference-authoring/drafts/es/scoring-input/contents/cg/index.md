# Nombre del documento: [Codificacion/Contenido] Cg

## Alias y busqueda

- Cg
- ropa
- clothing

## Definicion central

`Cg` se usa cuando el contenido principal es ropa, prenda o accesorio de vestir.
Lo decisivo es que la forma se entienda como algo que se lleva puesto.

## Condiciones de aplicacion

- Se aplica a sombrero, bota, guante, capa, vestido, uniforme u otra prenda reconocible.
- Tambien puede usarse con accesorios de uso corporal cuando se entienden como parte del vestir.
- Si la prenda tiene una referencia historica o cultural dominante, puede requerir tambien [`Ay`](ref://scoring-input/contents/Ay).
- Si el enfasis cae en valor decorativo o artistico, puede requerir tambien [`Art`](ref://scoring-input/contents/Art).

## Precauciones y distinciones

- `Cg` no se usa para cualquier objeto; hace falta que sea una prenda o accesorio de vestir.
- `Cg` y [`Ay`](ref://scoring-input/contents/Ay) pueden coexistir, pero `Ay` exige marco historico o cultural.
- `Cg` y [`Art`](ref://scoring-input/contents/Art) pueden coexistir, pero `Art` exige que la pieza se perciba como objeto artistico o decorativo.
