# Nombre del documento: [Codificación/Calidad formal] u

## Alias y búsqueda

- FQu
- única
- ajuste formal infrecuente

## Definición central

FQu se usa cuando la respuesta no es ordinaria, pero aun así mantiene un ajuste formal aceptable.
La respuesta puede ser menos frecuente, aunque todavía se ve con relativa rapidez y sin una distorsión importante de los contornos.

## Condiciones de aplicación

- La respuesta no encaja como ordinaria en la tabla.
- El objeto sigue siendo visualizable de manera plausible.
- El uso de los contornos no está excesivamente forzado.
- En respuestas fuera de tabla, `u` se favorece cuando el objeto se puede ver de forma rápida y relativamente fácil.

## Precauciones y distinciones

- La diferencia crítica frente a [`FQ-`](ref://scoring-input/fq/-) es el grado de forzamiento formal.
- `u` no implica mala forma; implica ajuste aceptable pero menos común.
- Si la respuesta aparece bien asentada como ordinaria, corresponde [`FQo`](ref://scoring-input/fq/o).
