# Nombre del documento: [Codificacion/Calidad formal] u

## Alias y busqueda

- FQu
- unica
- ajuste formal infrecuente

## Definicion central

FQu se usa cuando la respuesta no es ordinaria, pero aun asi mantiene un ajuste formal aceptable.  
La respuesta puede ser menos frecuente, aunque todavia se ve con relativa rapidez y sin una distorsion importante de los contornos.

## Condiciones de aplicacion

- La respuesta no encaja como ordinaria en la tabla.
- El objeto sigue siendo visualizable de manera plausible.
- El uso de los contornos no esta excesivamente forzado.
- En respuestas fuera de tabla, `u` se favorece cuando el objeto se puede ver de forma rapida y relativamente facil.

## Precauciones y distinciones

- La diferencia critica frente a [`FQ-`](ref://scoring-input/fq/-) es el grado de forzamiento formal.
- `u` no implica mala forma; implica ajuste aceptable pero menos comun.
- Si la respuesta aparece bien asentada como ordinaria, corresponde [`FQo`](ref://scoring-input/fq/o).
