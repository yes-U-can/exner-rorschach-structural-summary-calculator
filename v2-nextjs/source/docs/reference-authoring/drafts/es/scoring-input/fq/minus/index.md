# Nombre del documento: [Codificación/Calidad formal] -

## Alias y búsqueda

- FQ-
- menos
- ajuste formal pobre

## Definición central

FQ- se asigna cuando el objeto descrito no encaja adecuadamente con el área de la mancha y la respuesta fuerza los contornos o crea rasgos que no están realmente ahí.
Es la categoría de ajuste formal pobre o distorsionado.

## Condiciones de aplicación

- El objeto no se ve de manera rápida ni fácil.
- La respuesta depende de contornos arbitrarios o de una distorsión importante.
- La tabla o el juicio de extrapolación no sostienen un ajuste adecuado.
- En caso dudoso fuera de tabla, el manual favorece `-` antes que sobrevalorar el ajuste.

## Precauciones y distinciones

- `FQ-` no debe asignarse por mera rareza; la clave es la mala calidad de ajuste.
- Si la respuesta es rara pero todavía plausible y relativamente fácil de ver, corresponde [`FQu`](ref://scoring-input/fq/u).
- Una o dos respuestas `-` aisladas no agotan la interpretación; su peso depende de frecuencia y patrón.
