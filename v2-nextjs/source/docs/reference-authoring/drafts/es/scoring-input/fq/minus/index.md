# Nombre del documento: [Codificacion/Calidad formal] -

## Alias y busqueda

- FQ-
- menos
- ajuste formal pobre

## Definicion central

FQ- se asigna cuando el objeto descrito no encaja adecuadamente con el area de la mancha y la respuesta fuerza los contornos o crea rasgos que no estan realmente ahi.  
Es la categoria de ajuste formal pobre o distorsionado.

## Condiciones de aplicacion

- El objeto no se ve de manera rapida ni facil.
- La respuesta depende de contornos arbitrarios o de una distorsion importante.
- La tabla o el juicio de extrapolacion no sostienen un ajuste adecuado.
- En caso dudoso fuera de tabla, el manual favorece `-` antes que sobrevalorar el ajuste.

## Precauciones y distinciones

- `FQ-` no debe asignarse por mera rareza; la clave es la mala calidad de ajuste.
- Si la respuesta es rara pero todavia plausible y relativamente facil de ver, corresponde [`FQu`](ref://scoring-input/fq/u).
- Una o dos respuestas `-` aisladas no agotan la interpretacion; su peso depende de frecuencia y patron.
