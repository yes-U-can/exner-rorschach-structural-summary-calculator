# Nombre del documento: [Codificacion/Calidad formal] +

## Alias y busqueda

- FQ+
- ordinaria-elaborada
- plus form quality

## Definicion central

FQ+ se usa cuando la respuesta muestra un ajuste formal apropiado y, ademas, una articulacion de forma mas detallada y elaborada que la ordinaria.  
Es una respuesta bien ajustada y especialmente bien especificada.

## Condiciones de aplicacion

- La respuesta ya cumple un buen ajuste formal de base.
- El sujeto articula rasgos formales de manera mas rica o precisa que en una respuesta ordinaria.
- La elaboracion mejora la calidad de la respuesta sin perder ajuste formal.
- `FQ+` sigue siendo una categoria poco frecuente en comparacion con [`FQo`](ref://scoring-input/fq/o).

## Precauciones y distinciones

- No basta con que la respuesta suene compleja o imaginativa. La mejora debe estar en la articulacion formal.
- Si la respuesta es buena pero no claramente mas elaborada que la ordinaria, corresponde [`FQo`](ref://scoring-input/fq/o).
- No confunda `FQ+` con [`DQ+`](ref://scoring-input/dq/%2B). Uno evalua calidad formal; el otro, sintesis organizativa.
