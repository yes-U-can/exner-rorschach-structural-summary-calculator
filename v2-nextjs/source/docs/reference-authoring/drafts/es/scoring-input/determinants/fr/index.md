# Nombre del documento: [Codificación/Determinantes] Fr

## Alias y búsqueda

- Fr
- forma-reflejo
- form-reflection

## Definición central

`Fr` se usa cuando la forma organiza primero la respuesta y el sujeto además describe el objeto como un reflejo o una imagen especular.
La simetría tiene que estar implicada y la idea de reflejo debe estar explícitamente presente.

## Condiciones de aplicación

- La forma debe identificar primero el objeto principal.
- El sujeto tiene que hablar de reflejo, imagen en espejo o algo equivalente.
- Los dos objetos deben ser idénticos de manera explícita o implícita.
- Si solo hay dos objetos iguales por simetría, sin idea de reflejo, hay que reconsiderar [`(2)`](ref://scoring-input/pair).

## Precauciones y distinciones

- `Fr` no es igual a [`rF`](ref://scoring-input/determinants/rF); en `Fr` la forma domina más claramente.
- `Fr` no sustituye al código de par; cuando hay reflejo no se agrega [`(2)`](ref://scoring-input/pair).
- Si la respuesta solo dice que hay dos figuras parecidas, pero no habla de espejo o reflejo, no corresponde `Fr`.
- Si la respuesta depende más del efecto especular que de la forma del objeto, hay que revisar [`rF`](ref://scoring-input/determinants/rF).
