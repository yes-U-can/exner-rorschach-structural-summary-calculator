# Nombre del documento: [Codificacion/Determinantes] Fr

## Alias y busqueda

- Fr
- forma-reflejo
- form-reflection

## Definicion central

`Fr` se usa cuando la forma organiza primero la respuesta y el sujeto ademas describe el objeto como un reflejo o una imagen especular.  
La simetria tiene que estar implicada y la idea de reflejo debe estar explicitamente presente.

## Condiciones de aplicacion

- La forma debe identificar primero el objeto principal.
- El sujeto tiene que hablar de reflejo, imagen en espejo o algo equivalente.
- Los dos objetos deben ser identicos de manera explicita o implicita.
- Si solo hay dos objetos iguales por simetria, sin idea de reflejo, hay que reconsiderar [`(2)`](ref://scoring-input/pair).

## Precauciones y distinciones

- `Fr` no es igual a [`rF`](ref://scoring-input/determinants/rF); en `Fr` la forma domina mas claramente.
- `Fr` no sustituye al codigo de par; cuando hay reflejo no se agrega [`(2)`](ref://scoring-input/pair).
- Si la respuesta solo dice que hay dos figuras parecidas, pero no habla de espejo o reflejo, no corresponde `Fr`.
- Si la respuesta depende mas del efecto especular que de la forma del objeto, hay que revisar [`rF`](ref://scoring-input/determinants/rF).
