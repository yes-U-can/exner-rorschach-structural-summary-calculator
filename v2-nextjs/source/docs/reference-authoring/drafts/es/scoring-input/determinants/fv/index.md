# Nombre del documento: [Codificacion/Determinantes] FV

## Alias y busqueda

- FV
- forma-vista
- form-vista

## Definicion central

`FV` se usa cuando la forma organiza la respuesta y la cualidad de profundidad o relieve queda en segundo plano.  
La sensacion de dimension esta presente, pero el percepto sigue apoyandose primero en la forma.

## Condiciones de aplicacion

- La forma identifica el objeto principal.
- El sujeto agrega despues sensacion de profundidad, distancia o relieve.
- La dimension debe estar realmente articulada en la respuesta.
- Si la vista domina mas que la forma, hay que reconsiderar [`VF`](ref://scoring-input/determinants/VF).

## Precauciones y distinciones

- `FV` no equivale a [`VF`](ref://scoring-input/determinants/VF); cambia cual aspecto domina.
- Si la impresion dominante es textura, hay que revisar [`FT`](ref://scoring-input/determinants/FT).
- Si la impresion dominante es sombreado vago o malestar difuso, hay que revisar [`Y`](ref://scoring-input/determinants/Y).
- El uso interpretativo de esta familia reaparece despues en [`SumV`](ref://result-interpretation/lower-section/core/SumV).
