# Nombre del documento: [Codificación/Determinantes] FV

## Alias y búsqueda

- FV
- forma-vista
- form-vista

## Definición central

`FV` se usa cuando la forma organiza la respuesta y la cualidad de profundidad o relieve queda en segundo plano.
La sensación de dimensión está presente, pero el percepto sigue apoyándose primero en la forma.

## Condiciones de aplicación

- La forma identifica el objeto principal.
- El sujeto agrega después sensación de profundidad, distancia o relieve.
- La dimensión debe estar realmente articulada en la respuesta.
- Si la vista domina más que la forma, hay que reconsiderar [`VF`](ref://scoring-input/determinants/VF).

## Precauciones y distinciones

- `FV` no equivale a [`VF`](ref://scoring-input/determinants/VF); cambia cuál aspecto domina.
- Si la impresión dominante es textura, hay que revisar [`FT`](ref://scoring-input/determinants/FT).
- Si la impresión dominante es sombreado vago o malestar difuso, hay que revisar [`Y`](ref://scoring-input/determinants/Y).
- El uso interpretativo de esta familia reaparece después en [`SumV`](ref://result-interpretation/lower-section/core/SumV).
