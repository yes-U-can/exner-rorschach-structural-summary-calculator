# Nombre del documento: [Codificación/Determinantes] C'F

## Alias y búsqueda

- C'F
- color acromático-forma
- achromatic color-form

## Definición central

`C'F` se usa cuando el color acromático domina la respuesta y la forma queda en segundo plano.
La respuesta depende más de rasgos negros, blancos o grises usados como color que de la organización formal.

## Condiciones de aplicación

- El sujeto usa características acromáticas como base principal de la respuesta.
- Existe cierto apoyo formal, pero claramente secundario.
- Si la forma lleva el peso principal, debe reconsiderarse [`FC'`](ref://scoring-input/determinants/FC%27).
- Si no hay apoyo formal real, puede corresponder [`C'`](ref://scoring-input/determinants/C%27).

## Precauciones y distinciones

- `C'F` no es lo mismo que [`CF`](ref://scoring-input/determinants/CF); cambian tanto el tipo de color como la fuente perceptiva.
- El hecho de que el sujeto diga negro, blanco o gris no obliga por sí solo a codificar `C'F`.
- Si la respuesta se apoya más en distancia o relieve, hay que revisar [`V`](ref://scoring-input/determinants/V).
- Si la impresión principal es indefinida, nebulosa o de sombreado difuso, hay que revisar [`Y`](ref://scoring-input/determinants/Y).
