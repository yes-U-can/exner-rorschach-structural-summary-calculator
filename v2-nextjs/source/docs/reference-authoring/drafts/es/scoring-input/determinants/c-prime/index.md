# Nombre del documento: [Codificacion/Determinantes] C'

## Alias y busqueda

- C'
- color acromatico puro
- pure achromatic color

## Definicion central

`C'` se usa cuando la respuesta depende exclusivamente del color acromatico de la mancha.  
No hay apoyo formal suficiente y los rasgos negros, blancos o grises funcionan claramente como color.

## Condiciones de aplicacion

- La respuesta se apoya en negro, blanco o gris usados como color.
- No aparece organizacion formal suficiente para `FC'` o `C'F`.
- Si la encuesta muestra alguna forma secundaria real, hay que reconsiderar `C'F`.
- La decision exige que el uso de color acromatico sea claro e inequivoco.

## Precauciones y distinciones

- `C'` no es igual que [`C`](ref://scoring-input/determinants/C); ambos son color puro, pero uno es acromatico y el otro cromatico.
- Muchas palabras clave de acromatico tambien pueden aparecer en sombreado. Por eso hay que confirmar que funcionan como color.
- Si la respuesta apunta a profundidad o distancia, debe revisarse [`V`](ref://scoring-input/determinants/V).
- Si la respuesta apunta a sombreado difuso, negrura o impresion atmosferica, debe revisarse [`Y`](ref://scoring-input/determinants/Y).
