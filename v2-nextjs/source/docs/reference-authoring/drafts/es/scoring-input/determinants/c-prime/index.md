# Nombre del documento: [Codificación/Determinantes] C'

## Alias y búsqueda

- C'
- color acromático puro
- pure achromatic color

## Definición central

`C'` se usa cuando la respuesta depende exclusivamente del color acromático de la mancha.
No hay apoyo formal suficiente y los rasgos negros, blancos o grises funcionan claramente como color.

## Condiciones de aplicación

- La respuesta se apoya en negro, blanco o gris usados como color.
- No aparece organización formal suficiente para `FC'` o `C'F`.
- Si la encuesta muestra alguna forma secundaria real, hay que reconsiderar `C'F`.
- La decisión exige que el uso de color acromático sea claro e inequívoco.

## Precauciones y distinciones

- `C'` no es igual que [`C`](ref://scoring-input/determinants/C); ambos son color puro, pero uno es acromático y el otro cromático.
- Muchas palabras clave de acromático también pueden aparecer en sombreado. Por eso hay que confirmar que funcionan como color.
- Si la respuesta apunta a profundidad o distancia, debe revisarse [`V`](ref://scoring-input/determinants/V).
- Si la respuesta apunta a sombreado difuso, negrura o impresión atmosférica, debe revisarse [`Y`](ref://scoring-input/determinants/Y).
