# Nombre del documento: [Codificacion/Determinantes] Cn

## Alias y busqueda

- Cn
- color naming
- nombrar color

## Definicion central

`Cn` se usa cuando el sujeto nombra un color, pero no utiliza el color como verdadero determinante de la respuesta.  
Por eso `Cn` no entra en la familia cromatica `C / CF / FC` como codigo de color pleno.

## Condiciones de aplicacion

- El sujeto menciona un color de manera explicita.
- Esa mencion no sostiene la construccion perceptiva principal de la respuesta.
- El objeto seguiria siendo el mismo aunque se retirara la referencia al color.
- Si el color realmente participa en la respuesta, debe reconsiderarse [`FC`](ref://scoring-input/determinants/FC), [`CF`](ref://scoring-input/determinants/CF) o [`C`](ref://scoring-input/determinants/C).

## Precauciones y distinciones

- `Cn` no equivale a `C`. En `C`, el color es determinante real; en `Cn`, el color solo es nombrado.
- Una menc ion casual del color no debe inflarse hasta `FC` o `CF` si la respuesta no depende de ello.
- `Cn` no se incorpora a recuentos interpretativos como si fuera color cromatico pleno.
- Cuando exista duda, la pregunta clave es si el color hace que eso parezca eso o si el sujeto solo esta describiendo un rasgo accesorio.
