# Nombre del documento: [Codificación/Determinantes] F

## Alias y búsqueda

- F
- forma pura
- pure form

## Definición central

`F` se usa cuando la respuesta se sostiene solo por la forma.
No hay un uso operativo de movimiento, color, reflejo, textura, vista o dimensión adicional.

## Condiciones de aplicación

- La forma debe ser la única base explícita de la respuesta.
- En la encuesta, el sujeto debe seguir apoyándose en contorno, borde o configuración formal.
- Si aparece otro determinante real, `F` deja de ser suficiente por sí solo.
- La decisión final debe tomarse según lo que el sujeto dijo, no por una inferencia libre del examinador.

## Precauciones y distinciones

- `F` no es lo mismo que [`FQ`](ref://scoring-input/fq); `F` es determinante y `FQ` es calidad formal.
- Si la respuesta incluye profundidad creada por el contorno o el tamaño, hay que revisar [`FD`](ref://scoring-input/determinants/FD).
- Si la respuesta agrega textura, vista o sombreado difuso, hay que revisar [`FT`](ref://scoring-input/determinants/FT), [`V`](ref://scoring-input/determinants/V) o [`Y`](ref://scoring-input/determinants/Y).
- Una respuesta puede tener `F` dentro de una combinación compleja, pero no por eso deja de requerir revisar si otro determinante domina.
