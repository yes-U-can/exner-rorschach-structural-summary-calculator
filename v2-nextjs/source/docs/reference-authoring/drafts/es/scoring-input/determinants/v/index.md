# Nombre del documento: [Codificación/Determinantes] V

## Alias y búsqueda

- V
- vista pura
- pure vista

## Definición central

`V` se usa cuando la respuesta depende de profundidad o relieve sin apoyo formal suficiente.
La experiencia dimensional domina por sí misma y la forma no organiza realmente el percepto.

## Condiciones de aplicación

- La respuesta nace de una impresión de distancia, profundidad o relieve.
- No hay forma suficiente para `FV` o `VF`.
- La encuesta debe mostrar que esa cualidad dimensional es real y no una inferencia del examinador.
- Si aparece apoyo formal secundario, hay que reconsiderar `VF`.

## Precauciones y distinciones

- `V` no es lo mismo que la tarjeta `V`; son contextos distintos.
- Si la respuesta se apoya más en textura, hay que revisar [`T`](ref://scoring-input/determinants/T).
- Si la respuesta se apoya más en sombreado difuso o impresión vaga, hay que revisar [`Y`](ref://scoring-input/determinants/Y).
- El uso interpretativo de esta familia reaparece después en [`SumV`](ref://result-interpretation/lower-section/core/SumV).
