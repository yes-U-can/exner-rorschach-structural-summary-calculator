# Nombre del documento: [Codificación/Determinantes] FC'

## Alias y búsqueda

- FC'
- forma-color acromático
- achromatic form-color

## Definición central

`FC'` se usa cuando la forma organiza la respuesta y el color acromático queda en segundo plano.
Las características negras, blancas o grises tienen que funcionar claramente como color, no solo como sombreado o como descripción de área.

## Condiciones de aplicación

- La forma sostiene la identificación principal del objeto.
- Los rasgos acromáticos participan como color, pero con importancia secundaria.
- Si el color acromático domina la respuesta, hay que reconsiderar [`C'F`](ref://scoring-input/determinants/C%27F).
- Si casi no hay exigencia formal, puede corresponder [`C'`](ref://scoring-input/determinants/C%27).

## Precauciones y distinciones

- `FC'` no es lo mismo que [`FC`](ref://scoring-input/determinants/FC); el primero usa color acromático, el segundo color cromático.
- Palabras como negro, blanco, gris, claro u oscuro no bastan por sí solas; hay que confirmar que se usan como color.
- Si la respuesta apunta más a profundidad o distancia, debe revisarse [`V`](ref://scoring-input/determinants/V).
- Si la respuesta apunta más a atmósfera vaga, negrura o malestar difuso, debe revisarse [`Y`](ref://scoring-input/determinants/Y).
