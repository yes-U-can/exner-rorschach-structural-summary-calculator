# Nombre del documento: [Codificacion/Determinantes] FC'

## Alias y busqueda

- FC'
- forma-color acromatico
- achromatic form-color

## Definicion central

`FC'` se usa cuando la forma organiza la respuesta y el color acromatico queda en segundo plano.  
Las caracteristicas negras, blancas o grises tienen que funcionar claramente como color, no solo como sombreado o como descripcion de area.

## Condiciones de aplicacion

- La forma sostiene la identificacion principal del objeto.
- Los rasgos acromaticos participan como color, pero con importancia secundaria.
- Si el color acromatico domina la respuesta, hay que reconsiderar [`C'F`](ref://scoring-input/determinants/C%27F).
- Si casi no hay exigencia formal, puede corresponder [`C'`](ref://scoring-input/determinants/C%27).

## Precauciones y distinciones

- `FC'` no es lo mismo que [`FC`](ref://scoring-input/determinants/FC); el primero usa color acromatico, el segundo color cromatico.
- Palabras como negro, blanco, gris, claro u oscuro no bastan por si solas; hay que confirmar que se usan como color.
- Si la respuesta apunta mas a profundidad o distancia, debe revisarse [`V`](ref://scoring-input/determinants/V).
- Si la respuesta apunta mas a atmosfera vaga, negrura o malestar difuso, debe revisarse [`Y`](ref://scoring-input/determinants/Y).
