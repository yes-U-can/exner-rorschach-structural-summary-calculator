# Nombre del documento: [Codificacion/Localizacion] DdS

## Alias y busqueda

- DdS
- detalle inusual con espacio blanco
- Dd + S

## Definicion central

`DdS` se usa cuando la respuesta parte de un detalle inusual y ademas integra espacio blanco.  
Combina dos decisiones: area poco frecuente y uso real del blanco.

## Condiciones de aplicacion

- El area basica es `Dd`.
- El espacio blanco cumple una funcion especifica dentro del percepto.
- La participacion del blanco debe poder justificarse con la encuesta o la formulacion del sujeto.

## Precauciones y distinciones

- Si el area basica es `D`, corresponde [`DS`](ref://scoring-input/location/DS).
- Si no hay uso real del blanco, corresponde solo [`Dd`](ref://scoring-input/location/Dd).
- `DdS` no se asigna por incluir blanco de forma pasiva o accidental.
