# Nombre del documento: [Codificación/Localización] DdS

## Alias y búsqueda

- DdS
- detalle inusual con espacio blanco
- Dd + S

## Definición central

`DdS` se usa cuando la respuesta parte de un detalle inusual y además integra espacio blanco.
Combina dos decisiones: área poco frecuente y uso real del blanco.

## Condiciones de aplicación

- El área básica es `Dd`.
- El espacio blanco cumple una función específica dentro del percepto.
- La participación del blanco debe poder justificarse con la encuesta o la formulación del sujeto.

## Precauciones y distinciones

- Si el área básica es `D`, corresponde [`DS`](ref://scoring-input/location/DS).
- Si no hay uso real del blanco, corresponde solo [`Dd`](ref://scoring-input/location/Dd).
- `DdS` no se asigna por incluir blanco de forma pasiva o accidental.
