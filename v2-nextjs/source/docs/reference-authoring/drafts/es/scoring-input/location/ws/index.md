# Nombre del documento: [Codificacion/Localizacion] WS

## Alias y busqueda

- WS
- global con espacio blanco
- W + S

## Definicion central

`WS` se usa cuando la respuesta emplea toda la mancha e integra ademas una zona de espacio blanco.  
La clave no es solo que haya blanco dentro del area, sino que ese blanco tenga una funcion real en la respuesta.

## Condiciones de aplicacion

- La respuesta sigue siendo global.
- El espacio blanco participa activamente en la construccion del objeto.
- La integracion del blanco queda explicitada por el sujeto o confirmada con claridad en la encuesta.

## Precauciones y distinciones

- No codifique `S` solo porque el sujeto senale una zona que contiene blanco si no le asigna un papel especifico.
- Si la respuesta usa espacio blanco pero no toda la mancha, corresponde [`DS`](ref://scoring-input/location/DS) o [`DdS`](ref://scoring-input/location/DdS).
- `WS` puede coexistir con codigos organizativos o Z, pero no los reemplaza.
