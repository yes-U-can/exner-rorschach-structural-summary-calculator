# Nombre del documento: [Codificación/Localización] WS

## Alias y búsqueda

- WS
- global con espacio blanco
- W + S

## Definición central

`WS` se usa cuando la respuesta emplea toda la mancha e integra además una zona de espacio blanco.
La clave no es solo que haya blanco dentro del área, sino que ese blanco tenga una función real en la respuesta.

## Condiciones de aplicación

- La respuesta sigue siendo global.
- El espacio blanco participa activamente en la construcción del objeto.
- La integración del blanco queda explicitada por el sujeto o confirmada con claridad en la encuesta.

## Precauciones y distinciones

- No codifique `S` solo porque el sujeto señale una zona que contiene blanco si no le asigna un papel específico.
- Si la respuesta usa espacio blanco pero no toda la mancha, corresponde [`DS`](ref://scoring-input/location/DS) o [`DdS`](ref://scoring-input/location/DdS).
- `WS` puede coexistir con códigos organizativos o Z, pero no los reemplaza.
