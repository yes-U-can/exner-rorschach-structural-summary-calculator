# Nombre del documento: [Codificación/Localización] D

## Alias y búsqueda

- D
- detalle usual
- detalle frecuente

## Definición central

`D` se usa cuando la respuesta se basa en un área de detalle utilizada con frecuencia por muchos sujetos.
La decisión depende de la frecuencia de uso del área, no de su tamaño aparente.

## Condiciones de aplicación

- La respuesta no usa toda la mancha.
- El área utilizada corresponde a un detalle usual de la tabla.
- Si se combinan varias áreas D y la combinación ya existe como área D reconocida, la codificación sigue siendo `D`.

## Precauciones y distinciones

- `D` y [`Dd`](ref://scoring-input/location/Dd) no se diferencian por tamaño.
- Si la combinación de varias áreas D crea una zona no habitual para un solo objeto, puede corresponder `Dd`.
- Si el detalle frecuente integra espacio blanco, corresponde [`DS`](ref://scoring-input/location/DS).
