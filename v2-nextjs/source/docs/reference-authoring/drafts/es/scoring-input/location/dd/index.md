# Nombre del documento: [Codificación/Localización] Dd

## Alias y búsqueda

- Dd
- detalle inusual
- detalle poco frecuente

## Definición central

`Dd` se usa cuando la respuesta recae en un área de detalle utilizada con poca frecuencia.
Es el código para detalles no habituales, incluso cuando el área sea relativamente grande.

## Condiciones de aplicación

- La respuesta no es `W` ni `D`.
- El área utilizada no corresponde a un detalle usual de la tabla.
- Si una combinación de áreas D forma una zona no habitual para un solo objeto, esa combinación puede codificarse `Dd`.

## Precauciones y distinciones

- `Dd` no significa pequeño; significa poco frecuente.
- Si el sujeto usa varias áreas D para objetos separados, puede mantenerse `D` y registrarse la síntesis en DQ.
- Si el detalle inusual integra espacio blanco, corresponde [`DdS`](ref://scoring-input/location/DdS).
