# Nombre del documento: [Codificacion/Localizacion] DS

## Alias y busqueda

- DS
- detalle usual con espacio blanco
- D + S

## Definicion central

`DS` se usa cuando la respuesta parte de un detalle usual e integra espacio blanco como parte real del percepto.  
El blanco debe cumplir una funcion especifica, no solo aparecer dentro del area senalada.

## Condiciones de aplicacion

- El area basica corresponde a un detalle usual.
- El espacio blanco participa en la respuesta.
- La encuesta o la formulacion del sujeto deja claro que el blanco fue usado de manera activa.

## Precauciones y distinciones

- Si el area basica es inusual, corresponde [`DdS`](ref://scoring-input/location/DdS).
- Si no hay uso real de espacio blanco, corresponde solo [`D`](ref://scoring-input/location/D).
- `DS` no debe asignarse por presencia accidental de blanco dentro del contorno.
