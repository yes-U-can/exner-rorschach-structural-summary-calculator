# Nombre del documento: [Codificación/Localización] DS

## Alias y búsqueda

- DS
- detalle usual con espacio blanco
- D + S

## Definición central

`DS` se usa cuando la respuesta parte de un detalle usual e integra espacio blanco como parte real del percepto.
El blanco debe cumplir una función específica, no solo aparecer dentro del área señalada.

## Condiciones de aplicación

- El área básica corresponde a un detalle usual.
- El espacio blanco participa en la respuesta.
- La encuesta o la formulación del sujeto deja claro que el blanco fue usado de manera activa.

## Precauciones y distinciones

- Si el área básica es inusual, corresponde [`DdS`](ref://scoring-input/location/DdS).
- Si no hay uso real de espacio blanco, corresponde solo [`D`](ref://scoring-input/location/D).
- `DS` no debe asignarse por presencia accidental de blanco dentro del contorno.
