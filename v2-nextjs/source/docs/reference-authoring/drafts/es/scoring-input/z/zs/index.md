# Nombre del documento: [Codificación/Z] ZS

## Alias y búsqueda

- ZS
- integración del espacio blanco
- Z con espacio blanco

## Definición central

`ZS` se usa cuando la actividad organizativa depende de integrar espacio blanco con otras áreas de la mancha.
El espacio blanco debe tener un papel específico dentro del percepto.

## Condiciones de aplicación

- La respuesta usa forma.
- El espacio blanco se integra con otras partes del blot.
- El sujeto asigna al blanco una función real en el objeto o escena.
- El blanco no constituye por sí solo toda la respuesta.

## Precauciones y distinciones

- Si el blanco solo está contenido dentro del área señalada, pero no participa de verdad, no corresponde `ZS`.
- Respuestas dadas solo al espacio blanco no reciben `Z`.
- `ZS` suele sobrecodificarse cuando el examinador supone integración sin respaldo suficiente de la encuesta.
