# Nombre del documento: [Codificacion/Z] ZW

## Alias y busqueda

- ZW
- Z global
- actividad organizativa global

## Definicion central

`ZW` se usa cuando la respuesta es global y cumple los requisitos de Z.  
La base es una respuesta `W` con calidad evolutiva compatible.

## Condiciones de aplicacion

- La respuesta usa toda la mancha.
- La respuesta implica uso de forma.
- La DQ de la respuesta es `+`, `v/+` u `o`.
- La respuesta no es `Wv`.

## Precauciones y distinciones

- Una respuesta global no recibe `ZW` automaticamente. Si es `Wv`, no recibe Z.
- `ZW` depende de la combinacion entre localizacion global y DQ compatible.
- Si la organizacion principal se basa en detalles adyacentes, distantes o espacio blanco, puede corresponder otro subtipo de Z.
