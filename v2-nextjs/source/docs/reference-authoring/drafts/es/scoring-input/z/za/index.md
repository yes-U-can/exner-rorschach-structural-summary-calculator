# Nombre del documento: [Codificación/Z] ZA

## Alias y búsqueda

- ZA
- detalle adyacente
- integración adyacente

## Definición central

`ZA` se usa cuando la respuesta integra áreas adyacentes de la mancha en una relación significativa.
La organización se apoya en unir partes contiguas del blot.

## Condiciones de aplicación

- La respuesta usa forma.
- La organización une áreas adyacentes del blot.
- Los elementos aparecen separados o distinguibles, pero relacionados de manera significativa.
- La respuesta no se limita a listar partes cercanas sin integración.

## Precauciones y distinciones

- La cercanía física entre áreas no basta por sí sola; debe haber relación organizada.
- Si las áreas organizadas son no adyacentes, corresponde [`ZD`](ref://scoring-input/z/ZD).
- Si la organización principal depende del espacio blanco, corresponde [`ZS`](ref://scoring-input/z/ZS).
