# Nombre del documento: [Codificacion/Z] ZD

## Alias y busqueda

- ZD
- detalle distante
- integracion distante

## Definicion central

`ZD` se usa cuando la respuesta integra areas no adyacentes de la mancha en una relacion significativa.  
La actividad organizativa depende de unir partes separadas del blot.

## Condiciones de aplicacion

- La respuesta usa forma.
- La organizacion conecta areas no contiguas.
- Los elementos separados forman una escena o relacion significativa.
- La distancia entre las partes es relevante para el codigo.

## Precauciones y distinciones

- Si las areas son contiguas, corresponde [`ZA`](ref://scoring-input/z/ZA).
- Si la organizacion principal se basa en todo el blot, puede corresponder [`ZW`](ref://scoring-input/z/ZW).
- `ZD` no se asigna por simple descripcion de varias zonas lejanas sin integracion real.
