# Nombre del documento: [Codificación/Z] ZD

## Alias y búsqueda

- ZD
- detalle distante
- integración distante

## Definición central

`ZD` se usa cuando la respuesta integra áreas no adyacentes de la mancha en una relación significativa.
La actividad organizativa depende de unir partes separadas del blot.

## Condiciones de aplicación

- La respuesta usa forma.
- La organización conecta áreas no contiguas.
- Los elementos separados forman una escena o relación significativa.
- La distancia entre las partes es relevante para el código.

## Precauciones y distinciones

- Si las áreas son contiguas, corresponde [`ZA`](ref://scoring-input/z/ZA).
- Si la organización principal se basa en todo el blot, puede corresponder [`ZW`](ref://scoring-input/z/ZW).
- `ZD` no se asigna por simple descripción de varias zonas lejanas sin integración real.
