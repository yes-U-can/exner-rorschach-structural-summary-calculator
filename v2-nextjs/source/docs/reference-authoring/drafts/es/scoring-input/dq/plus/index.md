# Nombre del documento: [Codificación/Calidad evolutiva] +

## Alias y búsqueda

- DQ+
- respuesta de síntesis (+)
- síntesis con demanda formal

## Definición central

DQ+ se asigna cuando la respuesta sintetiza dos o más objetos separados pero relacionados y, además, por lo menos uno de ellos exige una forma específica.
La clave no es solo que haya varios objetos, sino que la respuesta forme una unidad organizada.

## Condiciones de aplicación

- Hay más de un objeto o parte diferenciada dentro de la respuesta.
- Los objetos aparecen en una relación significativa entre si.
- Por lo menos uno de los objetos implica demanda formal específica.
- La síntesis debe sentirse estructurada, no simplemente yuxtapuesta.

## Precauciones y distinciones

- Dos objetos parecidos por simetría, sin una relación significativa, no bastan para `DQ+`; en ese caso suele corresponder [`DQo`](ref://scoring-input/dq/o).
- Si existe síntesis, pero ninguno de los objetos exige forma específica, corresponde [`DQv/+`](ref://scoring-input/dq/v/%2B).
- `DQ+` no significa lo mismo que [`FQ+`](ref://scoring-input/fq/%2B). El primero habla de organización; el segundo de calidad formal.
