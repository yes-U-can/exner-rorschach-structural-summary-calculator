# Nombre del documento: [Codificación/Calidad evolutiva] v/+

## Alias y búsqueda

- DQv/+
- síntesis vaga
- síntesis sin demanda formal

## Definición central

DQv/+ se usa cuando la respuesta muestra síntesis entre objetos separados pero relacionados, pero ninguno de los objetos exige una forma específica.
Hay organización, pero el material sintetizado sigue siendo formalmente vago.

## Condiciones de aplicación

- La respuesta combina dos o más elementos en una sola escena o unidad.
- Los elementos están separados pero guardan una relación significativa.
- Ninguno de los objetos implicados requiere una forma específica.
- La respuesta sigue siendo de síntesis, aunque los objetos pertenezcan a clases difusas como humo, nube, agua o materia extendida.

## Precauciones y distinciones

- Si por lo menos uno de los objetos exige forma específica, debe codificarse [`DQ+`](ref://scoring-input/dq/%2B).
- Si no hay síntesis significativa y el objeto sigue siendo difuso, corresponde [`DQv`](ref://scoring-input/dq/v).
- No confunda `v/+` con una descripción plural sin relación clara entre los elementos.
