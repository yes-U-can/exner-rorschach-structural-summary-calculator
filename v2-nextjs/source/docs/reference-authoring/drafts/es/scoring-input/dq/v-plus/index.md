# Nombre del documento: [Codificacion/Calidad evolutiva] v/+

## Alias y busqueda

- DQv/+
- sintesis vaga
- sintesis sin demanda formal

## Definicion central

DQv/+ se usa cuando la respuesta muestra sintesis entre objetos separados pero relacionados, pero ninguno de los objetos exige una forma especifica.  
Hay organizacion, pero el material sintetizado sigue siendo formalmente vago.

## Condiciones de aplicacion

- La respuesta combina dos o mas elementos en una sola escena o unidad.
- Los elementos estan separados pero guardan una relacion significativa.
- Ninguno de los objetos implicados requiere una forma especifica.
- La respuesta sigue siendo de sintesis, aunque los objetos pertenezcan a clases difusas como humo, nube, agua o materia extendida.

## Precauciones y distinciones

- Si por lo menos uno de los objetos exige forma especifica, debe codificarse [`DQ+`](ref://scoring-input/dq/%2B).
- Si no hay sintesis significativa y el objeto sigue siendo difuso, corresponde [`DQv`](ref://scoring-input/dq/v).
- No confunda `v/+` con una descripcion plural sin relacion clara entre los elementos.
