# Nombre del documento: [Codificacion/Calidad evolutiva] v

## Alias y busqueda

- DQv
- respuesta vaga
- vaga sin sintesis

## Definicion central

DQv se asigna cuando el objeto descrito no exige una forma especifica y la respuesta tampoco establece una sintesis significativa.  
La impresion global puede ser reconocible como categoria amplia, pero sigue siendo formalmente difusa.

## Condiciones de aplicacion

- El objeto pertenece a una clase que puede adoptar muchas formas posibles.
- La formulacion verbal no introduce una especificidad formal suficiente.
- No aparece una relacion sintetica significativa entre varios objetos.
- La respuesta conserva un caracter general o difuso.

## Precauciones y distinciones

- Si los objetos vagos se integran en una sintesis significativa, corresponde [`DQv/+`](ref://scoring-input/dq/v/%2B).
- Si la descripcion obliga a una forma especifica, incluso sin sintesis, corresponde [`DQo`](ref://scoring-input/dq/o).
- En interpretacion, el seguimiento posterior de este estilo de codificacion aparece en [`DQv`](ref://result-interpretation/lower-section/processing/DQ_v_proc).
