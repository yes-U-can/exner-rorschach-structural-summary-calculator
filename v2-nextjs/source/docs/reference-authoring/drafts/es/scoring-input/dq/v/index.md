# Nombre del documento: [Codificación/Calidad evolutiva] v

## Alias y búsqueda

- DQv
- respuesta vaga
- vaga sin síntesis

## Definición central

DQv se asigna cuando el objeto descrito no exige una forma específica y la respuesta tampoco establece una síntesis significativa.
La impresión global puede ser reconocible como categoría amplia, pero sigue siendo formalmente difusa.

## Condiciones de aplicación

- El objeto pertenece a una clase que puede adoptar muchas formas posibles.
- La formulación verbal no introduce una especificidad formal suficiente.
- No aparece una relación sintética significativa entre varios objetos.
- La respuesta conserva un carácter general o difuso.

## Precauciones y distinciones

- Si los objetos vagos se integran en una síntesis significativa, corresponde [`DQv/+`](ref://scoring-input/dq/v/%2B).
- Si la descripción obliga a una forma específica, incluso sin síntesis, corresponde [`DQo`](ref://scoring-input/dq/o).
- En interpretación, el seguimiento posterior de este estilo de codificación aparece en [`DQv`](ref://result-interpretation/lower-section/processing/DQ_v_proc).
