# Nombre del documento: [Codificacion/Puntuaciones especiales] FABCOM1

## Alias y busqueda

- FABCOM1
- fabulized combination level 1
- combinacion fabulada nivel 1

## Definicion central

`FABCOM1` es un special score de nivel 1 para relaciones fabulas o inverosimiles entre elementos de una respuesta.
No describe una sola combinacion incongruente dentro de un objeto, sino una relacion distorsionada entre ideas, acciones o partes de la escena.

## Condiciones de aplicacion

- Usa `FABCOM1` cuando la respuesta organiza una relacion improbable o forzada entre elementos que deberian conectarse de otra manera.
- El problema central es la logica fabulizada del conjunto, no solo una mezcla incongruente dentro de un mismo objeto.
- La extraneza debe ser observable, pero todavia no tan severa como para exigir `FABCOM2`.
- Si dudas entre nivel 1 y nivel 2, reserva `FABCOM2` para la distorsion mas extrema.

## Precauciones y distinciones

- Si la rareza principal es una combinacion incongruente dentro de una sola unidad perceptiva, revisa antes `INCOM1`.
- Si dos perceptos incompatibles aparecen fundidos como una misma percepcion, revisa `CONTAM`.
- `FABCOM1` no se usa solo porque una respuesta parezca fantastica o imaginativa.
- La clave es que la relacion entre elementos quede forzada o poco plausible dentro del response.
