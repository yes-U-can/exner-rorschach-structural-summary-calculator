# Nombre del documento: [Codificación/Puntuaciones especiales] MOR

## Alias y búsqueda

- MOR
- morbid content
- contenido mórbido

## Definición central

`MOR` se usa cuando el objeto aparece como muerto, roto, arruinado, lesionado, deteriorado o investido de una cualidad mórbida marcada.
El código apunta a daño, degradación o tono patológico atribuido directamente al contenido.

## Condiciones de aplicación

- Se aplica cuando el sujeto describe algo muerto, destruido, herido, podrido, quebrado, aplastado o claramente arruinado.
- También se considera cuando la respuesta atribuye una cualidad mórbida o patológica directa al objeto.
- `MOR` se agrega como puntuación especial y no reemplaza el contenido o el determinante principal.
- Puede coexistir con otros códigos si la respuesta muestra tanto contenido específico como tono mórbido.

## Precauciones y distinciones

- Un contenido feo, raro o desagradable no basta por sí solo para `MOR`.
- Si la acción principal es atacar o herir en movimiento, primero se considera [`AG`](ref://scoring-input/special-score/AG); si el foco está en el estado dañado, se considera `MOR`.
- En respuestas humanas, `MOR` puede relacionarse después con [`PHR`](ref://scoring-input/gphr/PHR), pero no la determina automáticamente.
