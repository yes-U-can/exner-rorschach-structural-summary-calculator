# Nombre del documento: [Codificacion/Puntuaciones especiales] MOR

## Alias y busqueda

- MOR
- morbid content
- contenido morbido

## Definicion central

`MOR` se usa cuando el objeto aparece como muerto, roto, arruinado, lesionado, deteriorado o investido de una cualidad morbida marcada.
El codigo apunta a dano, degradacion o tono patologico atribuido directamente al contenido.

## Condiciones de aplicacion

- Se aplica cuando el sujeto describe algo muerto, destruido, herido, podrido, quebrado, aplastado o claramente arruinado.
- Tambien se considera cuando la respuesta atribuye una cualidad morbida o patologica directa al objeto.
- `MOR` se agrega como puntuacion especial y no reemplaza el contenido o el determinante principal.
- Puede coexistir con otros codigos si la respuesta muestra tanto contenido especifico como tono morbido.

## Precauciones y distinciones

- Un contenido feo, raro o desagradable no basta por si solo para `MOR`.
- Si la accion principal es atacar o herir en movimiento, primero se considera [`AG`](ref://scoring-input/special-score/AG); si el foco esta en el estado danado, se considera `MOR`.
- En respuestas humanas, `MOR` puede relacionarse despues con [`PHR`](ref://scoring-input/gphr/PHR), pero no la determina automaticamente.
