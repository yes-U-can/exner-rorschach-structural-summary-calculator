# Nombre del documento: [Codificación/Puntuaciones especiales] INCOM1

## Alias y búsqueda

- INCOM1
- incongruous combination level 1
- combinación incongruente nivel 1
- combinación incongruente leve

## Definición central

`INCOM1` es un código especial de nivel 1 para combinaciones incongruentes dentro de una misma respuesta.
Se usa cuando los elementos unidos no encajan bien entre si, pero la extrañeza todavía no alcanza el grado más grave de `INCOM2`.

## Condiciones de aplicación

- Considera `INCOM1` cuando dos partes, cualidades o funciones quedan combinadas de manera incongruente dentro del mismo objeto o de la misma escena.
- La combinación debe sonar impropia o poco coherente, no solo poco común.
- Si la respuesta resulta rara pero todavía puede seguirse con cierta lógica, `INCOM1` suele ser más apropiado que `INCOM2`.
- Cuando la incongruencia se limita a una palabra aislada o a una expresión verbal rara, primero hay que revisar si corresponde mejor `DV1` o `DV2`.

## Precauciones y distinciones

- `INCOM1` no se asigna solo porque la respuesta sea creativa, fantástica o poco frecuente.
- Si el problema principal es una relación imposible entre partes de la respuesta, pero con una organización más claramente deformada, revisa `FABCOM1`.
- Si dos realidades incompatibles quedan fundidas como un mismo percepto, revisa `CONTAM`.
- Si dudas entre nivel 1 y nivel 2, usa `INCOM1` de forma conservadora y deja `INCOM2` para la incongruencia más severa.
