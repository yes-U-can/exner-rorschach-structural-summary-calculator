# Nombre del documento: [Codificacion/Puntuaciones especiales] INCOM1

## Alias y busqueda

- INCOM1
- incongruous combination level 1
- combinacion incongruente nivel 1
- combinacion incongruente leve

## Definicion central

`INCOM1` es un special score de nivel 1 para combinaciones incongruentes dentro de una misma respuesta.
Se usa cuando los elementos unidos no encajan bien entre si, pero la extraneza todavia no alcanza el grado mas grave de `INCOM2`.

## Condiciones de aplicacion

- Considera `INCOM1` cuando dos partes, cualidades o funciones quedan combinadas de manera incongruente dentro del mismo objeto o de la misma escena.
- La combinacion debe sonar impropia o poco coherente, no solo poco comun.
- Si la respuesta resulta rara pero todavia puede seguirse con cierta logica, `INCOM1` suele ser mas apropiado que `INCOM2`.
- Cuando la incongruencia se limita a una palabra aislada o a una expresion verbal rara, primero hay que revisar si corresponde mejor `DV1` o `DV2`.

## Precauciones y distinciones

- `INCOM1` no se asigna solo porque la respuesta sea creativa, fantastica o poco frecuente.
- Si el problema principal es una relacion imposible entre partes de la respuesta, pero con una organizacion mas claramente deformada, revisa `FABCOM1`.
- Si dos realidades incompatibles quedan fundidas como un mismo percepto, revisa `CONTAM`.
- Si dudas entre nivel 1 y nivel 2, usa `INCOM1` de forma conservadora y deja `INCOM2` para la incongruencia mas severa.
