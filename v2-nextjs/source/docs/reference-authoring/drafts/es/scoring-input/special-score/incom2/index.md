# Nombre del documento: [Codificacion/Puntuaciones especiales] INCOM2

## Alias y busqueda

- INCOM2
- incongruous combination level 2
- combinacion incongruente nivel 2
- combinacion incongruente severa

## Definicion central

`INCOM2` es un special score de nivel 2 para combinaciones incongruentes claramente mas graves y extranas que `INCOM1`.
Se usa cuando la union entre partes o cualidades resulta marcadamente inverosimil o ilogica dentro de la respuesta.

## Condiciones de aplicacion

- Usa `INCOM2` cuando la combinacion incongruente es intensa, llamativa y dificil de sostener como una variacion leve.
- La respuesta debe mostrar una mezcla de elementos que no solo suena impropia, sino claramente extrana o severamente desajustada.
- Si la duda real es entre nivel 1 y nivel 2, `INCOM2` debe reservarse para el caso mas grave.
- La presencia de `INCOM2` suele anticipar relacion con `Lv2` y con el peso de `WSum6`, aunque el codigo primero se decide en la fase de codificacion.

## Precauciones y distinciones

- `INCOM2` no se usa solo porque una respuesta sea larga, rara o altamente imaginativa.
- Si la alteracion principal es una organizacion distorsionada entre ideas o acciones dentro de la respuesta, revisa `FABCOM2`.
- Si dos perceptos incompatibles quedan fusionados en una sola percepcion contaminada, revisa `CONTAM`.
- Si la rareza se limita mas a formulacion verbal extrana que a una combinacion incongruente del percepto, compara con `DV2`.
