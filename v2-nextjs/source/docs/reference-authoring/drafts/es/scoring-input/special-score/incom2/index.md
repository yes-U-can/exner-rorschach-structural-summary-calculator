# Nombre del documento: [Codificación/Puntuaciones especiales] INCOM2

## Alias y búsqueda

- INCOM2
- incongruous combination level 2
- combinación incongruente nivel 2
- combinación incongruente severa

## Definición central

`INCOM2` es un código especial de nivel 2 para combinaciones incongruentes claramente más graves y extrañas que `INCOM1`.
Se usa cuando la unión entre partes o cualidades resulta marcadamente inverosímil o ilógica dentro de la respuesta.

## Condiciones de aplicación

- Usa `INCOM2` cuando la combinación incongruente es intensa, llamativa y difícil de sostener como una variación leve.
- La respuesta debe mostrar una mezcla de elementos que no solo suena impropia, sino claramente extraña o severamente desajustada.
- Si la duda real es entre nivel 1 y nivel 2, `INCOM2` debe reservarse para el caso más grave.
- La presencia de `INCOM2` suele anticipar relación con `Lv2` y con el peso de `WSum6`, aunque el código primero se decide en la fase de codificación.

## Precauciones y distinciones

- `INCOM2` no se usa solo porque una respuesta sea larga, rara o altamente imaginativa.
- Si la alteración principal es una organización distorsionada entre ideas o acciones dentro de la respuesta, revisa `FABCOM2`.
- Si dos perceptos incompatibles quedan fusionados en una sola percepción contaminada, revisa `CONTAM`.
- Si la rareza se limita más a formulación verbal extraña que a una combinación incongruente del percepto, compara con `DV2`.
