# Nombre del documento: [Codificación/Puntuaciones especiales] PSV

## Alias y búsqueda

- PSV
- perseveración
- perseveration

## Definición central

`PSV` se usa cuando el sujeto persevera y repite de manera rígida la misma respuesta o el mismo tipo de respuesta.
No se refiere a simple consistencia; apunta a repetición estereotipada o mecánica.

## Condiciones de aplicación

- Se considera cuando la misma respuesta o un patrón muy cercano se repite de forma perseverativa.
- Puede observarse dentro de la misma lámina o a través de varias láminas.
- Lo importante no es solo la semejanza, sino la cualidad de repetición rígida.
- El significado interpretativo posterior se conecta con la ruta [`PSV`](ref://result-interpretation/lower-section/processing/PSV).

## Precauciones y distinciones

- Una respuesta parecida por economía o preferencia no basta por sí sola para `PSV`.
- La codificación pide repetición insistente o mecánica, no solo un estilo estable.
- `PSV` describe la presencia de perseveración; no explica por sí mismo su causa.
