# Nombre del documento: [Codificacion/Puntuaciones especiales] PSV

## Alias y busqueda

- PSV
- perseveracion
- perseveration

## Definicion central

`PSV` se usa cuando el sujeto persevera y repite de manera rigida la misma respuesta o el mismo tipo de respuesta.
No se refiere a simple consistencia; apunta a repeticion estereotipada o mecanica.

## Condiciones de aplicacion

- Se considera cuando la misma respuesta o un patron muy cercano se repite de forma perseverativa.
- Puede observarse dentro de la misma lamina o a traves de varias laminas.
- Lo importante no es solo la semejanza, sino la cualidad de repeticion rigida.
- El significado interpretativo posterior se conecta con la ruta [`PSV`](ref://result-interpretation/lower-section/processing/PSV).

## Precauciones y distinciones

- Una respuesta parecida por economia o preferencia no basta por si sola para `PSV`.
- La codificacion pide repeticion insistente o mecanica, no solo un estilo estable.
- `PSV` describe la presencia de perseveracion; no explica por si mismo su causa.
