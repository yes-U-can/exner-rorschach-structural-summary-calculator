# Nombre del documento: [Codificacion/Puntuaciones especiales] DR1

## Alias y busqueda

- DR1
- deviant response level 1
- respuesta desviada nivel 1

## Definicion central

`DR1` se usa cuando la respuesta, tomada como conjunto, se aparta del curso esperado de forma leve.
La desviacion no esta solo en una palabra, sino en la organizacion o desarrollo general de la respuesta.

## Condiciones de aplicacion

- Se considera cuando la respuesta incluye digresion, desvio o articulacion impropia en un grado leve.
- La perturbacion afecta a la respuesta completa mas que a una sola expresion verbal.
- Si la desviacion parece centrarse solo en la formulacion verbal, primero se revisan [`DV1`](ref://scoring-input/special-score/DV1) y [`DV2`](ref://scoring-input/special-score/DV2).
- Si la desorganizacion global es claramente mas severa, tambien se revisa [`DR2`](ref://scoring-input/special-score/DR2).

## Precauciones y distinciones

- Una respuesta larga o complicada no basta por si sola para `DR1`.
- `DR1` se diferencia de `DV` porque mira la desviacion del response completo, no solo una eleccion verbal.
- Si existe duda entre nivel 1 y nivel 2, se mantiene el criterio mas conservador.
