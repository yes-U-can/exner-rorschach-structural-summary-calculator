# Nombre del documento: [Codificación/Puntuaciones especiales] DR1

## Alias y búsqueda

- DR1
- deviant response level 1
- respuesta desviada nivel 1

## Definición central

`DR1` se usa cuando la respuesta, tomada como conjunto, se aparta del curso esperado de forma leve.
La desviación no está solo en una palabra, sino en la organización o desarrollo general de la respuesta.

## Condiciones de aplicación

- Se considera cuando la respuesta incluye digresión, desvío o articulación impropia en un grado leve.
- La perturbación afecta a la respuesta completa más que a una sola expresión verbal.
- Si la desviación parece centrarse solo en la formulación verbal, primero se revisan [`DV1`](ref://scoring-input/special-score/DV1) y [`DV2`](ref://scoring-input/special-score/DV2).
- Si la desorganización global es claramente más severa, también se revisa [`DR2`](ref://scoring-input/special-score/DR2).

## Precauciones y distinciones

- Una respuesta larga o complicada no basta por sí sola para `DR1`.
- `DR1` se diferencia de `DV` porque considera la desviación de la respuesta completa, no solo una elección verbal.
- Si existe duda entre nivel 1 y nivel 2, se mantiene el criterio más conservador.
