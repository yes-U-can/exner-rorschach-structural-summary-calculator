# Nombre del documento: [Codificacion/Puntuaciones especiales] AG

## Alias y busqueda

- AG
- aggressive movement
- movimiento agresivo

## Definicion central

`AG` se usa cuando una respuesta con movimiento muestra una accion claramente agresiva.
Lo importante es que la agresion aparezca como acto en curso, no solo como cualidad vaga del objeto.

## Condiciones de aplicacion

- La respuesta debe incluir movimiento, normalmente [`M`](ref://scoring-input/determinants/M), [`FM`](ref://scoring-input/determinants/FM) o [`m`](ref://scoring-input/determinants/m).
- La accion debe ser agresiva de forma observable: golpear, atacar, desgarrar, empujar violentamente, morder o herir.
- El codigo se usa para agresion como movimiento, no solo para dano ya consumado.
- Si la accion agresiva es realizada conjuntamente por mas de una figura, `AG` puede coexistir con [`COP`](ref://scoring-input/special-score/COP).

## Precauciones y distinciones

- Un adjetivo hostil sin accion clara no basta para `AG`.
- Si el foco principal es que el objeto ya esta roto, muerto o deteriorado, primero se considera [`MOR`](ref://scoring-input/special-score/MOR).
- `AG` tampoco decide automaticamente [`PHR`](ref://scoring-input/gphr/PHR); esa decision requiere un paso adicional.
