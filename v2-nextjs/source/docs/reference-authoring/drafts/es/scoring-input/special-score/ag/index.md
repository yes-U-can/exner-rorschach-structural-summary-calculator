# Nombre del documento: [Codificación/Puntuaciones especiales] AG

## Alias y búsqueda

- AG
- aggressive movement
- movimiento agresivo

## Definición central

`AG` se usa cuando una respuesta con movimiento muestra una acción claramente agresiva.
Lo importante es que la agresión aparezca como acto en curso, no solo como cualidad vaga del objeto.

## Condiciones de aplicación

- La respuesta debe incluir movimiento, normalmente [`M`](ref://scoring-input/determinants/M), [`FM`](ref://scoring-input/determinants/FM) o [`m`](ref://scoring-input/determinants/m).
- La acción debe ser agresiva de forma observable: golpear, atacar, desgarrar, empujar violentamente, morder o herir.
- El código se usa para agresión como movimiento, no solo para daño ya consumado.
- Si la acción agresiva es realizada conjuntamente por más de una figura, `AG` puede coexistir con [`COP`](ref://scoring-input/special-score/COP).

## Precauciones y distinciones

- Un adjetivo hostil sin acción clara no basta para `AG`.
- Si el foco principal es que el objeto ya está roto, muerto o deteriorado, primero se considera [`MOR`](ref://scoring-input/special-score/MOR).
- `AG` tampoco decide automáticamente [`PHR`](ref://scoring-input/gphr/PHR); esa decisión requiere un paso adicional.
