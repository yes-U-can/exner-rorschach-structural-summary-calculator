# Nombre del documento: [Codificacion/Puntuaciones especiales] PER

## Alias y busqueda

- PER
- respuestas personalizadas
- personalized answers

## Definicion central

`PER` se usa cuando el sujeto introduce conocimiento personal, experiencia propia o referencia autobiografica para justificar o aclarar su respuesta.
Lo importante es que esa informacion personal funcione como apoyo directo de la codificacion.

## Condiciones de aplicacion

- Se considera cuando el sujeto dice que sabe que es asi por experiencia personal, aprendizaje previo o conocimiento propio.
- La informacion personal debe usarse para justificar o sostener la respuesta.
- Los pronombres personales por si solos no bastan para `PER`.
- El codigo se conecta despues con [`PER`](ref://result-interpretation/lower-section/interpersonal/PER) y puede revisarse junto con [`Xu%`](ref://result-interpretation/lower-section/mediation/Xu_percent).

## Precauciones y distinciones

- Decir "yo veo" o "me parece" no alcanza por si solo para `PER`.
- `PER` no es un codigo de contenido; marca una manera personalizada de justificar la respuesta.
- Una explicacion personal que no esta ligada a la respuesta concreta no debe recibir `PER`.
