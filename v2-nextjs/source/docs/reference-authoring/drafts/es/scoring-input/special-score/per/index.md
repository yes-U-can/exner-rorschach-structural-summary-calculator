# Nombre del documento: [Codificación/Puntuaciones especiales] PER

## Alias y búsqueda

- PER
- respuestas personalizadas
- personalized answers

## Definición central

`PER` se usa cuando el sujeto introduce conocimiento personal, experiencia propia o referencia autobiográfica para justificar o aclarar su respuesta.
Lo importante es que esa información personal funcione como apoyo directo de la codificación.

## Condiciones de aplicación

- Se considera cuando el sujeto dice que sabe que es así por experiencia personal, aprendizaje previo o conocimiento propio.
- La información personal debe usarse para justificar o sostener la respuesta.
- Los pronombres personales por sí solos no bastan para `PER`.
- El código se conecta después con [`PER`](ref://result-interpretation/lower-section/interpersonal/PER) y puede revisarse junto con [`Xu%`](ref://result-interpretation/lower-section/mediation/Xu_percent).

## Precauciones y distinciones

- Decir "yo veo" o "me parece" no alcanza por sí solo para `PER`.
- `PER` no es un código de contenido; marca una manera personalizada de justificar la respuesta.
- Una explicación personal que no está ligada a la respuesta concreta no debe recibir `PER`.
