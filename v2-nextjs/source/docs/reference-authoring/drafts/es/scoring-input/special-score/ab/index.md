# Nombre del documento: [Codificación/Puntuaciones especiales] AB

## Alias y búsqueda

- AB
- abstracciones
- abstractions

## Definición central

`AB` se usa cuando la respuesta se aleja del objeto concreto y pasa a una formulación abstracta, general o conceptualmente vagamente definida.
La idea central es que el sujeto nombra una abstracción o una cualidad global en vez de un objeto claramente delimitado.

## Condiciones de aplicación

- Se considera cuando la respuesta nombra algo como "una emoción", "una idea", "una fuerza", "una sensación" u otra formulación abstracta.
- Puede aparecer cerca de [`Hx`](ref://scoring-input/contents/Hx) cuando la respuesta apunta a experiencia humana general y no a persona concreta.
- También puede coexistir con [`Art`](ref://scoring-input/contents/Art) o [`Ay`](ref://scoring-input/contents/Ay) si el sujeto transforma el contenido en una categoría muy abstracta.
- El código se agrega como puntuación especial y no reemplaza el contenido principal.

## Precauciones y distinciones

- No toda respuesta artística o simbólica recibe `AB`.
- `Hx` por sí solo no obliga a codificar `AB`; hace falta que la formulación sea realmente abstracta.
- Una respuesta rara pero concreta no se codifica `AB` solo por sonar sofisticada.
