# Nombre del documento: [Codificacion/Puntuaciones especiales] AB

## Alias y busqueda

- AB
- abstracciones
- abstractions

## Definicion central

`AB` se usa cuando la respuesta se aleja del objeto concreto y pasa a una formulacion abstracta, general o conceptualmente vagamente definida.
La idea central es que el sujeto nombra una abstraccion o una cualidad global en vez de un objeto claramente delimitado.

## Condiciones de aplicacion

- Se considera cuando la respuesta nombra algo como "una emocion", "una idea", "una fuerza", "una sensacion" u otra formulacion abstracta.
- Puede aparecer cerca de [`Hx`](ref://scoring-input/contents/Hx) cuando la respuesta apunta a experiencia humana general y no a persona concreta.
- Tambien puede coexistir con [`Art`](ref://scoring-input/contents/Art) o [`Ay`](ref://scoring-input/contents/Ay) si el sujeto transforma el contenido en una categoria muy abstracta.
- El codigo se agrega como puntuacion especial y no reemplaza el contenido principal.

## Precauciones y distinciones

- No toda respuesta artistica o simbolica recibe `AB`.
- `Hx` por si solo no obliga a codificar `AB`; hace falta que la formulacion sea realmente abstracta.
- Una respuesta rara pero concreta no se codifica `AB` solo por sonar sofisticada.
