# Nombre del documento: [Codificacion/Puntuaciones especiales] DV1

## Alias y busqueda

- DV1
- deviant verbalization level 1
- verbalizacion desviada nivel 1

## Definicion central

`DV1` se usa cuando la expresion verbal de la respuesta resulta extraña o desviada, pero en un grado leve.
La idea central es que la formulacion suena rara o impropia sin llegar a una alteracion grave.

## Condiciones de aplicacion

- Se considera cuando la respuesta presenta uso verbal desviado, formulacion impropia o combinacion verbal extrana en un nivel relativamente leve.
- El problema principal esta en la expresion, no en una desorganizacion amplia de toda la respuesta.
- Si la extraneza verbal es mas marcada, primero se revisa [`DV2`](ref://scoring-input/special-score/DV2).
- Si la perturbacion parece afectar a la estructura completa de la respuesta, tambien se revisan [`DR1`](ref://scoring-input/special-score/DR1) y [`DR2`](ref://scoring-input/special-score/DR2).

## Precauciones y distinciones

- Una respuesta larga o torpe no basta por si sola para `DV1`.
- Si la duda entre nivel 1 y nivel 2 no se resuelve con claridad, se mantiene el criterio mas conservador.
- `DV1` se conecta despues con medidas como [`Sum6`](ref://result-interpretation/lower-section/ideation/Sum6) y [`WSum6`](ref://result-interpretation/lower-section/ideation/WSum6), pero la decision de codificacion sigue siendo local a la respuesta.
