# Nombre del documento: [Codificación/Puntuaciones especiales] CONTAM

## Alias y búsqueda

- CONTAM
- contamination
- contaminación

## Definición central

`CONTAM` se usa cuando dos perceptos incompatibles quedan fusionados como una sola respuesta contaminada.
No describe solo una mezcla rara, sino una integración imposible dentro de un mismo percepto final.

## Condiciones de aplicación

- Usa `CONTAM` cuando una misma respuesta une de forma contaminada dos percepciones que normalmente deberían mantenerse separadas.
- La clave es que el producto final se presenta como una sola percepción fusionada, no como dos ideas simplemente comparadas.
- La respuesta debe resultar imposible o internamente contradictoria por la propia fusión.
- `CONTAM` representa una forma especialmente grave de distorsión y luego pesa en `Sum6` y `WSum6`.

## Precauciones y distinciones

- Si el problema principal es una combinación incongruente dentro de la respuesta, pero no una fusión contaminada, revisa `INCOM2`.
- Si la extrañeza principal es una relación fabulizada entre elementos, revisa `FABCOM2`.
- `CONTAM` no se usa solo porque la respuesta sea muy imaginativa o simbólica.
- Debe existir una contaminación perceptiva real, no solo una descripción metafórica llamativa.
