# Nombre del documento: [Codificacion/Puntuaciones especiales] CONTAM

## Alias y busqueda

- CONTAM
- contamination
- contaminacion

## Definicion central

`CONTAM` se usa cuando dos perceptos incompatibles quedan fusionados como una sola respuesta contaminada.
No describe solo una mezcla rara, sino una integracion imposible dentro de un mismo percepto final.

## Condiciones de aplicacion

- Usa `CONTAM` cuando una misma respuesta une de forma contaminada dos percepciones que normalmente deberian mantenerse separadas.
- La clave es que el producto final se presenta como una sola percepcion fusionada, no como dos ideas simplemente comparadas.
- El response debe sonar imposible o internamente contradictorio por la fusion misma.
- `CONTAM` representa una forma especialmente grave de distorsion y luego pesa en `Sum6` y `WSum6`.

## Precauciones y distinciones

- Si el problema principal es una combinacion incongruente dentro de la respuesta, pero no una fusion contaminada, revisa `INCOM2`.
- Si la extraneza principal es una relacion fabulizada entre elementos, revisa `FABCOM2`.
- `CONTAM` no se usa solo porque la respuesta sea muy imaginativa o simbolica.
- Debe existir una contaminacion perceptiva real, no solo una descripcion metaforica llamativa.
