# [해석/Upper Section] D

## 별칭/검색어

- D
- common detail
- 보통 세부

## 핵심 정의

`D`는 반점의 보통 세부를 사용한 반응의 빈도다. 해석에서는 일상적이고 실용적인 정보 단위에 주의를 두는 정도를 가늠하는 기본 지표로 본다.

## 해석 포인트

- `D`는 현실적이고 보편적인 단위에 초점을 두는 경향과 연결되기 쉽다.
- 적절한 `D` 비중은 전체와 세부의 균형 잡힌 접근과 잘 맞을 수 있다.
- `D`가 두드러지면 구체적이고 통상적인 부분을 먼저 다루는 경향을 생각한다.

## 주의사항/감별 기준

- `D`가 높아도 이것만으로 적응적이라 할 수 없다. `Dd`, `P`, `Xu%`가 함께 중요하다.
- `D`가 낮고 `Dd`가 높으면 특이하고 비전형적인 단서에 더 끌릴 가능성을 본다.

## 상호 참조

- `D`는 `W:D:Dd` 전체 분포 안에서만 안전하게 읽힌다.
- `D`가 많다고 해서 유능함, 보수성, 현실감각을 단정하지 않는다.

## 근거 메모

- `W`
- `Dd`
- `P`
- `Xu%`
- `W_D_Dd`
