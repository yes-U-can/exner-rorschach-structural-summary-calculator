# [해석/Upper Section] S

## 별칭/검색어

- S
- space responses
- 공백 반응

## 핵심 정의

`S`는 공백을 주요 위치로 사용한 반응의 빈도다. 상부 영역에서는 주의 초점이 얼마나 역방향적이거나 비정형적으로 배치되는지 살펴보는 단서로 읽는다.

## 해석 포인트

- `S`는 통상적 형태뿐 아니라 공백 자체를 적극적으로 쓰는 방식과 연결된다.
- 어느 정도의 `S`는 독립적 초점, 비판적 주의, 역방향 탐색 가능성을 시사할 수 있다.
- `S`가 많으면 대립적, 비순응적, 혹은 비전형적 접근이 강해지는지 추가 확인한다.

## 주의사항/감별 기준

- 높은 `S`를 곧바로 반항성으로 읽지 않는다. 창의적 탐색인지, 대립인지, 불만인지 구분해야 한다.
- `S`가 높고 `S-%`, `X-%`, `AG`, `PER` 등도 맞물리면 해석 무게가 달라질 수 있다.

## 상호 참조

- `S`는 맥락 의존성이 크다.
- 위치로서의 `S`와 정서/매개 해석 변수는 분리해서 보고, 나중에 다시 연결한다.
