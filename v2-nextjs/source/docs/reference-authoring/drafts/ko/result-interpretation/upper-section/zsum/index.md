# 문서명: [해석/Upper Section] ZSum

## 별칭/검색어

- ZSum
- weighted Z sum
- 가중 Z 총합

## 핵심 정의

`ZSum`은 각 `Z` 반응에 부여된 가중치를 모두 더한 값이다. 조직화 시도의 단순 빈도보다, 전체적으로 얼마나 많은 조직화 에너지가 투입되었는지를 보여준다.

## 해석 포인트

- `ZSum`은 조직화 시도의 총량을 본다.
- 값이 높으면 자극장 관계를 더 많이 엮고 계산하려는 경향을 생각할 수 있다.
- 값이 낮으면 조직화 시도가 단순하거나 적었을 수 있다.
- 핵심은 `ZSum` 자체보다 `ZEst`와의 차이에서 나온 `Zd`다.

## 높거나 낮을 때 주의

- 높더라도 `Zf`가 함께 높아 생긴 자연스러운 결과인지, 또는 `Zd`가 과도한 양수로 가는지 확인해야 한다.
- 낮더라도 무조건 처리장애로 보지 않는다. 기록 길이, 반응 단순화, 과제 몰입도도 영향을 준다.

## 단독 해석 한계

- `ZSum`은 단독 의미가 약하다.
- `ZSum`만 떼어 읽기보다 `Zf -> ZEst -> Zd`의 흐름으로 읽는 것이 안전하다.
