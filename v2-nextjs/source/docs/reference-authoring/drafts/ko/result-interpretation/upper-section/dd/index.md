# [해석/Upper Section] Dd

## 별칭/검색어

- Dd
- unusual detail
- 드문 세부

## 핵심 정의

`Dd`는 반점의 드물거나 비전형적인 세부를 사용한 반응의 빈도다. 해석에서는 피검자가 덜 흔한 단서에 얼마나 자주 주의를 두는지 보는 지표다.

## 해석 포인트

- `Dd`가 어느 정도 있으면 흔치 않은 요소를 포착하는 주의 폭을 시사할 수 있다.
- `Dd`가 많으면 독특한 초점, 비전형적 관심, 지나친 주변 탐색 가능성을 생각한다.
- `Dd`가 매우 적으면 특이 요소 탐색이 적고 통상적 단위에 머무를 수 있다.

## 주의사항/감별 기준

- 높은 `Dd`는 창의성, 집요함, 비효율, 비전형성 중 무엇인지 추가 확인이 필요하다.
- `Dd`가 높고 `P`가 낮으며 `Xu%`가 높으면 비전형적 지각 스타일 쪽으로 더 기울 수 있다.

## 상호 참조

- `Dd` 하나만으로 창의성이나 병리성을 정하지 않는다.
- `W:D:Dd`, `P`, `Xu%`, `Zd`를 함께 본다.
