# 문서명: [해석/특수지표] HVI

## 별칭/검색어

- HVI
- Hypervigilance Index
- 과잉경계지수
- 과민지수

## 핵심 정의

`HVI`는 과도한 경계와 대인관계의 거리두기에 관련된 것으로 제안된 변수의 조작적 조합이다.
경험적 지지는 약하고 일관되지 않으므로, 양성만으로 경계심, 불신, 특정 성격 특성을 추론하지 않는다.

## 해석 포인트

- 양성은 우선 조작적 변수 조합이 충족됐다는 사실만 뜻한다.
- 면접이나 행동 관찰에서 경계 행동이 독립적으로 확인될 때에만 낮은 확신의 가설로 검토한다.
- `SumT_inter`, `Zf`, `Zd`, `S`, `H_ratio`, `HumanCont`는 판정의 구성 요소로 확인한다. 이 값들이 모였다는 사실만으로 임상적 의미가 확정되지는 않는다.
- 실제 위험, 외상 경험, 최근 피해, 문화적으로 적절한 대인 거리 같은 상황 요인을 우선해서 평가한다.

## 높거나 낮을 때 주의

- 양성도 과잉경계나 편집성을 확정하지 못하며, 음성도 임상적인 경계심을 배제하지 못한다.
- `T = 0`을 포함한 판정 조건은 성격이나 진단의 직접적인 증거가 아니다.
- 문화, 직업, 상황에 맞는 신중함을 병리화하지 않도록 주의한다.

## 단독 해석 한계

- 지표만으로 경계의 유무, 원인, 지속성을 결정할 수 없다.
- 구성 변수로 돌아가 면접, 행동 관찰, 생활환경, 병력과 대조한다.

## 함께 볼 변수

- `SumT_inter`
- `Zf_proc`
- `Zd_proc`
- `S_aff`
- `H_ratio`
