# 문서명: [해석/특수지표] OBS

## 별칭/검색어

- OBS
- Obsessive Style Index
- 강박스타일지수

## 핵심 정의

`OBS`는 정밀성, 세부 집중, 조직화와 관련된 것으로 제안된 변수의 조작적 조합이다.
강박적 성격 특성, 완벽주의, 경직성, 기능 손상을 추론할 구성개념 타당도 근거는 불충분하다.

## 해석 포인트

- 양성은 조작적 변수 조합이 충족됐다는 사실만 뜻한다.
- 양성에서 특정한 강박적 양식이나 적응상의 장점·단점을 추론하지 않는다.
- `Zf`, `Zd`, `Dd`, `P`, `FQ+`는 판정을 구성하는 변수로 확인한다. 이 값들이 모인 사실을 별도의 타당화 근거로 다시 세지 않는다.
- 완벽주의, 과도한 통제, 경직성, 주관적 고통, 기능 손상은 면접과 행동 관찰에서 독립적으로 확인한다.

## 높거나 낮을 때 주의

- `OBS`는 조작적인 양성·음성만 나타내며, 강박장애나 강박성 성격장애의 진단이 아니다.
- 음성도 완벽주의나 강박 증상을 배제하지 못한다.
- 문화, 직업 훈련, 검사에 임하는 태도의 영향을 검토한다.

## 단독 해석 한계

- 현재 근거로는 지표만 보고 제안된 임상적 의미를 진술할 수 없다.
- 구성 변수를 확인하되, 관련 특성의 유무는 독립된 정보원에서 판단한다.
