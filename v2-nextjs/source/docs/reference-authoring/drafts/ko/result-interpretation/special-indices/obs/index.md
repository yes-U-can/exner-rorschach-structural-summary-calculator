# 문서명: [해석/Special Indices] Obsessive Style Index

## 별칭/검색어

- OBS
- Obsessive Style Index
- 강박스타일지수

## 핵심 정의

`OBS`는 완벽주의적 통제, 과도한 세부집중, 규칙 의존, 지나친 조직화 경향을 yes/no 형태로 요약하는 constellation이다. 양성이면 정보처리와 판단에서 precision과 control을 과하게 추구하는 style을 의심한다.

## 해석 포인트

- 양성인 경우 detail orientation, overincorporation, rule-bound checking, perfectionistic control이 강할 수 있다.
- 이 style은 적응적으로는 신중함과 정확성을 높일 수 있지만, 지나치면 융통성을 떨어뜨리고 부담을 키운다.
- `Dd`, `Zf`, `Zd`, `Popular`, `FQ+` 같은 정교함 지표들이 한쪽으로 모일 때 의미가 분명해진다.

## 높거나 낮을 때 주의

- `OBS`는 yes/no 지수다. 양성이면 스타일적 시사점이 크지만, 음성이라고 강박적 특징이 없다고 단정하지 않는다.
- 강박장애 진단을 대신하지 않는다.
- 높은 통제가 문화적으로 장려되는 환경, 검사 태도, 직업 훈련의 영향도 고려해야 한다.

## 단독 해석 한계

- `OBS`는 style의 방향을 알려 줄 뿐, 그 사람이 실제로 불안 때문에 통제하는지 성격상 정밀한지까지는 구분하지 않는다.
- `Zf`, `Zd`, `Dd`, `P`, `FQ+`를 함께 봐야 한다.

## 함께 볼 변수

- `Zf_proc`
- `Zd_proc`
- `Dd`
- `P`
- `FQ+`
