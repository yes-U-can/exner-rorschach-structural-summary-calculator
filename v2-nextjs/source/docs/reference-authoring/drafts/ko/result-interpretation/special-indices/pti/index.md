# 문서명: [해석/Special Indices] PTI

## 별칭/검색어

- PTI
- Perceptual-Thinking Index
- 지각사고지수

## 핵심 정의

`PTI`는 지각 왜곡과 사고 특이성이 함께 모일 때 thought disorder 또는 psychotic-like slippage 가능성을 점검하기 위한 composite index다. lower section의 mediation과 ideation 변수를 함께 끌어와 읽는 대표적인 특수지수다.

## 해석 포인트

- `PTI`가 양성이면 현실검증, 사고의 논리성, 개념 연결 방식 가운데 하나 이상이 흔들릴 가능성을 더 적극적으로 검토한다.
- mediation 쪽 부정확성과 ideation 쪽 특수점수가 동시에 높을수록 PTI의 의미가 더 강해진다.
- 이는 정신병적 장애 가능성을 생각하게 할 수 있지만, 급성 스트레스, 중독, 발달적 취약성, 심한 정서 과부하도 비슷한 패턴을 만들 수 있다.

## 높거나 낮을 때 주의

- 양성이라고 곧바로 schizophrenia를 확정하지 않는다.
- 음성이라고 해서 사고장애 가능성이 완전히 배제되는 것도 아니다.
- 반응수가 적거나 자료가 빈약하면 false negative가 생길 수 있다.

## 단독 해석 한계

- `PTI`는 경고지수이지 진단지수가 아니다.
- 반드시 `WSum6`, `Lv2`, `M-`, `X-%`, `XA%` 같은 하위 변수로 다시 내려가 확인해야 한다.

## 함께 볼 변수

- `WSum6`
- `Lv2`
- `M_minus`
- `X_minus_percent`
- `XA_percent`
