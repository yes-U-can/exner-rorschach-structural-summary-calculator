# 문서명: [해석/특수지표] PTI

## 별칭/검색어

- PTI
- Perceptual-Thinking Index
- 지각사고지수

## 핵심 정의

`PTI`는 지각의 부정확성과 사고 과정의 이상에 관련된 여러 변수를 조합한 선별 지표다.
현실검증의 어려움, 관념의 비논리성, 지각 정확도의 저하를 우선해서 확인할 필요가 있는지 보여 준다.

## 해석 포인트

- 양성이면 지각의 정확성, 사고의 논리성, 개념 연결 방식을 더 자세히 확인한다.
- `WSum6`, `Lv2`, `M-`, `X-%`, `XA%` 가운데 어떤 값과 반응이 판정에 기여했는지 살펴본다.
- 여러 구성 변수가 같은 방향을 보일수록 우려는 커질 수 있지만, 지표만으로 사고장애나 정신병적 장애를 진단하지 않는다.
- 급성 스트레스, 물질의 영향, 발달 요인, 심한 정서적 과부하, 짧거나 질이 낮은 반응 기록 등 다른 설명도 검토한다.

## 높거나 낮을 때 주의

- 양성만으로 조현병이나 다른 정신병적 장애를 확정하지 않는다.
- 음성도 임상적으로 중요한 사고와 지각의 문제를 배제하지 못한다.
- 양성·음성 표지보다 구성 변수와 실제 반응 기록의 질을 우선한다.

## 단독 해석 한계

- `PTI`는 보조적인 경고 신호이며 면접, 행동 관찰, 병력과 통합해야 한다.
- `WSum6`, `Lv2`, `M-`, `X-%`, `XA%`를 확인한 뒤 결론의 강도를 정한다.

## 함께 볼 변수

- `WSum6`
- `Lv2`
- `M_minus`
- `X_minus_percent`
- `XA_percent`
