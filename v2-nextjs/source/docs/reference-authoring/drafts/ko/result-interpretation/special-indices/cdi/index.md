# 문서명: [해석/특수지표] CDI

## 별칭/검색어

- CDI
- Coping Deficit Index
- 대처결함지수

## 핵심 정의

`CDI`는 대처 자원과 대인 기능에 관련된 것으로 제안된 여러 변수를 조합한 조작적 선별 지표다.
경험적 지지는 제한적이며, 양성은 현재의 대처와 대인 기능을 자세히 확인하라는 신호로만 사용한다.

## 해석 포인트

- 양성이면 현재의 부담, 실제 대처 행동, 도움의 활용, 대인관계의 어려움, 주변의 지지 자원을 확인한다.
- `Food`, `SumT_inter`, `ISO_Index`, `PureH`, `EA` 가운데 어떤 구성 변수가 판정에 기여했는지 살펴본다.
- 면접과 행동 관찰에서 확인되는 어려움을 우선하며, 지표만으로 무력감, 미숙함, 의존성, 대인능력 부족을 추정하지 않는다.
- `DEPI`도 양성이면 우울 증상과 대처·대인 기능의 문제를 각각 직접 평가한다.

## 높거나 낮을 때 주의

- `CDI`는 우울지수도 진단지표도 아니다.
- 고정된 성격 특성이나 발달적 미숙함의 증거로 사용하지 않는다.
- 연령, 발달 단계, 문화, 현재 요구 수준, 사회적 지지에 따른 다른 설명을 검토한다.

## 단독 해석 한계

- 양성만으로 대처 자원의 부족이나 그 원인을 확정할 수 없다.
- 구성 변수로 돌아가 면접, 행동 관찰, 현재 생활 상황과 대조한다.

## 함께 볼 변수

- `Food`
- `SumT_inter`
- `ISO_Index`
- `PureH`
- `EA`
