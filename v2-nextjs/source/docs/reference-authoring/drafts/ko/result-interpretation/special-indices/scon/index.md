# 문서명: [해석/Special Indices] Suicide Constellation

## 별칭/검색어

- S-CON
- Suicide Constellation
- 자살가능성 지수

## 핵심 정의

`S-CON`은 여러 cluster에서 자살 관련 위험 신호가 함께 모이는지를 actuarial하게 점검하는 constellation이다. 위기평가를 대신하지는 않지만, 구조요약만으로도 먼저 확인해야 할 경고 신호를 빠르게 보여 준다.

## 해석 포인트

- 일반적으로 `S-CON` 값이 8 이상이면, 자살한 사람들에게서 보였던 특징들과 유사한 패턴이 나타난 것으로 본다.
- 이 지수는 한 cluster 안에 묶이지 않고 stress, self-regard, affect, control 관련 변수들을 함께 끌어온다.
- 양성이면 현재 suicidal ideation, 계획, 수단 접근성, 최근 상실, 보호요인을 즉시 추가 확인해야 한다.

## 높거나 낮을 때 주의

- 8 미만이라고 해서 자살위험이 없다고 단정하지 않는다.
- 이 지수는 대체로 15세 이상에서 먼저 참고하고, 14세 이하에는 demonstrated usefulness가 없다.
- 반응수가 적은 기록에서는 false negative가 더 쉬워질 수 있다.

## 단독 해석 한계

- `S-CON`은 actuarial warning일 뿐, 임상 면접과 안전평가를 대체하지 않는다.
- 양성일수록 오히려 지금-여기 위험평가를 더 직접적으로 해야 한다.
