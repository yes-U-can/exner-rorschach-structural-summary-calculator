# 문서명: [해석/Special Indices] Special Indices

## 별칭/검색어

- special indices
- 특수지수
- constellations

## 핵심 정의

`Special Indices`는 lower section 여러 변수를 한데 모아 만든 composite screen이다. 각각의 지수는 특정 임상 문제를 직접 진단하기보다, 여러 변수들이 한 방향으로 함께 모일 때 어떤 임상적 경고를 더 우선적으로 살펴봐야 하는지 알려 주는 actuarial shortcut에 가깝다.

## 해석 포인트

- `PTI`는 사고와 지각의 와해 가능성을, `DEPI`는 우울 관련 특징을, `CDI`는 대처와 관계 기능의 취약성을 더 집중해서 보게 한다.
- `S-CON`은 자살 위험과 관련된 actuarial warning이고, `HVI`는 과도한 경계와 경계적 스캐닝 양식을, `OBS`는 강박적 과통제와 완벽주의적 조직화를 시사한다.
- 이 지수들은 단일 변수보다 강한 경고 효과를 내지만, 그 대신 false positive와 false negative 가능성도 항상 남는다.

## 읽을 때 주의

- 특수지수는 "진단 버튼"이 아니다. 양성이라고 바로 진단을 확정하지 않는다.
- 반응수가 너무 적은 기록은 constellation의 정확도를 떨어뜨릴 수 있다.
- `S-CON`은 특히 연령과 현재 위기 맥락을 같이 봐야 한다.

## 단독 해석 한계

- 특수지수는 여러 cluster의 요약 신호이므로, 양성일수록 오히려 하위 변수로 다시 내려가서 확인해야 한다.
- 최종 해석은 `core`, `ideation`, `affect`, `interpersonal`, `self-perception`을 다시 읽는 방식으로 닫아야 한다.

## 관련 하위 변수

- `DEPI`
- `CDI`
- `SCON`
- `HVI`
- `OBS`
