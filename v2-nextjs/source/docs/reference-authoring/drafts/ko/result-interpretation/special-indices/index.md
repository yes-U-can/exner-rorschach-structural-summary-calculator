# 문서명: [해석/특수지표] 특수지표

## 별칭/검색어

- special indices
- 특수지수
- constellations

## 핵심 정의

`Special Indices`는 여러 구성 변수를 조합해 특정 임상 영역을 추가로 확인할 필요가 있는지 알려 주는 조작적 선별 지표다.
각 지표는 진단이나 성격 특성의 증거가 아니라, 면접과 행동 관찰에서 무엇을 더 확인할지 정리하는 보조 신호로 사용한다.

## 해석 포인트

- `PTI`는 지각과 사고 과정의 이상을 더 자세히 확인하게 하는 지표로, 특수지표 가운데 경험적 지지가 비교적 뚜렷하다.
- `S-CON`이 양성이거나 현재 위험이 의심되면 지수 해석보다 직접적인 자살위험 평가와 안전 확보를 우선한다.
- `DEPI`와 `CDI`의 경험적 지지는 제한적이므로, 양성은 각각 우울 증상과 대처·대인 기능을 추가 평가하라는 신호로만 사용한다.
- `HVI`의 지지는 약하고 일관되지 않으며, `OBS`는 제안된 강박적 특성을 추론할 근거가 불충분하다.

## 읽을 때 주의

- 특수지수는 자동 진단 도구가 아니다. 양성 기준을 충족했다는 사실과 임상적 의미가 확인됐다는 결론을 구분한다.
- 지수마다 근거의 강도가 다르므로 같은 무게로 해석하지 않는다.
- 양성에도 위양성이, 음성에도 위음성이 생길 수 있다.
- 반응 기록의 길이와 질, 연령, 발달 수준, 문화, 현재 상황을 함께 고려한다.

## 단독 해석 한계

- 특수지수는 여러 영역을 압축한 신호이므로, 양성일수록 구성 변수로 돌아가 어떤 값이 판정에 기여했는지 확인한다.
- 최종 결론은 중핵, 관념, 정서, 대인관계, 자기지각 영역과 면접·행동 자료를 통합해 내린다.

## 관련 하위 변수

- `PTI`
- `DEPI`
- `CDI`
- `SCON`
- `HVI`
- `OBS`
