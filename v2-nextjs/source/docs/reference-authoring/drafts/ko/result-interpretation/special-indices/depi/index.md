# 문서명: [해석/Special Indices] DEPI

## 별칭/검색어

- DEPI
- Depression Index
- 우울지수

## 핵심 정의

`DEPI`는 우울 관련 특징이 구조요약 전체에 얼마나 넓게 퍼져 있는지 보는 composite index다. affect, stress burden, self-regard, helplessness와 관련된 변수들이 한 방향으로 겹칠 때 양성으로 올라온다.

## 해석 포인트

- 일반적으로 `DEPI`에서 5개 이상 조건이 양성이면 우울 관련 특징을 더 본격적으로 검토한다.
- 양성인 경우 슬픔, 무기력, 후회, 자기비난, 정서 위축, hopelessness 같은 주제가 함께 나타날 가능성을 생각한다.
- `CDI`까지 같이 양성이면 단순한 기분저하보다 coping failure와 helplessness가 더 크게 작동할 수 있다.

## 높거나 낮을 때 주의

- `DEPI`가 양성이라고 곧바로 major depression을 진단하지 않는다.
- `DEPI`가 낮아도 우울 문제가 없다고 결론내리지 않는다. false negative가 가능하다.
- 최근 상실, 반응성 우울, 급성 위기에서는 하위 변수를 더 세밀하게 읽어야 한다.

## 단독 해석 한계

- 이 지수는 depressive feature의 가능성을 요약해 줄 뿐, 우울의 원인과 지속성, 심각도까지 확정하지는 않는다.
- 반드시 `SumV`, `MOR`, `Afr`, `CDI`, 정서 cluster를 함께 확인해야 한다.
