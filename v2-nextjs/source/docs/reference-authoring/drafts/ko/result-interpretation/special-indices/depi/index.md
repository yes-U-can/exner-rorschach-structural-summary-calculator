# 문서명: [해석/특수지표] DEPI

## 별칭/검색어

- DEPI
- Depression Index
- 우울지수

## 핵심 정의

`DEPI`는 우울과 관련된 것으로 제안된 여러 변수를 조합한 조작적 선별 지표다.
경험적 지지는 제한적이며, 양성 결과는 우울 증상을 직접 확인할 필요가 있음을 알리는 신호로만 사용한다.

## 해석 포인트

- 양성이면 현재 기분, 흥미와 즐거움의 저하, 수면·식욕 변화, 활력, 죄책감, 자살사고를 면접에서 직접 확인한다.
- 증상의 시작 시점, 지속 기간, 심각도, 일상 기능에 미치는 영향도 지수와 별도로 평가한다.
- `SumV`, `SumY`, `MOR_self`, `Afr` 가운데 어떤 구성 변수가 판정에 기여했는지 확인한다.
- `CDI`도 양성이면 우울 증상과 대처·대인 기능을 각각 평가한다. 두 지표의 양성만으로 무력감이나 특정 병태를 추정하지 않는다.

## 높거나 낮을 때 주의

- 양성은 우울장애의 진단이 아니며, 특정 우울 증상이 존재한다는 증거도 아니다.
- 음성도 임상적으로 중요한 우울을 배제하지 못한다.
- 사별, 급성 스트레스, 신체질환, 약물이나 물질의 영향 등 다른 설명을 함께 검토한다.

## 단독 해석 한계

- 구성 변수로 돌아가 면접, 행동 관찰, 병력과 대조한 뒤 결론의 강도를 정한다.
- `SumV`, `SumY`, `MOR_self`, `Afr`, `CDI`와 정서 영역을 함께 확인한다.

## 함께 볼 변수

- `SumV`
- `SumY`
- `MOR_self`
- `Afr`
- `CDI`
