# 문서명: [해석/Core] SumCprime

## 별칭/검색어

- SumCprime
- SumC'
- 무채색 합

## 핵심 정의

`SumCprime`는 무채색 determinant(`FC'`, `C'F`, `C'`)의 총합이다. core 해석에서는 내면화된 불쾌 정서, 우울한 기분톤, 눌린 dysphoria가 얼마나 부담으로 작동하는지 보여 주는 right-side `eb` 요소로 본다.

## 해석 포인트

- `SumCprime`가 높으면 슬픔, 침울함, 회색빛 정서 경험이 더 많이 내부에 머물 가능성을 생각한다.
- `SumCprime`는 `WSumC`와 비교될 때 의미가 더 커진다. 내부에 머무는 정서가 표현 가능한 정서를 압도하는지 보기 때문이다.
- `SumCprime` 증가는 `es`와 `AdjD`를 악화시키는 dysphoric load의 일부가 될 수 있다.
- `C'` 계열은 정서 표현의 자유로움보다 내면화와 억제를 먼저 시사하는 쪽으로 읽는다.

## 높거나 낮을 때 주의

- 높은 `SumCprime`이 곧 임상적 우울장애를 뜻하지는 않는다.
- 낮거나 0이어도 depression을 배제하지 않는다. 특히 감정을 잘 억누르는 기록에서는 false negative가 가능하다.
- `SumCprime`는 `Col-Shd Blend`, `SumShd`, `Afr`, `WSumC`와 함께 봐야 affective tone이 안정된다.

## 단독 해석 한계

- `SumCprime` 하나만으로 mood disorder를 판단하지 않는다.
- internalized dysphoria의 무게는 `SumCprime:WSumC`, `DEPI`, `AdjD`와 함께 읽어야 한다.

## 함께 볼 변수

- `es`
- `AdjD`
- `SumC_WSumC`
- `Afr`
