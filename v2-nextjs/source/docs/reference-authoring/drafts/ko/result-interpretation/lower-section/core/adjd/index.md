# 문서명: [해석/Core] AdjD

## 별칭/검색어

- AdjD
- Adjusted D
- Adj D

## 핵심 정의

`AdjD`는 `EA`와 `Adjusted es`의 관계를 척도화한 차이 점수다. `m`과 `Y`처럼 상황에 따라 크게 흔들릴 수 있는 요소를 보정한 뒤에도 스트레스 과부하가 남는지 확인하는, 더 지속적이고 기저적인 통제·스트레스 내성 지표다.

## 해석 포인트

- `AdjD = 0`이면 비교적 지속적인 수준에서는 자원과 요구가 대체로 균형을 이루고 있을 가능성이 높다.
- `AdjD < 0`이면 상황적 요인을 조금 걷어내고도 과부하가 남는 패턴으로, 보다 만성적이고 오래된 distress를 의심한다.
- `D < AdjD`이면 현재 distress 중 일부가 `m`, `Y` 같은 상황성 자극 때문에 더 악화되어 있음을 시사할 수 있다.
- `AdjD`는 `D`보다 trait-like하게 읽지만, 그렇다고 완전히 고정된 성격 변수로만 보지는 않는다.

## 높거나 낮을 때 주의

- `AdjD = 0`도 무조건 건강 지표는 아니다. chronic but stabilized pattern이나 변화 저항성과 함께 나타날 수 있다.
- `AdjD < 0`는 persistent overload를 강하게 시사하지만, 그 구체적 성질은 `eb`, `m`, `Y`, `SumCprime` 등으로 풀어 봐야 한다.
- `D`와 `AdjD`의 차이가 클수록 상황성 스트레스 개입을 더 적극적으로 의심한다.

## 단독 해석 한계

- `AdjD` 하나만으로 치료 필요성이나 예후를 단정하지 않는다.
- `AdjD`는 `D`, `AdjEs`, `EA`, `es`와 짝지어 봐야 의미가 생긴다.
