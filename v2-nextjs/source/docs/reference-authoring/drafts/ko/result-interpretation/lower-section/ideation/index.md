# 문서명: [해석/Lower Section/Ideation] Ideation

## 별칭/검색어

- ideation
- 사고
- 관념화

## 핵심 정의

`Ideation`은 기록 속 생각의 방향, 인간운동 표상의 질, 그리고 사고 미끄러짐의 빈도와 심각도를 함께 읽는 하부 묶음이다. 해석에서는 `a:p`, `Ma:Mp`, `2AB+Art+Ay`, `MOR`, `Sum6`, `Lv2`, `WSum6`, `M-`, `Mnone`를 함께 보며 사고가 얼마나 현실에 잘 매여 있는지와 어떤 방향으로 기울어지는지 점검한다.

## 해석 포인트

- `a:p`, `Ma:Mp`는 상상과 문제 해결에서 능동성보다 수동성이 더 앞서는지 본다.
- `2AB+Art+Ay`는 사고가 직접 경험보다 추상화, 양식화, 개념화 쪽으로 얼마나 기울어지는지 본다.
- `MOR`는 생각과 표상에 손상, 파괴, 실패, 비관의 색조가 얼마나 자주 섞이는지 본다.
- `Sum6`, `Lv2`, `WSum6`는 사고 미끄러짐의 양과 심각도를 읽는다.
- `M-`, `Mnone`는 인간에 관한 상상과 사고가 현실적으로 조직되는지, 아니면 왜곡되거나 풀려 버리는지 보여 준다.

## 높거나 낮을 때 주의

- 이 묶음은 `M`이 적거나 기록이 너무 짧으면 정보량이 급격히 줄어든다.
- `Sum6`가 조금 높아도 `Lv2`가 없고 `Mediation`이 안정적이면 심각도를 과장하지 않는다.
- 반대로 값 하나만 높지 않아도 `M-`, `Mnone`, `Lv2`가 같이 보이면 해석 톤을 더 보수적으로 잡는다.

## 단독 해석 한계

- `Ideation`은 생각의 질을 말해 주지만, 그 생각이 지각 왜곡 때문인지, 정서 과부하 때문인지, 대인 맥락 때문인지는 단독으로 정하지 않는다.
- 따라서 `Mediation`, `Processing`, `PTI`, `SCON`, `Core`를 함께 봐야 한다.
