# 문서명: [해석/Mediation] XA%

## 별칭/검색어

- XA%
- Form Appropriate Extended
- 적절형태확장비율

## 핵심 정의

`XA%`는 전체 반응 가운데 형태 사용이 대체로 적절한 반응이 얼마나 되는지를 보는 비율이다. 종합체계 기준에서는 `FQ +`, `o`, `u`를 포함한 넓은 범위의 적절한 형태 사용 비율로 계산한다.

## 해석 포인트

- `XA%`가 높으면 기록 전반에서 형태 사용이 대체로 자극에 맞게 유지된다고 본다.
- `XA%`가 낮으면 반점 번역 과정 전반에 미끄러짐이나 부정확성이 넓게 섞였을 가능성을 생각한다.
- `WDA%`와 비교하면 어려움이 흔한 영역에서도 나타나는지, 아니면 더 복잡한 상황에서 두드러지는지 가늠할 수 있다.

## 높거나 낮을 때 주의

- `XA%`가 높아도 `Xu%`가 높으면 적절하지만 다소 특이한 번역이 섞일 수 있다.
- `XA%`가 낮다고 곧바로 심한 왜곡으로 보지 않는다. `X-%`가 높아야 실제 왜곡 근거가 더 강해진다.
- 반응 수가 적으면 비율이 쉽게 흔들린다.

## 단독 해석 한계

- `XA%`만으로는 관습성, 특이성, 왜곡을 서로 분리해 설명하기 어렵다.
- 반드시 `WDA%`, `X+%`, `Xu%`, `X-%`와 함께 읽어야 한다.

## 함께 볼 변수

- `WDA%`
- `X+%`
- `Xu%`
- `X-%`
- `P`
