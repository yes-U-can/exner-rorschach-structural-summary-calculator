# [해석/Core] EBPer

## 별칭/검색어

- EBPer
- EB Pervasive
- 편재적 EB

## 핵심 정의

`EBPer`는 `EB`에서 큰 값을 작은 값으로 나눈 비율이다. `EB` 스타일이 분명할 때만 계산하며, 선호하는 대처 채널이 단순 선호 수준을 넘어 얼마나 편재적이고 고정적으로 우세한지 보여 준다.

## 해석 포인트

- `EBPer < 2.5`이면 선호하는 스타일은 있어도 어느 정도 유연성이 남아 있다고 본다.
- `EBPer >= 2.5`이면 `introversive` 또는 `extratensive` 스타일이 지나치게 우세해, 다른 채널을 쓰는 유연성이 줄어든 상태를 의심한다.
- 편재적 `introversive`는 사고와 숙고가 감정 표현보다 훨씬 앞서는 패턴과 연결될 수 있다.
- 편재적 `extratensive`는 감정과 즉각적 반응이 숙고보다 훨씬 앞서는 패턴과 연결될 수 있다.

## 주의사항/감별 기준

- `EBPer`는 `EB` 스타일이 분명하지 않으면 해석하지 않는다.
- `2.5` 이상이라고 해도 자동으로 병리로 보지 않는다. 핵심은 유연성 감소와 채널 편중이다.
- `EA`가 낮거나 짧은 기록이면 비율이 실제보다 과장되어 보일 수 있다.

## 상호 참조

- `EBPer`만으로 적응 실패를 결론내리지 않는다.
- 편재성이 실제로 위험한지 보려면 `EA`, `es`, `D`, 그리고 어떤 채널이 빈약한지(`M` 또는 `WSumC`)를 함께 봐야 한다.
