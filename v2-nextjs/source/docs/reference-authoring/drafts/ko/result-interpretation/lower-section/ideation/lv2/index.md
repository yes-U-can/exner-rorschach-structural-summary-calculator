# 문서명: [해석/Ideation] Lv2

## 별칭/검색어

- Lv2
- Level 2 special scores
- 2수준 특수점수 수

## 핵심 정의

`Lv2`는 Level 2 특수점수가 몇 개 있는지를 세는 값이다. 해석에서는 사고 이상이 단순한 어색함 수준을 넘어, 더 기괴하고 심한 미끄러짐으로 나타나는 정도를 본다.

## 해석 포인트

- `Lv2`가 0이면 심한 수준의 특수점수는 없다는 뜻이다.
- 1개만 있어도 사고 질을 더 조심해서 봐야 한다.
- 여러 개가 있으면 비논리성과 기괴함의 심각도가 높아졌을 가능성이 크다.

## 높거나 낮을 때 주의

- `Lv2`는 빈도보다 심각도 쪽 정보가 강하다.
- Level 2 하나만으로 정신병적 상태를 단정하지는 않지만, `WSum6`, `M-`, `X-%`, `PTI`와 함께 나오면 경고 강도가 높아진다.
- `CONTAM`은 Level 2로 집계되는 항목과 구조가 다르지만, 임상적 경고는 비슷하거나 더 강할 수 있어 함께 확인한다.

## 단독 해석 한계

- `Lv2`는 심각도 지표 하나일 뿐이며, 실제 의미를 판단하려면 어떤 특수점수가 Level 2였는지 내용을 열어 봐야 한다.
