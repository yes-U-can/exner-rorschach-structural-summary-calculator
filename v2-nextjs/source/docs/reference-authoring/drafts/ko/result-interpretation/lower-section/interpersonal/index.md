# 문서명: [해석/Lower Section/Interpersonal] Interpersonal

## 별칭/검색어

- interpersonal
- 인간관계부
- 대인관계

## 핵심 정의

`Interpersonal`은 구조요약에서 사람이 다른 사람을 얼마나 찾고, 얼마나 가깝게 지내고 싶어 하며, 관계를 협동적으로 보는지 아니면 공격적이고 멀어진 것으로 보는지를 읽는 묶음이다. 이 묶음은 `COP`, `AG`, `a:p`, `Food`, `SumT`, 인간 내용, 고립지수 같은 변수를 함께 본다.

## 해석 포인트

- `COP`와 `AG`는 사람 사이의 상호작용을 협동적으로 예상하는지, 갈등적으로 예상하는지의 방향을 보여 준다.
- `SumT_inter`와 `Food`는 친밀감 욕구나 의존 욕구가 관계 장면에서 얼마나 활성화되어 있는지 읽을 때 중요하다.
- `HumanCont`와 `PureH`는 사람에 대한 관심의 양과, 사람 개념이 실제 경험에 더 가까운지 상상에 더 기대는지의 차이를 보여 준다.
- `PER`는 대인 장면을 볼 때 자기 기준과 자기 경험을 얼마나 강하게 끌어오는지 알려 준다.
- `ISO_Index`는 사회적 거리두기, 위축, 고립 경향을 읽을 때 보조 축으로 쓴다.

## 높거나 낮을 때 주의

- 이 묶음의 변수들은 서로 겹친다. 예를 들어 `SumT_inter`가 높아도 `COP`가 낮고 `ISO_Index`가 높으면, 관계 욕구가 충족되기보다 결핍과 거리감으로 나타날 수 있다.
- `HumanCont`가 높다고 항상 건강한 친사회성을 뜻하지는 않는다. 경계심이나 과도한 사람 주시가 반영될 수도 있다.
- `AG`가 높다고 바로 실제 공격 행동을 단정하지 않는다. 우선은 대인 장면을 갈등적으로 조직하는 경향으로 읽는다.

## 단독 해석 한계

- 인간관계부는 자기지각부와 겹치는 정보가 많아서 두 묶음을 같이 읽어야 한다.
- 해석은 반드시 `core`, `affect`, `ideation`의 부담과 자원 상태 위에서 읽어야 한다.

## 함께 볼 변수

- `SumT`
- `a:p`
- `H ratio`
- `CDI`
- `DEPI`
