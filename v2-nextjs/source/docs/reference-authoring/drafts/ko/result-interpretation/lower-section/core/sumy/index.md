# 문서명: [해석/Core] SumY

## 별칭/검색어

- SumY
- diffuse shading total
- 확산 음영 합

## 핵심 정의

`SumY`는 diffuse shading determinant의 총합이다. core 해석에서는 막연한 긴장, 압도감, 마비감, 통제하기 어려운 상황적 distress가 얼마나 현재 심리적 부담으로 작동하는지 보여 주는 right-side `eb` 요소로 본다.

## 해석 포인트

- `SumY`가 높으면 막연한 불안, 압박감, helplessness-like distress가 커졌을 가능성을 생각한다.
- `SumY`는 `m`과 함께 상대적으로 상황 의존적이고 흔들리기 쉬운 burden으로 읽는다.
- `SumY`가 `es`를 밀어 올리면 `D`를 빠르게 떨어뜨릴 수 있다.
- `Y > 1` 수준은 stress-related paralysis나 hopelessness와 연결될 가능성이 커진다.

## 높거나 낮을 때 주의

- 높은 `SumY`가 곧 우울장애나 trauma를 확정하는 것은 아니다.
- 낮거나 0이어도 distress가 없다고 단정하지 않는다. 다른 shading이나 `m` 쪽으로 나타날 수 있다.
- `SumY`는 trait보다 state 성격을 더 의심해야 한다.

## 단독 해석 한계

- `SumY` 하나만으로 hopelessness를 단정하지 않는다.
- 현재 overload와 situational component를 보려면 `es`, `D`, `AdjEs`, `m`을 함께 봐야 한다.
