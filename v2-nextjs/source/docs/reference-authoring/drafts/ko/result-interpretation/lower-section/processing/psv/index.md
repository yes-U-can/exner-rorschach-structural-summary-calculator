# 문서명: [해석/Processing] PSV

## 별칭/검색어

- PSV
- Perseveration
- 보속

## 핵심 정의

`PSV`는 처리 단계에서 주의 이동이 잘 되지 않고 이전 틀을 반복하는 경향을 보여 주는 변수다. 해석에서는 스캔 효율, 주의 전환, 반응 세트의 경직성을 읽을 때 중요하다.

## 해석 포인트

- 상위 종합체계 기준에서는 `PSV`가 0이면 보통 처리 효율을 기본선으로 본다.
- `PSV`가 있으면, `Zd` 값과 무관하게 처리 효율 문제를 다시 점검한다.
- 흔한 경우인 `PSV = 1`은 검사 중 일부 시점에서 주의 전환이 잘 안 되는 장면이 있었음을 시사한다.
- `PSV`가 1보다 크면 더 뚜렷한 주의 이동의 어려움이나 경직성을 의심한다.

## 높거나 낮을 때 주의

- `PSV`는 채점상 존재 여부 자체가 중요하므로 단순 빈도 해석보다 맥락을 함께 본다.
- within-card PSV인지, 내용 보속인지에 따라 임상적 무게는 다를 수 있다.
- `PSV`가 있다고 곧바로 사고장애로 읽지 않는다.

## 단독 해석 한계

- `PSV`는 처리 효율 경고를 주지만 전체 인지 수준을 대신하지 않는다.
- 반드시 `Zd`, `DQ 분포`, `Mediation`, `Special Scores`와 함께 읽어야 한다.

## 함께 볼 변수

- `Zd`
- `DQ+`
- `DQv`
- `WSum6`
- `X-%`
