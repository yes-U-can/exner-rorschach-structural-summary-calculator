# 문서명: [해석/Processing] DQv

## 별칭/검색어

- DQv
- Vague Developmental Quality
- 미성숙 발달질

## 핵심 정의

`DQv`는 처리 단계에서 반응 조직화가 덜 성숙하고 덜 정교한 수준으로 머무는 횟수를 보여 준다. 해석에서는 processing quality가 어느 지점에서 단순하고 미숙한 쪽으로 흔들리는지 읽는다.

## 해석 포인트

- `DQv`가 많으면 처리의 질이 때때로 덜 성숙하고 덜 정교한 수준으로 떨어질 수 있다.
- 상위 종합체계 기준에서는 `DQv`와 `DQv/+`의 합이 기대보다 많으면 processing quality가 유의하게 falter한다고 본다.
- 성인 기록에서 이런 양상이 두드러지면 입력을 조직하는 방식이 자주 단순화되거나 거칠어질 가능성을 생각한다.

## 높거나 낮을 때 주의

- `DQv`는 `DQv/+`와 함께 보는 것이 더 안전하다.
- 낮다고 해서 자동으로 sophisticated processing을 뜻하지는 않는다. `DQ+`, `Zd`, `PSV`를 같이 봐야 한다.
- 아동 기록에서는 성인보다 빈도가 다르게 보일 수 있다.

## 단독 해석 한계

- `DQv`는 processing falter의 단서를 주지만, 왜 그것이 생기는지까지 설명하지 못한다.
- 반드시 `DQ+`, `DQv/+`, `PSV`, `Mediation`과 함께 읽어야 한다.

## 함께 볼 변수

- `DQv/+`
