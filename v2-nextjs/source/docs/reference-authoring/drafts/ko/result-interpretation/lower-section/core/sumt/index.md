# 문서명: [해석/Core] SumT

## 별칭/검색어

- SumT
- texture total
- 질감 합

## 핵심 정의

`SumT`는 texture determinant의 총합이다. core 해석에서는 친밀감, 접촉, 따뜻한 유대에 대한 욕구가 현재 부담과 결핍의 형태로 얼마나 활성화되어 있는지 보여 주는 right-side `eb` 요소로 본다.

## 해석 포인트

- `SumT`가 높으면 가까운 관계와 정서적 접촉에 대한 욕구가 현재 심리적 부담으로 올라와 있을 가능성을 생각한다.
- `SumT`는 단순한 사교성보다 "원하지만 충분히 충족되지 않는 closeness need"를 더 시사하는 편이다.
- `SumT`는 `es`를 올리는 요소라서, 자원 부족과 만나면 distress를 키울 수 있다.
- core 문맥에서는 stress load 쪽을 먼저 읽고, 관계 양상 자체는 later interpersonal 문서에서 더 구체화한다.

## 높거나 낮을 때 주의

- 높은 `SumT`가 곧 친밀한 관계 능력이 좋다는 뜻은 아니다.
- 낮거나 0이라고 해서 친밀 욕구가 없다고 단정하지 않는다. shading 전반을 잘 쓰지 않는 기록일 수도 있다.
- `T = 0` 해석과 `SumT` 해석은 구분해야 한다.

## 단독 해석 한계

- `SumT`는 core에서는 burden/need 측면만 먼저 읽는다.
- 실제 대인관계 스타일과 attachment 방식은 `SumT_inter`, `COP`, `HumanCont`와 함께 봐야 한다.
