# 문서명: [해석/Core] SumV

## 별칭/검색어

- SumV
- vista total
- 차원성 합

## 핵심 정의

`SumV`는 vista determinant의 총합이다. core 해석에서는 자기 자신이나 자신의 행위에 대해 비판적이고 후회 섞인 정서 경험이 어느 정도 부담으로 작동하는지 보여 주는 right-side `eb` 요소로 본다.

## 해석 포인트

- `SumV`가 높으면 self-critical affect와 후회, 불만, 자기비난이 현재 부담으로 올라와 있을 가능성을 생각한다.
- `SumV`는 core에서는 distress source로, self-perception에서는 self-regard issue로 다시 읽는다.
- `SumV`가 `es`를 올리고 `AdjD`를 낮추는 쪽으로 작동하면, self-criticism이 overload의 일부임을 시사할 수 있다.
- `V`는 존재 자체보다 얼마나 반복되는지가 중요하다.

## 높거나 낮을 때 주의

- `SumV > 0`이 늘 trait self-hatred를 뜻하는 것은 아니다. 최근 실패, 후회, 노출 사건 때문에 반응적으로 올라올 수도 있다.
- 낮거나 0이어도 자기비판이 없다고 단정하지 않는다.
- `SumV`는 `Fr+rF`, Egocentricity, `MOR` 같은 자기지각 변수와 함께 봐야 더 정확하다.

## 단독 해석 한계

- `SumV`는 core에서 distress burden만 말해 줄 뿐, self-structure 전체를 설명하지는 않는다.
- self-perception 쪽 의미는 `SumV_self` 문서와 함께 읽어야 한다.

## 함께 볼 변수

- `es`
- `AdjD`
- `SumV_self`
- `MOR`
