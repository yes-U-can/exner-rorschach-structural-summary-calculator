# [해석/Processing] W:D:Dd

## 별칭/검색어

- W:D:Dd
- Economy Index
- 경제지수

## 핵심 정의

`W:D:Dd`는 전체 위치(`W`), 흔한 세부(`D`), 드문 세부(`Dd`)를 어떤 비중으로 사용하는지 보여 주는 관계값이다. 처리 구역에서는 자극을 넓게 볼지, 흔한 부분에 머물지, 세세한 드문 부분까지 파고드는지를 읽는다.

## 해석 포인트

- `W`가 두드러지면 큰 틀이나 전체 구성을 먼저 다루려는 경향을 생각한다.
- `D`가 중심이면 비교적 실용적이고 흔한 부분 위주로 처리하는 편일 수 있다.
- `Dd`가 많으면 드문 세부에 관심이 많이 가거나 초점이 지나치게 미세해질 가능성을 본다.
- 이 값은 입력의 경제성을 보여 주므로, `W:M`과 함께 보면 목표 수준과 실제 처리 범위를 같이 읽기 좋다.

## 주의사항/감별 기준

- 특정 위치 하나가 많다고 곧바로 병리로 읽지 않는다.
- `R`이 많으면 세 값 모두 자연히 증가할 수 있으므로 절대량보다 비중을 본다.
- `Dd`가 많아도 반드시 세밀함만 뜻하는 것은 아니며, 우회·집착·비효율 가능성도 함께 본다.

## 상호 참조

- `W:D:Dd`만으로는 처리의 질, 효율, 성숙도를 다 설명하지 못한다.
- 반드시 `Zd`, `W:M`, `DQ 분포`, `Location`과 함께 읽어야 한다.
