# [해석/Core] AdjEs

## 별칭/검색어

- AdjEs
- Adjusted es
- Adj es

## 핵심 정의

`AdjEs`는 `es`에서 상황적으로 크게 흔들릴 수 있는 `m`과 `Y`의 초과분을 보정해 만든 adjusted demand 값이다. 현재 부담 중 무엇이 상대적으로 더 지속적인 요구인지, 무엇이 일시적 상황 스트레스인지 가늠하는 중간 지표다.

## 해석 포인트

- `AdjEs`가 높으면 상황성 요소를 일부 걷어내고도 부담이 꽤 남아 있다는 뜻일 수 있다.
- `es`는 높은데 `AdjEs`가 상대적으로 내려가면, 현재 overload의 일부가 일시적 상황 스트레스와 연결됐을 가능성을 생각한다.
- `AdjEs`는 `AdjD`를 계산하는 직접 재료이므로 `AdjD` 해석 전에 먼저 확인해야 한다.
- `AdjEs`는 `m`과 `Y`가 얼마나 `D`를 끌어내렸는지 이해하는 데 도움을 준다.

## 주의사항/감별 기준

- `AdjEs`가 낮다고 부담이 적다고 단정하지 않는다. 실제 현재 고통은 `es`와 `D` 쪽에서 더 크게 나타날 수 있다.
- `AdjEs`가 높으면 more chronic demand를 의심할 수 있지만, 어떤 질의 부담인지 보려면 `eb` 구성과 shading/movement 세부를 같이 봐야 한다.
- `AdjEs`는 독립 해석 변수라기보다 `D`와 `AdjD`를 연결하는 보정 변수다.

## 상호 참조

- `AdjEs` 하나만으로 현재 distress 수준을 판단하지 않는다.
- `es`, `D`, `AdjD`, `m`, `Y`를 함께 놓고 비교해야 의미가 생긴다.
