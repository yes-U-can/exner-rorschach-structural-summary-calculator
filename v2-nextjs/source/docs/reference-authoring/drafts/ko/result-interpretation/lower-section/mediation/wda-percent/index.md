# [해석/Mediation] WDA%

## 별칭/검색어

- WDA%
- Form Appropriate Common Areas
- 공통영역적절형태비율

## 핵심 정의

`WDA%`는 `W`와 `D` 영역 반응 가운데 형태 사용이 적절한 반응의 비율이다. 흔히 더 공유되기 쉬운 영역에서 지각 번역이 얼마나 안정적인지 보는 변수다.

## 해석 포인트

- `WDA%`가 높으면 비교적 흔하고 공유된 단서가 있는 자리에서는 지각이 안정적으로 유지된다고 본다.
- `WDA%`가 낮으면 공통 영역에서도 매개의 어려움이 나타난다는 뜻이므로 더 주의 깊게 본다.
- `XA%`보다 `WDA%`가 상대적으로 좋으면 어려움이 주로 비전형적이거나 더 복잡한 반응에서 커질 수 있다.

## 주의사항/감별 기준

- `WDA%`는 `W`와 `D`에만 한정되므로 전체 기록의 질을 대신하지 않는다.
- `W`나 `D` 반응 수가 적으면 값이 불안정할 수 있다.
- `WDA%`가 낮을수록 shared cue가 있어도 번역이 흔들릴 가능성을 의심하지만, 단독으로 병리 수준을 단정하지 않는다.

## 상호 참조

- `WDA%`는 공통 영역에서의 안정성을 보여 줄 뿐, 전체 기록의 특이성이나 왜곡 양상을 다 설명하지 못한다.
- 반드시 `XA%`, `X+%`, `X-%`, `FQ`와 함께 읽어야 한다.
