# 문서명: [해석/Lower Section] Lower Section

## 별칭/검색어

- lower section
- 하부
- 하부 영역
- 구조요약 하부

## 핵심 정의

`Lower Section`은 구조요약의 하부 묶음으로, 피검자의 자원, 스트레스 조절, 사고, 정서, 매개, 대인관계, 자기지각을 cluster 단위로 읽어 가는 중심 해석 영역이다. 상부가 초기 스캔이라면, 하부는 실제 의미를 붙여 나가는 본론에 가깝다.

## 해석 포인트

- 하부는 `core -> ideation -> affect -> mediation -> processing -> interpersonal -> selfPerception` 같은 cluster 흐름으로 읽는 것이 안전하다.
- 개별 수치보다, 서로 연결된 변수 묶음이 어떤 패턴을 이루는지가 더 중요하다.
- 상부에서 보인 경고 신호가 하부에서 실제로 유지되는지, 약화되는지, 다른 의미로 바뀌는지를 확인한다.

## 높거나 낮을 때 주의

- 하부 변수는 임상적 의미가 커 보여도, 단일 변수만으로 결론 내리면 오독 위험이 크다.
- 방어성, 짧은 기록, 문화권 차이, 최근 스트레스는 하부 변수 해석을 직접 흔든다.

## 단독 해석 한계

- 하부 각 cluster는 서로 분리된 장이 아니라 연결된 영역이다.
- 예를 들어 `Lambda`, `D`, `AdjD`, `SumY`, `MOR`, `PER`는 서로 다른 페이지에 있어도 실제 해석에서는 함께 움직일 수 있다.
