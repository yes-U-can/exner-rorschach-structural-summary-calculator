# 문서명: [해석/Interpersonal] Isolation Index

## 별칭/검색어

- ISO Index
- Isolation Index
- Isolate/R
- 고립지수

## 핵심 정의

`ISO_Index`는 `(Bt + 2Cl + Ge + Ls + 2Na) / R`로 계산하는 비율이다. 해석에서는 사회적 관계로부터 물러서거나, 상호작용 참여가 적거나, 정서적으로 고립된 위치를 취하는 경향을 가늠할 때 쓴다.

## 해석 포인트

- 대체로 `.25` 미만이면 이 변수만으로 고립 결론을 내리지 않는다.
- `.25`에서 `.30`대는 사회적 참여가 적은 편일 수는 있지만, 곧바로 부적응으로 보지 않는다.
- 대략 `1/3` 이상이면 실제 사회적 고립이나 두드러진 철수 가능성을 더 진지하게 검토한다.
- `COP`가 낮고 `ISO_Index`가 높으면 관계 장면을 멀고 적게 참여하는 방식으로 조직할 가능성이 커진다.

## 높거나 낮을 때 주의

- 이 변수는 성격 스타일, 정서상태, 환경적 철수 모두의 영향을 받을 수 있다.
- `R`이 적으면 비율 변화가 크게 보일 수 있다.
- `ISO_Index`가 높아도 반드시 관계 욕구가 없는 것은 아니다. `SumT_inter`가 함께 높으면 욕구는 있으나 참여는 줄어든 상태일 수 있다.

## 단독 해석 한계

- 고립의 질은 `COP`, `HumanCont`, `Food`, `CDI`를 함께 봐야 더 분명하다.

## 함께 볼 변수

- `COP`
- `HumanCont`
- `SumT_inter`
- `CDI`
- `DEPI`
