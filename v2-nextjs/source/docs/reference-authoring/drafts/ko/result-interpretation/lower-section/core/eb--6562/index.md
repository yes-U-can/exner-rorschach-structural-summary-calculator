# [해석/Core] eb

## 별칭/검색어

- eb
- Experience Base
- 경험기저

## 핵심 정의

`eb`는 `FM + m : SumShd` 관계로 표시되는 경험기저다. 피검자가 현재 겪는 자극 요구가 주로 침투적 생각과 압박된 욕구 쪽인지, 아니면 불쾌한 정서와 긴장 쪽인지 그 구성비를 보여 준다.

## 해석 포인트

- 왼쪽(`FM + m`)이 크면 충족되지 않은 욕구, 침투적 생각, 통제되지 않는 걱정이 더 두드러질 수 있다.
- 오른쪽(`SumShd`)이 크면 불쾌한 정서, 긴장, 예민함, 부담감이 더 두드러질 수 있다.
- `eb`는 "어떤 종류의 요구가 더 많은가"를 보여 주는 구성비라서, 전체 부담의 양인 `es`와는 역할이 다르다.
- `eb`의 구성은 이후 `D`, `AdjEs`, `AdjD`를 읽을 때 스트레스의 성격을 세분하는 단서가 된다.

## 주의사항/감별 기준

- `eb`는 합계가 아니라 비율이므로, 값이 작아 보여도 특정 성질이 뚜렷할 수 있다.
- `FM`, `m`, `C'`, `T`, `V`, `Y`가 각각 무엇으로 구성됐는지 풀어 봐야 한다.
- 반응 수가 적은 기록에서는 `eb` 비율이 쉽게 흔들릴 수 있다.

## 상호 참조

- `eb`는 요구의 성질을 말할 뿐, 전체 요구량을 직접 말하지 않는다.
- 실제 과부하 여부는 `es`, `EA`, `D`, `AdjD`를 함께 봐야 한다.
