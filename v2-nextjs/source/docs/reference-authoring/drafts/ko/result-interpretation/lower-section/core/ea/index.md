# [해석/Core] EA

## 별칭/검색어

- EA
- Experience Actual
- 경험실제

## 핵심 정의

`EA`는 `EB`의 두 값을 합한 자원 지표다. 즉 피검자가 사고 채널과 정서 채널을 이용해 일상 요구를 다룰 때 실제로 동원할 수 있는 기본 적응 자원이 얼마나 되는지를 보여 준다.

## 해석 포인트

- `EA`가 적절하면 문제 해결과 감정 처리에 쓸 수 있는 자원이 평균 범위 안에 있을 가능성이 높다.
- `EA`가 낮으면 생활 요구를 다룰 수 있는 여유가 부족해, 긴장과 좌절에 더 쉽게 흔들릴 수 있다.
- `EA`가 충분하면 일상 과제를 비교적 버텨낼 기반이 있다는 뜻이지만, 그것만으로 자동 적응을 의미하지는 않는다.
- `EA`는 `D`, `AdjD`의 해석 발판이 되므로 스트레스 과부하를 읽기 전 먼저 확인해야 한다.

## 주의사항/감별 기준

- `EA`가 충분해 보여도 `M < 2` 또는 `WSumC < 2.5`처럼 한 채널이 빈약하면 자원 구성이 치우쳐 있을 수 있다.
- `EA < 6`은 보통 자원 부족을 시사하지만, 짧고 `Lambda`가 높은 기록에서는 의도적 축소나 방어적 절제가 반영됐을 가능성을 같이 본다.
- `EA`가 높아도 `EBPer`가 강하거나 `es`가 매우 높으면 실제 적응은 불안정할 수 있다.

## 상호 참조

- `EA`는 자원량을 말해 주지만, 현재 요구가 그 자원을 넘어서는지는 `es`, `D`, `AdjD`를 함께 봐야 한다.
- `EA` 하나만으로 coping deficit, 만성 불안정, 대인관계 실패를 결정하지 않는다.
