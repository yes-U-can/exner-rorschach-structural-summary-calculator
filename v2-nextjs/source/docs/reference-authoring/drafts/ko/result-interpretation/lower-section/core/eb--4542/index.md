# [해석/Core] EB

## 별칭/검색어

- EB
- Erlebnistypus
- Experience Balance
- 경험균형

## 핵심 정의

`EB`는 `M : WSumC` 관계로 표시되는 경험균형이다. 피검자가 문제를 다루고 결정을 내릴 때 주로 사고와 숙고 쪽(`M`)을 더 쓰는지, 아니면 정서적 표현과 반응 쪽(`WSumC`)을 더 쓰는지 보여 주는 기본 대처 스타일 지표다.

## 해석 포인트

- `M`이 더 우세하면 보통 `introversive` 방향, `WSumC`가 더 우세하면 보통 `extratensive` 방향으로 읽는다.
- 한쪽이 분명히 우세하지만 지나치게 편재적이지 않으면, 선호하는 스타일이 있으면서도 어느 정도 유연성이 남아 있다고 본다.
- 뚜렷한 우세가 잡히지 않으면 `ambitent`로 읽으며, 이때는 융통성이라기보다 일관된 대처 방식의 부족을 의심한다.
- `EB`는 기록의 정서-사고 채널 배치를 먼저 보여 주므로, 이후 `affect`, `ideation`, `core` 읽기 순서를 잡는 출발점이 된다.

## 주의사항/감별 기준

- 한쪽 값이 크다고 곧바로 건강하거나 병리적이라고 단정하지 않는다.
- 차이가 매우 크면 단순한 스타일 선호가 아니라 `EBPer` 수준의 편재성을 따로 점검해야 한다.
- 짧은 기록, 높은 `Lambda`, 낮은 `EA`가 함께 있으면 `EB`가 실제 스타일보다 빈약하거나 왜곡되어 보일 수 있다.

## 상호 참조

- `EB`만으로 적응 수준이나 진단을 판단하지 않는다.
- `EB`는 "어느 채널을 더 선호하는가"를 말할 뿐, 그 채널을 실제로 충분히 쓸 자원이 있는지는 `EA`, `es`, `D`, `AdjD`를 함께 봐야 한다.
