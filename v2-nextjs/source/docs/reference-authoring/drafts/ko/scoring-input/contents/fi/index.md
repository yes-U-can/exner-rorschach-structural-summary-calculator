# 문서명: [부호화/내용] Fi

## 별칭/검색어

- Fi
- 불
- fire

## 핵심 정의

`Fi`는 반응의 내용이 불이나 연기일 때 쓰는 content code다. flame, smoke, burning image처럼 fire/smoke 자체가 중심일 때 `Fi`를 준다.

## 채점/적용 조건

- 불길, 화염, 연기처럼 fire or smoke image이면 `Fi`를 준다.
- 타오르는 장면이든 연기 기둥이든 불/연기 자체가 핵심이면 `Fi`에 해당한다.
- 폭발 장면이 핵심이면 `Ex`를 먼저 검토한다.
- 구름처럼 보인다면 `Cl`, 안개나 박무라면 `Na`와 구분한다.

## 주의사항/감별 기준

- `Fi`와 `Ex`는 둘 다 역동적 장면일 수 있지만, `Fi`는 불/연기이고 `Ex`는 폭발이다.
- `Fi`와 `Cl`은 둘 다 떠 있는 형상처럼 보일 수 있지만, `Cl`은 cloud content다.
- smoke를 막연한 흐림으로 읽지 말고, 실제로 연기로 지각되었는지 확인해야 한다.
