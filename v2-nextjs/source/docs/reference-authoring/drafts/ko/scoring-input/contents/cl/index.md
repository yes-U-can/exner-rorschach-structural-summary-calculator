# 문서명: [부호화/내용] Cl

## 별칭/검색어

- Cl
- 구름
- clouds

## 핵심 정의

`Cl`은 반응의 내용이 구름일 때 쓰는 content code다. cloud 자체를 보았을 때만 `Cl`을 주고, 안개나 박무 같은 변형은 다른 code로 구분한다.

## 채점/적용 조건

- 구름으로 지각된 반응이면 `Cl`을 준다.
- 하늘의 구름 덩어리, 구름층, 뭉친 cloud image가 전형적이다.
- cloud 자체가 핵심 대상이어야 한다.
- 안개, 박무, mist, fog처럼 구름의 변형 지각이면 `Na`를 먼저 검토한다.

## 주의사항/감별 기준

- `Cl`은 구름이고, 안개나 박무는 `Na`로 분리한다.
- `Cl`과 `Fi`는 둘 다 공기 중에 퍼진 형태처럼 보일 수 있지만, `Fi`는 불이나 연기다.
- 단순히 흐릿하거나 퍼져 보인다는 이유만으로 `Cl`을 붙이지 않는다.
