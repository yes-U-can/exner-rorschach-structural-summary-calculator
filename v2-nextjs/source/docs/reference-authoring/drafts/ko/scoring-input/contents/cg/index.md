# 문서명: [부호화/내용] Cg

## 별칭/검색어

- Cg
- 의류
- clothing

## 핵심 정의

`Cg`는 반응의 내용이 모자, 장화, 넥타이, 자켓처럼 의류나 의복 항목일 때 쓰는 content code다. 몸에 착용하는 물건으로 지각되었는지가 핵심이다.

## 채점/적용 조건

- 모자, 장화, 허리띠, 드레스, 넥타이, 자켓, 바지, 스카프처럼 의류 항목이면 `Cg`를 준다.
- 몸에 착용하는 물건으로 인식되는 것이 기준이다.
- 특정 역사적·문화적 상징이 강한 복식이면 기본 `Cg`에 더해 `Ay`를 검토할 수 있다.
- 단순 장식 예술품으로 본 경우는 `Art`와 구분한다.

## 주의사항/감별 기준

- `Cg`와 `Ay`는 함께 갈 수 있지만, `Cg`는 의류 자체이고 `Ay`는 문화·역사 의미다.
- `Cg`와 `Art`는 모두 시각적 물건이지만, `Art`는 예술 작품이나 장식 대상일 때 붙는다.
- 몸에 걸치는 물건이 아니라 일반 생활용품이면 다른 content route를 먼저 검토한다.
