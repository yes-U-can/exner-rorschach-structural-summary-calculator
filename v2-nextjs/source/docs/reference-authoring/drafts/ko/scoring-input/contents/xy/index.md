# 문서명: [부호화/내용] Xy

## 별칭/검색어

- Xy
- X-ray
- 엑스레이

## 핵심 정의

`Xy`는 반응의 내용이 X-ray일 때 쓰는 content code다. X-ray로 지각된 경우에는 골격이나 장기가 보여도 `An`을 2차로 더하지 않는다.

## 채점/적용 조건

- 엑스레이 사진, 투시 영상, X-ray image로 보았을 때 `Xy`를 준다.
- 내용 안에 뼈나 장기가 보여도 X-ray라는 매체 인식이 핵심이면 `Xy`를 사용한다.
- `Xy`를 채점할 때는 `An`을 2차 내용으로 추가하지 않는다.

## 주의사항/감별 기준

- `Xy`와 `An`은 모두 내부 구조처럼 보일 수 있지만, X-ray인지 해부학적 구조 자체인지가 다르다.
- X-ray로 본 순간 `An`을 같이 더하지 않는다는 점이 중요하다.
- 이후 해석에서는 `[해석/SelfPerception] An_Xy` route와 연결될 수 있다.
