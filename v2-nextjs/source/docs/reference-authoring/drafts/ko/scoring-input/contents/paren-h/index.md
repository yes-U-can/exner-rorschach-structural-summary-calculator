# 문서명: [부호화/내용] (H)

## 별칭/검색어

- (H)
- imaginary human
- 가공의 전체 인간

## 핵심 정의

(H)는 허구적이거나 신화적인 전체 인간 형상을 지각했을 때 주는 human content이다. 요정, 악마, 천사, 거인, 유령처럼 사람 비슷하지만 현실 인물로 보지 않는 경우가 여기에 들어간다.

## 채점/적용 조건

- 반응의 중심 대상이 전체 인간 형상이어야 한다.
- 그 인간 형상이 허구적, 신화적, 비현실적이어야 한다.
- 특정 전설 인물이나 신화 인물로 지각되면 필요시 `Ay`를 2차로 붙일 수 있다.
- 부분만 보였다면 `(H)`가 아니라 `(Hd)`를 재검토한다.

## 주의사항/감별 기준

- `(H)`는 `H`와 달리 현실적 인간이 아니다.
- `(H)`는 `(Hd)`와 달리 전체 인간 형상이다.
- 인간처럼 행동하는 공상과학 존재라 해도 전체 형상이 중심이면 `(H)`를 우선 검토한다.
- 허구적 인간 내용이 있다고 해서 자동으로 `PHR`가 되는 것은 아니다.
- 내용의 핵심이 정서나 감정 경험이면 `(H)`보다 `Hx`를 다시 본다.
