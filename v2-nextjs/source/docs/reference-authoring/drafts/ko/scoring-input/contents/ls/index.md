# 문서명: [부호화/내용] Ls

## 별칭/검색어

- Ls
- 풍경
- landscape

## 핵심 정의

`Ls`는 산, 섬, 동굴, 사막, 바닷속 장면처럼 실제 풍경이나 경관을 내용으로 볼 때 쓰는 content code다. 지도나 일반 자연 현상과는 구분해서 본다.

## 채점/적용 조건

- 산, 산맥, 언덕, 섬, 동굴, 바위, 사막, 늪, 바닷속 풍경 같은 landscape scene이면 `Ls`를 준다.
- 실제 경관이나 scenery가 핵심이어야 한다.
- 지도처럼 공간을 도식화한 표상이면 `Ge`를 먼저 검토한다.
- 물, 비, 안개, 하늘처럼 자연 환경이 핵심이면 `Na`와의 우선 규칙을 확인한다.

## 주의사항/감별 기준

- `Ls`는 실제 풍경이고 `Ge`는 지도다.
- `Na`가 함께 들어가면 `Na`가 `Ls`보다 우선한다.
- `Bt`와 `Ls`가 함께 보이고 `Na`가 없으면 둘 중 하나만 채점한다.
