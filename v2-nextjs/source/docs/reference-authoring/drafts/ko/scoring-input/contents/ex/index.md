# 문서명: [부호화/내용] Ex

## 별칭/검색어

- Ex
- 폭발
- explosion

## 핵심 정의

`Ex`는 반응의 내용이 폭발이나 폭발 장면일 때 쓰는 content code다. 터짐, blast, 폭발 사건 자체가 중심일 때 `Ex`를 준다.

## 채점/적용 조건

- 폭발, blast, 터지는 장면으로 본 경우 `Ex`를 준다.
- 불꽃놀이처럼 폭발성 장면도 `Ex`에 포함할 수 있다.
- 사건이나 장면의 핵심이 폭발이어야 한다.
- 단순한 불이나 연기 지각이면 `Fi`를 먼저 검토한다.

## 주의사항/감별 기준

- `Ex`와 `Fi`는 함께 떠오를 수 있지만, `Ex`는 폭발 사건이고 `Fi`는 불이나 연기다.
- 단순히 퍼져 보이는 장면이나 cloud-like image만으로 `Ex`를 붙이지 않는다.
- 폭발의 결과보다 폭발 자체를 본 경우에 `Ex`를 준다.
