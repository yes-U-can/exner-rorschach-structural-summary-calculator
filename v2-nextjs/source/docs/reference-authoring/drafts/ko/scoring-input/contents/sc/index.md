# 문서명: [부호화/내용] Sc

## 별칭/검색어

- Sc
- 과학
- science

## 핵심 정의

`Sc`는 과학과 관련되거나 과학의 직접적·간접적 산물로 지각된 내용을 부호화할 때 쓰는 content code다. 과학기기, 기계, 탈것, 무기, 우주선 같은 대상이 여기에 들어간다.

## 채점/적용 조건

- 망원경, 현미경, 로켓, 우주선, 자동차, 전구, TV, 레이더 기지, 배, 기차, 비행기, 무기 등은 `Sc`를 준다.
- 과학 또는 기술의 산물로 이해되는지가 핵심이다.
- 보기 드문 대상이라도 표준 content category에 맞으면 `Id`보다 `Sc`를 우선한다.
- 어떤 대상은 `Art`와 함께 갈 수도 있다.

## 주의사항/감별 기준

- 드물어 보여도 표준 범주에 맞으면 `Id`로 보내지 않는다.
- test tube 같은 반응은 흔치 않아도 `Sc`에 들어간다.
- carousel 같은 드문 내용도 `Sc` 또는 `Art + Sc`로 처리할 수 있으므로 바로 `Id`로 가지 않는다.
