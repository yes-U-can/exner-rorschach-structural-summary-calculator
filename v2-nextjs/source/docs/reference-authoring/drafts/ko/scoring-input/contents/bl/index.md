# 문서명: [부호화/내용] Bl

## 별칭/검색어

- Bl
- 혈액
- blood

## 핵심 정의

`Bl`은 반응의 내용이 인간이나 동물의 피, 즉 혈액 자체일 때 쓰는 content code다. 몸의 내부 구조 전체가 아니라 혈액이 지각의 핵심일 때 `Bl`을 준다.

## 채점/적용 조건

- 인간의 피나 동물의 피로 본 경우 `Bl`을 준다.
- 피가 흘러나오는 장면, 핏자국, 붉은 피 덩어리처럼 혈액이 중심 대상이면 `Bl`에 해당한다.
- 혈액의 출처가 인간이든 동물이든 `Bl`을 사용할 수 있다.
- 장기, 골격, 근육처럼 해부학적 구조 자체가 중심이면 `An`을 먼저 검토한다.

## 주의사항/감별 기준

- `Bl`과 `An`은 모두 신체 관련 내용처럼 보일 수 있지만, `Bl`은 혈액이고 `An`은 해부학적 구조다.
- `Bl`은 인간이나 동물의 피를 다 포함하므로 `H`나 `A`와 별개 축의 내용 코드다.
- 붉은 색을 보았다는 이유만으로 `Bl`을 붙이지 말고, 실제로 혈액으로 지각되었는지 확인해야 한다.
- `Bl`은 determinant의 color coding과 혼동하지 않는다.
