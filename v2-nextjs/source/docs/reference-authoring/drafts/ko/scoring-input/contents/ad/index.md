# 문서명: [부호화/내용] Ad

## 별칭/검색어

- Ad
- 부분 동물
- animal detail

## 핵심 정의

`Ad`는 반응의 내용이 동물의 불완전한 형태나 일부일 때 쓰는 content code다. 말발굽, 개 머리, 가재의 집게, 동물 가죽처럼 전체가 아니라 부분일 때 `Ad`를 준다.

## 채점/적용 조건

- 반응이 동물 전체가 아니라 일부나 부분 구조를 가리키면 `Ad`를 준다.
- 머리, 발굽, 발톱, 피부, 꼬리처럼 부분 지각이 전형적이다.
- 현실 동물의 부분이어야 한다.
- 허구적이거나 신화적인 동물 부분이면 `(Ad)`를 먼저 검토한다.
- 골격, 장기, 근육처럼 해부학적 내부 구조가 중심이면 `An`을 검토한다.

## 주의사항/감별 기준

- `Ad`와 `A`는 둘 다 동물이지만, 부분인지 전체인지가 다르다.
- `Ad`와 `(Ad)`는 둘 다 동물 부분이지만, 현실 동물 부분인지 허구적 동물 부분인지가 다르다.
- `Ad`와 `Hd`는 둘 다 부분 형태지만, 동물 부분인지 인간 부분인지가 다르다.
- 동물 껍질이나 피부는 `Ad`로 갈 수 있지만, 내부 장기나 골격이면 `An`을 먼저 본다.
