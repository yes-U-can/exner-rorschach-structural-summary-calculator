# 문서명: [부호화/내용] A

## 별칭/검색어

- A
- 전체동물
- whole animal

## 핵심 정의

`A`는 반응의 내용이 현실적 전체 동물 형태일 때 쓰는 content code다. 동물로 보았더라도 부분이 아니라 전체 형상이어야 하고, 허구적 동물도 아니어야 한다.

## 채점/적용 조건

- 반응이 전체 동물 형상을 포함하면 `A`를 준다.
- 실제 동물로 이해되는 전체 형태여야 한다.
- 말 전체, 새 전체, 곰 전체처럼 몸 전체가 하나의 지각 단위로 잡혀야 한다.
- 동물의 일부만 보이면 `Ad`를 먼저 검토한다.
- 허구적이거나 신화적인 동물 전체면 `(A)`를 준다.

## 주의사항/감별 기준

- `A`와 `(A)`는 둘 다 전체 동물이지만, 현실 동물인지 허구적 동물인지가 다르다.
- `A`와 `Ad`는 둘 다 동물이지만, 전체 형상인지 부분 형상인지가 다르다.
- 동물 가면처럼 동물의 일부나 허구적 부분으로 다루는 지각은 `(Ad)` 쪽을 먼저 본다.
- `A`는 내용 부호이므로 `Popular`, determinant, FQ와 섞어 생각하지 않는다.
