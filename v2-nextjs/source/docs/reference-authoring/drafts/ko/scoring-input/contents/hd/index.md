# 문서명: [부호화/내용] Hd

## 별칭/검색어

- Hd
- human detail
- 부분 인간

## 핵심 정의

Hd는 인간의 부분만 지각했을 때 주는 human detail content이다. 팔, 다리, 손가락, 발, 머리 없는 몸통처럼 불완전한 인간 형상이 중심일 때 사용한다.

## 채점/적용 조건

- 반응이 전체 인간이 아니라 인간의 일부를 가리켜야 한다.
- 그 부분이 현실적 인간의 일부로 이해되어야 한다.
- 부분 내용이 반응의 중심이면 `Hd`를 우선 준다.
- 허구적 인간 부분이면 `Hd`가 아니라 `(Hd)`를 재검토한다.

## 주의사항/감별 기준

- `Hd`는 `H`와 달리 전체 인간이 아니다.
- `Hd`는 `(Hd)`와 달리 허구적 인간 부분이 아니다.
- `Hd`는 해부 content `An`과 다르다. 하나는 사람의 외형 일부이고, 다른 하나는 해부학적 내부 구조다.
- 인간 일부가 나온다고 해서 자동으로 `PHR`가 되는 것은 아니다.
- 내용의 중심이 감정이나 감각 경험이면 `Hd`보다 `Hx`를 다시 본다.
