# 문서명: [부호화/내용] Na

## 별칭/검색어

- Na
- 자연
- nature

## 핵심 정의

`Na`는 `Bt`나 `Ls`로 따로 부호화되지 않는 자연 환경 전반을 내용으로 볼 때 쓰는 content code다. 해, 달, 하늘, 물, 눈, 비, 안개, 무지개 같은 자연 현상이 전형적이다.

## 채점/적용 조건

- 해, 달, 행성, 하늘, 물, 바다, 호수, 강, 얼음, 눈, 비, 안개, 박무, 무지개, 폭풍, 태풍, 밤, 빗방울처럼 자연 환경 내용이면 `Na`를 준다.
- 자연 현상이나 자연 환경 자체가 중심이어야 한다.
- `Na`가 `Bt`나 `Ls`와 함께 포함되면 `Na`만 채점한다.
- 반응에 `Na`가 없고 `Bt`와 `Ls`가 함께 있으면 둘 중 하나만 채점한다.

## 주의사항/감별 기준

- `Na`는 `Bt`와 `Ls`보다 우선한다.
- 구름 자체는 `Cl`, 안개나 박무는 `Na`로 구분한다.
- 자연 풍경 속 식물이나 바위가 보여도 `Na`가 중심이면 `Bt`, `Ls`를 함께 적지 않는다.
