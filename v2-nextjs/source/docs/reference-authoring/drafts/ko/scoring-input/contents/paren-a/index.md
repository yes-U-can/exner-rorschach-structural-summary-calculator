# 문서명: [부호화/내용] (A)

## 별칭/검색어

- (A)
- 전체동물(가공의, 신화적인)
- fictional whole animal

## 핵심 정의

`(A)`는 반응의 내용이 허구적이거나 신화적인 전체 동물 형태일 때 쓰는 content code다. 용, 일각수, 날개 달린 말처럼 현실 동물이 아닌 전체 동물 형상이 여기에 들어간다.

## 채점/적용 조건

- 반응이 전체 동물 형상을 포함해야 한다.
- 그 동물이 신화적, 전설적, 만화적, 환상적 동물로 이해되면 `(A)`를 준다.
- 일각수, 용, 마술 개구리, 천마 같은 예가 전형적이다.
- 허구적이어도 전체 형상이 아니라 일부만 보이면 `(Ad)`를 먼저 검토한다.

## 주의사항/감별 기준

- `(A)`와 `A`는 둘 다 전체 동물이지만, 현실성 여부가 다르다.
- `(A)`와 `(Ad)`는 둘 다 허구적 동물이지만, 전체 형상인지 부분 형상인지가 다르다.
- 동물 가면은 전체 동물이라기보다 허구적 동물 부분으로 보아 `(Ad)` 쪽을 먼저 확인한다.
- content code는 반응의 대상 분류이므로 determinant나 FQ와 섞지 않는다.
