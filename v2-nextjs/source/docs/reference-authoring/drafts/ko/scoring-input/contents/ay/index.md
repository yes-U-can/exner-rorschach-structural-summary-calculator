# 문서명: [부호화/내용] Ay

## 별칭/검색어

- Ay
- 인류(학)
- anthropology

## 핵심 정의

`Ay`는 반응의 내용이 특정 문화적 관련성이나 역사적 의미를 가진 대상으로 지각될 때 쓰는 content code다. 단순한 사람, 동물, 물건이 아니라 문화사적 맥락이 핵심일 때 붙인다.

## 채점/적용 조건

- 대상이 특정 역사, 문화, 민속, 전통, 문명과 연결된 의미를 가질 때 `Ay`를 준다.
- 토템, Roman helmet, Magna Carta, Napoleon의 모자, Cleopatra의 왕관, 선사 시대 도끼 같은 예가 전형적이다.
- 역사적 인물이나 특정 문화권 인물이 중심이면 기본 human content에 `Ay`를 2차 내용으로 덧붙일 수 있다.
- 문화적 상징이나 역사적 유물로 본 경우, 단순한 일반 물건으로 보았을 때보다 `Ay`가 우선된다.

## 주의사항/감별 기준

- `Ay`와 `Art`는 둘 다 문화적 느낌이 있을 수 있지만, `Ay`는 역사적·문화적 맥락 자체가 핵심이다.
- `Ay`는 단순히 오래된 물건처럼 보인다는 이유만으로 붙이지 않는다.
- 사람이나 동물을 보더라도, 특정 문화권·역사 인물·의례적 상징으로 지각되었을 때만 `Ay`를 추가한다.
- `Ay`는 보통 다른 content와 함께 붙는 경우가 많으므로 단독 코드처럼만 생각하지 않는다.
