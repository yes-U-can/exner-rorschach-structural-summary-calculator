# 문서명: [부호화/내용] Fd

## 별칭/검색어

- Fd
- 음식물
- food

## 핵심 정의

`Fd`는 반응의 내용이 먹을 수 있는 음식일 때 쓰는 content code다. 사람에게 흔한 음식뿐 아니라, 동물이 자기 종에 자연스러운 먹이를 먹는 장면도 `Fd`에 들어갈 수 있다.

## 채점/적용 조건

- 튀긴 닭, 아이스크림, 새우튀김, 채소, 솜사탕, 껌, 스테이크, 생선 요리처럼 먹을 수 있는 음식이면 `Fd`를 준다.
- 동물이 자기 종에 자연스러운 먹이를 먹는 장면도 `Fd`에 포함할 수 있다.
- 음식 자체가 반응의 중심 대상이어야 한다.
- 단순히 식물이나 동물을 본 것이면 `Bt`, `A`, `Ad`를 먼저 검토한다.

## 주의사항/감별 기준

- `Fd`는 content code이고 `FD`는 determinant이므로, 표기가 비슷해도 전혀 다른 항목이다.
- `Fd`와 `Bt`는 모두 먹을 수 있는 대상처럼 보일 수 있지만, 식물 자체를 보면 `Bt`, 음식으로 보면 `Fd`다.
- 동물을 보았더라도 초점이 “먹이”라면 `Fd`를 검토하고, 동물 자체가 중심이면 `A`나 `Ad`를 본다.
- 이후 해석에서는 `[해석/Interpersonal] Food` route와 연결되지만, coding 단계에서는 음식 내용 자체만 판단한다.
