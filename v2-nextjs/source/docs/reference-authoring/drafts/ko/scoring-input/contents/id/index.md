# 문서명: [부호화/내용] Id

## 별칭/검색어

- Id
- Idiographic contents
- 특이 내용

## 핵심 정의

`Id`는 표준적인 content category 어느 쪽에도 맞지 않는 특이한 내용을 기록할 때 쓰는 idiographic content code다. 다만 `Id`로 가기 전에 기존 표준 범주에 들어갈 수 없는지 반드시 먼저 확인해야 한다.

## 채점/적용 조건

- 반응 내용이 표준 content category에 들어가지 않을 때 `Id`를 검토한다.
- 먼저 모든 표준 범주를 확인한 뒤에도 맞지 않을 때만 `Id`를 쓴다.
- `Id`로 들어가는 내용은 written-out form으로 함께 기록한다.

## 주의사항/감별 기준

- 보기 드문 내용이라고 해서 곧바로 `Id`로 보내지 않는다.
- test tube는 드물어도 `Sc`에 들어간다.
- carousel처럼 드문 내용도 `Sc`나 `Art + Sc`로 처리하는 편이 `Id`보다 적절할 수 있다.
- `Id`는 예외 처리용이지, 익숙하지 않다는 이유로 남겨두는 임시 통로가 아니다.
