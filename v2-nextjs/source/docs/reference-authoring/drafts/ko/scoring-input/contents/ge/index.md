# 문서명: [부호화/내용] Ge

## 별칭/검색어

- Ge
- 지형
- geography
- map

## 핵심 정의

`Ge`는 반응의 내용이 지도(map)일 때 쓰는 content code다. 실제 산이나 바다 풍경을 본 것이 아니라, 지정되었든 아니든 “지도”로 지각했을 때 `Ge`를 준다.

## 채점/적용 조건

- 반응이 지도, 지도 그림, 특정 지역 지도처럼 지리적 표상으로 보이면 `Ge`를 준다.
- 특정 지도를 말하든 일반적인 지도라고 하든 기준은 같다.
- 실제 지형 장면이 아니라 “map representation”이 핵심이어야 한다.
- 산, 섬, 동굴, 사막 같은 실제 풍경 장면이면 `Ls`를 먼저 검토한다.

## 주의사항/감별 기준

- `Ge`는 지도이고 `Ls`는 실제 풍경이다.
- 한국어 가이드에는 `지형` 표기가 보이지만, 기준 문장은 지도 지각을 가리키므로 map 여부를 먼저 확인해야 한다.
- 실제 산맥이나 바다 풍경을 보았다고 해서 `Ge`를 주지 않는다.
