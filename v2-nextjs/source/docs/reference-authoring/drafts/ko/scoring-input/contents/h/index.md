# 문서명: [부호화/내용] H

## 별칭/검색어

- H
- human
- 전체 인간

## 핵심 정의

H는 전체 인간 형태를 현실적인 사람으로 지각했을 때 주는 human content이다. 실제 인물, 일반 사람, 역사적 인물처럼 완전한 인간 형상이 중심이어야 한다.

## 채점/적용 조건

- 반응의 중심 대상이 전체 인간 형상이어야 한다.
- 현실적 인간으로 이해되어야 한다.
- 역사적 인물이나 특정 인물로 지각되면 `Ay`를 2차로 추가할 수 있다.
- 사람의 일부만 보였다면 `H`가 아니라 `Hd`를 재검토한다.

## 주의사항/감별 기준

- `H`는 `(H)`와 달리 허구적, 신화적 인간이 아니다.
- `H`는 `Hd`와 달리 부분 인간이 아니라 전체 인간이다.
- `H`는 `Hx`와 달리 인간의 감정이나 경험이 아니라 인간 형상 자체가 내용이다.
- 인간 형상이 있다고 해서 자동으로 `GHR`나 `PHR`가 되는 것은 아니다.
- 그림 속 사람처럼 매체가 핵심이면 `Art`가 1차가 될 수도 있으므로 중심 구조를 다시 본다.
