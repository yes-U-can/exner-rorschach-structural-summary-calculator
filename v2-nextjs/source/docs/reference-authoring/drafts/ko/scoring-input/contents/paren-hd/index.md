# 문서명: [부호화/내용] (Hd)

## 별칭/검색어

- (Hd)
- imaginary human detail
- 가공의 부분 인간

## 핵심 정의

(Hd)는 허구적이거나 신화적인 인간의 부분만 지각했을 때 주는 human detail content이다. 악마의 머리, 천사의 눈, 마녀의 팔처럼 현실 인간 부분이 아닌 경우가 여기에 들어간다.

## 채점/적용 조건

- 반응이 전체 인간이 아니라 인간 부분만 가리켜야 한다.
- 그 부분이 허구적, 신화적, 비현실적 인간의 일부여야 한다.
- 부분 내용이 반응의 중심이면 `(Hd)`를 우선 준다.
- 전체 허구 인간이면 `(Hd)`가 아니라 `(H)`를 재검토한다.

## 주의사항/감별 기준

- `(Hd)`는 `Hd`와 달리 현실적 인간 부분이 아니다.
- `(Hd)`는 `(H)`와 달리 전체 인간이 아니라 부분 인간이다.
- 공상과학적 인간 부분이라도 비현실적 인간 일부면 `(Hd)`가 가능하다.
- `(Hd)` 역시 자동으로 `PHR`가 되는 것은 아니다.
- 감정, 공포, 사랑 같은 인간 경험을 지각한 경우는 `(Hd)`보다 `Hx`를 다시 본다.
