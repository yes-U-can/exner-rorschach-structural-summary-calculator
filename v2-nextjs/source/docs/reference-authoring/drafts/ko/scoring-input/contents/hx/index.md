# 문서명: [부호화/내용] Hx

## 별칭/검색어

- Hx
- human experience
- 인간 경험

## 핵심 정의

Hx는 인간 형상 그 자체가 아니라 인간의 정서, 감각, 경험을 내용으로 지각했을 때 주는 content이다. 사랑, 미움, 우울, 행복, 공포, 소리, 냄새처럼 인간 경험이 중심일 때 사용한다.

## 채점/적용 조건

- 반응의 핵심이 인간의 감정, 감각, 경험이어야 한다.
- 전체 인간이나 부분 인간 형상이 중심이 아니라 경험 내용이 중심이어야 한다.
- 피검자가 실제로 정서나 감각 경험을 내용으로 말했다고 확인해야 한다.
- 인간 형상 자체가 중심이면 `Hx`보다 `H`나 `Hd` 계열을 먼저 검토한다.

## 주의사항/감별 기준

- `Hx`는 `H`나 `Hd`와 달리 인간 형상이 아니라 인간 경험이 내용이다.
- 많은 `Hx` 반응은 `AB` 특수점수와 함께 나올 수 있지만, 두 coding은 같은 것이 아니다.
- 감정 단어가 들어가도 실제로 내용 중심이 사람 형상이면 `Hx`를 남발하지 않는다.
- `Hx`는 내용 code이며, 정서 해석 문장과 직접 같은 뜻은 아니다.
- 사람의 소리나 냄새처럼 감각 경험이 중심이면 `Hx`를 우선 검토한다.
