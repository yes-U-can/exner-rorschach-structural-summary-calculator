# 문서명: [부호화/내용] Sx

## 별칭/검색어

- Sx
- 성
- sex

## 핵심 정의

`Sx`는 성 기관이나 성 활동과 관련된 내용을 볼 때 쓰는 content code다. 보통은 2차 내용으로 붙고, 주 내용은 `H`, `Hd`, `An`인 경우가 많다.

## 채점/적용 조건

- 음경, 질, 고환, 유방, 월경, 유산, 성교처럼 성 기관이나 성 활동 내용이면 `Sx`를 준다.
- 성 관련 의미가 실제로 지각의 핵심이거나 분명히 언급되어야 한다.
- 보통은 2차 내용으로 붙으며, 주 내용은 `H`, `Hd`, `An`인 경우가 많다.

## 주의사항/감별 기준

- 사람이나 신체를 본 것만으로 `Sx`를 주지 않는다.
- `An`은 해부학적 구조, `Sx`는 성 기관/성 활동이라는 점을 구분해야 한다.
- 유방은 단순히 성별 확인용 인간 묘사일 때와 성적 내용으로 지각될 때를 구분한다.
