# [부호화/Z] ZS

## 별칭/검색어

- ZS
- space organization
- 공백 조직화

## 핵심 정의

`ZS`는 공백이 다른 반점 부분과 의미 있게 통합되어 대상이나 장면의 일부로 기능할 때 부여하는 조직화 활동 유형이다. 핵심은 공백이 실제로 사용되었는지가 명시되는가이다.

## 채점/적용 조건

- 형태 사용이 포함되어 있어야 한다.
- 공백과 하나 이상의 반점 부분이 함께 사용되어야 한다.
- 공백이 단순 배경이 아니라 대상이나 장면의 실제 구성 요소로 설명되어야 한다.
- 카드별 `ZS` 가중치는 카드 번호에 따라 다르다.

## 주의사항/감별 기준

- 공백을 둘러싸기만 한 반응이나 단순한 윤곽 닫힘은 `ZS`가 아니다.
- 카드 III와 X의 얼굴 반응처럼 공백이 자동으로 통합된 것처럼 보이는 경우는 과잉 채점에 주의한다.
- 공백이 실제로 흰 칠, 눈, 호수처럼 설명될 때에만 `ZS`를 준다.
- 공백만 단독 사용한 반응은 `ZS`가 아니고, 다른 반점 부분과의 통합이 필요하다.
