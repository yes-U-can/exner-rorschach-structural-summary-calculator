# [부호화/Z] ZA

## 별칭/검색어

- ZA
- adjacent detail organization
- 인접 부분 조직화

## 핵심 정의

`ZA`는 두 개 이상 인접한 세부 영역을 의미 있게 통합했을 때 부여하는 조직화 활동 유형이다. 전체 반응이 아니더라도, 서로 가까운 부분들이 하나의 장면이나 대상으로 묶여 있으면 `ZA`를 검토한다.

## 채점/적용 조건

- 형태 사용이 포함되어 있어야 한다.
- 두 개 이상 인접한 세부 영역이 하나의 의미 있는 지각으로 통합되어 있어야 한다.
- 단순 병렬 나열이 아니라, 부분들 사이의 관계가 드러나야 한다.
- 경우에 따라 하나의 넓은 영역 안에서 부분을 나누어 조직적으로 통합한 반응도 `ZA`로 볼 수 있다.
- 카드별 `ZA` 가중치는 카드 번호에 따라 다르다.

## 주의사항/감별 기준

- 가까운 부분을 각각 따로 말한 것만으로는 `ZA`가 아니다.
- 물리적으로 붙어 있어도 지각된 대상들이 서로 떨어진 관계라면 `ZD`가 더 적절할 수 있다.
- 공백이 핵심 통합 요소라면 `ZS`를 먼저 검토한다.
- 같은 반응이 둘 이상의 `Z` 기준을 만족하면 카드별 더 높은 값의 유형을 선택한다.
