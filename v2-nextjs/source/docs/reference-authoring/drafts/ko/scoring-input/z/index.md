# [부호화/Z] Z

## 별칭/검색어

- Z
- 조직화 활동
- organizational activity

## 핵심 정의

`Z`는 반응에서 사용된 반점 요소들이 서로 의미 있게 통합될 때 부여하는 조직화 활동 부호군이다. `Z`가 성립하려면 형태 사용이 포함되어야 하며, 실제 채점 유형은 `ZW`, `ZA`, `ZD`, `ZS` 중 하나로 기록한다.

## 채점/적용 조건

- 반응에 형태 사용이 포함되어 있어야 한다.
- 다음 중 최소 하나를 만족하면 `Z`를 검토한다.
- `W` 반응이면서 발달질 `+`, `v/+`, `o`를 포함하는 경우
- 두 개 이상 인접한 세부 영역을 의미 있게 통합하는 경우
- 두 개 이상 비인접 세부 영역을 의미 있게 통합하는 경우
- 공백이 다른 반점 부분과 의미 있게 통합되는 경우
- 한 반응이 두 가지 이상 준거를 동시에 만족하면, 그 카드에서 더 높은 가중치를 주는 `Z` 유형을 선택한다.
- 순수한 `C`, `T`, `Y`, `V` 반응처럼 형태 사용이 없는 반응은 `Z`로 채점하지 않는다.
- 질문 단계 정보는 `Z` 여부를 결정할 때 필요한 범위에서만 사용한다.

## 주의사항/감별 기준

- 단순히 여러 부분을 나열한 것만으로는 `Z`가 되지 않는다. 부분들 사이의 의미 있는 관계가 드러나야 한다.
- `W`를 썼다고 해서 자동으로 `ZW`가 되지 않는다. `DQ` 조건까지 확인해야 한다.
- `Wv` 반응에는 `Z`를 주지 않는다.
- `Z`는 위치(Location) 부호가 아니라 조직화 활동 부호다. `W`, `D`, `Dd`, `S`와 역할이 다르다.
- `ZS`는 공백이 실제로 대상이나 장면의 일부로 통합되었을 때만 준다. 단순한 윤곽 닫힘(closure)은 `ZS`가 아니다.
