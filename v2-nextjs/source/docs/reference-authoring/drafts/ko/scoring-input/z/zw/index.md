# [부호화/Z] ZW

## 별칭/검색어

- ZW
- whole-blot organization
- 전체 반응 조직화

## 핵심 정의

`ZW`는 `W` 반응이면서 발달질이 `+`, `v/+`, `o` 중 하나일 때 부여하는 전체 반응 조직화 유형이다. 즉, 전체 반점을 형태적으로 조직해 사용했다는 점이 확인될 때의 `Z` 부호다.

## 채점/적용 조건

- 반응이 `W`여야 한다.
- 발달질이 `+`, `v/+`, `o` 중 하나여야 한다.
- 형태 사용이 포함되어 있어야 한다.
- 전체 반점이 의미 있는 하나의 조직화된 지각으로 사용되어야 한다.
- 카드별 `ZW` 가중치는 고정값이 아니라 카드 번호에 따라 다르다.
- 같은 반응이 다른 `Z` 준거도 동시에 만족하면, 카드별로 더 높은 가중치를 주는 유형을 선택한다.

## 주의사항/감별 기준

- `W`라고 해서 자동으로 `ZW`가 되는 것은 아니다. `Wv`는 `ZW`가 아니다.
- 전체를 썼더라도 의미 있는 조직화가 없으면 `Z`를 주지 않는다.
- 단순한 전체 명명은 `Location W`의 정보이고, `ZW`는 조직화 활동까지 확인된 경우다.
- 인접 세부 통합이나 공백 통합이 더 강하게 드러나면 `ZA`나 `ZS`가 최종 선택될 수 있다.
