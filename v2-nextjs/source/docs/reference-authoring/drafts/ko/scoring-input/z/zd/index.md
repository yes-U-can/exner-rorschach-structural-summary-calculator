# [부호화/Z] ZD

## 별칭/검색어

- ZD
- distant detail organization
- 원격 부분 조직화

## 핵심 정의

`ZD`는 두 개 이상 비인접 세부 영역을 의미 있게 통합했을 때 부여하는 조직화 활동 유형이다. 서로 떨어진 부분들을 하나의 장면이나 관계로 묶어 보는 방식이 핵심이다.

## 채점/적용 조건

- 형태 사용이 포함되어 있어야 한다.
- 두 개 이상 비인접 세부 영역이 하나의 의미 있는 지각으로 연결되어야 한다.
- 반응 속 대상들이 서로 떨어져 지각되는 관계이면 `ZD`를 우선 검토한다.
- 카드별 `ZD` 가중치는 카드 번호에 따라 다르다.

## 주의사항/감별 기준

- 여러 부분이 등장해도 서로의 관계가 드러나지 않으면 `ZD`가 아니다.
- 반점 부분이 물리적으로 붙어 있어도, 지각된 대상들이 인접한 관계가 아니면 `ZA` 대신 `ZD`가 될 수 있다.
- 공백이 통합 요소의 핵심이면 `ZS`를 먼저 검토한다.
- `ZD`는 후속 해석에서 `ZSum`, `Zd`와 연결되므로 coding 단계에서 과잉 채점하지 않는 것이 중요하다.
