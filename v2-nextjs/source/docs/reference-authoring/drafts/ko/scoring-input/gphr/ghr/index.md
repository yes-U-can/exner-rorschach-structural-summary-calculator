# [부호화/GHR-PHR] GHR

## 별칭/검색어

- GHR
- good human representational response
- 양호한 인간 표상

## 핵심 정의

GHR은 인간 표상 반응 중에서 비교적 적절하고 덜 왜곡된 인간 표상이 확인될 때 주는 부호이다. 다만 이것은 단독 직관으로 찍는 코드가 아니라, 다른 coding을 모두 끝낸 뒤 순차 판정표에 따라 결정해야 한다.

## 채점/적용 조건

- 먼저 반응이 인간 표상 반응인지 확인한다.
- `Pure H`에 적절한 형태질이 있고, 심한 인지적 특수점수나 `AG`, `MOR`가 없으면 GHR 쪽으로 간다.
- 남은 인간 표상 반응 중 `COP`가 있고 `AG`가 없으면 GHR 쪽으로 간다.
- 남은 인간 표상 반응이 카드 III, IV, VII, IX에서 `Popular`이면 GHR 쪽 판단 요소가 된다.
- 위의 배제 규칙들을 통과한 남은 인간 표상 반응은 최종적으로 GHR가 될 수 있다.

## 주의사항/감별 기준

- `GHR`는 “좋은 사람”이라는 뜻이 아니라 상대적으로 적응적인 인간 표상 coding이다.
- `H`가 있다고 해서 자동으로 `GHR`가 되는 것은 아니다.
- `COP`가 있더라도 `AG`가 함께 있으면 다시 주의 깊게 봐야 한다.
- `Hd`, `An`, `MOR`, 심한 cognitive special score가 있으면 `PHR` 조건을 먼저 재검토한다.
- 판정은 고정 순서가 중요하므로, 확인된 GHR 조건만 따로 떼어 체크리스트처럼 쓰면 안 된다.
