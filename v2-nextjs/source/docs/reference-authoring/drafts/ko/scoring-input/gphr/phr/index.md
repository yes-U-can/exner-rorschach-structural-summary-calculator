# 문서명: [부호화/GHR-PHR] PHR

## 별칭/검색어

- PHR
- poor human representational response
- 취약한 인간 표상

## 핵심 정의

PHR은 인간 표상 반응 중에서 왜곡, 손상, 부적절성, 불안정성이 더 두드러질 때 주는 부호이다. 이것도 역시 단순 인상으로 정하는 것이 아니라, 다른 coding이 끝난 뒤 순차 판정표에 따라 결정해야 한다.

## 채점/적용 조건

- 먼저 반응이 인간 표상 반응인지 확인한다.
- `FQ-`나 `FQ none`이면 PHR 쪽으로 바로 기울 수 있다.
- 형태질이 괜찮아도 `ALOG`, `CONTAM`, level 2 cognitive special score가 있으면 PHR 쪽 판단 요소가 된다.
- 남은 인간 표상 반응에 `FABCOM`, `MOR`, `An`이 있으면 PHR를 우선 검토한다.
- 남은 인간 표상 반응에 `AG`, `INCOM`, `DR`, 또는 `Hd`가 있으면 PHR 쪽 판단 요소가 된다.

## 주의사항/감별 기준

- `PHR`는 “나쁜 사람”이라는 뜻이 아니라 더 왜곡되거나 취약한 인간 표상 coding이다.
- `Hd`는 `(Hd)`와 다르지만, 순차 판정에서는 `Hd`가 별도 PHR 요소가 될 수 있다.
- `PHR`는 인간 내용 하나만으로 정하지 않고 형태질, cognitive special score, 공격성, morbid성까지 함께 본다.
- `pair`나 `reflection`이 있다고 해서 자동으로 `PHR`는 아니다.
- `GHR`와 `PHR`는 동시에 주지 않고, 순차 판정 끝에 하나만 남겨야 한다.
