# [부호화/카드] II

## 별칭/검색어

- Card II
- card II
- card 2
- 카드 II
- 2번 카드

## 핵심 정의

`카드 II`는 초기 카드들 가운데서 적색 요소가 뚜렷하게 들어오는 카드다. 이 문서는 카드 II에서 색채, 위치, 조직화가 coding에 어떻게 영향을 주는지에 초점을 둔다.

## 채점/적용 조건

- 카드 II에서는 빨간 부분이 자주 언급되므로, 색이 실제로 반응 형성에 쓰였는지 확인해 `FC / CF / C`를 판단한다.
- 색이 단지 위치를 가리키는 말로만 쓰이면 색채 결정인을 주지 않는다.
- 카드 II의 일부 반응은 흰 부분과 유채색 부분을 함께 조직화하여 `Z` 기준을 충족할 수 있다.
- 사람이나 동물이 서로 관계를 맺는 반응이 나올 수 있지만, movement coding은 실제 행동이 분명할 때만 준다.

## 주의사항/감별 기준

- 빨간색이 보인다고 해서 자동으로 `C` 계열 결정인을 주면 안 된다.
- 카드 II는 여러 부분이 끊겨 보여 하나의 반응인지 여러 반응인지 헷갈릴 수 있으므로 inquiry에서 통합 여부를 확인한다.
- 협동적 장면이 나와도 실제 협동 운동이 명확하지 않으면 `COP`를 주지 않는다.
