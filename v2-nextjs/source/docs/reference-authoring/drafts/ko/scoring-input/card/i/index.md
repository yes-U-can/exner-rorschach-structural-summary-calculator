# [부호화/카드] I

## 별칭/검색어

- Card I
- card I
- card 1
- 카드 I
- 1번 카드

## 핵심 정의

`카드 I`는 종합체계 10장 중 첫 번째 카드다. 이 문서는 카드 I의 상징 해석이 아니라, 카드 I에서 자주 생기는 `부호화상 주의점`을 짧게 정리한 카드별 coding note다.

## 채점/적용 조건

- 카드 I에서는 반점 전체를 쓰는 `W` 반응이 자주 나타난다. 특히 전체 반점을 날개 달린 대상으로 보는 반응이 흔하다.
- 카드 I의 대표적인 전체 반응 범주는 `박쥐`, `나비`, `새` 같은 날개 달린 대상이다.
- 카드 I에서 `박쥐`가 흔한 이유에는 무채색 채색과 윗부분 돌출부가 함께 작용한다.
- 얼굴 반응에서 흰 부분을 `눈`이나 `입`으로 쓰는 경우가 있는데, 다른 반점 부분도 함께 써야 `Z` 기준을 검토할 수 있다.

## 주의사항/감별 기준

- 카드 I에서 흔한 대답처럼 보여도, 실제로는 `전체 반점`을 썼는지 확인한 뒤에만 `Popular`을 검토한다.
- 카드 I와 카드 V는 둘 다 `박쥐/나비`가 흔하지만, 두 카드의 선호 분포는 같지 않다.
- `박쥐처럼 보인다`고 말할 때 회색이나 검은색을 언급했다고 해서 자동으로 색채 결정인이 되는 것은 아니다. 색이 `대상을 보이게 한 요인`으로 실제 사용되었는지를 따로 확인해야 한다.
