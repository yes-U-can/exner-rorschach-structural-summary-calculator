# [부호화/카드] III

## 별칭/검색어

- Card III
- card III
- card 3
- 카드 III
- 3번 카드

## 핵심 정의

`카드 III`는 사람 둘의 장면이나 상호작용이 자주 보고되는 카드다. 이 문서는 카드 III에서 `인간 내용`, `운동`, `관계 장면`을 coding할 때의 핵심 기준만 다룬다.

## 채점/적용 조건

- 카드 III에서는 `두 사람` 또는 `두 인물`이 어떤 일을 하는 반응이 흔하다.
- 떨어져 보이는 부분이라도 의미 있게 조직되면 하나의 반응으로 coding할 수 있다.
- 협동 운동이 분명하면 `COP`, 공격적 운동이 분명하면 `AG`를 검토한다.
- 인간 반응이 자주 나오므로 `H`, `GHR/PHR` 같은 사람 관련 coding도 함께 확인한다.

## 주의사항/감별 기준

- 카드 III라고 해서 인간 내용이나 `COP`가 자동으로 붙지는 않는다.
- `사람 두 명과 나비`처럼 여러 대상이 언급되면 하나의 통합 반응인지, 따로 떨어진 반응인지 inquiry에서 확인해야 한다.
- 단순히 두 대상이 나란히 있다는 이유만으로 협동 장면으로 coding하면 안 된다.
