# 문서명: [부호화/카드] VI

## 별칭/검색어

- Card VI
- card VI
- card 6
- 카드 VI
- 6번 카드

## 핵심 정의

`카드 VI`는 동물 가죽, 털가죽, 융단 같은 반응이 자주 나오는 카드다. 이 문서는 카드 VI에서 `형태`와 `질감`을 어떻게 구분할지에 초점을 둔 coding note다.

## 채점/적용 조건

- 카드 VI에서는 `animal skin / hide / pelt / rug` 같은 반응이 흔하므로 `Popular` 여부를 먼저 확인한다.
- `가죽`, `털 같은 질감`, `보송보송하다` 같은 표현이 실제 반응 형성에 쓰였으면 `T / TF / FT`를 검토한다.
- 반대로 피검자가 음영을 분명히 언급하지 않고 윤곽이나 형태를 중심으로 반응하면 `F` 쪽이 맞을 수 있다.
- 동물 가죽이나 피부는 content coding에서 `Ad`를 함께 검토한다.
- 죽은 동물, 손상된 동물처럼 훼손이 강조되면 `MOR` 검토가 필요하다.

## 주의사항/감별 기준

- 카드 VI라고 해서 `피부/가죽` 반응이 자동으로 texture determinant가 되는 것은 아니다.
- 피검자가 음영 특징을 명시하지 않으면, 형태 반응으로 남겨야 하는 경우가 있다.
- `죽은 동물`은 단순 동물 내용과 다르므로 손상, 부패, 훼손이 드러나는지 따로 본다.
