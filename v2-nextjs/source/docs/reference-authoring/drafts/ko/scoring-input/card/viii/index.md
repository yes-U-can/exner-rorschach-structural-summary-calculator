# [부호화/카드] VIII

## 별칭/검색어

- Card VIII
- card VIII
- card 8
- 카드 VIII
- 8번 카드

## 핵심 정의

`카드 VIII`는 유채색 카드들 중에서 `동물 형상`이 매우 잘 잡히는 카드다. 이 문서는 카드 VIII에서 동물 content, 인기반응, 색채 coding을 함께 확인하는 기준을 정리한다.

## 채점/적용 조건

- 카드 VIII의 D1 영역은 전체 동물 형상으로 자주 보고되므로 `Popular`과 `A`를 먼저 확인한다.
- 카드 VIII에서는 색채가 반응 형성에 실제로 기여했는지 확인해 `FC / CF / C`를 검토한다.
- 동물이 단순히 보인 것인지, 서로 관계를 이루거나 다른 배경과 조직화된 것인지에 따라 `Z` 가능성이 달라진다.
- 털 같다, 보송보송하다 같은 표현이 추가되더라도 실제 결정인은 inquiry로 확인한 뒤 정한다.

## 주의사항/감별 기준

- 카드 VIII는 동물이 흔하다고 해서 모든 동물 반응이 `Popular`이 되는 것은 아니다.
- 색을 언급한 것과 색을 결정인으로 사용한 것은 다르다.
- 같은 D1 반응이라도 locale에 따라 세부 예시는 달라질 수 있으므로 현재 문서는 coding 원칙만 남긴다.
