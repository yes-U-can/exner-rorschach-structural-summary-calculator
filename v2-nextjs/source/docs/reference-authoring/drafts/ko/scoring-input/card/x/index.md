# [부호화/카드] X

## 별칭/검색어

- Card X
- card X
- card 10
- 카드 X
- 10번 카드

## 핵심 정의

`카드 X`는 여러 색과 여러 조각이 함께 들어 있는 복합 카드다. 이 문서는 카드 X에서 인기반응, 색채 coding, 형태 왜곡 위험을 점검하는 기준만 남긴다.

## 채점/적용 조건

- 카드 X의 D1 영역에서는 `거미`나 `게` 같은 반응이 흔하므로 `Popular`을 확인한다.
- 카드 X에서는 청색이나 다색 요소가 주의를 끌 수 있으므로 `FC / CF / C`를 실제 사용 여부에 따라 판단한다.
- 여러 조각을 한 번에 묶어 설명하면 형태가 흐려질 수 있으므로 `FQ`를 조심해서 본다.
- 뒤집어서 전체 얼굴처럼 보는 반응은 존재하지만, 윤곽을 새로 만들어 낸 경우 `-` 가능성을 검토해야 한다.

## 주의사항/감별 기준

- 카드 X는 복잡하다고 해서 자동으로 특수점수로 가는 카드가 아니다.
- 단지 색이 많다는 사실만으로 색채 결정인을 주지 않는다.
- 여러 곤충이나 해양생물을 한꺼번에 말할 때는 전체 반응에서 핵심이 무엇인지 먼저 정리한 뒤 형태질을 판단한다.
