# [부호화/카드] IX

## 별칭/검색어

- Card IX
- card IX
- card 9
- 카드 IX
- 9번 카드

## 핵심 정의

`카드 IX`는 후기 유채색 카드들 중에서도 조직화가 어렵게 느껴질 수 있는 카드다. 이 문서는 카드 IX에서 흔한 사람형 반응과 형태질 점검 포인트를 coding note로 정리한다.

## 채점/적용 조건

- 카드 IX에서는 사람 또는 사람 비슷한 형상이 흔한 영역이 있으므로 `Popular` 여부를 확인한다.
- 반응이 실제 인간인지, 인간 같은 허구적 형상인지에 따라 `H`와 `(H)`를 구분한다.
- 인간형 장면이 실제 운동을 포함하면 `M`을 검토한다.
- 카드 IX는 형태가 덜 분명하게 느껴질 수 있으므로 `FQ` 판단을 조심해서 확인한다.

## 주의사항/감별 기준

- 카드 IX에서 반응이 막연하다고 해서 곧바로 약한 형태질이나 특수점수로 가면 안 된다.
- 사람처럼 보인다고 해도 실제 인간인지, 인간형 상상물인지 구분해야 한다.
- 카드 IX는 감정적으로 복잡하게 느껴질 수 있지만, 이 문서는 그 해석이 아니라 coding 기준만 다룬다.
