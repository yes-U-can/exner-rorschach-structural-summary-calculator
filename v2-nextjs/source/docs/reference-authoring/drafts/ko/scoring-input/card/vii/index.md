# [부호화/카드] VII

## 별칭/검색어

- Card VII
- card VII
- card 7
- 카드 VII
- 7번 카드

## 핵심 정의

`카드 VII`는 사람 머리나 얼굴 반응이 자주 나오는 카드다. 이 문서는 카드 VII에서 사람 내용, 인기반응, 조직화 판단이 어떻게 연결되는지만 coding 관점에서 정리한다.

## 채점/적용 조건

- 카드 VII에서는 `사람 머리/얼굴` 계열 반응이 흔하므로 `Popular`을 먼저 확인한다.
- 반응이 완전한 인간이 아니라 머리나 얼굴의 부분이면 `Hd`가 우선 검토 대상이다.
- 두 사람이 어떤 관계를 가진 장면으로 조직되면 단순 병렬 반응보다 높은 조직화가 될 수 있다.
- 실제 인간 운동이 분명하면 `M`을 검토한다.
- 사람 관련 반응은 필요하면 `GHR/PHR`와도 연결해 본다.

## 주의사항/감별 기준

- 카드 VII에서 흔한 사람 반응이 나온다고 해서 자동으로 완전한 인간 내용 `H`를 주면 안 된다.
- 단지 두 얼굴이 있다는 것과, 두 사람이 의미 있는 관계를 형성한다는 것은 다르다.
- 카드 VII는 카드별 coding note이지, 관계 문제를 상징적으로 설명하는 문서가 아니다.
