# [부호화/카드] IV

## 별칭/검색어

- Card IV
- card IV
- card 4
- 카드 IV
- 4번 카드

## 핵심 정의

`카드 IV`는 어두운 음영 차이와 큰 전체 형태가 함께 눈에 들어오는 카드다. 이 문서는 카드 IV에서 `W`, `형태`, `음영 묘사`를 coding할 때의 주의점을 정리한다.

## 채점/적용 조건

- 카드 IV에서는 반점 전체를 쓰는 반응이 자주 나올 수 있으므로 실제 사용 위치를 먼저 확인한다.
- 어둡다, 부드럽다, 벨벳 같다, 주름 같다 같은 표현이 나오면 `Texture`인지, 아니면 여러 음영을 섞은 blend인지 inquiry에서 확인한다.
- 같은 반응 안에서 질감만 확인되면 `T` 계열 한 종류만 준다.
- 형태가 분명한 전체 반응이라도 형태질은 따로 판단해야 한다.

## 주의사항/감별 기준

- 음영을 설명하는 말이 여러 개 나와도 실제 지각 내용이 하나이면 자동으로 blend로 가지 않는다.
- 카드 IV의 묘사는 카드 상징 해석과 분리해서 다뤄야 한다. 이 문서는 coding page다.
- 카드 IV는 `카드 V`와 이름이 비슷한 `결정인 V`와 무관하다.
