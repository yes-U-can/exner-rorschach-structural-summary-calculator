# [위치] S

## 별칭/검색어

- S
- white space
- space response

## 핵심 정의

`S`는 흰 여백이 반응의 지각 구성에 실질적으로 사용되었음을 나타내는 위치 부호다.

## 채점/적용 조건

- 흰 여백이 단순한 배경이 아니라 지각 대상의 일부로 사용된다.
- Inquiry에서 흰 여백이 반응 형성에 어떤 역할을 했는지 확인한다.
- 실제 사용 영역에 맞추어 `S`, `WS`, `DS`, `DdS`를 구분한다.

## 주의사항/감별 기준

- 배경을 언급했다는 사실만으로 `S`를 부호화하지 않는다. 흰 여백이 지각 구성에 관여해야 한다.
- 잉크반점 영역과 흰 여백을 함께 사용했다면 해당 기본 위치와 결합한 `WS`, `DS`, `DdS`를 검토한다.
- `S`의 해석은 위치 부호 하나만으로 단정하지 않고 구조요약의 전체 배열 안에서 다룬다.
