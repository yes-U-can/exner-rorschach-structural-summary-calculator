# [위치] WS

## 별칭/검색어

- WS
- whole blot with white space

## 핵심 정의

`WS`는 잉크반점 전체를 사용하면서 흰 여백도 지각 구성에 실질적으로 포함할 때 부호화한다. 흰 여백 사용이 결합된 `W` 반응이다.

## 채점/적용 조건

- 잉크반점 전체가 반응에 사용된다.
- 흰 여백이 지각 구성에서 실제 역할을 한다.
- 표현만으로 추정하지 않고 Inquiry에서 흰 여백의 관여를 확인한다.

## 주의사항/감별 기준

- 배경을 막연히 언급했다는 이유만으로 `WS`를 부호화하지 않는다. 흰 여백이 구조적으로 관여해야 한다.
- 흔한 세부영역과 흰 여백을 사용하면 [`DS`](ref://scoring-input/location/DS), 드문 세부영역과 흰 여백을 사용하면 [`DdS`](ref://scoring-input/location/DdS)를 검토한다.
- 흰 여백 없이 전체 반점만 사용하면 [`W`](ref://scoring-input/location/W)로 구분한다.
