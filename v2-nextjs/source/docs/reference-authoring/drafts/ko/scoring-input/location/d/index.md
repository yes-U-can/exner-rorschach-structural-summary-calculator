# [위치] D

## 별칭/검색어

- D
- common detail

## 핵심 정의

`D`는 사람들이 흔히 사용하는 잉크반점의 세부영역을 바탕으로 반응을 구성할 때 부호화한다. 영역의 물리적 크기보다 사용 빈도가 핵심 기준이다.

## 채점/적용 조건

- 반응이 통상적으로 자주 사용되는 세부영역을 바탕으로 한다.
- Location Sheet 또는 Inquiry에서 사용 영역을 확인한다.
- 전체 반점이나 드물게 사용하는 세부영역에 해당하지 않아야 한다.

## 주의사항/감별 기준

- `D`와 `Dd`는 크고 작음이 아니라 흔한 사용과 드문 사용으로 구분한다.
- 흔한 세부영역에 흰 여백이 실질적으로 결합되면 [`DS`](ref://scoring-input/location/DS)를 검토한다.
- `D` 자체가 단순성이나 정상성을 뜻하지는 않는다.
