# [위치] DS

## 별칭/검색어

- DS
- common detail with white space

## 핵심 정의

`DS`는 흔히 사용하는 세부영역과 흰 여백을 함께 사용하여 반응을 구성할 때 부호화한다. 흰 여백 사용이 결합된 `D` 반응이다.

## 채점/적용 조건

- 반응의 주된 영역은 흔히 사용하는 세부영역이다.
- 흰 여백이 지각 구성에서 실제 역할을 한다.
- Inquiry에서 세부영역과 흰 여백의 결합 사용을 확인한다.

## 주의사항/감별 기준

- 흰 여백을 부수적으로 언급했다는 이유만으로 `DS`를 부호화하지 않는다.
- 전체 반점과 흰 여백을 사용하면 [`WS`](ref://scoring-input/location/WS), 드문 세부영역과 흰 여백을 사용하면 [`DdS`](ref://scoring-input/location/DdS)를 검토한다.
- 흰 여백 없이 흔한 세부영역만 사용하면 [`D`](ref://scoring-input/location/D)로 구분한다.
