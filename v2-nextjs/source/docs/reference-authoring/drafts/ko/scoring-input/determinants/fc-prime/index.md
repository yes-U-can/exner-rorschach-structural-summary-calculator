# [부호화/결정인] FC'

## 별칭/검색어

- FC'
- form-achromatic color
- 형태-무채색

## 핵심 정의

FC'는 형태가 먼저 대상을 조직하고, 무채색 특징이 그 대상을 2차적으로 정교화할 때 주는 무채색 색채 결정인이다. 무채색 반응 가운데서는 비교적 분명하게 구별되는 편이며, 기본 대상은 형태로도 어느 정도 유지된다.

## 채점/적용 조건

- 형태 단서가 먼저 대상을 성립시키고 있어야 한다.
- `검다`, `희다`, `회색`, `어둡다`, `밝다` 같은 무채색 단서가 그 대상을 보강해야 한다.
- 무채색 단서가 빠져도 기본 형태가 어느 정도 유지되는지 질문 단계에서 확인한다.
- 피검자가 무채색 특징을 실제로 사용했다고 분명히 말할 때만 `FC'`를 준다.

## 주의사항/감별 기준

- `FC'`는 `C'F`와 달리 형태 우세가 더 분명하다.
- `FC'`는 `FC`와 기호가 비슷하지만, 유채색이 아니라 무채색에 근거한 결정인이다.
- `어둡다`, `밝다`가 깊이감이나 거리감을 뜻하면 `V`를 먼저 검토한다.
- `어둡다`, `밝다`가 막연한 음영 인상이나 불편한 분위기를 뜻하면 `Y`를 검토한다.
- 단순히 반점의 어느 부분이 더 어둡다고 위치를 가리킨 것만으로는 `FC'`를 주지 않는다.
