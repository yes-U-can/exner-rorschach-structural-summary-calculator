# [부호화/결정인] C'

## 별칭/검색어

- C'
- pure achromatic color
- 순수 무채색

## 핵심 정의

C'는 반점의 검정, 회색, 흰색 같은 무채색 특징만으로 반응이 형성될 때 주는 순수 무채색 색채 결정인이다. 형태 단서는 실질적으로 보이지 않아야 하며, 무채색 반응 중에서도 매우 드문 편이다.

## 채점/적용 조건

- 반응이 무채색 특징 자체에 근거해야 한다.
- 질문 단계에서 형태를 보여 주는 추가 설명이 거의 없어야 한다.
- 피검자가 자발적으로 검정, 흰색, 회색 같은 무채색 단서를 분명히 언급해야 한다.
- 질문 단계에서 형태 조직이 확인되면 `C'`가 아니라 `C'F`를 재검토한다.

## 주의사항/감별 기준

- `C'`는 `C'F`와 달리 형태 보조가 거의 없거나 없어야 한다.
- `C'`는 `C`와 달리 유채색이 아니라 무채색에 근거한 순수 색채 반응이다.
- `어둡다`, `밝다`가 깊이감이나 거리감을 뜻하면 `V`일 수 있다.
- `어둡다`, `밝다`가 막연한 음영 인상이나 불안감을 뜻하면 `Y`일 수 있다.
- 단순한 위치 설명이나 배경 묘사만으로는 `C'`를 주지 않는다.
