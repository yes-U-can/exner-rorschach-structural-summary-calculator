# [결정인] FM

## 별칭/검색어

- FM
- animal movement

## 핵심 정의

`FM`은 동물이나 비인간 생명체의 통상적인 움직임이 지각될 때 사용하는 결정인이다.

## 입력 코드 규칙

`FM`은 구조요약에서 동물운동의 전체 빈도를 표시하는 합계 항목이다. 개별 반응은 능동형 `FMa`, 수동형 `FMp`, 능동-수동형 `FMa-p` 중 하나로 구분해 입력한다. 능동형·수동형 표기가 없는 `FM`은 개별 반응에 넣는 완전한 부호로 사용하지 않는다.

## 채점/적용 조건

- 반응에 실제 움직임이 지각되어야 한다.
- 움직이는 주체는 동물이나 비인간 생명체이며, 그 종에 자연스러운 행동을 한다.
- 명백히 인간적이거나 의인화된 행동은 `FM`보다 [`M`](ref://scoring-input/determinants/M)을 검토한다.

## 주의사항/감별 기준

- 동물 내용이 있다는 사실만으로 `FM`을 부호화하지 않는다. 움직임이 표현되어야 한다.
- 인간적이거나 의인화된 움직임은 `M`, 무생물의 움직임은 [`m`](ref://scoring-input/determinants/m)과 구분한다.
- 움직임의 주체와 행동 방식은 피검자의 설명을 기준으로 판단한다.
