# 문서명: [부호화/결정인] T

## 별칭/검색어

- T
- pure texture
- 순수 질감

## 핵심 정의

T는 음영 특징이 형태와 거의 결합되지 않은 채 순수한 질감으로만 지각될 때 주는 texture determinant이다. `얼음`, `비단`, `기름`, `푹신한 울`처럼 촉감이나 재질 인상만 남고, 형태 조직은 거의 드러나지 않아야 한다.

## 채점/적용 조건

- 반응의 핵심이 형태가 아니라 촉감, 재질, 결 인상이어야 한다.
- 질문 단계에서 피검자가 실제로 질감을 근거로 반응했다고 확인되어야 한다.
- 형태를 보여 주는 설명이 실질적으로 나타나지 않아야 한다.
- 형태 조직이 확인되면 `T`가 아니라 `TF` 또는 `FT`를 재검토한다.

## 주의사항/감별 기준

- `T`는 `TF`와 달리 형태 보조가 거의 없거나 없어야 한다.
- `T`는 `F`와 달리 형태 조직이 핵심이 아니다.
- `부드럽다`, `차갑다`, `뜨겁다` 같은 표현은 색이나 내용 연상일 수도 있으므로 자동으로 `T`를 주지 않는다.
- 굴곡, 깊이, 입체감이 강조되면 `V`를 먼저 검토한다.
- 막연하고 확산된 음영 인상이 핵심이면 `Y`를 먼저 검토한다.
