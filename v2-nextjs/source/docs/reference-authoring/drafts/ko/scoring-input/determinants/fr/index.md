# 문서명: [부호화/결정인] Fr

## 별칭/검색어

- Fr
- form-reflection
- 형태-반사

## 핵심 정의

Fr는 형태가 먼저 특정 대상을 성립시키고, 그 대상이 반사된 것으로 해석될 때 주는 reflection determinant이다. 반점의 대칭성이 사용되지만, 실제로는 `비친 모습`, `거울상`, `물에 비친 것` 같은 반사 개념이 분명히 들어가야 한다.

## 채점/적용 조건

- 형태가 먼저 특정 내용을 identify해야 한다.
- 피검자가 반사 개념을 직접 말하거나, 그와 동등한 표현을 사용해야 한다.
- 반점의 대칭성이 실제로 반사 해석의 근거가 되어야 한다.
- 단순히 두 개가 보인다는 말만으로는 `Fr`를 주지 않고 `pair` 가능성을 검토한다.

## 주의사항/감별 기준

- `Fr`는 `rF`보다 형태 우세가 더 분명하다.
- `Fr`는 `pair`와 다르다. `pair`는 두 개가 보인다는 뜻이고, `Fr`는 반사되었다는 개념이 들어가야 한다.
- 반사가 비대칭 음영에 근거하면 `V`나 `Y`가 더 적절할 수 있다.
- 움직임이 함께 나타날 수 있지만, 반사 coding 자체는 별도로 확인해야 한다.
- `양쪽에 하나씩 있다`는 말만으로는 `Fr`를 주지 않는다.
