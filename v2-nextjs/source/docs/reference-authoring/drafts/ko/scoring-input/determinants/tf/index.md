# 문서명: [부호화/결정인] TF

## 별칭/검색어

- TF
- texture-form
- 질감-형태

## 핵심 정의

TF는 음영 특징이 먼저 질감으로 지각되고, 형태는 2차적으로만 정교화될 때 주는 texture determinant이다. 대상은 어느 정도 말해지지만, 반응을 실제로 이끈 것은 `가죽 같다`, `퍼석하다`, `털조각 같다` 같은 질감 인상이다.

## 채점/적용 조건

- 질감 단서가 반응 형성의 핵심이어야 한다.
- 형태는 대상을 약하게 보조하거나 뒤늦게 정교화하는 수준이어야 한다.
- 반응 문장과 질문 단계에서 질감이 먼저 나오는지, 형태가 뒤에 붙는지 함께 본다.
- 대상 이름이 붙어 있어도, 실제로는 질감 인상이 먼저였는지 Inquiry에서 확인한다.

## 주의사항/감별 기준

- `TF`는 `FT`보다 질감 우세가 강하다.
- `TF`는 `T`와 달리 약한 형태 조직이 남아 있다.
- 피검자가 실제로 음영을 근거로 질감을 보았는지 확인하지 못하면 `TF`로 서둘러 coding 하지 않는다.
- 같은 `거칠다`, `울퉁불퉁하다`라도 깊이나 굴곡이 강조되면 `V`일 수 있다.
- 모호하고 흐릿한 불편감이 핵심이면 `Y`를 먼저 검토한다.
