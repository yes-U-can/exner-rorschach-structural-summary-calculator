# 문서명: [부호화/결정인] F

## 별칭/검색어

- F
- pure form
- 순수 형태

## 핵심 정의

F는 형태만으로 반응이 성립할 때 주는 순수 형태 determinant이다. 다른 결정인이 실제로 확인되지 않고, 피검자가 주로 윤곽과 모양에 근거해 대상을 지각했을 때 `F` 단독으로 coding 한다.

## 채점/적용 조건

- 반응이 형태 특징에 기초해야 한다.
- 색채, 음영, 움직임, 반사, 차원성 같은 다른 determinant가 확인되지 않아야 한다.
- 질문 단계에서 피검자가 모양이나 윤곽을 근거로 반응했다고 확인해야 한다.
- 다른 determinant 단서가 분명하면 `F` 단독이 아니라 해당 determinant를 함께 검토한다.

## 주의사항/감별 기준

- 대부분의 반응은 형태를 포함하므로, `F`는 “형태가 있다”가 아니라 “형태만 있다”는 뜻이다.
- `F`는 `FQ`와 다르다. `F`는 결정인이고, `FQ`는 형태질이다.
- `F`는 `FD`와 다르다. `FD`는 형태만으로 3차원성이 성립하는 경우다.
- `피부`, `가죽`, `털` 같은 반응은 자동으로 texture가 아니며, 실제로 음영 근거가 없으면 `F`일 수 있다.
- 혼합 반응에서 `F`는 드물게 들어가므로, 다른 determinant가 실제로 확인되는지 먼저 다시 본다.
