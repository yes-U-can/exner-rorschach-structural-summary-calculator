# 문서명: [부호화/결정인] YF

## 별칭/검색어

- YF
- shading-form
- 음영-형태

## 핵심 정의

YF는 diffuse shading이 먼저 반응을 이끌고, 형태는 2차적으로만 정교화될 때 주는 determinant이다. 반응 내용은 대개 `구름`, `그림자`, `연기`, `x-ray 같은 것`처럼 구체성이 낮고, 피검자는 먼저 흐림이나 음영 인상에 끌린다.

## 채점/적용 조건

- 음영 특징이 반응 형성의 핵심이어야 한다.
- 형태는 약하게 보조되거나 뒤늦게만 붙어야 한다.
- 반응 문장과 질문 단계에서 음영 인상이 먼저 나오는지, 형태가 뒤에 붙는지 함께 본다.
- 형태를 말하더라도, 실제 강조점이 막연한 shading 인상인지 Inquiry에서 확인한다.

## 주의사항/감별 기준

- `YF`는 `FY`보다 음영 우세가 강하다.
- `YF`는 `Y`와 달리 약한 형태 조직이 남아 있다.
- 깊이나 입체감이 강조되면 `VF/V`를 먼저 검토한다.
- 촉감이나 재질이 강조되면 `TF/T`를 먼저 검토한다.
- `구름`, `연기`처럼 내용이 애매한 반응은 자동으로 `YF`가 아니라, 형태를 묘사하려는 의도가 있었는지 먼저 본다.
