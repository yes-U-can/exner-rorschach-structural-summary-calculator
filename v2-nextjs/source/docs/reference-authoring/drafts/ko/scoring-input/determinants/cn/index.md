# [결정인] CN

## 별칭/검색어

- Cn
- color naming

## 핵심 정의

`Cn`은 피검자가 색의 이름을 말했지만, 그 색이 반응을 실제로 형성한 결정인은 아닐 때 사용하는 부호입니다. 색이 반응 형성에 관여한 `FC`, `CF`, `C`와 구분합니다.

## 채점/적용 조건

- 피검자가 특정 색을 명시적으로 언급합니다.
- 그 색 언급이 반응의 주된 지각 구성을 만들지는 않습니다.
- 색 언급을 빼도 무엇으로 보았는지는 그대로 유지됩니다.
- 색이 실제로 반응 형성에 관여했다면 `FC`, `CF`, `C`를 다시 검토합니다.

## 주의사항/감별 기준

- `Cn`은 순수색채 결정인 `C`와 같지 않습니다.
- 반응이 색에 의존하지 않는데 단순한 색 언급을 `FC`나 `CF`로 확대 부호화하지 않습니다.
- 애매하면 색 때문에 그렇게 보였는지, 이미 본 대상의 부수적 특징으로 색만 덧붙였는지 Inquiry에서 확인합니다.

**후속 계산 경계:** Lower Section의 관례적 라벨은 `FC:CF+C`이지만, 화면 비율은 `FC:(CF+C+Cn)`으로 계산합니다. 따라서 Cn 한 개는 화면 우변을 1 증가시킵니다. 이 화면 표기용 합계를 일반 색채 합계로 재사용하지 않습니다. `WSumC`, S-CON의 `CF+C > FC`, Color-Shading blend는 각각 별도 식을 사용하며 Cn을 제외합니다.
