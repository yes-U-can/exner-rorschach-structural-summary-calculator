# 문서명: [부호화/결정인] V

## 별칭/검색어

- V
- pure vista
- 순수 vista

## 핵심 정의

V는 형태와 거의 결합되지 않은 채, 음영 특징만으로 깊이감이나 입체감이 지각될 때 주는 순수 vista determinant이다. `깊어 보인다`, `튀어나와 보인다`, `원근이 느껴진다` 같은 차원성 자체가 반응의 핵심이어야 하며, 비교적 드문 반응이다.

## 채점/적용 조건

- 반응의 핵심이 형태가 아니라 깊이, 차원성, 원근 인상이어야 한다.
- 질문 단계에서 피검자가 실제로 음영 차이에 근거해 입체감을 보았다고 확인되어야 한다.
- 형태를 보여 주는 설명이 실질적으로 나타나지 않아야 한다.
- 형태 조직이 확인되면 `V`가 아니라 `VF` 또는 `FV`를 재검토한다.

## 주의사항/감별 기준

- `V`는 `VF`와 달리 형태 보조가 거의 없거나 없어야 한다.
- `V`는 `T`와 달리 촉감이나 재질보다 입체감이 핵심이다.
- `V`는 `Y`와 달리 막연한 음영 인상이 아니라 깊이감, 거리감, 차원성이 강조된다.
- 피검자가 단순히 어두운 부분이나 밝은 부분만 가리킨 것이라면 `V`로 바로 coding 하지 않는다.
- 카드 번호 `V`와 determinant `V`는 전혀 다른 문맥이므로 문서를 섞지 않는다.
