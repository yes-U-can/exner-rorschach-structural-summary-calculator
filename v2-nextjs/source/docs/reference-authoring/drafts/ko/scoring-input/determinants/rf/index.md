# 문서명: [부호화/결정인] rF

## 별칭/검색어

- rF
- reflection-form
- 반사-형태

## 핵심 정의

rF는 반점의 대칭성과 반사 인상이 먼저 반응을 이끌고, 형태는 모호하게만 보조될 때 주는 reflection determinant이다. 반응 내용은 대개 구체성이 낮고, `호수에 비친 풍경`, `반사된 바위 같은 것`처럼 반사 개념이 먼저 나온다.

## 채점/적용 조건

- 대칭성과 반사 인상이 반응 형성의 핵심이어야 한다.
- 형태는 약하게만 남아 있거나 모호해야 한다.
- 피검자가 반사 개념을 실제로 사용했다고 질문 단계에서 확인해야 한다.
- 단순히 두 개가 보인다는 말만으로는 `rF`를 주지 않고 `pair`를 먼저 검토한다.

## 주의사항/감별 기준

- `rF`는 `Fr`보다 반사 우세가 강하다.
- `rF`는 `pair`와 다르다. `pair`는 동일 대상 두 개이고, `rF`는 반사된 한 대상의 관계 개념이 들어간다.
- 비대칭 반사는 대개 `V`나 `Y`와 연결될 수 있으므로, 반사 근거가 truly symmetry인지 확인한다.
- 형태가 꽤 구체적이면 `Fr`를 다시 검토한다.
- `바위 두 개`, `구름 두 개`처럼 개수만 말한 반응은 `rF`가 아니다.
