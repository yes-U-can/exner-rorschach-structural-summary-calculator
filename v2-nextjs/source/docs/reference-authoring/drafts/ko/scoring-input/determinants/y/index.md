# 문서명: [부호화/결정인] Y

## 별칭/검색어

- Y
- pure shading
- 순수 음영

## 핵심 정의

Y는 형태와 거의 결합되지 않은 채, 밝고 어두운 음영 특징만으로 반응이 형성될 때 주는 순수 diffuse shading determinant이다. `안개`, `어둠`, `연기`처럼 막연하고 비구체적인 음영 인상 자체가 핵심이어야 하며, 매우 드문 반응이다.

## 채점/적용 조건

- 반응의 핵심이 형태가 아니라 일반적인 음영 인상이어야 한다.
- 질문 단계에서 피검자가 실제로 밝고 어두운 특징에 근거해 반응했다고 확인되어야 한다.
- 형태를 보여 주는 설명이 실질적으로 나타나지 않아야 한다.
- 형태 조직이 확인되면 `Y`가 아니라 `YF` 또는 `FY`를 재검토한다.

## 주의사항/감별 기준

- `Y`는 `YF`와 달리 형태 보조가 거의 없거나 없어야 한다.
- `Y`는 `V`와 달리 깊이감이나 원근보다 막연한 shading 인상이 핵심이다.
- `Y`는 `T`와 달리 촉감이나 재질이 아니라 비특정적 음영 인상이 핵심이다.
- `연기`, `안개`, `구름` 같은 단어만으로 자동 coding 하지 말고, 형태를 만들려는 시도가 있었는지 확인한다.
- 카드 X의 파란 영역처럼 색채가 큰 역할을 하는 경우에는 색채 determinant와의 구분도 다시 본다.
