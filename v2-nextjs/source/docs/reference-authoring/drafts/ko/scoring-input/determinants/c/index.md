# [결정인] C

## 별칭/검색어

- C
- pure color

## 핵심 정의

`C`는 유채색만으로 반응이 결정되고 형태는 실질적으로 관여하지 않을 때 사용하는 순수색채 결정인이다.

## 채점/적용 조건

- 유채색이 반응을 형성하는 주된 근거다.
- 형태는 반응 형성에 실질적으로 기여하지 않는다.
- 형태가 부차적으로라도 관여하면 [`CF`](ref://scoring-input/determinants/CF)를 검토한다.

## 주의사항/감별 기준

- 단순히 색채를 언급했다는 이유만으로 `C`를 부호화하지 않는다. 반응 자체가 색채에 의존해야 한다.
- 형태가 조금이라도 반응을 지탱하면 `C`보다 `CF`가 적절할 수 있다.
- 해석할 때는 `FC`, `CF`와 함께 유채색 결정인 전체의 배열을 확인한다.
