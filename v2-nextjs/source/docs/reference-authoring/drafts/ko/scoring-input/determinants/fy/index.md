# 문서명: [부호화/결정인] FY

## 별칭/검색어

- FY
- form-shading
- 형태-음영

## 핵심 정의

FY는 형태가 먼저 대상을 조직하고, diffuse shading이 그 대상을 2차적으로 정교화할 때 주는 determinant이다. 반응의 기본 대상은 비교적 구체적이지만, `더럽다`, `그을렸다`, `x-ray처럼 보인다` 같은 일반적 음영 인상이 뒤에 붙는다.

## 채점/적용 조건

- 반응의 기본 대상이 형태로 먼저 분명하게 잡혀 있어야 한다.
- 음영 특징이 그 대상에 막연한 어둠, 흐림, 그을림, 그림자 느낌을 보강해야 한다.
- 질문 단계에서 피검자가 실제로 음영 인상을 사용했다고 확인해야 한다.
- 음영 정교화가 빠져도 기본 형태가 어느 정도 유지되는지 확인한다.

## 주의사항/감별 기준

- `FY`는 `YF`보다 형태 우세가 더 분명하다.
- `FY`는 `F`와 달리 음영 정교화가 실제로 확인되어야 한다.
- 깊이감, 원근, 입체감이 핵심이면 `FV/VF/V`를 먼저 검토한다.
- 촉감, 재질, 털 느낌이 핵심이면 `FT/TF/T`를 먼저 검토한다.
- `구름`은 형태와 음영이 모두 가능하므로, 실제 강조점이 형태인지 흐림인지 확인한다.
