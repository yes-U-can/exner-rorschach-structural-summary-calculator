# 문서명: [부호화/결정인] FT

## 별칭/검색어

- FT
- form-texture
- 형태-질감

## 핵심 정의

FT는 형태가 먼저 대상을 조직하고, 음영 특징이 그 대상의 질감을 2차적으로 정교화할 때 주는 texture determinant이다. 대상의 기본 형태는 비교적 분명하고, `털 같다`, `벨벳 같다`, `부드럽다` 같은 질감이 뒤에 붙는다.

## 채점/적용 조건

- 반응의 기본 대상이 형태로 먼저 분명하게 잡혀 있어야 한다.
- 음영 특징이 그 대상의 촉감이나 질감을 보강해야 한다.
- 질문 단계에서 피검자가 질감 단서를 실제로 사용했다고 확인해야 한다.
- 질감 단서가 빠져도 기본 형태가 어느 정도 유지되는지 확인한다.

## 주의사항/감별 기준

- `FT`는 `TF`보다 형태 우세가 더 분명하다.
- `FT`는 `F`와 달리 질감 정교화가 실제로 확인되어야 한다.
- `부드럽다`, `거칠다`, `털 같다`는 단어만으로 자동 coding 하지 말고, 음영에서 온 질감인지 확인한다.
- 깊이감, 거리감, 입체감이 핵심이면 `V`를 먼저 검토한다.
- 막연한 어둡고 흐린 느낌이 핵심이면 `Y`를 먼저 검토한다.
