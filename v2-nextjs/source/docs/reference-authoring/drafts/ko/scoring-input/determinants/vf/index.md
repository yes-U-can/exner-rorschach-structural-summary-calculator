# 문서명: [부호화/결정인] VF

## 별칭/검색어

- VF
- vista-form
- vista-형태

## 핵심 정의

VF는 음영 특징이 먼저 깊이나 차원성을 만들고, 형태는 2차적으로만 정교화될 때 주는 vista determinant이다. 반응은 어느 정도 대상 이름을 갖지만, 실제로 반응을 이끈 것은 `협곡처럼 깊다`, `비구름처럼 층이 진다`, `지도처럼 입체적으로 보인다` 같은 차원성 인상이다.

## 채점/적용 조건

- 깊이, 원근, 입체감이 반응 형성의 핵심이어야 한다.
- 형태는 대상을 약하게 보조하거나 뒤늦게 정교화하는 수준이어야 한다.
- 반응 문장과 질문 단계에서 차원성 단서가 먼저 나오는지, 형태가 뒤에 붙는지 함께 본다.
- 대상 이름이 구체적이더라도 실제 강조점이 음영 기반 차원성인지 Inquiry에서 확인한다.

## 주의사항/감별 기준

- `VF`는 `FV`보다 vista 우세가 강하다.
- `VF`는 `V`와 달리 약한 형태 조직이 남아 있다.
- `TF`와 헷갈릴 수 있으므로, 피검자가 말한 것이 촉감인지 입체감인지 먼저 구분한다.
- 막연한 확산 음영이나 정서적 불편감이 핵심이면 `Y`를 먼저 검토한다.
- 형태가 꽤 분명하더라도, 강조점이 음영 차원성이면 `VF`가 될 수 있다.
