# [부호화/결정인] Determinants

## 별칭/검색어

- Determinants
- 결정인
- 반응 결정요인

## 핵심 정의

Determinants는 피검자가 무엇 때문에 그 대상을 보았는지를 기록하는 부호군이다. 반응을 성립시킨 핵심 자극 속성이 움직임인지, 색채인지, 무채색인지, 음영인지, 순수 형태인지를 구분해 적는다.

## 채점/적용 조건

- 질문 단계에서는 피검자가 실제로 사용했다고 확인된 자극 속성만 coding 한다.
- Determinants는 Location, DQ, FQ, Content와 별개의 축이므로 섞어 쓰지 않는다.
- 같은 반응에 여러 결정인이 확인되면 혼합 결정인으로 기록하되, 말로 드러난 요소만 포함한다.
- 현재 이 개요 문서에서는 순수 형태(`F`), movement(`M`, `FM`, `m`), chromatic color(`FC`, `CF`, `C`, `Cn`), achromatic color(`FC'`, `C'F`, `C'`), texture shading(`FT`, `TF`, `T`), vista shading(`FV`, `VF`, `V`), diffuse shading(`FY`, `YF`, `Y`), reflection/형태차원(`Fr`, `rF`, `FD`)을 우선 정리한다.

## 주의사항/감별 기준

- `FC`와 `FC'`는 둘 다 형태가 우세하지만, 하나는 유채색이고 다른 하나는 무채색이다.
- `C`와 `C'`는 둘 다 순수 색채 반응이지만, `C`는 유채색이고 `C'`는 검정, 회색, 흰색 같은 무채색에 근거한다.
- `FT`, `TF`, `T`는 모두 질감 반응이지만, 형태가 우세한지, 질감이 우세한지, 형태가 사실상 없는지에 따라 갈린다.
- `FV`, `VF`, `V`는 모두 깊이감이나 입체감 반응이지만, 형태가 우세한지, 음영이 우세한지, 형태가 거의 없는지에 따라 갈린다.
- `FY`, `YF`, `Y`는 모두 diffuse shading 반응이지만, 형태가 우세한지, 음영이 우세한지, 형태가 거의 없는지에 따라 갈린다.
- `Fr`와 `rF`는 둘 다 reflection 반응이지만, 형태가 우세한지 대칭성/반사 인상이 우세한지에 따라 갈린다.
- `FD`는 `FV/VF/V`처럼 차원성과 관련되지만, 음영이 아니라 형태만으로 3차원성이 성립해야 한다.
- `어둡다`, `밝다` 같은 단어는 무채색 색채가 아니라 `V`나 `Y` 같은 음영 결정인을 뜻할 수도 있으므로 질문 단계에서 꼭 구분한다.
- `부드럽다`, `거칠다`, `털 같다` 같은 표현도 자동으로 `T` 계열이 되는 것은 아니며, `V`나 `Y`와의 감별이 필요하다.
- `뒤에 있다`, `깊다`, `튀어나온다`, `멀리 있다` 같은 표현도 자동으로 `V` 계열이 되는 것은 아니며, 실제 강조점이 입체감인지 확인해야 한다.
- `연기`, `안개`, `그림자`, `흐림` 같은 표현도 자동으로 `Y` 계열이 되는 것은 아니며, 실제 강조점이 막연한 음영 인상인지 확인해야 한다.
- `반사되어 보인다`, `비친다` 같은 표현도 실제로 reflection 개념이 사용되었는지 확인해야 하며, 단순한 쌍반응과 섞지 않아야 한다.
- 해석 문서의 `Lambda`, `FM`, `m`, `FC:CF+C`, `Pure C`, `SumC'`, `SumT`, `SumV`, `SumY`, `Fr+rF`, `FD`는 coding 규칙과 분리해서 읽어야 한다.
