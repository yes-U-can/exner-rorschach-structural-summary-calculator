# [부호화/결정인] C'F

## 별칭/검색어

- C'F
- achromatic color-form
- 무채색-형태

## 핵심 정의

C'F는 무채색 특징이 먼저 반응을 이끌고, 형태는 2차적으로만 보조될 때 주는 무채색 색채 결정인이다. 반점의 검정, 회색, 흰색 특성이 없었다면 그 반응이 잘 성립하지 않는 경우가 많다.

## 채점/적용 조건

- 무채색 특징이 반응 성립의 핵심 단서여야 한다.
- 형태는 대상을 약하게 보조하거나 뒤늦게 정교화하는 수준이어야 한다.
- 반점의 무채색 특징이 사라지면 같은 반응을 유지하기 어려운지 질문 단계에서 확인한다.
- `연기`, `안개`처럼 모호한 대상은 실제로 무채색이 핵심인지, 아니면 음영 인상인지 더 확인한다.

## 주의사항/감별 기준

- `C'F`는 `FC'`보다 무채색 우세가 강하다.
- `C'F`는 `C'`와 달리 약한 형태 조직이 남아 있다.
- `CF`와 기호 구조는 비슷하지만, `CF`는 유채색이고 `C'F`는 무채색이다.
- `연기`, `안개`, `흐릿함`은 흔히 `Y`나 `V`와 겹쳐 보이므로 자동으로 `C'F`로 주지 않는다.
- 질문 단계에서 피검자가 깊이감이나 명암 차이를 강조하면 shading 계열을 먼저 검토한다.
