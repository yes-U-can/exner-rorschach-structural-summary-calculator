# 문서명: [부호화/형태질] FQ

## 별칭/검색어

- Form Quality
- 형태질
- FQ
- `+`, `o`, `u`, `-`, `none`

## 핵심 정의

FQ는 피검자가 지각한 대상이 반점의 실제 윤곽과 얼마나 잘 들어맞는지를 평가하는 부호다.  
즉, 반응의 조직 수준을 보는 DQ와 달리, FQ는 `형태의 합치도`를 본다.

## 채점/적용 조건

- 형태질은 결정인 부호의 끝부분에 붙여 기록한다.
- 기본 범주는 `+`, `o`, `u`, `-`, `none`이다.
- 채점의 출발점은 형태질 표와 기존 반응 목록의 기준이다.
- 핵심 판단 질문은 다음과 같다.
  - 이 대상이 반점 윤곽과 적절하게 맞는가
  - 그 적절성이 보통 수준인가, 비상투적이지만 허용 가능한가, 왜곡적인가
  - 애초에 형태를 근거로 채점할 수 있는 반응인가

## 주의사항/감별 기준

- FQ는 DQ와 다르다.
- DQ가 조직화 수준이라면, FQ는 형태 적합성이다.
- `FQ+`는 드물며, 단순히 내용이 복잡하거나 좋아 보인다고 부여하지 않는다.
- `FQu`와 `FQ-`의 핵심 차이는 `쉽고 빠르게 볼 수 있는가`와 `윤곽 사용이 적절한가`에 있다.
- `FQnone`은 형태가 나쁘다는 뜻이 아니라, 애초에 형태질을 줄 수 없는 반응이라는 뜻이다.
- 공란은 `FQnone`이 아니라 미완성 입력이다. 형태질을 줄 수 없는 반응이라면 `none`을 직접 선택한다.
