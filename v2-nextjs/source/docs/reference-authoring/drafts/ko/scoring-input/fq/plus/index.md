# 문서명: [부호화/형태질] +

## 별칭/검색어

- FQ+
- 형태질 +
- plus form quality

## 핵심 정의

FQ+는 반점 윤곽과 잘 맞는 보통 반응 가운데서도, 피검자가 통상적인 수준보다 더 많은 형태 세부를 정확하게 확인한 경우에 부여하는 형태질 코드다.  
즉, `잘 맞는다`를 넘어서 `특히 정교하게 잘 맞는다`는 뜻이다.

## 채점/적용 조건

- 반응은 기본적으로 반점 윤곽과 적절하게 들어맞아야 한다.
- 보통 반응 목록에 해당하거나, 그에 준하는 적절한 형태 사용이어야 한다.
- 피검자가 일반적인 수준보다 더 많은 형태 세부를 분명하게 확인해야 한다.
- 대부분의 적절한 반응은 `o`이므로, `+`는 드문 예외로 본다.

## 주의사항/감별 기준

- `+`는 형태질 코드이므로 `DQ+`와 다르다.
- 복잡하거나 인상적인 내용이라고 해서 자동으로 `FQ+`가 되지 않는다.
- 핵심은 `형태 세부의 정교한 확인`이지, 단순한 길이감이나 화려함이 아니다.
- 확신이 부족하면 무리하게 `+`로 올리지 말고 `o`를 먼저 검토한다.
