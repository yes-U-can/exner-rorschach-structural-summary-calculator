# 문서명: [부호화/형태질] -

## 별칭/검색어

- FQ-
- 형태질 -
- minus form quality

## 핵심 정의

FQ-는 반응이 반점 영역의 구조와 충분히 맞지 않거나, 왜곡되고 임의적이며 비현실적인 형태 사용을 보일 때 부여하는 형태질 코드다.  
즉, 지각된 대상과 반점 윤곽의 합치도가 낮다는 뜻이다.

## 채점/적용 조건

- 반응이 반점 윤곽과 적절하게 맞지 않는다.
- 형태 사용이 왜곡적이거나 임의적이다.
- 때로는 실제 윤곽이 없는 곳에서 자의적 윤곽을 만들어 쓰기도 한다.
- 표에 없는 반응을 검토할 때, 검사자가 쉽고 빠르게 볼 수 없다면 `-`를 검토한다.

## 주의사항/감별 기준

- `-`는 `u`보다 심하게 부적절한 형태 사용이다.
- 검사자가 애매하다고 느끼는 반응은 `-`를 더 보수적으로 검토하는 것이 원칙에 가깝다.
- 단, 반응 하나의 `-`만으로 과도한 해석을 해서는 안 된다.
- 중요한 것은 빈도, 군집, 특정 내용으로의 쏠림 여부다.
