# [부호화/형태질] none

## 별칭/검색어

- FQnone
- 형태질 none
- no form quality coding

## 핵심 정의

FQnone은 반응에 대해 형태질을 부여할 수 없을 때 쓰는 표기다.  
즉, 형태가 나쁘다는 뜻이 아니라 `형태를 근거로 채점하지 않는 반응`이라는 뜻이다.

## 채점/적용 조건

- 반응이 형태 윤곽을 근거로 조직되지 않는다.
- 구조적 요약에서는 어떤 형태질로도 부호화되지 않는 반응을 따로 집계한다.
- 순수 색채나 순수 음영처럼 형태 근거가 사실상 빠진 반응에서 검토한다.

## 주의사항/감별 기준

- `none`은 `-`와 다르다.
- `-`는 형태를 썼지만 부적절한 경우이고, `none`은 애초에 형태질을 줄 만한 형태 근거가 없는 경우다.
- 형태가 일부라도 실제로 반응 형성에 기여했다면 `none`보다 다른 FQ를 먼저 검토해야 한다.
