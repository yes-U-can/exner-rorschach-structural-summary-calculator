# [부호화] score

## 별칭/검색어

- score
- scoring
- 채점
- 부호화
- 반응 채점

## 핵심 정의

`score`는 로샤 반응을 숫자로 점수화하는 뜻이 아니라, 각 반응을 종합체계의 기호들로 바꾸어 기록하는 절차를 뜻한다.

## 채점/적용 조건

- 프로토콜 작성을 마친 뒤 각 반응을 카드 순서와 반응 번호 순서대로 부호화한다.
- 한 반응마다 위치, 발달질, 결정인, 형태질, 내용, 추가 coding 요소를 확인한다.
- 각 반응을 정확히 부호화해야 뒤에서 구조요약의 빈도, 비율, 백분율, 유도값을 계산할 수 있다.
- 핵심은 `반응 하나를 어떤 coding 조합으로 기록할 것인가`를 일관되게 정하는 것이다.

## 주의사항/감별 기준

- 로샤의 `score`는 다른 검사처럼 숫자만 주는 작업이 아니다.
- 확인되지 않은 것은 가정해서 부호화하지 않는다.
- inquiry에서 확인된 내용만 coding에 사용해야 한다.
- coding 단계와 interpretation 단계를 섞지 않는다. 지금 단계에서는 의미 해석보다 정확한 symbol 기록이 우선이다.
