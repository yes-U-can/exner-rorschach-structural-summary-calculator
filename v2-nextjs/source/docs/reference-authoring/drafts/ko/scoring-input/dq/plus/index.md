# 문서명: [부호화/발달질] +

## 별칭/검색어

- DQ+
- synthesized response (+)
- 발달질 +

## 핵심 정의

DQ+는 반점의 하나 이상의 부분이 정교하게 조직되어 하나의 응답으로 결합된 경우에 부여하는 발달질 코드다.  
둘 이상의 대상이 `개별적이지만 관련된` 방식으로 묘사되고, 그중 적어도 하나는 명확한 형태 요구를 가져야 한다.

## 채점/적용 조건

- 둘 이상의 대상이나 부분이 하나의 응답 안에서 의미 있게 연결되어야 한다.
- 각 대상은 서로 따로 떨어진 이름표가 아니라, 관계가 있는 것으로 기술되어야 한다.
- 결합된 대상 중 적어도 하나는 `specific form demand`를 가져야 한다.
- 예시 유형:
  - “담장 위에 앉은 두 마리 새”
  - 둘 이상의 구체적 대상이 상호작용하거나 배치 관계를 가지는 반응

## 주의사항/감별 기준

- 단지 대상이 두 개 보인다고 해서 자동으로 `+`가 되지는 않는다.
- “두 마리 새”처럼 관계가 없으면 `o`를 먼저 검토한다.
- 둘 이상의 대상을 결합했더라도 어느 대상도 뚜렷한 형태 요구가 없으면 `v/+`를 검토한다.
- `+`는 발달질 부호이므로, 형태가 매우 잘 맞는다는 뜻의 `FQ+`와 혼동하면 안 된다.
