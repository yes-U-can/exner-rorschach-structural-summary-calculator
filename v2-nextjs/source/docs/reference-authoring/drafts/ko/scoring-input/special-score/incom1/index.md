# 문서명: [부호화/특수점수] INCOM1

## 별칭/검색어

- INCOM1
- incongruous combination level 1
- 부당결합 수준 1
- 모순적 결합 수준 1

## 핵심 정의

`INCOM1`은 반점 영역이나 심상 요소가 하나의 대상 안에서 부적절하게 결합되지만, 그 기괴성이 비교적 경미한 수준 1 부당결합 특수점수다.

## 채점/적용 조건

- 서로 잘 어울리지 않는 신체 부위, 속성, 형태 요소가 하나의 대상 안에 묶이면 `INCOM1`을 검토한다.
- 색과 형태의 조합이 부적절하지만 수준 2만큼 현저하게 기괴하지 않으면 `INCOM1`을 준다.
- 핵심은 `하나의 그럴듯하지 않은 대상`이 만들어졌는가이다.
- 수준 1과 수준 2가 애매하면 보수적으로 `INCOM1`을 준다.

## 주의사항/감별 기준

- 둘 이상의 대상 사이에 받아들이기 어려운 관계를 만드는 경우는 `INCOM`보다 `FABCOM`을 먼저 본다.
- 한 반점 영역 안에서 두 인상이 오염적으로 겹쳐 하나의 반응이 되면 `CONTAM`과 구분해야 한다.
- `INCOM1`은 이상한 결합이 있지만, 수준 2처럼 분명히 기괴하다고 느껴질 정도는 아니다.
- 단순한 말실수나 느슨한 표현만으로는 자동으로 `INCOM`을 주지 않는다.
