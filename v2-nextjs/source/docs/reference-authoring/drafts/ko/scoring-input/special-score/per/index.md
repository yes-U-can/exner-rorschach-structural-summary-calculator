# 문서명: [부호화/특수점수] PER

## 별칭/검색어

- PER
- personalized answers
- 개인화 반응
- 자기관여

## 핵심 정의

`PER`는 피검자가 반응을 정당화하거나 더 그럴듯하게 만들기 위해 자신의 지식, 경험, 익숙함을 직접 끌어올 때 붙이는 특수점수다.

## 채점/적용 조건

- "나는 예전에 그런 걸 봤다", "우리는 학교에서 배웠다"처럼 개인 경험이나 익숙함을 근거로 반응을 뒷받침하면 `PER`를 준다.
- 일인칭 대명사가 없더라도, 사적인 경험이나 배경지식을 끌어와 반응을 정당화하면 `PER`를 검토한다.
- 핵심은 대상을 본 것 자체보다, 그 지식이나 경험이 반응의 근거로 사용되고 있는가이다.

## 주의사항/감별 기준

- "내가 보기엔", "내 생각엔"처럼 말버릇처럼 붙는 표현만으로는 `PER`가 아니다.
- `I`, `my`, `we` 같은 인칭 표현이 들어 있어도 반응 정당화 기능이 없으면 `PER`를 주지 않는다.
- 개인적 설명이 반응과 직접 연결되지 않고 단순한 잡담에 가깝다면 `PER`가 아니다.
- `PER`는 내용 범주가 아니라 정당화 방식에 붙는 특수점수이므로, 내용 코드와 별도로 판단한다.
