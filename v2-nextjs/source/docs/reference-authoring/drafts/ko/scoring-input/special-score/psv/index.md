# 문서명: [부호화/특수점수] PSV

## 별칭/검색어

- PSV
- perseveration
- 보속적 반응

## 핵심 정의

`PSV`는 이전 반응의 틀이나 대상을 반복해서 다시 쓰는 보속이 나타날 때 붙이는 특수점수다. 핵심은 단순한 수정이 아니라, 같은 반응이 경직되게 반복되는가이다.

## 채점/적용 조건

- 한 카드 안에서 연속된 두 반응이 위치, 발달질, 결정인, 형태질, 내용 범주까지 사실상 같은 틀을 반복하면 `PSV`를 준다.
- 다른 카드에서라도 이전에 본 대상을 같은 대상으로 다시 확인하면 내용 보속으로 `PSV`를 검토한다.
- 여러 카드에 걸쳐 거의 기계적으로 같은 대상을 반복하면 기계적 보속으로 `PSV`를 준다.
- 세 유형은 구분해서 설명할 수 있지만, 부호는 모두 `PSV` 하나로 적는다.

## 주의사항/감별 기준

- 단순히 더 적절한 대안을 고르는 수정 반응은 `PSV`가 아니다.
- Within-card 보속은 반응이 연속해서 나와야 하며, 특수점수를 제외한 기본 부호 틀이 사실상 같아야 한다.
- 기계적 보속은 매우 짧고 단순한 기록에서 자주 보이므로, 기록 전체의 타당성도 함께 점검해야 한다.
- `PSV`는 반복의 존재를 부호화하는 것이지, 그 원인을 해석 단계 없이 단정하는 코드는 아니다.
