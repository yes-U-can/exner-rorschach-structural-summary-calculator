# [부호화/특수점수] Special Scores

## 별칭/검색어

- Special Scores
- 특수점수
- 반응 특수 부호

## 핵심 정의

`Special Scores`는 위치, 발달질, 결정인, 형태질, 내용과는 별도로 반응의 특수한 성질을 표시하는 보조 부호군이다. 이 범주는 사고 특이성, 대인관계적 상호작용, 손상성, 자기참조성 같은 추가 단서를 coding 단계에서 붙일 때 사용한다.

## 채점/적용 조건

- 특수점수는 기본 coding을 대신하지 않고 추가로 붙는다.
- 반응 안에 해당 특징이 명확하게 드러날 때만 부여한다.
- 이번 개요 문서는 현재 우선 작성이 끝난 `DV1`, `DV2`, `DR1`, `DR2`, `INCOM1`, `INCOM2`, `FABCOM1`, `FABCOM2`, `ALOG`, `CONTAM`, `COP`, `AG`, `MOR`, `AB`, `PER`, `CP`, `PSV`를 중심으로 정리한다.
- `COP`, `AG`는 모두 운동 반응(`M`, `FM`, `m`)과 함께 검토한다.
- 특수점수는 나중 해석과 연결되지만, coding 단계에서는 먼저 정의와 적용 조건을 정확히 고정해야 한다.

## 주의사항/감별 기준

- 특수점수는 한 반응의 분위기를 느슨하게 요약하는 라벨이 아니라, 명시적 기준을 만족할 때만 주는 코드다.
- movement가 있다고 해서 자동으로 `COP`나 `AG`가 붙지 않는다.
- 같은 반응에 둘 이상의 특수점수가 함께 붙을 수 있지만, 각각의 조건을 따로 확인해야 한다.
- coding과 interpretation을 섞지 않는다. `특수점수가 많다`는 사실 자체는 해석 단계에서 다시 검토한다.
