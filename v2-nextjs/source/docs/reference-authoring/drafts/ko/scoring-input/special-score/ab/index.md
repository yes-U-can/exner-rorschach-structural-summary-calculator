# 문서명: [부호화/특수점수] AB

## 별칭/검색어

- AB
- abstractions
- 추상화 반응

## 핵심 정의

`AB`는 반응이 구체적인 사물을 단순히 지각하는 수준을 넘어, 인간의 정서·감각 경험이나 분명한 상징적 의미를 표현할 때 붙이는 특수점수다.

## 채점/적용 조건

- `Hx` 내용으로 인간의 감정이나 감각 경험 자체를 말할 때 `AB`를 검토한다.
- 대상이 무엇을 상징하거나 대표한다고 분명하게 진술하면 `AB`를 준다.
- 핵심은 추상적인 느낌이 아니라, 구체적인 상징적 표상이 분명하게 언어화되었는가이다.
- `AB`는 내용 코드와 별도로 추가되는 특수점수다.

## 주의사항/감별 기준

- 막연하게 추상적이거나 분위기만 말한 반응은 `AB`가 아니다.
- 추상화된 그림이나 예술 작품을 말하더라도, 무엇을 상징하는지 분명하지 않으면 `AB`를 주지 않는다.
- `Hx`가 있다고 해서 모두 `AB`는 아니다. 인간 경험 내용이 추상적 표상으로 쓰였는지 확인해야 한다.
- `Art`, `Ay`와 함께 나타날 수 있지만, 그 코드들 자체가 `AB`를 자동으로 만들지는 않는다.
