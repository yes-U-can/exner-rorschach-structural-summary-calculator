# 문서명: [부호화/특수점수] INCOM2

## 별칭/검색어

- INCOM2
- incongruous combination level 2
- 부당결합 수준 2
- 모순적 결합 수준 2

## 핵심 정의

`INCOM2`는 하나의 대상 안에 묶인 요소들의 결합이 매우 부적절하고 분명히 기괴해서, 수준 2로 평가되는 부당결합 특수점수다.

## 채점/적용 조건

- 서로 어울릴 수 없는 부위나 속성이 한 대상 안에서 매우 기괴하게 결합되면 `INCOM2`를 준다.
- 반응 전체가 수준 1보다 훨씬 더 부자연스럽고 뚜렷하게 이상하게 들리면 `INCOM2`를 검토한다.
- 색과 형태의 조합이 현저하게 부적절하고 기괴하면 `INCOM2` 가능성을 본다.

## 주의사항/감별 기준

- 수준 2인지 확신이 없으면 `INCOM2`가 아니라 `INCOM1`을 준다.
- `INCOM2`도 여전히 `하나의 대상` 내부 결합 문제다. 둘 이상의 대상 사이 관계 이상은 `FABCOM2`와 구분해야 한다.
- 단일 영역 안에서 두 인상이 오염적으로 합쳐지면 `CONTAM`과 구분해야 한다.
- `INCOM2`는 Level 2 special score이므로 이후 `Lv2`, `WSum6` 문맥과 연결되지만, coding 단계에서는 먼저 반응의 기괴성을 본다.
