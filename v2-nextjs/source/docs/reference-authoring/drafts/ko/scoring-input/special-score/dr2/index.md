# 문서명: [부호화/특수점수] DR2

## 별칭/검색어

- DR2
- deviant response level 2
- 일탈적 반응 수준 2

## 핵심 정의

`DR2`는 일탈적 반응 가운데 판단의 왜곡이나 표현의 기괴성이 뚜렷해서, 사고의 이완이나 비논리성이 수준 2로 드러나는 특수점수다.

## 채점/적용 조건

- 부적절한 어구나 우회적 반응이 현저하게 기괴하고, 반응의 판단이나 표현 방식이 뚜렷하게 비정상적으로 벗어나면 `DR2`를 준다.
- 수준 2는 단순히 길거나 복잡한 반응이 아니라, 반응 전체가 분명히 이상하게 느껴질 정도의 일탈성을 보여야 한다.
- `DV` 성격이 함께 들어 있어도 전체가 `DR` 기준을 충족하면 `DR`만 기록한다.
- `DR2`는 반응의 실질적 내용과 언어 표현만 보고 판단한다.

## 주의사항/감별 기준

- 수준 2인지 확신이 없으면 `DR2`가 아니라 `DR1`을 준다.
- 잘 조직된 장문 반응이나 교육 수준이 높아 보이는 표현을 `DR2`로 과잉 채점하면 안 된다.
- `DR2`는 `DV2`보다 더 넓은 반응 단위에서 우회성, 비논리성, 일탈된 판단이 드러난다.
- `DR2`라고 해서 자동으로 별도 해석을 붙이지 않고, 이후 `Sum6`, `Lv2`, `WSum6` 문맥에서 다시 본다.
