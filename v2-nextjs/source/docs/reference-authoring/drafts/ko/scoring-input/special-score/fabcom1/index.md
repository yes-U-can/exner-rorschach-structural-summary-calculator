# 문서명: [부호화/특수점수] FABCOM1

## 별칭/검색어

- FABCOM1
- fabulized combination level 1
- 우화적 결합 수준 1

## 핵심 정의

`FABCOM1`은 반점에서 보이는 둘 이상의 대상 사이에 받아들이기 어려운 관계를 만들지만, 그 기괴성이 수준 2만큼 현저하지는 않은 수준 1 우화적 결합 특수점수다.

## 채점/적용 조건

- 반응에 독립된 둘 이상의 대상이 들어 있고, 그들 사이 관계가 비현실적이거나 부적절하면 `FABCOM1`을 검토한다.
- 핵심은 `한 대상 내부의 이상한 결합`이 아니라 `둘 이상의 대상 사이의 이상한 관계`다.
- 관계가 이상하지만 수준 2처럼 명백하게 기괴하지 않으면 `FABCOM1`을 준다.
- 두 대상이 상호작용하거나 연결되는 방식이 과제 맥락에서 무리하게 느껴질 때 사용한다.

## 주의사항/감별 기준

- 하나의 대상 내부에서 부위나 속성이 부적절하게 결합된 경우는 `FABCOM`보다 `INCOM`을 먼저 본다.
- 단일 영역에서 두 인상이 오염적으로 겹쳐 하나의 반응이 되면 `CONTAM`과 구분해야 한다.
- 수준 2인지 확신이 없으면 `FABCOM2`가 아니라 `FABCOM1`을 준다.
- 단순히 상상력이 풍부한 서술이라고 해서 자동으로 `FABCOM`을 주지 않는다.
