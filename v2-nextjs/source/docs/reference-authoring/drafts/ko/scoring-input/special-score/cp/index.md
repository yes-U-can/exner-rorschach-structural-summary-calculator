# [부호화/특수점수] CP

## 별칭/검색어

- CP
- color projection
- 색채투사

## 핵심 정의

`CP`는 무채색 반점이나 무채색 반점 영역을 유채색으로 지각했을 때 붙이는 특수점수다. 핵심은 실제 색이 없는 곳에 색이 있다고 반응했는가이다.

## 채점/적용 조건

- 피검자가 무채색 영역을 빨강, 보라, 파랑처럼 유채색으로 확인하면 `CP`를 준다.
- 원반응에서 색채 표현이 흐릿하면 질문 단계에서 어떤 색을 본 것인지 다시 확인한다.
- `CP`는 드문 반응이며, 결정인과 별도로 추가되는 특수점수다.
- 많은 경우 확산 음영 계열 결정인과 함께 검토된다.

## 주의사항/감별 기준

- 실제 유채색 영역을 본 반응은 `CP`가 아니다. 그런 경우는 색채 결정인 `FC`, `CF`, `C`를 검토한다.
- `CP`가 붙는 반응에는 `FC`, `CF`, `C`를 함께 주지 않는다. 반점 자체가 무채색이기 때문이다.
- 질문 단계에서 말실수로 정정되면 `CP`를 주지 않는다.
- 색채시 이상이 의심되면 별도 확인이 필요하며, 색채시가 정상인데도 무채색을 유채색으로 고집하면 `DV`를 검토한다.
