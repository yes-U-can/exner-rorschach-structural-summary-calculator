# 문서명: [부호화/특수점수] FABCOM2

## 별칭/검색어

- FABCOM2
- fabulized combination level 2
- 우화적 결합 수준 2

## 핵심 정의

`FABCOM2`는 둘 이상의 대상 사이에 맺어진 관계가 매우 비현실적이고 분명히 기괴해서, 수준 2로 평가되는 우화적 결합 특수점수다.

## 채점/적용 조건

- 둘 이상의 대상이 포함된 반응에서 그 관계가 현저하게 비정상적이고 기괴하면 `FABCOM2`를 준다.
- 불가능한 투시적 응답처럼 현실 검증이 크게 흔들린 관계 설정도 `FABCOM2`를 검토한다.
- 반응 전체가 수준 1보다 훨씬 더 이상하고 뚜렷하게 기괴하게 들릴 때 수준 2 가능성을 본다.

## 주의사항/감별 기준

- 수준 2인지 확신이 없으면 `FABCOM2`가 아니라 `FABCOM1`을 준다.
- 하나의 대상 내부 결합 문제는 `FABCOM`보다 `INCOM2`와 구분해야 한다.
- 단일 영역에서 두 인상이 오염적으로 융합되면 `CONTAM`과 구분해야 한다.
- `FABCOM2`는 Level 2 special score이므로 이후 `Lv2`, `WSum6`와 연결되지만, coding 단계에서는 먼저 반응 자체의 기괴성을 본다.
