# 문서명: [부호화/특수점수] ALOG

## 별칭/검색어

- ALOG
- inappropriate logic
- 부당추론
- 부적절한 논리

## 핵심 정의

`ALOG`는 피검자가 자신의 반응을 정당화하기 위해 충분한 단서 없이 부자연스럽고 느슨한 추론을 자발적으로 사용할 때 붙이는 특수점수다.

## 채점/적용 조건

- 반응을 설명하거나 방어하는 과정에서 근거 없는 논리적 비약이 드러나면 `ALOG`를 검토한다.
- 반점의 크기나 공간적 요소를 근거로 삼아 부자연스러운 추론을 할 때 `ALOG`가 흔히 나타난다.
- 핵심은 `무엇을 봤는가`보다 `왜 그것이라고 주장하는가`의 논리 손상이다.
- 피검자가 자발적으로 그 논리를 말해야 한다.

## 주의사항/감별 기준

- 질문단계에서 검사자의 추가 질문 때문에 나온 비논리적 설명은 `ALOG`로 채점하지 않는다.
- `ALOG`는 이상한 명명이나 결합 자체보다, 반응을 정당화하는 논리의 손상에 초점이 있다.
- 방어적이거나 왜곡된 설명처럼 보여도, 더 적절한 다른 코드가 있으면 그쪽을 먼저 검토한다.
- `ALOG`는 Level 2 special score는 아니지만, 이후 `Sum6`, `WSum6` 문맥과 연결된다.
