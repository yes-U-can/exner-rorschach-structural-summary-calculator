# 문서명: [부호화/특수점수] CONTAM

## 별칭/검색어

- CONTAM
- contamination
- 오염 반응

## 핵심 정의

`CONTAM`은 두 개 이상의 인상이 현실성을 무시한 채 하나의 단일 반응으로 오염적으로 융합될 때 붙이는 특수점수다. `INCOM`이나 `FABCOM`보다 더 기괴하고 부적절한 결합으로 본다.

## 채점/적용 조건

- 하나의 반점 영역에서 나온 두 인상이 서로 겹쳐져 단일한 반응으로 합쳐지면 `CONTAM`을 검토한다.
- 각각 따로 말하면 성립할 수 있는 두 인상이, 융합 과정에서 서로의 적절성을 손상시키면 `CONTAM` 가능성이 높다.
- 핵심은 `두 인상의 오염적 융합`이지, 단순히 이상한 조합 자체가 아니다.
- 신조어나 매우 독특한 언어를 통해 오염된 대상을 기술하는 경우가 함께 나타날 수 있다.

## 주의사항/감별 기준

- `INCOM`은 여러 요소를 하나의 그럴듯하지 않은 대상으로 묶는 것이고, `CONTAM`은 한 영역에서 두 인상이 겹쳐 하나의 반응이 되는 것이다.
- 둘 이상의 독립된 대상 사이에 이상한 관계를 만드는 경우는 `FABCOM`과 구분해야 한다.
- `CONTAM`은 매우 심한 기괴성의 신호지만, coding 단계에서는 먼저 반응 형식 자체가 오염적 융합인지 확인해야 한다.
- `CONTAM`은 이후 `Sum6`, `WSum6` 문맥에서 다시 검토한다.
