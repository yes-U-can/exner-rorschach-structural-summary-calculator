# 문서명: [부호화/특수점수] MOR

## 별칭/검색어

- MOR
- morbid content
- 손상반응
- 병적 내용

## 핵심 정의

`MOR`는 대상이 죽었거나, 파괴되었거나, 손상되었거나, 망가진 상태로 확인되거나, 우울하고 침울한 성질이 대상에 직접 부여될 때 붙이는 특수점수다.

## 채점/적용 조건

- 대상이 죽은 것, 부서진 것, 찢어진 것, 불탄 것, 썩은 것, 상처 입은 것, 망가진 것으로 분명하게 확인되면 `MOR`를 준다.
- 사람, 동물, 식물, 사물처럼 내용 종류와 상관없이 손상 또는 파괴 상태가 반응의 핵심이면 `MOR`를 검토한다.
- "슬픈 나무", "침울한 집"처럼 우울하거나 황폐한 성질이 대상에 직접 붙으면 `MOR`를 준다.
- `MOR`는 위치, 결정인, 내용 코드를 대신하지 않고 추가로 붙는 특수점수다.

## 주의사항/감별 기준

- 무섭다, 불쾌하다, 이상하다는 느낌만으로는 `MOR`가 아니다. 대상의 손상성이나 병적 성질이 명시되어야 한다.
- 공격 행동이 일어나는 장면은 `AG`를 먼저 검토하고, 그 결과 대상이 실제로 손상되거나 죽은 것으로 제시될 때 `MOR`를 함께 검토한다.
- 해부 내용 `An`이나 인간 세부 `Hd`가 있다고 해서 자동으로 `MOR`가 되지 않는다.
- 인간 반응에서 `MOR`가 붙으면 `PHR` 가능성이 높아질 수 있지만, 그것만으로 자동 결정하지 않는다.
