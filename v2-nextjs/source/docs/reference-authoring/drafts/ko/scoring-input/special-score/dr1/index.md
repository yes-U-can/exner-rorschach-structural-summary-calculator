# 문서명: [부호화/특수점수] DR1

## 별칭/검색어

- DR1
- deviant response level 1
- 일탈적 반응 수준 1

## 핵심 정의

`DR1`은 `DV`보다 더 많은 요소를 포함한 일탈적 반응 가운데, 부적절한 어구나 우회적 전개가 보이지만 기괴성이 상대적으로 약한 수준 1 특수점수다.

## 채점/적용 조건

- 반응 안에 과제와 직접 관련되지 않는 부적절한 어구가 섞여 있으면 `DR1`을 검토한다.
- 대상을 말하는 과정이 지나치게 우회적이어서 초점이 흐려지지만, 수준 2만큼 현저하게 기괴하지 않으면 `DR1`을 준다.
- `DR`은 `DV`보다 더 많은 요소를 포함한 반응에 붙인다. 짧고 단순한 이상 표현만 있으면 먼저 `DV` 가능성을 본다.
- 같은 반응에 `DV` 성격이 함께 보여도 전체가 `DR` 기준을 충족하면 `DR`만 기록한다.
- 수준 1과 수준 2가 애매하면 보수적으로 `DR1`을 준다.

## 주의사항/감별 기준

- 적절하게 정교화된 반응은 길어도 `DR`이 아니다. 핵심은 장황함 자체가 아니라 과제에서 벗어나는지이다.
- `DR1`은 `DV1`보다 반응 구조가 더 길고 복잡하다.
- `DR1`과 `DR2`의 구분은 반응의 내용과 표현 방식만 보고 판단한다.
- 나이, 교육 수준, 문화적 배경은 수준 1과 2를 나누는 직접 기준이 아니다.
