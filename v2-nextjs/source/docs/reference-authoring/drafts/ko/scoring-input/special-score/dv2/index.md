# 문서명: [부호화/특수점수] DV2

## 별칭/검색어

- DV2
- deviant verbalization level 2
- 일탈언어반응 수준2
- 일탈표현 수준2

## 핵심 정의

`DV2`는 짧은 언어 수준의 일탈이 뚜렷하고 현저하게 기괴할 때 붙이는 수준 2 특수점수다. 핵심은 표현의 이상함이 분명히 눈에 띄고, 수준 1보다 훨씬 두드러진다는 점이다.

## 채점/적용 조건

- 신조어나 동어반복이 매우 기괴하고 현저하면 `DV2`를 준다.
- 반응이 짧은 편이지만 표현 자체가 중등도 이상으로 해리적이거나 이상하게 들리면 `DV2`를 검토한다.
- 수준 2는 보통 점수자가 바로 눈치챌 만큼 두드러지는 반응에 준다.

## 주의사항/감별 기준

- 수준 2인지 확실하지 않으면 `DV2` 대신 `DV1`을 준다.
- 반응이 짧지 않고 여러 요소가 섞여 이상한 경우는 `DR2`를 먼저 검토한다.
- `DV2`는 Level 2 special score에 포함되므로 나중 해석에서 `Lv2`, `WSum6`와 연결되지만, coding 단계에서는 먼저 반응 자체의 기괴성을 본다.
- 나이, 교육 수준, 문화 배경은 수준 2 판정의 직접 기준으로 쓰지 않는다.
