# 문서명: [부호화/특수점수] DV1

## 별칭/검색어

- DV1
- deviant verbalization level 1
- 일탈언어반응 수준1
- 일탈표현 수준1

## 핵심 정의

`DV1`은 반응의 표현 방식에 짧지만 분명한 일탈이 있을 때 붙이는 수준 1 특수점수다. 핵심은 반응이 짧은 언어 수준에서 약하게 이상하거나 부적절하다는 점이다.

## 채점/적용 조건

- 반응이 비교적 짧은 형태인데, 신조어나 부적절한 말 만들기, 어색한 중복 표현이 드러나면 `DV1`을 검토한다.
- 동어반복이나 어법에 맞지 않는 중복이 있지만, 기괴성이 심하지 않으면 `DV1`을 준다.
- 수준 1과 2를 나눌 때는 반응 자체의 기괴성과 표현 강도만 본다.

## 주의사항/감별 기준

- 반응이 더 길고 산만하며 여러 요소를 포함하는 이상한 응답이면 `DV`보다 `DR`를 먼저 검토한다.
- 같은 반응 안에 `DV` 성격이 들어 있어도 전체가 `DR` 기준을 충족하면 `DR`만 기록한다.
- 수준 2인지 확신이 없으면 보수적으로 `DV1`을 준다.
- 나이, 교육 수준, 문화 배경은 수준 1과 2를 나누는 직접 기준으로 쓰지 않는다.
