# 문서명: [부호화/형태질] o

## 별칭/검색어

- FQo
- 형태질 o
- ordinary form quality

## 핵심 정의

FQo는 반점 윤곽과 적절하게 들어맞는 보통 수준의 형태질 코드다.  
형태질 채점에서 가장 흔한 기본값에 가깝다.

## 채점/적용 조건

- 대상이 반점 윤곽과 적절하게 맞아야 한다.
- 보통 반응 목록에 해당하거나, 그와 동등한 수준의 적절한 윤곽 사용이어야 한다.
- W나 D 영역에서는 비교적 널리 보고되는 반응이 여기에 해당한다.
- Dd 영역에서도 기준을 만족하면 `o`가 가능하다.

## 주의사항/감별 기준

- `o`는 `+`보다 덜 정교하지만 여전히 적절한 형태 사용이다.
- `o`는 `u`보다 더 관습적이고 평범하다.
- `o`는 `DQo`와 다르다. 하나는 형태질, 다른 하나는 발달질이다.
- 여러 대상이 섞인 반응에서는 중요도가 더 낮은 세부 때문에 전체 FQ를 과도하게 낮추지 않도록 주의한다.

## 상호 참조

- `[부호화/형태질] +`
- `[부호화/형태질] u`
- `[부호화/형태질] -`
- `[부호화/발달질] o`

## 근거 메모

- 세부 근거 대조와 용어 판단 기록은 내부 작업 메모에서 관리한다.
- 공개 참조 문서 반영 단계에서는 내부 provenance가 노출되지 않도록 분리한다.
