# Nome do documento: [Codificação/Códigos Especiais] CONTAM

## Apelidos e busca

- CONTAM
- contaminação
- contaminação perceptiva

## Definição central

`CONTAM` é usado quando duas percepções incompatíveis se fundem numa única resposta contaminada.
Ele não descreve apenas uma mistura estranha, mas uma integração impossível num único percepto final.

## Condições de aplicação

- Use `CONTAM` quando uma resposta juntar duas percepções de forma contaminada, como se fossem uma percepção única.
- O ponto principal é que o produto final e apresentado como um único percepto fundido, e não como duas ideias apenas comparadas.
- A resposta deve soar impossível ou internamente contraditória por causa da fusão.
- `CONTAM` representa forma particularmente severa de distorção e depois pesa em `Sum6` e `WSum6`.

## Cuidados e diferenciação

- Se o problema principal for combinação incongruente dentro da resposta, mas sem fusão contaminada, revise `INCOM2`.
- Se a principal estranheza for uma relação fabulizada entre elementos, revise `FABCOM2`.
- `CONTAM` não é usado só porque a resposta é muito imaginativa ou simbólica.
