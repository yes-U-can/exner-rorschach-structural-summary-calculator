# Nome do documento: [Codificação/Escores Especiais] AB

## Apelidos e busca

- AB
- abstrações
- resposta abstrata

## Definição central

`AB` é usado quando a resposta se afasta de um objeto concreto e passa para formulação abstrata, generalizada ou frouxamente conceitual.
O ponto principal é que a pessoa nomeia uma abstração ou qualidade ampla, em vez de um objeto claramente delimitado.

## Condições de aplicação

- Considere `AB` quando a resposta nomeia algo como emoção, ideia, força, sensação ou outra formulação abstrata.
- Ele pode aparecer perto de [`Hx`](ref://scoring-input/contents/Hx) quando a resposta aponta para experiência humana generalizada, e não para pessoa específica.
- Também pode coexistir com [`Art`](ref://scoring-input/contents/Art) ou [`Ay`](ref://scoring-input/contents/Ay) quando o conteúdo vira categoria altamente abstrata.
- Esse código é acrescentado como escore especial e não substitui o conteúdo principal.

## Cuidados e diferenciação

- Nem toda resposta simbólica ou artística recebe `AB`.
- `Hx` sozinho não exige `AB`; a formulação precisa ser realmente abstrata.
- Uma resposta incomum, mas ainda concreta, não deve receber `AB` só porque parece sofisticada.
