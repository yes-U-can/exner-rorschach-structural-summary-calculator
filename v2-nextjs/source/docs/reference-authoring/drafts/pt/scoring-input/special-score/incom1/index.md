# Nome do documento: [Codificação/Códigos Especiais] INCOM1

## Apelidos e busca

- INCOM1
- combinação incongruente nível 1
- incongruência leve

## Definição central

`INCOM1` é um código especial de nível 1 para combinações incongruentes dentro da mesma resposta.
Ele é usado quando os elementos unidos não combinam bem, mas a estranheza ainda não chega ao nível mais severo de `INCOM2`.

## Condições de aplicação

- Considere `INCOM1` quando duas partes, qualidades ou funções forem combinadas de modo incongruente dentro do mesmo objeto ou cena.
- A combinação precisa soar imprópria ou mal encaixada, e não apenas incomum.
- Se a resposta ainda puder ser seguida com alguma lógica, `INCOM1` costuma ser mais apropriado do que `INCOM2`.
- Quando a estranheza ficar restrita a palavra isolada ou frase verbal esquisita, revise antes `DV1` ou `DV2`.

## Cuidados e diferenciação

- `INCOM1` não é dado só porque a resposta é criativa, fantástica ou incomum.
- Se o problema principal for uma relação impossível entre partes da resposta, com organização mais distorcida, revise `FABCOM1`.
- Se duas realidades incompatíveis forem fundidas numa única percepção contaminada, revise `CONTAM`.
- Se houver dúvida entre nível 1 e nível 2, use `INCOM1` de modo conservador e reserve `INCOM2` para a incongruência mais severa.
