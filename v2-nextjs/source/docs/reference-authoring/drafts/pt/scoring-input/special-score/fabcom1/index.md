# Nome do documento: [Codificação/Códigos Especiais] FABCOM1

## Apelidos e busca

- FABCOM1
- combinação fabulizada nível 1
- fabulizacao leve

## Definição central

`FABCOM1` é um código especial de nível 1 para relações fabulizadas ou implausíveis entre elementos da resposta.
Ele não descreve apenas uma mistura incongruente dentro de um objeto, mas uma relação distorcida entre ideias, ações ou partes da cena.

## Condições de aplicação

- Use `FABCOM1` quando a resposta organizar uma relação improvável ou forçada entre elementos que deveriam conectar-se de outro modo.
- O problema central está na lógica fabulizada do arranjo como um todo, e não apenas numa mistura incongruente dentro de um único objeto.
- A estranheza precisa ser observável, mas ainda não tão severa a ponto de exigir `FABCOM2`.
- Se houver dúvida entre nível 1 e nível 2, reserve `FABCOM2` para a distorção mais extrema.

## Cuidados e diferenciação

- Se a estranheza principal estiver numa combinação incongruente dentro de uma única unidade perceptiva, revise `INCOM1`.
- Se duas percepções incompatíveis forem fundidas numa única percepção contaminada, revise `CONTAM`.
- `FABCOM1` não é usado só porque a resposta parece fantástica ou imaginativa.
