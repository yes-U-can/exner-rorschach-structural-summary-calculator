# Nome do documento: [Codificacao/Escores Especiais] FABCOM1

## Apelidos e busca

- FABCOM1
- combinacao fabulizada nivel 1
- fabulizacao leve

## Definicao central

`FABCOM1` e um escore especial de nivel 1 para relacoes fabulizadas ou implausiveis entre elementos da resposta.
Ele nao descreve apenas uma mistura incongruente dentro de um objeto, mas uma relacao distorcida entre ideias, acoes ou partes da cena.

## Condicoes de aplicacao

- Use `FABCOM1` quando a resposta organizar uma relacao improvavel ou forcada entre elementos que deveriam conectar-se de outro modo.
- O problema central esta na logica fabulizada do arranjo como um todo, e nao apenas numa mistura incongruente dentro de um unico objeto.
- A estranheza precisa ser observavel, mas ainda nao tao severa a ponto de exigir `FABCOM2`.
- Se houver duvida entre nivel 1 e nivel 2, reserve `FABCOM2` para a distorcao mais extrema.

## Cuidados e diferenciacao

- Se a estranheza principal estiver numa combinacao incongruente dentro de uma unica unidade perceptiva, revise `INCOM1`.
- Se duas percepcoes incompativeis forem fundidas numa unica percepcao contaminada, revise `CONTAM`.
- `FABCOM1` nao e usado so porque a resposta parece fantastica ou imaginativa.
