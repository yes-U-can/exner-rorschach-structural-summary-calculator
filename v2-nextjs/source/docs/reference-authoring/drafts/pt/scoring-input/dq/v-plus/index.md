# Nome do documento: [Codificação/Qualidade Desenvolvimental] v/+

## Apelidos e busca

- v/+
- DQv/+
- síntese sem demanda formal

## Definição central

`DQv/+` é usado quando a resposta mostra síntese, mas os objetos envolvidos não exigem demanda formal específica.
A resposta é organizada, embora os objetos permaneçam relativamente vagos.

## Condições de aplicação

- Mais de um objeto ou parte e combinado numa resposta organizada.
- Os objetos estão ligados de modo significativo.
- Nenhum objeto envolvido exige claramente demanda formal específica.
- A resposta é mais do que uma simples listagem ou repetição.

## Cuidados e diferenciação

- Se ao menos um objeto exige demanda formal específica, use [`DQ+`](ref://scoring-input/dq/%2B).
- Se não há síntese significativa, mas o objeto exige demanda formal específica, use [`DQo`](ref://scoring-input/dq/o).
- Se não há síntese significativa e a resposta permanece vaga, use [`DQv`](ref://scoring-input/dq/v).
