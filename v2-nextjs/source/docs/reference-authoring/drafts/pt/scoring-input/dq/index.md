# Nome do documento: [Codificação/Qualidade Desenvolvimental] DQ

## Apelidos e busca

- DQ
- Qualidade Desenvolvimental
- codificação DQ

## Definição central

Qualidade Desenvolvimental (`DQ`) codifica quão organizada ou elaborada a resposta está como percepto.
O foco é a estrutura da resposta, não se o ajuste formal e bom ou ruim.

## Condições de aplicação

- Escolha um código DQ para cada resposta pontuável.
- Decida se existe síntese significativa.
- Decida se pelo menos um objeto exige demanda formal específica.
- Use o conjunto de quatro códigos: `+`, `v/+`, `o`, `v`.

## Cuidados e diferenciação

- `DQ` não é o mesmo que [`FQ`](ref://scoring-input/fq). `DQ` trata da organização; `FQ` trata do ajuste formal.
- O mesmo símbolo pode ter outro sentido em outra família. [`DQ+`](ref://scoring-input/dq/%2B) não é o mesmo construto que [`FQ+`](ref://scoring-input/fq/%2B).
- Atribua `DQ` a partir da estrutura da resposta, não a partir da interpretação posterior.
