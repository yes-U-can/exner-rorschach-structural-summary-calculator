# Nome do documento: [Codificação/Conteúdo] (H)

## Apelidos e busca

- (H)
- humano imaginário
- figura humana mítica

## Definição central

`(H)` é codificado quando o percepto é uma figura humana inteira de ficção, fantasia ou mitologia.
Continua sendo figura humana inteira, mas não uma pessoa comum e realista.

## Condições de aplicação

- O objeto principal precisa ser uma figura humana inteira.
- A figura deve ser fantástica, mítica, ficcional ou claramente irreal.
- Exemplos típicos incluem anjos, demónios, gigantes, fantasmas ou humanoides de ficção.
- Se apenas parte dessa figura for descrita, reconsidere [`(Hd)`](ref://scoring-input/contents/%28Hd%29).

## Cuidados e diferenciação

- `(H)` não é o mesmo que [`H`](ref://scoring-input/contents/H); `H` é reservado a figuras humanas inteiras realistas.
- `(H)` não é o mesmo que [`(Hd)`](ref://scoring-input/contents/%28Hd%29); `(Hd)` é usado para detalhe humano imaginário.
- Máscara, roupa ou estilo artístico não mudam o código automaticamente se a pessoa ainda estiver descrevendo uma pessoa realista inteira.
- A decisão sobre [`GHR`](ref://scoring-input/gphr/GHR) ou [`PHR`](ref://scoring-input/gphr/PHR) continua sendo etapa separada.
