# Nome do documento: [Codificação/Conteúdo] Hd

## Apelidos e busca

- Hd
- detalhe humano
- parte humana

## Definição central

`Hd` é codificado quando o percepto é um detalhe humano incompleto, não uma pessoa inteira.

## Condições de aplicação

- A resposta precisa apontar para parte do corpo humano ou detalhe humano incompleto.
- O percepto não deve ser uma pessoa inteira realista.
- Se a resposta indicar pessoa inteira, reconsidere [`H`](ref://scoring-input/contents/H).
- Se o detalhe for fantástico ou mítico, reconsidere [`(Hd)`](ref://scoring-input/contents/%28Hd%29).

## Cuidados e diferenciação

- `Hd` não é o mesmo que [`H`](ref://scoring-input/contents/H); `H` exige figura humana inteira.
- `Hd` não é o mesmo que [`Hx`](ref://scoring-input/contents/Hx); `Hx` codifica experiência humana e não parte corporal.
- Não basta o percepto parecer "humano"; ele precisa ser detalhe ou parte humana.
