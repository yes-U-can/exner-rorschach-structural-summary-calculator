# Nome do documento: [Codificação/Conteúdo] (Hd)

## Apelidos e busca

- (Hd)
- detalhe humano imaginário
- parte humana mítica

## Definição central

`(Hd)` é codificado quando o percepto é um detalhe humano incompleto, mas fantástico, ficcional ou mítico.

## Condições de aplicação

- A resposta precisa apontar para parte ou detalhe humano incompleto.
- Esse detalhe deve ser claramente fantástico, mítico ou irreal.
- Se a figura for humana inteira e imaginária, reconsidere [`(H)`](ref://scoring-input/contents/%28H%29).
- Se o detalhe for humano realista, reconsidere [`Hd`](ref://scoring-input/contents/Hd).

## Cuidados e diferenciação

- `(Hd)` não é o mesmo que [`Hd`](ref://scoring-input/contents/Hd); a diferença central é o caráter imaginário.
- `(Hd)` não é o mesmo que [`Hx`](ref://scoring-input/contents/Hx); `Hx` não codifica parte corporal.
- Um detalhe artisticamente estilizado não se torna automaticamente `(Hd)` se continuar sendo detalhe humano realista.
