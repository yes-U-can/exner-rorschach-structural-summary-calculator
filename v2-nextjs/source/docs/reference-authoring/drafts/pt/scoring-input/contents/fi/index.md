# Nome do documento: [Codificacao/Conteudo] Fi

## Apelidos e busca

- Fi
- fogo
- fire
- fumaca

## Definicao central

`Fi` e usado quando o conteudo principal e fogo, chama, brasa ou fumaca.
O codigo recai no material ardente ou fumegante em si.

## Condicoes de aplicacao

- Use para fogo, incendio, chama, brasa, fumaca ou forma organizada como isso.
- A resposta precisa tratar a figura como fogo ou fumaca, nao apenas como evento explosivo.
- Se a resposta apontar para explosao, impacto ou evento, revise primeiro [`Ex`](ref://scoring-input/contents/Ex).
- Se a forma for tratada como nuvem, revise primeiro [`Cl`](ref://scoring-input/contents/Cl).

## Cuidados e diferenciacao

- `Fi` e [`Ex`](ref://scoring-input/contents/Ex) separam fogo/fumaca de evento explosivo.
- `Fi` e [`Cl`](ref://scoring-input/contents/Cl) separam fogo/fumaca de nuvem.
- Cor quente sozinha nao basta sem conteudo organizado como fogo ou fumaca.
