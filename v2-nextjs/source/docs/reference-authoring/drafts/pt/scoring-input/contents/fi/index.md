# Nome do documento: [Codificação/Conteúdo] Fi

## Apelidos e busca

- Fi
- fogo
- fire
- fumaça

## Definição central

`Fi` é usado quando o conteúdo principal é fogo, chama, brasa ou fumaça.
O código recai no material ardente ou fumegante em si.

## Condições de aplicação

- Use para fogo, incêndio, chama, brasa, fumaça ou forma organizada como isso.
- A resposta precisa tratar a figura como fogo ou fumaça, não apenas como evento explosivo.
- Se a resposta apontar para explosão, impacto ou evento, revise primeiro [`Ex`](ref://scoring-input/contents/Ex).
- Se a forma for tratada como nuvem, revise primeiro [`Cl`](ref://scoring-input/contents/Cl).

## Cuidados e diferenciação

- `Fi` e [`Ex`](ref://scoring-input/contents/Ex) separam fogo/fumaça de evento explosivo.
- `Fi` e [`Cl`](ref://scoring-input/contents/Cl) separam fogo/fumaça de nuvem.
- Cor quente sozinha não basta sem conteúdo organizado como fogo ou fumaça.
