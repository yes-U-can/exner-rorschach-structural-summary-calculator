# Nome do documento: [Codificação/Conteúdo] Bl

## Apelidos e busca

- Bl
- sangue
- blood

## Definição central

`Bl` é usado quando o conteúdo principal é sangue ou material nitidamente entendido como sangue.
O código aponta para o próprio sangue, não apenas para anatomia geral.

## Condições de aplicação

- Use quando a pessoa nomeia sangue, poça de sangue, jato de sangue ou mancha de sangue.
- O sangue precisa ser realmente mencionado ou claramente organizado como sangue na resposta.
- Se o foco estiver em parte corporal ou estrutura biológica, revise primeiro [`An`](ref://scoring-input/contents/An).
- Se o foco estiver em detalhe humano ferido ou mutilado, [`Hd`](ref://scoring-input/contents/Hd) pode ser necessário junto com `Bl`.

## Cuidados e diferenciação

- `Bl` não substitui [`An`](ref://scoring-input/contents/An) quando a resposta é principalmente anatômica.
- `Bl` também não substitui [`Hd`](ref://scoring-input/contents/Hd) quando o foco está em detalhe humano.
- Uma cor vermelha sozinha não justifica `Bl` sem conteúdo de sangue.
