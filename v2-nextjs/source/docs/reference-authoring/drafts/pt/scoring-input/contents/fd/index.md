# Nome do documento: [Codificação/Conteúdo] Fd

## Apelidos e busca

- Fd
- comida
- food
- objeto comestível

## Definição central

`Fd` é usado quando o conteúdo principal é comida ou algo entendido como comestível.
É um código de conteúdo, não um determinante.

## Condições de aplicação

- Use para refeição pronta, carne, bolo, fruta servida ou outro item claramente comestível.
- O código também pode entrar quando a cena se organiza em torno de comer ou de um objeto claramente alimentar.
- Se o foco principal for o animal em si, revise primeiro [`A`](ref://scoring-input/contents/A) ou outro conteúdo animal.
- Não confunda `Fd` com [`FD`](ref://scoring-input/determinants/FD), que pertence aos determinantes.

## Cuidados e diferenciação

- `Fd` como conteúdo e [`FD`](ref://scoring-input/determinants/FD) como determinante pertencem a eixos diferentes.
- `Fd` e conteúdos botânicos podem parecer próximos quando verduras são nomeadas, mas `Fd` exige organização como alimento.
- O significado interpessoal posterior de [`Food`](ref://result-interpretation/lower-section/interpersonal/Food) não altera a decisão de codificação.
