# Nome do documento: [Codificação/Conteúdo] Cg

## Apelidos e busca

- Cg
- roupa
- vestimenta
- wearable

## Definição central

`Cg` é usado quando o conteúdo principal é roupa, vestimenta ou acessório usável no corpo.
O ponto decisivo é a resposta entender a forma como algo que se veste.

## Condições de aplicação

- Use para chapéu, bota, luva, capa, vestido, uniforme ou outro item de roupa reconhecível.
- Acessórios corporais podem entrar em `Cg` quando a resposta os trata como parte do vestir.
- Se a roupa trouxer referência histórica ou cultural dominante, revise também [`Ay`](ref://scoring-input/contents/Ay).
- Se a resposta enfatizar valor artístico ou decorativo, revise também [`Art`](ref://scoring-input/contents/Art).

## Cuidados e diferenciação

- `Cg` não serve para qualquer objeto solto.
- `Cg` e [`Ay`](ref://scoring-input/contents/Ay) podem coexistir, mas `Ay` exige moldura cultural ou histórica.
- `Cg` e [`Art`](ref://scoring-input/contents/Art) podem coexistir, mas `Art` exige leitura artística ou decorativa.
