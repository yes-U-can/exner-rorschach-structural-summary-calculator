# Nome do documento: [Codificacao/Conteudo] Hx

## Apelidos e busca

- Hx
- experiencia humana
- estado humano

## Definicao central

`Hx` e codificado quando a resposta aponta para experiencia, estado, sentimento ou qualidade humana, e nao para uma pessoa ou parte corporal.

## Condicoes de aplicacao

- A resposta precisa se referir a experiencia ou qualidade humana.
- O foco nao deve ser uma figura humana inteira nem parte do corpo.
- Se a resposta indicar pessoa inteira, reconsidere [`H`](ref://scoring-input/contents/H) ou [`(H)`](ref://scoring-input/contents/%28H%29).
- Se a resposta indicar detalhe corporal, reconsidere [`Hd`](ref://scoring-input/contents/Hd) ou [`(Hd)`](ref://scoring-input/contents/%28Hd%29).

## Cuidados e diferenciacao

- `Hx` nao codifica pessoa inteira.
- `Hx` tambem nao codifica apenas "algo relacionado a gente" de forma vaga; a experiencia humana precisa estar claramente presente.
- `Hx` pode aparecer perto de temas interpessoais, mas continua sendo codigo de conteudo.
