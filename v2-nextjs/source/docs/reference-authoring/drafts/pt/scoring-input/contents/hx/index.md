# Nome do documento: [Codificação/Conteúdo] Hx

## Apelidos e busca

- Hx
- experiência humana
- estado humano

## Definição central

`Hx` é codificado quando a resposta aponta para experiência, estado, sentimento ou qualidade humana, e não para uma pessoa ou parte corporal.

## Condições de aplicação

- A resposta precisa se referir a experiência ou qualidade humana.
- O foco não deve ser uma figura humana inteira nem parte do corpo.
- Se a resposta indicar pessoa inteira, reconsidere [`H`](ref://scoring-input/contents/H) ou [`(H)`](ref://scoring-input/contents/%28H%29).
- Se a resposta indicar detalhe corporal, reconsidere [`Hd`](ref://scoring-input/contents/Hd) ou [`(Hd)`](ref://scoring-input/contents/%28Hd%29).

## Cuidados e diferenciação

- `Hx` não codifica pessoa inteira.
- `Hx` também não codifica apenas "algo relacionado a gente" de forma vaga; a experiência humana precisa estar claramente presente.
- `Hx` pode aparecer perto de temas interpessoais, mas continua sendo código de conteúdo.
