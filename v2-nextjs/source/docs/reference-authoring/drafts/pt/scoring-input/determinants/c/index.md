# Nome do documento: [Codificação/Determinantes] C

## Apelidos e busca

- C
- cor pura
- resposta por cor

## Definição central

`C` é usado quando a cor cromática funciona como determinante principal sem apoio formal real.
A percepção depende da cor mais do que da forma.

## Condições de aplicação

- A cor precisa participar como elemento principal da resposta.
- O apoio formal está ausente ou é muito fraco para manter `FC` ou `CF`.
- Se ainda houver forma secundária clara, reconsidere [`CF`](ref://scoring-input/determinants/CF).
- Se a forma dominar mais que a cor, reconsidere [`FC`](ref://scoring-input/determinants/FC).

## Cuidados e diferenciação

- `C` não é mera menção de cor; a cor precisa realmente fazer a resposta parecer daquele jeito.
- Quando há dúvida, pergunte se a resposta continuaria a mesma sem a cor.
- O uso interpretativo desse código aparece depois em [`Pure C`](ref://result-interpretation/lower-section/affect/PureC) e [`FC:CF+C`](ref://result-interpretation/lower-section/affect/FC_CF_C).
