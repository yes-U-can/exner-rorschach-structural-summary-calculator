# Nome do documento: [Codificação/Determinantes] FY

## Apelidos e busca

- FY
- forma-sombreamento
- sombreamento secundário

## Definição central

`FY` é usado quando a forma organiza a resposta e o sombreamento difuso fica secundário.

## Condições de aplicação

- A forma identifica o objeto principal.
- Depois a pessoa acrescenta uma qualidade difusa de sombra, nebulosidade ou incerteza.
- Essa qualidade precisa aparecer de fato na resposta ou no inquérito.
- Se o sombreamento carregar mais peso que a forma, reconsidere [`YF`](ref://scoring-input/determinants/YF).

## Cuidados e diferenciação

- `FY` não é o mesmo que [`FV`](ref://scoring-input/determinants/FV); um aponta para sombreamento difuso e o outro para profundidade.
- `FY` também não é o mesmo que [`FT`](ref://scoring-input/determinants/FT); um aponta para sombreamento e o outro para textura.
- Se a forma não organizar de verdade a resposta, o puro [`Y`](ref://scoring-input/determinants/Y) pode caber melhor.
