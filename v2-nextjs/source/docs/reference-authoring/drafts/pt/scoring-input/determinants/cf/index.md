# Nome do documento: [Codificação/Determinantes] CF

## Apelidos e busca

- CF
- cor-forma
- forma secundária a cor

## Definição central

`CF` é usado quando a cor cromática tem peso principal e a forma fica secundária.
A resposta ainda conserva algum apoio formal, mas a cor conduz mais claramente a percepção.

## Condições de aplicação

- A cor participa de modo dominante.
- A forma ainda está presente, mas com importância secundária.
- Se a forma dominar mais que a cor, reconsidere [`FC`](ref://scoring-input/determinants/FC).
- Se praticamente não houver uso real de forma, reconsidere [`C`](ref://scoring-input/determinants/C).

## Cuidados e diferenciação

- `CF` não é apenas resposta colorida; a cor precisa carregar o peso principal da percepção.
- A diferença em relação a `C` é que `CF` ainda conserva algum apoio formal.
- O uso interpretativo dessa família aparece depois em [`FC:CF+C`](ref://result-interpretation/lower-section/affect/FC_CF_C).
