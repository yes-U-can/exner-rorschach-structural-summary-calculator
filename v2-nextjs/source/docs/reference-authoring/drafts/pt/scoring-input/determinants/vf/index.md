# Nome do documento: [Codificação/Determinantes] VF

## Apelidos e busca

- VF
- vista-forma
- forma secundária a vista

## Definição central

`VF` é usado quando profundidade, distância ou relevo dominam a resposta e a forma fica secundária.

## Condições de aplicação

- A experiência de profundidade precisa carregar o peso principal.
- Ainda deve existir algum apoio formal, embora secundário.
- Se a forma dominar mais, reconsidere [`FV`](ref://scoring-input/determinants/FV).
- Se o apoio formal praticamente desaparecer, reconsidere [`V`](ref://scoring-input/determinants/V).

## Cuidados e diferenciação

- `VF` não é o mesmo que `V`; em `VF` ainda sobra alguma organização formal.
- Se a experiência principal for textura, revise a família [`T`](ref://scoring-input/determinants/T).
- O uso interpretativo dessa família aparece depois em [`SumV`](ref://result-interpretation/lower-section/core/SumV).
