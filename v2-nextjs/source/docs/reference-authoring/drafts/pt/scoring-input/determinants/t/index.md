# Nome do documento: [Codificação/Determinantes] T

## Apelidos e busca

- T
- textura pura

## Definição central

`T` é usado quando a textura funciona como qualidade principal sem apoio formal real.

## Condições de aplicação

- A experiência tátil ou de superfície precisa carregar a resposta.
- O apoio formal deve estar ausente ou muito fraco.
- Se ainda houver forma secundária clara, reconsidere [`TF`](ref://scoring-input/determinants/TF).
- Se a forma dominar mais, reconsidere [`FT`](ref://scoring-input/determinants/FT).

## Cuidados e diferenciação

- `T` não é apenas descrição de superfície; a qualidade tátil precisa realmente fazer a resposta parecer daquele jeito.
- O uso interpretativo dessa família aparece depois em [`SumT`](ref://result-interpretation/lower-section/core/SumT) e [`SumT_inter`](ref://result-interpretation/lower-section/interpersonal/SumT_inter).
