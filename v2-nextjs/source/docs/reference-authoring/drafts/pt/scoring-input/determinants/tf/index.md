# Nome do documento: [Codificação/Determinantes] TF

## Apelidos e busca

- TF
- textura-forma
- forma secundária a textura

## Definição central

`TF` é usado quando a textura tem peso principal e a forma fica secundária.
A resposta ainda conserva algum apoio formal, mas a experiência tátil ou de superfície domina mais.

## Condições de aplicação

- A textura ou qualidade tátil precisa carregar o peso principal.
- Ainda deve existir algum apoio formal, embora secundário.
- Se a forma dominar mais, reconsidere [`FT`](ref://scoring-input/determinants/FT).
- Se o apoio formal praticamente desaparecer, reconsidere [`T`](ref://scoring-input/determinants/T).

## Cuidados e diferenciação

- `TF` não é o mesmo que `T`; em `TF` ainda sobra alguma organização formal.
- Se a experiência principal for profundidade, revise a família [`V`](ref://scoring-input/determinants/V).
- O uso interpretativo dessa família aparece depois em [`SumT`](ref://result-interpretation/lower-section/core/SumT).
