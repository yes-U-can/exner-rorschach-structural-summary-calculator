# Nome do documento: [Codificação/Determinantes] FC'

## Apelidos e busca

- FC'
- forma-cor acromática

## Definição central

`FC'` é usado quando preto, branco ou cinza participam como cor acromática, mas a forma continua dominante.

## Condições de aplicação

- A forma deve guiar a identificação principal.
- Preto, branco ou cinza precisam funcionar como cor, não apenas como sombreamento qualquer.
- Se a cor acromática dominar mais que a forma, reconsidere [`C'F`](ref://scoring-input/determinants/C%27F).
- Se não houver apoio formal real, reconsidere [`C'`](ref://scoring-input/determinants/C%27).

## Cuidados e diferenciação

- `FC'` segue a mesma lógica relativa de `FC`, mas com cor acromática.
- Nem toda resposta com preto, branco ou cinza recebe cor acromática; esses tons precisam funcionar como cor.
- O uso interpretativo dessa família aparece depois em [`SumCprime`](ref://result-interpretation/lower-section/core/SumCprime).
