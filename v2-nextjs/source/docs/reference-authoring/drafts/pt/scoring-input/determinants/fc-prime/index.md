# Nome do documento: [Codificacao/Determinantes] FC'

## Apelidos e busca

- FC'
- forma-cor acromatica

## Definicao central

`FC'` e usado quando preto, branco ou cinza participam como cor acromatica, mas a forma continua dominante.

## Condicoes de aplicacao

- A forma deve guiar a identificacao principal.
- Preto, branco ou cinza precisam funcionar como cor, nao apenas como sombreamento qualquer.
- Se a cor acromatica dominar mais que a forma, reconsidere [`C'F`](ref://scoring-input/determinants/C%27F).
- Se nao houver apoio formal real, reconsidere [`C'`](ref://scoring-input/determinants/C%27).

## Cuidados e diferenciacao

- `FC'` segue a mesma logica relativa de `FC`, mas com cor acromatica.
- Nem toda resposta com preto, branco ou cinza recebe cor acromatica; esses tons precisam funcionar como cor.
- O uso interpretativo dessa familia aparece depois em [`SumCprime`](ref://result-interpretation/lower-section/core/SumCprime).
