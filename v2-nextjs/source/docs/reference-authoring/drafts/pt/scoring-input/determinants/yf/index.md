# Nome do documento: [Codificação/Determinantes] YF

## Apelidos e busca

- YF
- sombreamento-forma
- forma secundária ao sombreamento

## Definição central

`YF` é usado quando o sombreamento difuso domina a resposta e a forma fica secundária.

## Condições de aplicação

- A qualidade difusa de sombra, nebulosidade ou incerteza precisa carregar o peso principal.
- Ainda deve existir algum apoio formal, embora secundário.
- Se a forma dominar mais, reconsidere [`FY`](ref://scoring-input/determinants/FY).
- Se o apoio formal praticamente desaparecer, reconsidere [`Y`](ref://scoring-input/determinants/Y).

## Cuidados e diferenciação

- `YF` não é o mesmo que `Y`; em `YF` ainda sobra alguma organização formal.
- Se a experiência principal for profundidade, revise a família [`V`](ref://scoring-input/determinants/V).
- O uso interpretativo dessa família aparece depois em [`SumY`](ref://result-interpretation/lower-section/core/SumY).
