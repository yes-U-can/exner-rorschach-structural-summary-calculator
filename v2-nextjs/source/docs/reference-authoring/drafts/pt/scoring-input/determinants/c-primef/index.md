# Nome do documento: [Codificacao/Determinantes] C'F

## Apelidos e busca

- C'F
- cor acromatica-forma

## Definicao central

`C'F` e usado quando preto, branco ou cinza funcionam como cor acromatica dominante e a forma fica secundaria.

## Condicoes de aplicacao

- A cor acromatica precisa carregar o peso principal da resposta.
- Ainda deve existir algum apoio formal, embora secundario.
- Se a forma dominar mais, reconsidere [`FC'`](ref://scoring-input/determinants/FC%27).
- Se o apoio formal praticamente desaparecer, reconsidere [`C'`](ref://scoring-input/determinants/C%27).

## Cuidados e diferenciacao

- `C'F` segue a mesma logica relativa de `CF`, mas com preto, branco ou cinza funcionando como cor.
- Nao use `C'F` apenas porque a area parece escura; a qualidade acromatica precisa participar como cor.
- O uso interpretativo dessa familia aparece depois em [`SumCprime`](ref://result-interpretation/lower-section/core/SumCprime).
