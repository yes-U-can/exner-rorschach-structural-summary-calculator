# Nome do documento: [Codificação/Determinantes] C'F

## Apelidos e busca

- C'F
- cor acromática-forma

## Definição central

`C'F` é usado quando preto, branco ou cinza funcionam como cor acromática dominante e a forma fica secundária.

## Condições de aplicação

- A cor acromática precisa carregar o peso principal da resposta.
- Ainda deve existir algum apoio formal, embora secundário.
- Se a forma dominar mais, reconsidere [`FC'`](ref://scoring-input/determinants/FC%27).
- Se o apoio formal praticamente desaparecer, reconsidere [`C'`](ref://scoring-input/determinants/C%27).

## Cuidados e diferenciação

- `C'F` segue a mesma lógica relativa de `CF`, mas com preto, branco ou cinza funcionando como cor.
- Não use `C'F` apenas porque a área parece escura; a qualidade acromática precisa participar como cor.
- O uso interpretativo dessa família aparece depois em [`SumCprime`](ref://result-interpretation/lower-section/core/SumCprime).
