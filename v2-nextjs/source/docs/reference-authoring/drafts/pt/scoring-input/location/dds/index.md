# Nome do documento: [Codificação/Localização] DdS

## Apelidos e busca

- DdS
- detalhe incomum com espaço branco

## Definição central

`DdS` é usado quando a resposta depende de um detalhe incomum e também integra o espaço branco de modo significativo.
É `Dd` modificado por `S`.

## Condições de aplicação

- O percepto se baseia em um detalhe incomum.
- O espaço branco participa ativamente da resposta.
- A combinação é confirmada no inquérito.

## Cuidados e diferenciação

- Se a área é comum com espaço branco, use [`DS`](ref://scoring-input/location/DS), não `DdS`.
- Se não existe papel real do espaço branco, use [`Dd`](ref://scoring-input/location/Dd).
