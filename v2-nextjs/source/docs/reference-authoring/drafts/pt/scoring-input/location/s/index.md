# Nome do documento: [Codificação/Localização] S

## Apelidos e busca

- S
- uso de espaço branco
- integração de espaço

## Definição central

`S` indica uso significativo de espaço branco na resposta.
Ele não é um código básico independente; modifica outra decisão de localização.

## Condições de aplicação

- O espaço branco é usado ativamente como parte do percepto.
- O papel do espaço branco é confirmado no inquérito.
- Acrescente `S` ao código básico relevante: `WS`, `DS` ou `DdS`.

## Cuidados e diferenciação

- Mencionar fundo, sozinho, não basta. O espaço branco precisa funcionar como parte do percepto.
- `S` não deve ser codificado sozinho como se fosse decisão completa de localização.
- No Sumário Estrutural, `S` é o código que totaliza o uso de espaço branco; na entrada de cada resposta, registre sempre `WS`, `DS` ou `DdS`.
- A interpretação de `S` pertence ao padrão do Sumário Estrutural, não apenas à codificação de localização.
