# Nome do documento: [Codificação/Qualidade Formal] o

## Apelidos e busca

- o
- FQo
- ajuste formal ordinário

## Definição central

`FQo` é usado quando a resposta mostra ajuste formal bom e plausível, sem o nível especial de elaboração exigido para `FQ+`.
É a forma mais comum de ajuste bom.

## Condições de aplicação

- O objeto se ajusta bem aos contornos da mancha.
- A leitura é plausível sem forçar muito a área.
- A resposta não é especialmente elaborada a ponto de exigir `FQ+`.
- O ajuste é mais ordinário do que incomum.

## Cuidados e diferenciação

- Se a resposta é boa e especialmente bem articulada, use [`FQ+`](ref://scoring-input/fq/%2B).
- Se a resposta é incomum, mas ainda plausível, use [`FQu`](ref://scoring-input/fq/u).
- Se o ajuste depende de distorção clara ou contornos forçados, use [`FQ-`](ref://scoring-input/fq/-).
