# Nome do documento: [Codificação/Z] ZA

## Apelidos e busca

- ZA
- Z de detalhes adjacentes
- atividade organizacional adjacente

## Definição central

`ZA` é usado quando a atividade organizacional depende de colocar áreas adjacentes da mancha em relação significativa.
A resposta mostra organização entre partes próximas da mancha.

## Condições de aplicação

- A resposta usa forma.
- A resposta liga áreas adjacentes da mancha de modo significativo.
- A organização é mais do que simplesmente nomear partes próximas.
- `ZA` precisa refletir integração real, não apenas mudança de localização.

## Cuidados e diferenciação

- Áreas adjacentes, sozinhas, não garantem `ZA`; precisa haver atividade organizacional.
- Se a organização depende de áreas distantes, considere [`ZD`](ref://scoring-input/z/ZD).
- Se o espaço branco é o principal organizador, considere [`ZS`](ref://scoring-input/z/ZS).
