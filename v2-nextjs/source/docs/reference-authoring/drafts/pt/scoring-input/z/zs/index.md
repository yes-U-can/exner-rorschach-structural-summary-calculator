# Nome do documento: [Codificação/Z] ZS

## Apelidos e busca

- ZS
- Z com espaço branco
- atividade organizacional por espaço

## Definição central

`ZS` é usado quando a atividade organizacional depende de integração significativa do espaço branco.
A resposta precisa organizar o espaço branco como parte do percepto, não apenas nota-lo.

## Condições de aplicação

- A resposta usa forma.
- O espaço branco é integrado de modo significativo na resposta.
- A organização depende desse uso de espaço branco.
- A resposta se ajusta melhor a organização por espaço branco do que a `ZW`, `ZA` ou `ZD`.

## Cuidados e diferenciação

- Espaço branco visível, sozinho, não basta para `ZS`; o espaço branco precisa funcionar como parte do percepto organizado.
- Se o espaço branco é usado sem atividade organizacional em nível Z, codifique localização com `S`, mas não presuma `ZS`.
- Mantenha clara a diferença entre codificação de localização com `S` e codificação de atividade organizacional com `ZS`.
