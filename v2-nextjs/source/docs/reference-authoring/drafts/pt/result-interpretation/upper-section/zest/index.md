# Nome do documento: [Interpretação/Seção Superior] ZEst

## Apelidos e busca

- ZEst
- soma Z estimada
- soma Z esperada

## Definição central

`ZEst` é a `ZSum` estimada esperada a partir do valor observado de `Zf`.
Ele funciona mais como linha de base para calcular `Zd` do que como variável clínica independente.

## Pontos de interpretação

- `ZEst` funciona como um valor de referência.
- Ele permite comparar a organização real (`ZSum`) com aquilo que seria esperado.
- A ênfase interpretativa costuma recair sobre `Zd`, não sobre `ZEst` isoladamente.

## Cuidados e diferenciação

- Um `ZEst` alto muitas vezes apenas reflete que `Zf` também está alto.
- E melhor evitar frases como "ZEst alto" ou "ZEst baixo" sem explicar o papel comparativo.
