# Nome do documento: [Interpretação/Core] D

## Apelidos e busca

- D
- D score
- tolerância atual ao estresse

## Definição central

`D` é a diferença entre `EA` e `es`.
Ela resume se os recursos disponíveis bastam para manejar o nível atual de demanda sem sobrecarga clara.

## Pontos de interpretação

- `D = 0` costuma sugerir equilíbrio aproximado entre recursos e demanda atual.
- `D > 0` sugere margem adaptativa e menor risco imediato de ficar sobrecarregado pela situação atual.
- `D < 0` sugere que a demanda atual excede os recursos disponíveis e pode acompanhar tensão, irritabilidade ou menor tolerância a frustração.
- `D` é sensível a condições atuais, por isso pode mudar com estresse situacional recente.

## Cuidados e diferenciação

- `D = 0` não significa automaticamente bom ajuste; um protocolo simplificado ou defensivo pode parecer equilibrado.
- `D > 0` não prova, sozinho, força global estável.
- `D < 0` aponta para sobrecarga atual, mas a comparação com `AdjD` é necessária para julgar se a tensão é mais situacional ou mais persistente.
