# Document Name: [コーディング/場所] W

## Aliases / Search Terms

- W
- 全体反応
- 全体領域

## Core Definition

`W` は、反応がインクブロット全体を使って構成されているときにつけるコードです。
全体っぽく聞こえるだけでは足りず、実際に主要な領域が全体として使われている必要があります。

## Application Conditions

- 反応がブロット全体を使っている。
- 対象を作るうえで重要な領域が除かれていない。
- inquiry または location sheet で全体使用が確認できる。

## Cautions / Distinctions

- よく知られた全体反応でも、実際には一部を落としていれば `W` にしません。
- 白地も図として一緒に使うなら [`WS`](ref://scoring-input/location/WS) を考えます。
- `W` は場所コードであり、それだけで form quality や organization の高さを意味しません。
