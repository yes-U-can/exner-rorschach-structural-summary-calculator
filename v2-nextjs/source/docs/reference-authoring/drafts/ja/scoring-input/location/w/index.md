# [コーディング/場所] W

## 別名・検索語

- W
- 全体反応
- 全体領域

## 中核的定義

`W` は、反応がインクブロット全体を使って構成されているときにつけるコードです。
全体っぽく聞こえるだけでは足りず、実際に主要な領域が全体として使われている必要があります。

## 採点・適用条件

- 反応がブロット全体を使っています。
- 対象を作るうえで重要な領域が除かれていません。
- 質問段階または領域図で全体使用が確認できます。

## 注意点・鑑別

- よく知られた全体反応でも、実際には一部を落としていれば `W` にしません。
- 白地も図として一緒に使うなら [`WS`](ref://scoring-input/location/WS) を考えます。
- `W` は場所コードであり、それだけで形態水準や組織化の高さを意味しません。
