# Document Name: [コーディング/場所] Dd

## Aliases / Search Terms

- Dd
- 稀少部分
- 珍しい部分

## Core Definition

`Dd` は、location sheet 上であまり使われない珍しい小部分を使った反応につけるコードです。
小さいからではなく、その部分が慣習的に稀少領域であることが重要です。

## Application Conditions

- 反応が全体ではなく一部に基づいている。
- その部分が通常部分ではなく、稀少部分に分類される。
- inquiry でその限定された領域が確認できる。

## Cautions / Distinctions

- ふつうによく使われる部分なら [`D`](ref://scoring-input/location/D) です。
- 白地も図として使っていれば [`DdS`](ref://scoring-input/location/DdS) を検討します。
- 単に細かく説明しただけで、実際の場所が通常部分なら `Dd` にしません。
