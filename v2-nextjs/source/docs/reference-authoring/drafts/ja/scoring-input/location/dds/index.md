# Document Name: [コーディング/場所] DdS

## Aliases / Search Terms

- DdS
- 稀少部分+白地
- Dd+白地

## Core Definition

`DdS` は、珍しい小部分を使いながら、白地も図として取り込んでいる反応につけるコードです。
稀少部分であることと白地利用の両方が確認されなければなりません。

## Application Conditions

- 反応が稀少部分に基づいている。
- 白地が対象の一部として使われている。
- inquiry で稀少部分と白地の両方の使用が確認できる。

## Cautions / Distinctions

- 白地を使わない稀少部分反応なら [`Dd`](ref://scoring-input/location/Dd) です。
- 通常部分+白地なら [`DS`](ref://scoring-input/location/DS) です。
- 白地が背景にすぎない場合は `DdS` にしません。
