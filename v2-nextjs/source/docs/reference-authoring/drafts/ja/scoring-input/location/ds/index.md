# [コーディング/場所] DS

## 別名・検索語

- DS
- 通常部分+白地
- D+白地

## 中核的定義

`DS` は、よく使われる通常部分を使いながら、白地も図として取り込んだ反応につけるコードです。
部分反応であることと、白地利用の両方が確認される必要があります。

## 採点・適用条件

- 反応が全体ではなく通常部分に基づいています。
- 白地が対象の一部として使われています。
- 質問段階で通常部分と白地の両方の使用が確認できます。

## 注意点・鑑別

- 白地を使わない通常部分反応なら [`D`](ref://scoring-input/location/D) です。
- 珍しい小部分+白地なら [`DdS`](ref://scoring-input/location/DdS) を使います。
- 白地が背景説明だけなら `DS` にはしません。
