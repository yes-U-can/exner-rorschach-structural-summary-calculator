# [コーディング/場所] S

## 別名・検索語

- S
- 白地反応
- 白地使用

## 中核的定義

`S` は、白地そのものを図として使ったことを示す場所コードです。
白い部分が対象の輪郭や主要部分として機能している必要があります。

## 採点・適用条件

- 白地が対象の形成に実際に使われています。
- どの範囲を使ったかに応じて `WS`, `DS`, `DdS`, `S` のどれかに整理できます。
- 質問段階で白地利用が明確に確認できます。

## 注意点・鑑別

- 白地が単なる背景説明なら `S` 系列にはしません。
- 全体+白地なら [`WS`](ref://scoring-input/location/WS)、通常部分+白地なら [`DS`](ref://scoring-input/location/DS)、稀少部分+白地なら [`DdS`](ref://scoring-input/location/DdS) を使います。
- `S` は反抗性や非慣習性の解釈と関連づけられることがありますが、場所コードだけで単独解釈はしません。
