# Document Name: [採点/発達水準] +

## Aliases / Search Terms

- DQ+
- 統合反応 (+)
- 形態要求を伴う統合

## Core Definition

`DQ+` は、2 つ以上の別個の対象が意味のあるまとまりとして統合され、そのうち少なくとも 1 つが明確な形態要求をもつときに与えます。  
単に対象が複数あるだけではなく、組織化された全体になっていることが重要です。

## Application Conditions

- 反応の中に 2 つ以上の対象または部分が含まれています。
- それらが意味のある関係で結びついています。
- 少なくとも 1 つの対象が特定の形態要求をもちます。
- ただ並べただけではなく、まとまった統合として知覚されています。

## Cautions / Distinctions

- 2 つの対象がただ並行しているだけで、意味のある統合が弱いなら [`DQo`](ref://scoring-input/dq/o) を再検討します。
- 統合はあるが、どの対象にも明確な形態要求がないなら [`DQv/+`](ref://scoring-input/dq/v/%2B) を使います。
- `DQ+` は [`FQ+`](ref://scoring-input/fq/%2B) と同じではありません。`DQ+` は組織化、`FQ+` は形態適合を扱います。
