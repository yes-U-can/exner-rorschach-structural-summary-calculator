# Document Name: [採点/発達水準] v

## Aliases / Search Terms

- DQv
- あいまいな発達水準
- 非統合であいまいな反応

## Core Definition

`DQv` は、反応があいまいで、意味のある統合を示さないときに用います。  
標準の 4 区分の中では、構造化の程度がもっとも低い DQ コードです。

## Application Conditions

- 意味のある統合がありません。
- 対象に明確な形態要求もありません。
- 反応が比較的拡散的、印象的、または非特定的です。
- `DQo`, `DQv/+`, `DQ+` のどれにも当てはまりません。

## Cautions / Distinctions

- 対象に形態要求があるのに統合がないなら [`DQo`](ref://scoring-input/dq/o) を使います。
- 統合はあるが形態要求がないなら [`DQv/+`](ref://scoring-input/dq/v/%2B) を使います。
- 統合があり、かつ少なくとも 1 つの対象に形態要求があるなら [`DQ+`](ref://scoring-input/dq/%2B) を使います。
