# [コーディング/Z] ZD

## 別名・検索語

- ZD
- 遠隔領域組織化
- distant Z

## 中核的定義

`ZD` は、離れたブロット領域どうしを一つの意味ある反応にまとめたときにつける Z コードです。
離れていること自体ではなく、その距離をまたいで組織化が起きていることが大事です。

## 採点・適用条件

- 反応に形態の使用があります。
- 離れた、隣接していない領域が関係づけられています。
- その結びつきが意味ある統合として成立しています。

## 注意点・鑑別

- ただ二つの部分を別々に挙げただけなら `ZD` にしません。
- 隣接領域どうしの組織化なら [`ZA`](ref://scoring-input/z/ZA) を考えます。
- `ZD` は後の [`Zd`](ref://result-interpretation/upper-section/Zd) 解釈に影響するので慎重に固定します。
