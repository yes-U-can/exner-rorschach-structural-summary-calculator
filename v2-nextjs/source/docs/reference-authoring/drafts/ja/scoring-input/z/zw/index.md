# Document Name: [コーディング/Z] ZW

## Aliases / Search Terms

- ZW
- 全体組織化
- whole Z

## Core Definition

`ZW` は、全体反応の中で意味ある組織化活動が見られるときにつける Z コードです。
全体を使うだけでは足りず、全体のまとまり方が実際に確認できる必要があります。

## Application Conditions

- 反応が [`W`](ref://scoring-input/location/W) に基づいている。
- 反応に form use がある。
- 全体反応としての統合が意味ある形で成立している。
- 対応する `DQ` が組織化を支える水準にある。

## Cautions / Distinctions

- 全体反応でもまとまりが弱ければ `ZW` にしません。
- 組織化が隣接部分同士の関係なら [`ZA`](ref://scoring-input/z/ZA) の方が適切なことがあります。
- `ZW` は `W` とは別コードです。`W` は場所、`ZW` は組織化活動です。
