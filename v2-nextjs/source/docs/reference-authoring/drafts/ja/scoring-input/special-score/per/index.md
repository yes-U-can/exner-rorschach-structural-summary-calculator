# [コーディング/特殊スコア] PER

## 別名・検索語

- PER
- 個人化
- 個人的正当化

## 中核的定義

`PER` は、被検者が自分の知識、個人的経験、あるいは自分だけの事情を持ち出して、その反応を正当化したときに付ける特殊スコアです。
要点は、個人的情報が反応の根拠として直接使われていることです。

## 採点・適用条件

- 個人的経験、以前の学習、自分の知っていることを理由にして「だからこれはこうだ」と支えているときに検討します。
- 個人的情報は、その反応自体を支えるために使われている必要があります。
- 一人称が出ただけでは `PER` にはなりません。
- 後の解釈では [`PER`](ref://result-interpretation/lower-section/interpersonal/PER) や [`Xu%`](ref://result-interpretation/lower-section/mediation/Xu_percent) と一緒に見直します。

## 注意点・鑑別

- 「私はそう見える」「私はそう思う」だけでは `PER` を付けません。
- `PER` は内容符号ではなく、反応の正当化の仕方を記録する特殊スコアです。
- 個人的な発言でも、その反応の根拠に結びついていないなら `PER` にしません。
