# Document Name: [符号化/特殊得点] DR1

## Aliases / Search Terms

- DR1
- 軽度の逸脱反応
- Deviant Response 1

## Core Definition

`DR1` は、反応全体の流れや組み立てが軽度に逸れているときに付ける特殊得点です。  
問題は単語ひとつではなく、反応全体のまとまりや展開にあります。

## Application Conditions

- 反応全体に軽い脱線、まとまりの崩れ、不適切な展開が見えるときに検討します。
- 問題が語の選択だけでなく、反応全体の組み立てに及んでいるときに使います。
- 問題が言い回しだけに見えるなら [`DV1`](ref://scoring-input/special-score/DV1) や [`DV2`](ref://scoring-input/special-score/DV2) を先に見直します。
- 全体の乱れがもっと強いなら [`DR2`](ref://scoring-input/special-score/DR2) も検討します。

## Cautions / Distinctions

- 長い、複雑というだけでは `DR1` にしません。
- `DR1` は語の逸脱を中心に見る `DV` と違い、反応全体の流れの逸脱を記録します。
- 水準1か2か迷うときは、より保守的な水準を保ちます。
