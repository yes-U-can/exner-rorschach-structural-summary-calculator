# [コーディング/特殊スコア] AB

## 別名・検索語

- AB
- 抽象化
- 抽象的反応

## 中核的定義

`AB` は、反応が具体的な対象から離れ、抽象的、一般化された、概念的な表現へ移ったときに付ける特殊スコアです。
要点は「はっきりした物」を答える代わりに、性質、概念、気分、力のような抽象化された内容を答えていることです。

## 採点・適用条件

- 感情、概念、力、感覚、一般化された質のように、具体物ではない抽象的表現が中心になっているときに検討します。
- [`Hx`](ref://scoring-input/contents/Hx) の近くで、人そのものではなく一般化された人間経験を述べている場合に現れやすいです。
- [`Art`](ref://scoring-input/contents/Art) や [`Ay`](ref://scoring-input/contents/Ay) と一緒に、内容が高度に抽象化されているときにも使われます。
- これは特殊スコアとして追加するもので、主内容符号の代わりにはしません。

## 注意点・鑑別

- 象徴的、芸術的、難解というだけでは `AB` を付けません。
- `Hx` があるだけでは不十分で、言い方そのものが本当に抽象化されている必要があります。
- 変わった反応でも、対象がまだ具体的なら `AB` は付けません。
