# [コーディング/特殊スコア] MOR

## 別名・検索語

- MOR
- 病的内容
- 損傷内容
- 陰惨内容

## 中核的定義

`MOR`は、対象が死んでいる、壊れている、傷ついている、腐っている、荒廃しているなど、明確に病的・陰惨な質で表現されているときに付ける特殊スコアです。
対象に直接、損傷や劣化のトーンが与えられていることが中心です。

## 採点・適用条件

- 死体、破壊された物、傷ついた身体、腐敗したもの、砕けたもの、明らかに荒れ果てたものなどに使います。
- 病的、腐敗的、死や損傷を直接示す質が対象に付与されている必要があります。
- `MOR`は特殊スコアとして加えるもので、主内容や決定因を置き換えるものではありません。
- 他のコードと共存できる場合もありますが、`MOR`は病的質そのものを示します。

## 注意点・鑑別

- ただ醜い、不快、奇妙というだけでは`MOR`にはなりません。
- 攻撃して傷つける「行為」が中心なら、まず[`AG`](ref://scoring-input/special-score/AG)を再検討します。
- 人間反応では[`PHR`](ref://scoring-input/gphr/PHR)とつながることがありますが、`MOR`だけで自動的に決まるわけではありません。
