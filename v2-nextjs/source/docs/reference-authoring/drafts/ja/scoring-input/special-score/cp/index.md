# Document Name: [符号化/特殊得点] CP

## Aliases / Search Terms

- CP
- 色彩投影
- 有彩色投影

## Core Definition

`CP` は、インクブロットの無彩色部分に、被検者が赤、青、緑などの有彩色を見たときに付ける特殊得点です。  
要点は、その部分に実際の有彩色はないのに、被検者がそこへ色を投影していることです。

## Application Conditions

- 無彩色部分を赤、青、緑などの有彩色として述べたときに検討します。
- これは通常の色彩決定因ではなく、特殊得点として追加します。
- 単なる言い間違い、見誤り、聞き返し後の修正ではないかを先に確認します。
- 多くの場合、[`FY`](ref://scoring-input/determinants/FY)、[`YF`](ref://scoring-input/determinants/YF)、[`Y`](ref://scoring-input/determinants/Y) と区別しながら見直し、後の解釈では [`CP`](ref://result-interpretation/lower-section/affect/CP) とつながります。

## Cautions / Distinctions

- その色が実際に有彩色部分に存在するなら `CP` ではなく、まず [`FC`](ref://scoring-input/determinants/FC) など通常の色彩決定因を検討します。
- `CP` は、実際に色があるかのように `FC`、`CF`、`C` と混ぜて扱いません。
- 一般的に色のあるテーマを好むというだけでは `CP` にしません。
