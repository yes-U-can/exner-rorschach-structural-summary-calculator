# [コーディング/特殊スコア] DV1

## 別名・検索語

- DV1
- 軽度の逸脱言語化
- Deviant Verbalization 1

## 中核的定義

`DV1` は、反応の言い方や語の組み合わせが少し不自然、奇妙、不適切だが、重度ではないときに付ける特殊スコアです。
要点は、問題の中心が反応全体の崩れではなく、言語表現そのものにあることです。

## 採点・適用条件

- 語の使い方、言い回し、語の結びつきが軽度に逸脱しているときに検討します。
- 反応全体よりも、表現のしかた自体に問題があるときに使います。
- 逸脱がもっとはっきり強いなら [`DV2`](ref://scoring-input/special-score/DV2) を先に検討します。
- 反応全体の組み立てまで乱れているようなら [`DR1`](ref://scoring-input/special-score/DR1) や [`DR2`](ref://scoring-input/special-score/DR2) も見直します。

## 注意点・鑑別

- 長い反応、言いにくい反応というだけでは `DV1` にしません。
- 水準1か2かがはっきりしないときは、より保守的に `DV1` を保ちます。
- `DV1` は後に [`Sum6`](ref://result-interpretation/lower-section/ideation/Sum6) や [`WSum6`](ref://result-interpretation/lower-section/ideation/WSum6) に反映されますが、符号化判断はあくまで個々の反応単位で行います。
