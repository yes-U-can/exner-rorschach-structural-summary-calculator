# [コーディング/特殊スコア] FABCOM2

## 別名・検索語

- FABCOM2
- 重度の作話的結合
- Fabulized Combination 2

## 中核的定義

`FABCOM2` は、`FABCOM1` よりも明らかに強く、要素間の関係づけが非常に作話的で無理なときに付ける特殊スコアです。
要点は、反応全体の要素のつながり方がかなり不自然で支えにくいことです。

## 採点・適用条件

- 要素同士の関係が著しく非論理的で、軽い歪みでは説明できないときに使います。
- 問題の中心は、場面や要素同士の結びつき方にあります。
- `FABCOM2` は、この関係の歪みのうち重いほうの水準に限って使います。
- 後の解釈では [`Sum6`](ref://result-interpretation/lower-section/ideation/Sum6)、[`Lv2`](ref://result-interpretation/lower-section/ideation/Lv2)、[`WSum6`](ref://result-interpretation/lower-section/ideation/WSum6) に反映されます。

## 注意点・鑑別

- 問題がひとつの知覚単位の不調和結合なら、まず `INCOM2` を見直します。
- 両立しない知覚がひとつに融合しているなら `CONTAM` を見直します。
- 歪みがそこまで強くないなら `FABCOM1` にとどめます。
- 幻想的、詩的というだけでは `FABCOM2` にしません。
