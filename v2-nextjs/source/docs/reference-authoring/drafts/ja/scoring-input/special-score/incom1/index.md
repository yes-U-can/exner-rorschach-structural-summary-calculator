# Document Name: [符号化/特殊得点] INCOM1

## Aliases / Search Terms

- INCOM1
- 軽度の不調和結合
- Incongruous Combination 1

## Core Definition

`INCOM1` は、ひとつの反応の中で要素の組み合わせが不調和だが、まだ重度ではないときに付ける特殊得点です。  
要点は、結びつけられた部分同士がうまく噛み合っていないことです。

## Application Conditions

- 同じ物体や場面の中で、部分、性質、機能の結びつきが不自然に見えるときに検討します。
- 単に珍しいだけではなく、組み合わせそのものが合っていない必要があります。
- 奇妙でも、まだある程度は追えるなら `INCOM1` が `INCOM2` より適切なことが多いです。
- 問題が言葉づかいの奇妙さに限られるなら、まず `DV1` や `DV2` も見直します。

## Cautions / Distinctions

- 創造的、空想的、珍しいというだけでは `INCOM1` にしません。
- 問題が要素同士の作話的な関係づけにあるなら `FABCOM1` を見直します。
- 両立しない知覚がひとつに融合しているなら `CONTAM` を見直します。
- 水準1か2か迷うときは、より保守的に `INCOM1` を使います。
