# [コーディング/特殊スコア] COP

## 別名・検索語

- COP
- 協力運動
- 協同行為

## 中核的定義

`COP`は、運動反応の中で二つ以上の対象が協力的、相互扶助的、または共同的に行為しているときに付ける特殊スコアです。
中心になるのは、前向きで協調的な共同行為です。

## 採点・適用条件

- 反応に運動が含まれている必要があり、通常は[`M`](ref://scoring-input/determinants/M)、[`FM`](ref://scoring-input/determinants/FM)、[`m`](ref://scoring-input/determinants/m)が前提になります。
- 少なくとも二つ以上の対象や人物が相互作用していなければなりません。
- その相互作用が、明確に協力的、共有的、援助的である必要があります。
- 一緒に運ぶ、助け合う、押し合うのではなく押して運ぶ、共同で何かを行う、といった場面が典型です。
- 同じ共同行為が攻撃的でもある場合は、[`AG`](ref://scoring-input/special-score/AG)と併記されることがあります。

## 注意点・鑑別

- 二者が並んでいるだけ、向かい合っているだけでは`COP`にはなりません。
- 見つめ合う、話している、一緒にいるだけでは、協力行為が明確でない限り不足です。
- `COP`が付いても、自動的に[`GHR`](ref://scoring-input/gphr/GHR)が決まるわけではありません。
