# [コーディング/特殊スコア] DR2

## 別名・検索語

- DR2
- 重度の逸脱反応
- Deviant Response 2

## 中核的定義

`DR2` は、反応全体が `DR1` より明らかに強く逸脱し、まとまりのある流れから大きく外れているときに付ける特殊スコアです。
要点は、反応全体の構成がかなり乱れて見えることです。

## 採点・適用条件

- 反応全体に強く広い乱れがあり、軽度の逸脱ではないときに検討します。
- 乱れは [`DR1`](ref://scoring-input/special-score/DR1) よりも重い水準で見えます。
- 問題が単語や短い句に限られるなら [`DV1`](ref://scoring-input/special-score/DV1) や [`DV2`](ref://scoring-input/special-score/DV2) を先に見直します。
- 後の解釈では [`Sum6`](ref://result-interpretation/lower-section/ideation/Sum6)、[`Lv2`](ref://result-interpretation/lower-section/ideation/Lv2)、[`WSum6`](ref://result-interpretation/lower-section/ideation/WSum6) に反映されます。

## 注意点・鑑別

- 反応が長い、複雑というだけで `DR2` にしません。
- `DR2` には、反応全体に及ぶはっきりした崩れが必要です。
- 水準2の重さが明確でないなら、無理に `DR2` にしません。
