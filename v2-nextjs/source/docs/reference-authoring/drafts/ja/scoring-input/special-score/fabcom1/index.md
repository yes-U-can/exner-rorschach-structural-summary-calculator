# Document Name: [符号化/特殊得点] FABCOM1

## Aliases / Search Terms

- FABCOM1
- 軽度の作話的結合
- Fabulized Combination 1

## Core Definition

`FABCOM1` は、反応内の要素同士の関係づけが作話的で無理だが、まだ重度ではないときに付ける特殊得点です。  
ひとつの物体の中の不調和というより、場面や要素同士のつながり方そのものが不自然な場合に使います。

## Application Conditions

- 反応の中で、要素間の関係づけがありそうでなさそうな、無理な結びつきになっているときに使います。
- 問題の中心は、要素の結びつけ方の作話性にあります。
- 奇妙さは見えるが、まだ `FABCOM2` ほど強くないときに使います。
- 水準1か2か迷うときは、より重い歪みだけを `FABCOM2` に残します。

## Cautions / Distinctions

- 問題がひとつの知覚単位の中の不調和な組み合わせなら、まず `INCOM1` を見直します。
- 両立しない知覚がひとつに融合しているなら `CONTAM` を見直します。
- 想像的、物語的というだけでは `FABCOM1` にしません。
- 大事なのは、要素同士の関係づけが無理に作られていることです。
