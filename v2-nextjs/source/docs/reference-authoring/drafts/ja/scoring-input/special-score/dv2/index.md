# Document Name: [符号化/特殊得点] DV2

## Aliases / Search Terms

- DV2
- 重度の逸脱言語化
- Deviant Verbalization 2

## Core Definition

`DV2` は、言語表現の逸脱が `DV1` よりもはっきり強く、かなり不適切または奇妙に見えるときに付ける特殊得点です。  
要点は、言語の崩れが軽度の違和感ではなく、明確な重さを持っていることです。

## Application Conditions

- 言い方の奇妙さや不適切さが強く、軽い逸脱では済まないときに検討します。
- 問題の中心はやはり言語表現ですが、[`DV1`](ref://scoring-input/special-score/DV1) より重い水準です。
- 反応全体が広く乱れているようなら [`DR2`](ref://scoring-input/special-score/DR2) も見直します。
- 後の解釈では [`Sum6`](ref://result-interpretation/lower-section/ideation/Sum6)、[`Lv2`](ref://result-interpretation/lower-section/ideation/Lv2)、[`WSum6`](ref://result-interpretation/lower-section/ideation/WSum6) に反映されます。

## Cautions / Distinctions

- 重さが水準2だとはっきり言えないなら、無理に `DV2` にしません。
- 長い、複雑、難しい言い回しというだけでは `DV2` になりません。
- `DV2` は言語の逸脱に中心があり、反応全体の大きな崩れを示す `DR2` とは区別します。
