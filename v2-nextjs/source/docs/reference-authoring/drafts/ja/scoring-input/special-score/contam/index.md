# Document Name: [符号化/特殊得点] CONTAM

## Aliases / Search Terms

- CONTAM
- 汚染
- Contamination

## Core Definition

`CONTAM` は、両立しない二つの知覚がひとつの最終知覚の中に融合してしまったときに付ける特殊得点です。  
単に奇妙な混合ではなく、ひとつの反応の中で不可能な融合が起きていることが中心です。

## Application Conditions

- 本来は別々であるはずの二つの知覚が、ひとつの知覚として汚染的に結びついているときに使います。
- 重要なのは、比較や並置ではなく「ひとつの融合した知覚」として語られていることです。
- その融合自体のために、反応が不可能または自己矛盾的に聞こえる必要があります。
- `CONTAM` はかなり重い歪みで、後の解釈では [`Sum6`](ref://result-interpretation/lower-section/ideation/Sum6) や [`WSum6`](ref://result-interpretation/lower-section/ideation/WSum6) に重く影響します。

## Cautions / Distinctions

- 主な問題が不調和結合であって、融合そのものではないなら `INCOM2` を見直します。
- 主な問題が要素間の作話的関係づけなら `FABCOM2` を見直します。
- 想像的、象徴的というだけでは `CONTAM` にしません。
- 実際に知覚が汚染されて融合している必要があります。
