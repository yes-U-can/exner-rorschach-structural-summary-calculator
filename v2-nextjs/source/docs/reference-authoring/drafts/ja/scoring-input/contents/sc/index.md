# [コーディング/内容] Sc

## 別名・検索語

- Sc
- 科学
- 技術
- 機械

## 中核的定義

`Sc`は、反応が科学的・技術的・機械的な物として把握されたときに付ける内容コードです。
装置、機械、器具、技術製品として見られていることが中心になります。

## 採点・適用条件

- 顕微鏡、ロボット、機械、兵器、乗り物、実験器具、装置などに使います。
- 一般的な技術製品でも、装置や仕組みとして見ているなら`Sc`を検討します。
- 芸術的・装飾的な側面が強い場合は、[`Art`](ref://scoring-input/contents/Art)の併用を検討します。
- 珍しい内容でも、科学的・技術的に読めるなら、まず[`Id`](ref://scoring-input/contents/Id)ではなく`Sc`を検討します。

## 注意点・鑑別

- ただ「変わっている」だけでは`Sc`にはなりません。
- 試験管や機械部品などは、奇妙でもまず[`Sc`](ref://scoring-input/contents/Sc)です。
- 美術作品や象徴物としての読みが強いなら、[`Art`](ref://scoring-input/contents/Art)のほうが適切なことがあります。
