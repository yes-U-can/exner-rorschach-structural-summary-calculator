# Document Name: [コーディング/内容] Art

## Aliases / Search Terms

- Art
- 芸術
- 美術品
- 装飾物

## Core Definition

`Art` は、主内容が芸術作品、装飾物、美的産物として知覚されたときに付ける内容コードです。  
単に物を見たのではなく、芸術的・装飾的なものとして理解している点が重要です。

## Application Conditions

- 絵画、彫像、彫刻、模様、紋章、装飾品などに使います。
- 対象そのものと芸術的性質の両方を述べているなら、他の内容コードと併記できます。
- 主眼が解剖や生物構造なら [`An`](ref://scoring-input/contents/An) を先に見直します。
- 芸術作品というより歴史的・民族的・文化的参照が主なら [`Ay`](ref://scoring-input/contents/Ay) を先に見直します。

## Cautions / Distinctions

- `Art` と [`Ay`](ref://scoring-input/contents/Ay) は併記されることがありますが、同じ意味ではありません。
- `Art` は作品性・装飾性が中心で、`Ay` は文化的・歴史的文脈が中心です。
- 標本風・図解風の表現では [`An`](ref://scoring-input/contents/An) が主コードで `Art` が補助になることがあります。
