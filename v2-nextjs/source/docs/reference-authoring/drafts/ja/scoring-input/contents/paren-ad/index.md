# Document Name: [コーディング/内容] (Ad)

## Aliases / Search Terms

- (Ad)
- 空想動物部分
- 架空動物部分
- 神話的動物部分

## Core Definition

`(Ad)` は、空想的・神話的・架空の動物の部分が述べられたときに付ける内容コードです。  
動物クラスターの中で、部分性と空想性の両方を持つ位置づけです。

## Application Conditions

- 反応は動物全体ではなく、動物の一部を指していなければなりません。
- その対象は空想的・神話的・架空である必要があります。
- 典型例は、ドラゴンの翼、怪物の爪、神話的獣の角などです。
- 空想動物全体であれば [`(A)`](ref://scoring-input/contents/%28A%29) を先に検討します。
- 現実動物の部分であれば [`Ad`](ref://scoring-input/contents/Ad) を先に検討します。

## Cautions / Distinctions

- `(Ad)` と [`(A)`](ref://scoring-input/contents/%28A%29) は部分か全体かで分けます。
- `(Ad)` と [`Ad`](ref://scoring-input/contents/Ad) は空想か現実かで分けます。
- 空想的であっても、ここでは動物部分です。人間部分に見えるなら [`Hd`](ref://scoring-input/contents/Hd) と比較します。
