# Document Name: [コーディング/内容] Hd

## Aliases / Search Terms

- Hd
- 人物部分
- 人の一部
- 部分的人物

## Core Definition

`Hd` は、反応が人物全体ではなく人物の一部だけを指すときに付ける内容コードです。  
手、足、頭部、胴体の一部など、人間の断片的記述が対象です。

## Application Conditions

- 主対象が人物全体ではなく部分である必要があります。
- その部分は現実的人間の一部として理解されていなければなりません。
- 例として、手、腕、脚、指、頭、上半身の一部などが含まれます。
- 全体人物としてまとまるなら [`H`](ref://scoring-input/contents/H) に戻って見直します。

## Cautions / Distinctions

- `Hd` は [`H`](ref://scoring-input/contents/H) と違います。ここでは全体人物がありません。
- `Hd` は [`(Hd)`](ref://scoring-input/contents/%28Hd%29) と違います。`(Hd)` は空想的人物の部分です。
- `Hd` は [`Hx`](ref://scoring-input/contents/Hx) と違います。`Hx` は人間経験への言及です。
- `Hd` は人物表象の質と関わることが多いので、必要なら [`PHR`](ref://scoring-input/gphr/PHR) も見直します。
