# [コーディング/内容] Ge

## 別名・検索語

- Ge
- 地図
- 地理図
- 地理的配置

## 中核的定義

`Ge`は、反応が地図、地理図、地理的な配置図として把握されたときに付ける内容コードです。
実際の風景そのものではなく、「地図として読まれていること」が核心です。

## 採点・適用条件

- 世界地図、地域図、地形図、案内図のように、地理情報を示す図として理解されているときに使います。
- 特定の地域の地図でも、一般的な地図でも構いません。
- 重要なのは「風景」ではなく「地図表現」であることです。
- 山や海岸、景色の場面として見ているだけなら、まず[`Ls`](ref://scoring-input/contents/Ls)を再検討します。

## 注意点・鑑別

- `Ge`は[`Ls`](ref://scoring-input/contents/Ls)と混同しやすいですが、`Ge`は地図、`Ls`は風景です。
- 形が国境線っぽい、島っぽいというだけでは足りません。
- 空間配置の装飾や抽象模様を、地図として読んでいない場合は`Ge`にしません。
