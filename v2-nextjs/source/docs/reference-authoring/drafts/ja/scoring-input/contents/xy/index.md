# Document Name: [採点/内容] Xy

## Aliases / Search Terms

- Xy
- X線
- レントゲン
- 放射線画像

## Core Definition

`Xy`は、反応がX線写真やレントゲン画像として把握されたときに付ける内容コードです。  
対象そのものよりも、「X線として見ていること」に重心があります。

## Application Conditions

- X線写真、レントゲン像、透視画像として明確に述べられたときに使います。
- 被検者が「X線のように見える」と実際に示している必要があります。
- `Xy`を付けたときは、[`An`](ref://scoring-input/contents/An)を同時に加えません。
- 後の解釈では[`An + Xy`](ref://result-interpretation/lower-section/selfPerception/An_Xy)と結びつきます。

## Cautions / Distinctions

- `Xy`は[`An`](ref://scoring-input/contents/An)と同じではありません。`An`は解剖、`Xy`はX線的形式です。
- 骨や内臓を言っただけで、X線だと述べていなければ、まず`An`を再検討します。
- 同一反応で`Xy`が付く場合、`An`は併記しないという点が重要です。
