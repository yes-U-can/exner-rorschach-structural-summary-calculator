# [コーディング/内容] Id

## 別名・検索語

- Id
- 特異内容
- 個別的内容
- 特異的反応内容

## 中核的定義

`Id`は、標準的な内容カテゴリーのどれにも十分には当てはまらないときに、最後の手段として使う内容コードです。
「変わっている内容」一般のコードではなく、他の選択肢を検討した後の例外的コードです。

## 採点・適用条件

- 他の内容カテゴリーにより適切に入らないことを確認してから使います。
- `Id`を使うときは、反応内容をできるだけ原文に近く記録しておくのが望ましいです。
- 変わった内容でも、科学技術的に読めるなら、まず[`Sc`](ref://scoring-input/contents/Sc)を再検討します。
- 芸術作品や象徴物として成り立つなら、まず[`Art`](ref://scoring-input/contents/Art)を再検討します。

## 注意点・鑑別

- 奇妙に見えるだけで自動的に`Id`にはなりません。
- 試験管、機械、装置などは、まず[`Sc`](ref://scoring-input/contents/Sc)で検討します。
- 装飾物や象徴物なら、[`Art`](ref://scoring-input/contents/Art)のほうが適切なことがあります。
- `Id`は便利な逃げ道ではなく、管理された例外として使う必要があります。
