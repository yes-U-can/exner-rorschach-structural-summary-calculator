# [コーディング/内容] Cg

## 別名・検索語

- Cg
- 衣服
- 着衣
- 身につけるもの

## 中核的定義

`Cg` は、主内容が衣服、衣類、または身につける装身具として知覚されたときに付ける内容コードです。  
身体に着けるものとして理解されていることが決定点です。

## 採点・適用条件

- 帽子、ブーツ、手袋、マント、ドレス、制服など、明確な衣類に使います。
- 身体に着ける装身具が服飾の一部として扱われる場合も使えます。
- 歴史的・文化的参照が強ければ [`Ay`](ref://scoring-input/contents/Ay) の併記を検討します。
- 装飾性や芸術性が主なら [`Art`](ref://scoring-input/contents/Art) の併記を検討します。

## 注意点・鑑別

- `Cg` は単なる物体一般には使いません。衣服・装着物として同定されている必要があります。
- `Cg` と [`Ay`](ref://scoring-input/contents/Ay) は併記可能ですが、`Ay` には文化的・歴史的枠組みが必要です。
- `Cg` と [`Art`](ref://scoring-input/contents/Art) も併記可能ですが、`Art` には作品性・装飾性が必要です。
