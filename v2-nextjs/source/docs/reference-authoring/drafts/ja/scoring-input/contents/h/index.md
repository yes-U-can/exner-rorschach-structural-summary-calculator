# [コーディング/内容] H

## 別名・検索語

- H
- 人間
- 全体人物
- 現実的人物

## 中核的定義

`H` は、反応が現実的な全体人物を指すときに付ける内容コードです。
身体部分ではなく、一人の人間全体として知覚されている必要があります。

## 採点・適用条件

- 主対象が人物全体として述べられている必要があります。
- その人物は空想的存在ではなく、現実的人間として扱われていなければなりません。
- 歴史的・文化的に特定された人物像であれば、必要に応じて [`Ay`](ref://scoring-input/contents/Ay) を二次内容として併記します。
- 身体の一部だけなら [`Hd`](ref://scoring-input/contents/Hd) を再検討します。

## 注意点・鑑別

- `H` は [`(H)`](ref://scoring-input/contents/%28H%29) と違います。`(H)` は空想的・神話的人物全体です。
- `H` は [`Hd`](ref://scoring-input/contents/Hd) と違います。`Hd` は人物部分です。
- 人間内容があるだけで [`GHR`](ref://scoring-input/gphr/GHR) や [`PHR`](ref://scoring-input/gphr/PHR) は決まりません。
