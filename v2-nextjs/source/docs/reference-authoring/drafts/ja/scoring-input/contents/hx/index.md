# Document Name: [コーディング/内容] Hx

## Aliases / Search Terms

- Hx
- 人間経験
- 人間感情
- 人間的状態

## Core Definition

`Hx` は、人間そのものではなく、人間の感情・感覚・経験が明確に表現されたときに付ける内容コードです。  
焦点は人物像ではなく、人間的経験へのはっきりした帰属にあります。

## Application Conditions

- 被検者が感情、感覚、内的状態、人間的経験を明確に言い表している必要があります。
- その帰属は、検査者の推測ではなく、反応の中ではっきり述べられていなければなりません。
- 人物反応に二次的に付くこともありますが、中心が経験なら `Hx` を検討します。
- 象徴的で抽象度の高い表現では [`AB`](ref://scoring-input/special-score/AB) の見直しが役立つことがあります。

## Cautions / Distinctions

- `Hx` は [`H`](ref://scoring-input/contents/H) や [`Hd`](ref://scoring-input/contents/Hd) と違います。これらは人物像を扱います。
- 「怒っていそう」「疲れていそう」のような曖昧な印象だけでは足りない場合があります。
- `Hx` は検査者の読み込みだけで付けてはいけません。反応の中で人間経験が明示されている必要があります。
- 人間経験を象徴化しすぎた反応では [`AB`](ref://scoring-input/special-score/AB) との区別を確認します。
