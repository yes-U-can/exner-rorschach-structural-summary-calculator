# Document Name: [コーディング/内容] Bl

## Aliases / Search Terms

- Bl
- 血
- 出血

## Core Definition

`Bl` は、主内容が血そのもの、または出血として知覚されたときに付ける内容コードです。  
完全な解剖構造ではなく、血自体や血が流れているという考えが中心です。

## Application Conditions

- 血、血の流れ、血痕、出血といった表現に使います。
- その血が人間由来でも動物由来でもコードできます。
- 反応の中心が骨、器官、身体構造なら [`An`](ref://scoring-input/contents/An) を先に見直します。
- 血が人物や動物の一部として述べられる場合は、必要に応じて [`H`](ref://scoring-input/contents/H) や [`A`](ref://scoring-input/contents/A) と併記できます。

## Cautions / Distinctions

- `Bl` は色彩コードではありません。色 determinant の代わりにはなりません。
- `Bl` と [`An`](ref://scoring-input/contents/An) はどちらも生物学的に見えても、`Bl` は血、`An` は解剖構造が中心です。
- ただ赤く見えるだけでは足りません。被検者が血、またはそれに等しい内容を明示している必要があります。
