# [コーディング/内容] Ad

## 別名・検索語

- Ad
- 動物部分
- 動物の一部
- 部分動物

## 中核的定義

`Ad` は、動物全体ではなく、動物の一部や細部だけが述べられたときに付ける内容コードです。  
尾、頭、翼、爪など、部分的な動物特徴が中心になります。

## 採点・適用条件

- 反応の焦点が動物全体ではなく、その一部にある必要があります。
- その部分は外的特徴でも構いませんが、あくまで部分的動物特徴として理解されていなければなりません。
- 現実的な全体動物なら [`A`](ref://scoring-input/contents/A) を先に検討します。
- 空想動物の部分なら [`(Ad)`](ref://scoring-input/contents/%28Ad%29) を先に検討します。
- 解剖学的・内的構造的な焦点なら [`An`](ref://scoring-input/contents/An) を先に検討します。

## 注意点・鑑別

- `Ad` と [`A`](ref://scoring-input/contents/A) は部分か全体かで分けます。
- `Ad` と [`(Ad)`](ref://scoring-input/contents/%28Ad%29) は現実か空想かで分けます。
- `Ad` と [`Hd`](ref://scoring-input/contents/Hd) はどちらも部分コードですが、こちらは動物、`Hd` は人間です。
- 生物片すべてが `Ad` ではありません。解剖学的焦点が強ければ [`An`](ref://scoring-input/contents/An) の方が適切です。
