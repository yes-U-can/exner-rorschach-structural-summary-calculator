# [コーディング/内容] Ay

## 別名・検索語

- Ay
- 文化内容
- 歴史内容
- 人類学的内容

## 中核的定義

`Ay` は、特定の歴史的・民族的・文化的・人類学的参照を含むときに付ける内容コードです。  
人や物が見えただけでは足りず、文化的枠組みがはっきりしている必要があります。

## 採点・適用条件

- 文化圏、時代、文明に結びついた人物、衣服、武器、象徴、場面などに使います。
- 主対象が人や動物であれば、[`H`](ref://scoring-input/contents/H), [`(H)`](ref://scoring-input/contents/%28H%29), [`A`](ref://scoring-input/contents/A) と併記されることがあります。
- 判断の中心は見た目の珍しさではなく、文化的・歴史的な特定性です。
- 芸術作品や装飾品としての性格が主なら [`Art`](ref://scoring-input/contents/Art) を先に見直します。

## 注意点・鑑別

- `Ay` と [`Art`](ref://scoring-input/contents/Art) は近く見えても、`Ay` は文化的・歴史的文脈に依存します。
- 「異国風」に聞こえるだけでは `Ay` にしません。実際に同定可能な文化参照が必要です。
- `Ay` は単独コードよりも、他の内容コードへの追加コードとして働くことが多いです。
