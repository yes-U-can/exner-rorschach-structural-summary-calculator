# [コーディング/内容] Fi

## 別名・検索語

- Fi
- 火
- 煙

## 中核的定義

`Fi` は、主内容が火または煙として知覚されたときに付ける内容コードです。  
反応が炎、火事、煙、煙の柱としてまとめられている必要があります。

## 採点・適用条件

- 炎、火、燃焼、煙、煙の柱などが明確に述べられたときに使います。
- 主眼は火や煙そのものに置かれていなければなりません。
- 爆発や爆風が中心なら [`Ex`](ref://scoring-input/contents/Ex) を先に見直します。
- 雲として述べているなら [`Cl`](ref://scoring-input/contents/Cl) を先に見直します。

## 注意点・鑑別

- `Fi` と [`Ex`](ref://scoring-input/contents/Ex) は強さではなく、名指された内容で分けます。
- `Fi` と [`Cl`](ref://scoring-input/contents/Cl) は拡散形で似て見えても、一方は火/煙、もう一方は雲です。
- 赤や暗い色があるだけでは `Fi` を付けません。被検者が火または煙と言っている必要があります。
