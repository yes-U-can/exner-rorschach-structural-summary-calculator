# Document Name: [コーディング/内容] (Hd)

## Aliases / Search Terms

- (Hd)
- 空想的人物部分
- 神話的人物部分
- 想像上の人間部分

## Core Definition

`(Hd)` は、空想的・神話的・非現実的人物の部分だけが述べられたときに付ける内容コードです。  
`Hd` の空想版と考えると分かりやすいです。

## Application Conditions

- 主対象が人物全体ではなく部分である必要があります。
- その部分は、空想的・神話的・非現実的な人物に属していなければなりません。
- 例として、悪魔の頭、天使の目、魔女の腕、ファンタジー人物の手などが入ります。
- 全体人物として述べられていれば [`(H)`](ref://scoring-input/contents/%28H%29) を再検討します。

## Cautions / Distinctions

- `(Hd)` は [`Hd`](ref://scoring-input/contents/Hd) と違います。差は空想性・神話性にあります。
- `(Hd)` は [`(H)`](ref://scoring-input/contents/%28H%29) と違います。`(H)` は全体人物です。
- ただ奇妙な部分に見えるだけでは `(Hd)` にしません。実際に非現実的人物の部分として答えられている必要があります。
- 人物表象の質を見るときには [`PHR`](ref://scoring-input/gphr/PHR) の再検討が役立つことがあります。
