# [コーディング/内容] Sx

## 別名・検索語

- Sx
- 性的内容
- 性器
- 性的反応

## 中核的定義

`Sx`は、内容に性器、性的行為、または明確な性的意味が含まれるときに付ける内容コードです。
多くの場合、人間内容や解剖内容に追加される二次的な内容コードとして使われます。

## 採点・適用条件

- 性器、性的に強調された乳房、性交、明確な性的行為などに使います。
- 性的意味は被検者の反応の中に実際に示されていなければなりません。
- [`H`](ref://scoring-input/contents/H)、[`Hd`](ref://scoring-input/contents/Hd)、[`An`](ref://scoring-input/contents/An)と併記されることがあります。
- 単なる裸体や身体の形だけで、明確な性的意味がなければ自動的に`Sx`にはしません。

## 注意点・鑑別

- `Sx`は[`An`](ref://scoring-input/contents/An)とは違い、解剖ではなく性的意味に重心があります。
- `Sx`だけで全体的人間内容になるわけではなく、[`H`](ref://scoring-input/contents/H)や[`Hd`](ref://scoring-input/contents/Hd)が必要なことがあります。
- 検査者の推測だけで性的意味を付けてはいけません。
