# Document Name: [コーディング/内容] (H)

## Aliases / Search Terms

- (H)
- 空想的人物
- 神話的人物
- 想像上の人間

## Core Definition

`(H)` は、反応が空想的・神話的・非現実的な全体人物を指すときに付ける内容コードです。  
人物全体ではあるものの、普通の現実的人間ではありません。

## Application Conditions

- 主対象が人物全体として述べられている必要があります。
- その人物は神話・空想・フィクション・超自然など、非現実的存在として扱われていなければなりません。
- 例として、天使、悪魔、幽霊、妖精、巨人、SF 的な humanoid などが含まれます。
- 部分だけを述べるなら [`(Hd)`](ref://scoring-input/contents/%28Hd%29) を再検討します。

## Cautions / Distinctions

- `(H)` は [`H`](ref://scoring-input/contents/H) と違います。`H` は現実的人物全体です。
- `(H)` は [`(Hd)`](ref://scoring-input/contents/%28Hd%29) と違います。`(Hd)` は空想的人物の部分です。
- 奇妙に見えるというだけでは `(H)` にしません。実際に非現実的人物として答えられている必要があります。
- [`GHR`](ref://scoring-input/gphr/GHR) と [`PHR`](ref://scoring-input/gphr/PHR) は別の判定段階です。
