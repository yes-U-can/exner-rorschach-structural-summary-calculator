# Document Name: [採点/内容] Hh

## Aliases / Search Terms

- Hh
- 家庭用品
- 日用品
- 家庭内物品

## Core Definition

`Hh`は、反応の主内容が家庭内で使う器具や日用品として把握されたときに付ける内容コードです。  
衣類でも芸術作品でもなく、「家庭用品として読まれていること」がポイントです。

## Application Conditions

- コップ、皿、鍋、ランプ、台所用品、家庭用の道具などに使います。
- 家庭内で使う物であることが、反応の中心内容である必要があります。
- 装飾品や工芸品としての意味が前面に出るなら、[`Art`](ref://scoring-input/contents/Art)の併用や再検討が必要です。
- 動物の一部から作られた道具として強調されるなら、[`Ad`](ref://scoring-input/contents/Ad)と併記されることがあります。

## Cautions / Distinctions

- 衣類や身につける物は[`Cg`](ref://scoring-input/contents/Cg)をまず検討します。
- `Hh`と[`Art`](ref://scoring-input/contents/Art)は近づくことがありますが、同じではありません。
- `Hh`と[`Ad`](ref://scoring-input/contents/Ad)の重なりは、動物由来の家庭用品として表現されたときに限られます。
