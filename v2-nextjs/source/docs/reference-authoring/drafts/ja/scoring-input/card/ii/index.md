# Document Name: [符号化/カード] II

## Aliases / Search Terms

- Card II
- カードII
- カード2
- 図版II

## Core Definition

`Card II` は、早い段階で赤色部分が目立つカードです。  
このページは、その色の存在が符号化にどう影響しやすいかを整理するもので、自動ルールの一覧ではありません。

## Application Conditions

- 赤が実際に知覚形成に関わっているなら `FC`、`CF`、`C` を見直します。
- 色が位置情報として触れられただけで、知覚形成に使われていないなら色彩決定因は加えません。
- 暗部、明部、赤部の配置がまとまりの検討を必要にしやすいので `Z` も見直します。
- 人や動物が関係しているように見えても、運動は別に確認が必要です。

## Cautions / Distinctions

- 赤が見えたというだけで色コードを付けません。
- このカードは複数部分を見やすいので、ひとつの反応なのか複数反応なのかも確認します。
- 二つの人物が見えるだけでは `COP` や `AG` は付きません。
