# [コーディング/カード] IX

## 別名・検索語

- Card IX
- カードIX
- カード9
- 図版IX

## 中核的定義

`Card IX` は、良い形態を保つために比較的高い構成力を要しやすいカードです。
このページは、人間内容、運動、形式水準を見るときの焦点をまとめます。

## 採点・適用条件

- 人や人型に近い反応が出ることがあるので、`H` と `(H)` を見直します。
- 明確な人間行動が含まれるなら `M` を見直します。
- このカードでは形態が不安定になりやすいので、`FQ` を丁寧に見直します。
- 平凡反応が見えても、なじみだけでなく正式基準で確認します。

## 注意点・鑑別

- 人間らしく見えるだけでは、自動的に `H` にはしません。`(H)` のほうが合うこともあります。
- このカードの構成の難しさだけで、反応を不良形態と決めません。
- このページはカード解釈ではなく、符号化の見直し用です。
