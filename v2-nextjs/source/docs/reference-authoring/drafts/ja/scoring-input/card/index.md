# Document Name: [符号化/カード] Card

## Aliases / Search Terms

- Card
- カード
- 図版
- インクブロットカード

## Core Definition

`Card` は、包括システムの10枚のカードを見分けるための案内ページです。  
この系列はカードの象徴解釈をするためではなく、どのような反応がそのカードで出やすいかを符号化の補助として整理するために使います。

## Application Conditions

- カード文書は、よく出やすい反応、`Popular` の可能性、位置の傾向、決定因の確認ポイントを思い出すために使います。
- 実際の符号化は、あくまで被検者が本当に何を言ったかに基づいて決めます。
- 同じカードでも、実際の記述が違えばコードも変わります。
- route の骨組みは全言語で同じですが、本体と相互参照は言語ごとに変わり得ます。

## Cautions / Distinctions

- カード文書はブロットの象徴的意味を説明するページではありません。
- 「このカードだから自動的にこのコード」というルールとして読まないでください。
- カード番号の文字は他のコード記号と似ることがあるので、文脈を必ず明示して読みます。
- たとえば `Card V` は決定因 `V` とは別物です。
