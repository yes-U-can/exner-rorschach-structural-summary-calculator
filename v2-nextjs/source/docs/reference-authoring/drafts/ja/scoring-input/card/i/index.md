# Document Name: [符号化/カード] I

## Aliases / Search Terms

- Card I
- カードI
- カード1
- 図版I

## Core Definition

`Card I` はプロトコルの最初のカードです。  
このページは、開始時に出やすい反応を符号化の観点から整理するもので、カードの象徴解釈をするページではありません。

## Application Conditions

- `Card I` では全体反応の `W` が出やすいです。
- コウモリ、チョウ、広い動物形など、単純な形態反応が出やすいです。
- ブロット全体を使い、まとまりがよければ `F`、`FQ`、`Popular` を見直します。
- 小さい部分だけを使った場合、カードだけを理由に `Popular` にはしません。

## Cautions / Distinctions

- なじみのある反応に聞こえるだけで `Popular` を付けません。正式基準を満たす必要があります。
- `Card I` と `Card V` はコウモリやチョウ型の反応を共有しやすいですが、同じカードとして扱いません。
- 暗い色を言葉で触れただけでは、別の決定因は加えません。
