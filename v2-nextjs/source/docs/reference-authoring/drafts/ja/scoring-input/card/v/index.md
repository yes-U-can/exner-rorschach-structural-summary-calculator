# Document Name: [符号化/カード] V

## Aliases / Search Terms

- Card V
- カードV
- カード5
- 図版V

## Core Definition

`Card V` は、比較的はっきりした形態反応と全体の popular 反応を促しやすいカードです。  
このページは、そのカードで特に見直しやすい符号化ポイントをまとめます。

## Application Conditions

- `W` 反応が出やすいです。
- ブロット形態が比較的明瞭なので、`F` と `FQ` を見直しやすいです。
- このカードの popular 反応には、コウモリやチョウが含まれやすいです。
- ブロット全体を使い、まとまりがよければ、形式判断と一緒に `Popular` を見直します。

## Cautions / Distinctions

- `Card V` は Vista の決定因 `V` とは別物です。
- `Card I` と `Card V` は典型反応を一部共有しますが、同じカードとしてまとめません。
- 暗い、左右対称だと述べたというだけで、別の決定因を加えません。
