# [コーディング/カード] V

## 別名・検索語

- Card V
- カードV
- カード5
- 図版V

## 中核的定義

`Card V` は、比較的はっきりした形態反応と全体の平凡反応を促しやすいカードです。
このページは、そのカードで特に見直しやすい符号化ポイントをまとめます。

## 採点・適用条件

- `W` 反応が出やすいです。
- ブロット形態が比較的明瞭なので、`F` と `FQ` を見直しやすいです。
- このカードの平凡反応には、コウモリやチョウが含まれやすいです。
- ブロット全体を使い、まとまりがよければ、形式判断と一緒に `Popular` を見直します。

## 注意点・鑑別

- `Card V` はビスタの決定因 `V` とは別物です。
- `Card I` と `Card V` は典型反応を一部共有しますが、同じカードとしてまとめません。
- 暗い、左右対称だと述べたというだけで、別の決定因を加えません。
