# [コーディング/形態水準] o

## 別名・検索語

- FQo
- 通常の形態水準
- 標準的に良好な適合

## 中核的定義

`FQo` は、反応がインクに対して妥当で、標準的な良好適合を示すときに用います。
もっとも一般的な「良好な FQ」区分です。

## 採点・適用条件

- 対象が比較的すばやく、もっともらしく見えます。
- 大きなゆがみや無理な輪郭化なしに適合しています。
- `FQ+` といえるほどの特別な精密さはありません。
- [`FQu`](ref://scoring-input/fq/u) よりも明らかに安定した適合です。

## 注意点・鑑別

- 通常より精密に形態が組み立てられているなら [`FQ+`](ref://scoring-input/fq/%2B) を使います。
- 珍しいがまだ妥当なら [`FQu`](ref://scoring-input/fq/u) を検討します。
- `FQo` は [`DQo`](ref://scoring-input/dq/o) と同じではありません。片方は形態適合、もう片方は発達水準です。
