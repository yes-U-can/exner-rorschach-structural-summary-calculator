# Document Name: [採点/形態水準] +

## Aliases / Search Terms

- FQ+
- 高い形態適合
- 精緻で良好な適合

## Core Definition

`FQ+` は、反応が良好な形態適合を示し、しかも通常の良好反応よりも形態がより精密に記述されているときに用います。  
単に合っているだけでなく、形態の明確さが一段高い反応です。

## Application Conditions

- まず基本的に良好な形態適合があります。
- 被検者が形態特徴を通常より精密または豊かに表現しています。
- その精密さが適合を損なわず、むしろ支えています。
- 通常の良好反応で足りるなら [`FQo`](ref://scoring-input/fq/o) を使います。

## Cautions / Distinctions

- 複雑に聞こえるだけでは `FQ+` にしません。形態面での向上が必要です。
- 良好だが特に精密ではないなら [`FQo`](ref://scoring-input/fq/o) を使います。
- `FQ+` は [`DQ+`](ref://scoring-input/dq/%2B) と同じではありません。片方は形態適合、もう片方は組織化水準です。
