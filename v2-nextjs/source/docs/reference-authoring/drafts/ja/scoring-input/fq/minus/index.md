# [コーディング/形態水準] -

## 別名・検索語

- FQ-
- 低い形態適合
- 不良な形態適合

## 中核的定義

`FQ-` は、述べられた対象がインク領域に十分適合せず、輪郭を無理に使ったり、存在しない特徴を作っているときに与えます。
不良または歪んだ形態適合を示す区分です。

## 採点・適用条件

- 対象がすぐには自然に見えません。
- 反応が恣意的な輪郭づけや大きな歪曲に依存しています。
- 表や外挿によっても妥当な適合として支持しにくいです。
- 表外の迷うケースでは、適合を高く見積もりすぎないよう注意します。

## 注意点・鑑別

- 単に珍しいだけでは `FQ-` にしません。中心問題は適合の悪さです。
- 珍しいがまだ妥当なら [`FQu`](ref://scoring-input/fq/u) を使います。
- `FQ-` が 1 つ 2 つあるだけで解釈を決めつけません。頻度とパターンが重要です。
