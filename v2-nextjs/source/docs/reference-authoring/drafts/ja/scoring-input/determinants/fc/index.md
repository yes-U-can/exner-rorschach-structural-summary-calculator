# Document Name: [コーディング/決定因] FC

## Aliases / Search Terms

- FC
- 形態-色彩
- 色より形が優勢

## Core Definition

`FC` は、反応が主に形によって支えられ、色彩は二次的に加わっているときにつける determinant です。
色は関与しているが、対象同定の主軸は形にあります。

## Application Conditions

- 形が対象同定を主に導いている。
- 色彩も反応の一部だが、重要度は二次的である。
- 色が形より強いなら [`CF`](ref://scoring-input/determinants/CF) を再検討する。
- 形がほとんど働いていないなら [`C`](ref://scoring-input/determinants/C) の方が適切かもしれない。

## Cautions / Distinctions

- カードに色があるだけでは `FC` にしません。形優位の反応の中で色が実際に働く必要があります。
- `CF` との違いは、どちらが主な重みを担っているかです。
- この群の解釈的意味はあとで [`FC:CF+C`](ref://result-interpretation/lower-section/affect/FC_CF_C) で検討します。
