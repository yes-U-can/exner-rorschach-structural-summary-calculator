# Document Name: [コーディング/決定因] FM

## Aliases / Search Terms

- FM
- 動物運動
- 非人間運動

## Core Definition

`FM` は、動物に固有な運動や、人間的ではない生物的活動が知覚されているときにつける determinant です。
人間運動ではなく、種にふさわしい生き物の動きを表します。

## Application Conditions

- 動きが動物的であり、人間的活動ではない。
- 走る、飛ぶ、はねる、這うなど、種にふさわしい行為である。
- 架空の生物でも、活動の質が人間的でなければ `FM` を考える。
- 判定は受検者の記述に基づいて行う。

## Cautions / Distinctions

- 人間的活動なら [`M`](ref://scoring-input/determinants/M) を使います。
- 無生物の動きは [`m`](ref://scoring-input/determinants/m) です。
- 単に動物内容があるだけでは足りません。活動知覚が必要です。
