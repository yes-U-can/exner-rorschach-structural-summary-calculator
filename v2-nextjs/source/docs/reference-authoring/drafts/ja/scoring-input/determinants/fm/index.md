# [コーディング/決定因] FM

## 別名・検索語

- FM
- 動物運動
- 非人間運動

## 中核的定義

`FM` は、動物に固有な運動や、人間的ではない生物的活動が知覚されているときにつける決定因です。
人間運動ではなく、種にふさわしい生き物の動きを表します。

## 入力コードの規則

`FM` は構造一覧表で用いる動物運動の集計ラベルです。個々の反応には、運動の能動・受動の性質に応じて `FMa`、`FMp`、`FMa-p` のいずれかを入力します。添字のない `FM` は、個々の反応の完全な入力コードとして使用しません。

## 採点・適用条件

- 動きが動物的であり、人間的活動ではありません。
- 走る、飛ぶ、はねる、這うなど、種にふさわしい行為です。
- 架空の生物でも、活動の質が人間的でなければ `FM` を検討します。
- 判定は受検者の記述に基づいて行います。

## 注意点・鑑別

- 人間的活動なら [`M`](ref://scoring-input/determinants/M) を使います。
- 無生物の動きは [`m`](ref://scoring-input/determinants/m) です。
- 単に動物内容があるだけでは足りません。活動知覚が必要です。
