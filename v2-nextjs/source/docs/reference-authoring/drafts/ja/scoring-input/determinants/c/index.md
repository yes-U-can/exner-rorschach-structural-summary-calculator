# Document Name: [コーディング/決定因] C

## Aliases / Search Terms

- C
- 純色彩
- pure color

## Core Definition

`C` は、反応が色彩そのものにほぼ全面的に支えられ、形の支えが実質的にないときにつける determinant です。
色だけで反応が出ている純色彩反応です。

## Application Conditions

- 色彩が反応の中心である。
- 形が対象同定を支えるほど働いていない。
- 形が少しでも意味ある形で支えているなら `CF` や `FC` を再検討する。

## Cautions / Distinctions

- 色が見えるだけでは `C` にしません。形の寄与が実質的にない必要があります。
- 形が補助的に残るなら [`CF`](ref://scoring-input/determinants/CF) です。
- 形が主なら [`FC`](ref://scoring-input/determinants/FC) です。
