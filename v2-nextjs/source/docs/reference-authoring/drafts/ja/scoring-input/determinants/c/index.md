# [コーディング/決定因] C

## 別名・検索語

- C
- 純色彩
- pure color

## 中核的定義

`C` は、反応が色彩そのものにほぼ全面的に支えられ、形の支えが実質的にないときにつける決定因です。
色だけで反応が出ている純色彩反応です。

## 採点・適用条件

- 色彩が反応の中心です。
- 形が対象同定を支えるほど働いていません。
- 形が少しでも意味ある形で支えているなら `CF` や `FC` を再検討します。

## 注意点・鑑別

- 色が見えるだけでは `C` にしません。形の寄与が実質的にない必要があります。
- 形が補助的に残るなら [`CF`](ref://scoring-input/determinants/CF) です。
- 形が主なら [`FC`](ref://scoring-input/determinants/FC) です。
