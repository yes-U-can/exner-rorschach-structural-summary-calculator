# [コーディング/決定因] Determinants

## 別名・検索語

- Determinants
- 決定因
- 運動
- 色彩

## 中核的定義

決定因は、その反応が「何によってそう見えたか」をコード化する軸です。
何が見えたかではなく、形・運動・色・陰影・反射・奥行きなど、どのブロット特性が反応を支えたかを記録します。

## 採点・適用条件

- 判定は受検者の言葉と質問段階に基づき、検査者の自由推測だけで決めません。
- 一つの反応に複数の決定因が実際に含まれるなら複合的にコード化できます。
- 複合反応では、反応段階で現れた順序に従って決定因を並べます。
- 主な決定因には、運動 [`M`](ref://scoring-input/determinants/M)、[`FM`](ref://scoring-input/determinants/FM)、[`m`](ref://scoring-input/determinants/m) と、色彩 [`FC`](ref://scoring-input/determinants/FC)、[`CF`](ref://scoring-input/determinants/CF)、[`C`](ref://scoring-input/determinants/C) があります。

## 注意点・鑑別

- 決定因は [`FQ`](ref://scoring-input/fq) や領域の代わりではありません。別の軸です。
- 運動では「人間的か、動物的か、無生物的か」を区別します。
- 色彩では「形が主か、色が主か」を区別して `FC`, `CF`, `C` を分けます。
- 後続の無彩色, 質感, ビスタ, 拡散陰影もすべて同じ陰影群としてまとめず、別々に扱います。
- 解釈的意味はあとで [`FC:CF+C`](ref://result-interpretation/lower-section/affect/FC_CF_C), [`Pure C`](ref://result-interpretation/lower-section/affect/PureC), [`Lambda`](ref://result-interpretation/lower-section/core/Lambda), [`FM`](ref://result-interpretation/lower-section/core/FM), [`m`](ref://result-interpretation/lower-section/core/m) などで検討します。
