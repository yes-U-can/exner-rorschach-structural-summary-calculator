# [コーディング/決定因] CF

## 別名・検索語

- CF
- 色彩-形態
- 形より色が優勢

## 中核的定義

`CF` は、色彩が主な重みをもち、形は補助的に働いているときにつける決定因です。
形は残っているが、反応を押しているのは色の方です。

## 採点・適用条件

- 色彩が反応の中心です。
- 形もある程度関与しているが、主役ではありません。
- 形がもっと優勢なら [`FC`](ref://scoring-input/determinants/FC) を検討します。
- 形が実質的に使われていないなら [`C`](ref://scoring-input/determinants/C) を検討します。

## 注意点・鑑別

- `CF` は単なる色の言及ではありません。色が反応を駆動している必要があります。
- `FC` との違いは主導要因がどちらかです。
- 形がほぼ消えているなら `CF` ではなく `C` になります。
