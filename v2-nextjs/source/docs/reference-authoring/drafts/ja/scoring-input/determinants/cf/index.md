# Document Name: [コーディング/決定因] CF

## Aliases / Search Terms

- CF
- 色彩-形態
- 形より色が優勢

## Core Definition

`CF` は、色彩が主な重みをもち、形は補助的に働いているときにつける determinant です。
形は残っているが、反応を押しているのは色の方です。

## Application Conditions

- 色彩が反応の中心である。
- 形もある程度関与しているが、主役ではない。
- 形がもっと優勢なら [`FC`](ref://scoring-input/determinants/FC) を考える。
- 形が実質的に使われていないなら [`C`](ref://scoring-input/determinants/C) を考える。

## Cautions / Distinctions

- `CF` は単なる色の言及ではありません。色が反応を駆動している必要があります。
- `FC` との違いは主導要因がどちらかです。
- 形がほぼ消えているなら `CF` ではなく `C` になります。
