# Document Name: [コーディング/決定因] FC'

## Aliases / Search Terms

- FC'
- 無彩色形態-色彩
- 形優位の無彩色

## Core Definition

`FC'` は、形が反応を主に組織化し、黒・白・灰の無彩色が二次的に働いているときにつける determinant です。
この無彩色は、単なる陰影や領域説明ではなく、色として機能していなければなりません。

## Application Conditions

- 形が対象同定の主な支えになっている。
- 無彩色特徴が色として関与しているが、重みは二次的である。
- 無彩色がより優勢なら [`C'F`](ref://scoring-input/determinants/C%27F) を再検討する。
- 形の支えがほとんどないなら [`C'`](ref://scoring-input/determinants/C%27) の方が適切かもしれない。

## Cautions / Distinctions

- `FC'` は [`FC`](ref://scoring-input/determinants/FC) と同じではありません。前者は無彩色、後者は有彩色です。
- 黒、白、灰、明るい、暗いという言葉が出ても、それだけで `FC'` にはしません。色として働く必要があります。
- 奥行きや距離の感じが中心なら [`V`](ref://scoring-input/determinants/V) を見直します。
- ぼんやりした不安感や拡散的陰影が中心なら [`Y`](ref://scoring-input/determinants/Y) を見直します。
