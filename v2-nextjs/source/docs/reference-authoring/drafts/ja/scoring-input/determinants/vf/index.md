# Document Name: [コーディング/決定因] VF

## Aliases / Search Terms

- VF
- Vista-形態
- 奥行き優位

## Core Definition

`VF` は、奥行き・距離・立体感が反応を主に支え、形は補助的に残っているときにつける determinant です。
形はあるが、反応を押しているのは深さや遠近感の方です。

## Application Conditions

- Vista が反応の主要な基盤になっている。
- 形の支えもあるが、明らかに二次的である。
- 形が主なら [`FV`](ref://scoring-input/determinants/FV) を再検討する。
- 形の支えがほとんどないなら [`V`](ref://scoring-input/determinants/V) を考える。

## Cautions / Distinctions

- `VF` は `FV` より Vista の重みが強い反応です。
- 質感が中心なら [`TF`](ref://scoring-input/determinants/TF) や [`T`](ref://scoring-input/determinants/T) を見直します。
- 拡散的陰影が中心なら [`Y`](ref://scoring-input/determinants/Y) を見直します。
