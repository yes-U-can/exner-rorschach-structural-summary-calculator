# [コーディング/決定因] VF

## 別名・検索語

- VF
- Vista-形態
- 奥行き優位

## 中核的定義

`VF` は、奥行き・距離・立体感が反応を主に支え、形は補助的に残っているときにつける決定因です。
形はあるが、反応を押しているのは深さや遠近感の方です。

## 採点・適用条件

- ビスタが反応の主要な基盤になっています。
- 形の支えもあるが、明らかに二次的です。
- 形が主なら [`FV`](ref://scoring-input/determinants/FV) を再検討します。
- 形の支えがほとんどないなら [`V`](ref://scoring-input/determinants/V) を検討します。

## 注意点・鑑別

- `VF` は `FV` よりビスタの重みが強い反応です。
- 質感が中心なら [`TF`](ref://scoring-input/determinants/TF) や [`T`](ref://scoring-input/determinants/T) を見直します。
- 拡散的陰影が中心なら [`Y`](ref://scoring-input/determinants/Y) を見直します。
