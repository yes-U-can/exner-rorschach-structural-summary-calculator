# [コーディング/決定因] T

## 別名・検索語

- T
- 純質感
- pure texture

## 中核的定義

`T` は、反応がほぼ全面的に質感・触覚的性質に依存し、形の支えが十分でないときにつける決定因です。
柔らかい、べたつく、ふわふわするなどの触覚的経験が中心になります。

## 採点・適用条件

- 質感が反応の主要基盤になっています。
- `FT` や `TF` に必要な形の支えが足りません。
- 触覚的・表面感覚が明確に述べられています。

## 注意点・鑑別

- 形の支えがあるなら [`TF`](ref://scoring-input/determinants/TF) や [`FT`](ref://scoring-input/determinants/FT) を再検討します。
- 奥行きや距離感が中心なら [`V`](ref://scoring-input/determinants/V) ではないか確認します。
- `T` の意味づけはあとで [`SumT`](ref://result-interpretation/lower-section/core/SumT) や [`SumT_inter`](ref://result-interpretation/lower-section/interpersonal/SumT_inter) で見直します。
