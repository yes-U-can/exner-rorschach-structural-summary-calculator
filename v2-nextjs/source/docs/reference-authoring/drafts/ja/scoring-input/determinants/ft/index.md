# Document Name: [コーディング/決定因] FT

## Aliases / Search Terms

- FT
- 形態-質感
- 形優位の質感

## Core Definition

`FT` は、形が反応を主に組織化し、質感や触覚的性質が二次的に加わるときにつける determinant です。
ざらざら、柔らかい、布のようだといった感触があるが、まず形で対象が分かる反応です。

## Application Conditions

- 形が主な対象同定を支えている。
- そのうえで触覚・表面・質感の性質が述べられている。
- 質感手がかりは反応または inquiry の中で実際に語られている必要がある。
- 質感の方が重いなら [`TF`](ref://scoring-input/determinants/TF) を再検討する。

## Cautions / Distinctions

- `FT` と [`TF`](ref://scoring-input/determinants/TF) の違いは、どちらが主導しているかです。
- 実際の触覚成分がなければ、反応は単なる [`F`](ref://scoring-input/determinants/F) にとどまることがあります。
- 奥行きや距離感が中心なら [`V`](ref://scoring-input/determinants/V) を見直します。
- ぼんやりした陰影的雰囲気が中心なら [`Y`](ref://scoring-input/determinants/Y) を見直します。
