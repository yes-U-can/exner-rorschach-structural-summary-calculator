# Document Name: [コーディング/決定因] YF

## Aliases / Search Terms

- YF
- 拡散陰影-形態
- 陰影優位

## Core Definition

`YF` は、拡散的陰影が反応を主に支え、形は補助的に残っているときにつける determinant です。
影っぽさ、曇り、不定な陰影感が反応を押し、形は二次的です。

## Application Conditions

- 拡散陰影が反応の主要基盤になっている。
- 形の支えもあるが、明らかに二次的である。
- 形が主なら [`FY`](ref://scoring-input/determinants/FY) を再検討する。
- 形の支えがほとんどないなら [`Y`](ref://scoring-input/determinants/Y) を考える。

## Cautions / Distinctions

- `YF` は `FY` より陰影の重みが強い反応です。
- 奥行き・距離感が中心なら [`VF`](ref://scoring-input/determinants/VF) や [`V`](ref://scoring-input/determinants/V) を見直します。
- 質感が中心なら [`TF`](ref://scoring-input/determinants/TF) や [`T`](ref://scoring-input/determinants/T) を見直します。
