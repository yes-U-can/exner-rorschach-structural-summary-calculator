# [コーディング/決定因] YF

## 別名・検索語

- YF
- 拡散陰影-形態
- 陰影優位

## 中核的定義

`YF` は、拡散的陰影が反応を主に支え、形は補助的に残っているときにつける決定因です。
影っぽさ、曇り、不定な陰影感が反応を押し、形は二次的です。

## 採点・適用条件

- 拡散陰影が反応の主要基盤になっています。
- 形の支えもあるが、明らかに二次的です。
- 形が主なら [`FY`](ref://scoring-input/determinants/FY) を再検討します。
- 形の支えがほとんどないなら [`Y`](ref://scoring-input/determinants/Y) を検討します。

## 注意点・鑑別

- `YF` は `FY` より陰影の重みが強い反応です。
- 奥行き・距離感が中心なら [`VF`](ref://scoring-input/determinants/VF) や [`V`](ref://scoring-input/determinants/V) を見直します。
- 質感が中心なら [`TF`](ref://scoring-input/determinants/TF) や [`T`](ref://scoring-input/determinants/T) を見直します。
