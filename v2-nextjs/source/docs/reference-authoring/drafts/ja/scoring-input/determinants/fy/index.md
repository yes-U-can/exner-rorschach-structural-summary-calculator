# Document Name: [コーディング/決定因] FY

## Aliases / Search Terms

- FY
- 形態-拡散陰影
- 形優位の拡散陰影

## Core Definition

`FY` は、形が反応を主に組織化し、拡散的陰影が二次的に加わるときにつける determinant です。
影、曇り、不定な陰影感があるが、対象はまず形で支えられています。

## Application Conditions

- 形が主な対象同定を支えている。
- そのうえで拡散的陰影の質が加わっている。
- その陰影感は反応または inquiry の中で実際に述べられている。
- 陰影の方が重いなら [`YF`](ref://scoring-input/determinants/YF) を再検討する。

## Cautions / Distinctions

- `FY` は [`FV`](ref://scoring-input/determinants/FV) と同じではありません。こちらは拡散陰影、あちらは奥行きです。
- `FY` は [`FT`](ref://scoring-input/determinants/FT) とも違います。こちらは陰影、あちらは質感です。
- 形が実際には反応を組織化していないなら [`Y`](ref://scoring-input/determinants/Y) の方が適切です。
- この群の意味づけはあとで [`SumY`](ref://result-interpretation/lower-section/core/SumY) で検討します。
