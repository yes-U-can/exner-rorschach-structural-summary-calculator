# Document Name: [Coding/Determinants] rF

## Aliases / Search Terms

- rF
- reflection-form
- 反射優勢

## Core Definition

`rF` は、反射経験が反応の中心で、形態はそれを支える二次要素として使われているときに付ける determinant です。  
つまり、まず「映っている」「鏡のようだ」が前面に出ており、そのあとで形態が補われています。

## Application Conditions

- Inquiry で反射、鏡像、水面への映り込みなどがはっきり述べられている。
- 反応の説明で、形態より reflection の体験が先に立っている。
- 形態は反射体験を補う程度で、主役ではない。
- 形態が先なら [`Fr`](ref://scoring-input/determinants/Fr) に戻って検討する。

## Cautions / Distinctions

- `rF` は [`Fr`](ref://scoring-input/determinants/Fr) と逆で、反射が主、形態が従です。
- 単なる左右対称や pair 反応なら [`Pair`](ref://scoring-input/pair) を優先して見直します。
- 反射がないなら [`F`](ref://scoring-input/determinants/F) に戻ります。
- reflection coding は自己像や自己確認のテーマに接続しやすいので、後の自己知覚解釈も確認します。
