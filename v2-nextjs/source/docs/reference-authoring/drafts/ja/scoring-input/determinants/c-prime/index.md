# Document Name: [コーディング/決定因] C'

## Aliases / Search Terms

- C'
- 純無彩色
- pure achromatic color

## Core Definition

`C'` は、反応が無彩色そのものにほぼ全面的に依存しているときにつける determinant です。
十分な形の支えがなく、黒・白・灰がはっきり色として働いている必要があります。

## Application Conditions

- 反応が黒・白・灰を色として基盤にしている。
- `FC'` や `C'F` に必要な形の支えが足りない。
- inquiry で二次的形態が確認できるなら `C'F` を再検討する。
- 無彩色の使用が明確で疑いないことが必要である。

## Cautions / Distinctions

- `C'` は [`C`](ref://scoring-input/determinants/C) と同じではありません。どちらも純色彩ですが、こちらは無彩色です。
- 無彩色語は shading でも出やすいので、本当に色として機能しているか確認します。
- 奥行きや距離の感じが中心なら [`V`](ref://scoring-input/determinants/V) を見直します。
- 拡散的陰影や大気的トーンが中心なら [`Y`](ref://scoring-input/determinants/Y) を見直します。
