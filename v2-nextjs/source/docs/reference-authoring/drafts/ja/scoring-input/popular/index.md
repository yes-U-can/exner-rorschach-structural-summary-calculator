# Document Name: [コーディング/Popular] Popular

## Aliases / Search Terms

- Popular
- P
- 人気反応
- 慣習的反応

## Core Definition

`Popular` は、その体系の基準集団で高頻度に見られる反応を示すコードです。
コーディングでは、内容と場所が定義済みの popular 基準に実際に合うときだけ `P` をつけます。

## Application Conditions

- 反応が体系で認められた popular 反応に一致している。
- 似ているだけでは足りず、必要な内容と場所条件に合っている。
- `P` は他の主コードに付け加える形で別に記録する。
- 臨床的直感ではなく、体系の popular 一覧に基づいて判断する。

## Cautions / Distinctions

- 内容や場所が意味ある形でずれていれば、popular に近く見えても `P` はつけません。
- カードの向きが関係する popular もあるので、上下位置を崩していないか確認します。
- `P` の多寡や欠如の意味づけはあとで解釈します。コーディング段階では「基準に合うか」だけを決めます。
- 文化差で説明文は変わり得ますが、コーディング判断はその locale の運用規則に従います。
