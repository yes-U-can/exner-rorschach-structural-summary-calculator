# [コーディング/Popular] Popular

## 別名・検索語

- Popular
- P
- 人気反応
- 慣習的反応

## 中核的定義

`Popular` は、その体系の基準集団で高頻度に見られる反応を示すコードです。
コーディングでは、内容と場所が定義済みの平凡反応基準に実際に合うときだけ `P` をつけます。

## 採点・適用条件

- 反応が体系で認められた平凡反応に一致しています。
- 似ているだけでは足りず、必要な内容と場所条件に合っています。
- `P` は他の主コードに付け加える形で別に記録します。
- 臨床的直感ではなく、体系の平凡反応一覧に基づいて判断します。

## 注意点・鑑別

- 内容や場所が意味ある形でずれていれば、平凡反応に近く見えても `P` はつけません。
- カードの向きが関係する平凡反応もあるので、上下位置を崩していないか確認します。
- `P` の多寡や欠如の意味づけはあとで解釈します。コーディング段階では「基準に合うか」だけを決めます。
- 文化的文脈は解釈に影響し得ますが、コーディング判断は適用する包括システムの基準に従います。
