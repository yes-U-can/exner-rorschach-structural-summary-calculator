# Document Name: [採点/GHR-PHR] GHR

## Aliases / Search Terms

- GHR
- 良好な人間表象
- 良好な人間表象反応

## Core Definition

`GHR`は、決定手順の中で比較的まとまりがあり、より適応的に見える人間表象反応に付くコードです。  
自由な臨床印象で決めるのではなく、`PHR`に押しやる条件に捕まらなかった結果として決まります。

## Application Conditions

- まず反応が人間表象領域に入っていることを確認します。
- 純粋な`H`反応で、`FQ+`、`FQo`、`FQu`があり、強い認知特殊スコアや[`AG`](ref://scoring-input/special-score/AG)、[`MOR`](ref://scoring-input/special-score/MOR)がない場合は、`GHR`に向かいやすいです。
- [`COP`](ref://scoring-input/special-score/COP)がある反応も、`AG`がなければ`GHR`に入ることがあります。
- カードIII、IV、VII、IXのpopular反応も、先に`PHR`条件に捕まっていなければ`GHR`方向に働きます。
- 先に`PHR`基準に該当しなかった人間表象反応は、最終的に`GHR`で閉じます。

## Cautions / Distinctions

- 反応が社交的・感じの良い内容に見えるだけでは`GHR`にはなりません。
- `H`があるだけで自動的に`GHR`になるわけではなく、形態水準や特殊スコアも確認が必要です。
- 攻撃性、病的内容、低い形態水準、強い認知特殊スコアがある場合は、まず[`PHR`](ref://scoring-input/gphr/PHR)を再検討します。
- 判断は常に決定手順の中で行い、単一ルールだけで決めません。
