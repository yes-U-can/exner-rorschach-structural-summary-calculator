# Document Name: [解釈/Upper Section] Zd

## Aliases / Search Terms

- Zd
- Z difference
- Z差

## Core Definition

`Zd` は、実際の組織化量 `ZSum` と期待される組織化量 `ZEst` の差です。
`Z` 系列の中では、組織化効率をもっとも直接的に示す指標です。

## Interpretation Points

- `Zd` がゼロ近辺にあるときは、頻度と負荷のバランスが比較的保たれていることがあります。
- 明らかな正値は、過剰な取り込み、過度の点検、場の elaboration のしすぎを示すことがあります。
- 明らかな負値は、取り込み不足、急ぎすぎ、刺激場の点検不足と並ぶことがあります。

## Cautions / Distinctions

- 正の値をそのまま適応的な慎重さとみなしてはいけません。
- 負の値も、そのまま障害と訳してはいけません。
- 意味づけは `Lambda`、`R`、`Zf`、`ZSum` と一緒に見ると安定します。
