# Document Name: [解釈/Upper Section] Upper Section

## Aliases / Search Terms

- upper-section
- upper section
- 構造一覧上段

## Core Definition

`Upper Section` は、`Zf`、`ZSum`、`ZEst`、`Zd`、`W`、`D`、`Dd`、`S`、そして `DQ` 分布を最初に確認する structural summary の部分です。
刺激場をどう整理するか、どこに注意が向きやすいか、反応が全体としてどの程度まとまっているかを最初に見渡す入口として働きます。

## Interpretation Points

- `Z` 系列は、組織化努力の頻度、総量、効率を見積もる助けになります。
- `W`、`D`、`Dd`、`S` は、全体、よく見られる細部、珍しい細部、白地のどこに注意が向いたかを示します。
- `DQ` 分布は、反応全体の統合度や構成のまとまり具合を要約します。
- このブロックは通常、[`processing`](ref://result-interpretation/lower-section/processing) や [`mediation`](ref://result-interpretation/lower-section/mediation) とつながります。

## Cautions / Distinctions

- `Zf` や `ZSum` だけで結論を出さず、通常は `Zd` まで合わせて見ます。
- `W` が多いことを、そのまま効率的な広域探索とみなしてはいけません。
- `Dd` や `S` が多いことも、それだけで病理とは言えません。
- `DQ+` が低い場合でも、短いプロトコルや強い防衛性が関係することがあります。
