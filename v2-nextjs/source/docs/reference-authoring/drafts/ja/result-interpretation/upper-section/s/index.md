# Document Name: [解釈/Upper Section] S

## Aliases / Search Terms

- S
- space responses
- 白地反応

## Core Definition

`S` は、白地を主位置として用いた反応の頻度です。
`Upper Section` では、逆方向の焦点化、反対的姿勢、あるいは場の非慣習的な探索の手がかりとして読まれます。

## Interpretation Points

- ある程度の `S` は、独立した視点や批判的探索と並ぶことがあります。
- `S` が高い場合は、反対性、非同調性、他の人が見ない場所を見る傾向を示すことがあります。
- `S` が低い場合は、白地がプロトコル内で重要な資源ではなかったことを示すだけです。

## Cautions / Distinctions

- `S` が高いからといって、すぐ反抗性だけに還元してはいけません。
- `S` が `S-%`、`X-%`、`AG`、`PER` と一緒に上がる場合は、衝突や非効率の方へ意味が傾くことがあります。
- 良い mediation と形式統制が保たれているなら、柔軟性や独自性の表れであることもあります。
