# [解釈/Upper Section] D

## 別名・検索語

- D
- common detail
- 通常細部

## 中核的定義

`D` は、ブロットの一般的な細部に対して与えられた反応の頻度です。
解釈では、実用的で共有されやすい単位に注意が向きやすいかを見る基本指標になります。

## 解釈の要点

- 釣り合いの取れた `D` は、具体的で扱いやすい要素へ焦点を当てられることと並ぶことがあります。
- `D` が目立つ場合は、共通で使いやすい部分を選びやすい様式を示すことがあります。
- `D` が `W` や `Dd` より明確に優勢なら、探索的というより実用的な処理様式が前に出ている可能性があります。

## 注意点・鑑別

- `D` が高いことだけで、適応や現実検討の良さを証明するわけではありません。
- `D` が高くても `P` が低い、あるいは `Xu%` が高いなら、見かけほど慣習的ではない可能性があります。
- `D` が低い場合も、それだけで読むのではなく `W` と `Dd` の関係で見ます。
