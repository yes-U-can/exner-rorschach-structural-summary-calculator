# Document Name: [解釈/Upper Section] ZEst

## Aliases / Search Terms

- ZEst
- estimated Z sum
- 予測ZSum

## Core Definition

`ZEst` は、観察された `Zf` から期待される `ZSum` の推定値です。
独立した臨床変数というより、`Zd` を計算するための基準値として働きます。

## Interpretation Points

- `ZEst` は基準値として使われます。
- 実際の組織化量 `ZSum` と、期待される水準との差を比べるために必要です。
- 解釈の焦点は通常 `ZEst` 単独ではなく `Zd` に置かれます。

## Cautions / Distinctions

- 高い `ZEst` は、しばしば `Zf` も高いことを反映しているだけです。
- `高い ZEst` や `低い ZEst` という言い方だけで意味づけるより、比較役割を説明する方が安全です。
