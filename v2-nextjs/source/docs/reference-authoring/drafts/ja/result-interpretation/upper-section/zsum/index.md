# Document Name: [解釈/Upper Section] ZSum

## Aliases / Search Terms

- ZSum
- weighted Z sum
- 総Z得点

## Core Definition

`ZSum` は、`Z` 得点が付いた反応に割り当てられた重みの総和です。
組織化がどのくらいの回数起きたかだけでなく、全体としてどれだけの組織化努力が投入されたかを表します。

## Interpretation Points

- `ZSum` は、組織化に向けられた総エネルギーを要約します。
- 高値は、刺激場の関係を積極的にまとめようとする傾向に並ぶことがあります。
- 低値は、単純な反応や組織化努力の少なさと並ぶことがあります。
- 主な解釈価値は `ZEst` との比較にあり、その差が `Zd` につながります。

## Cautions / Distinctions

- `Zf` も高い場合には、高い `ZSum` が十分に妥当なことがあります。
- 低い `ZSum` をそのまま処理障害と読んではいけません。
- 大事なのは `ZEst` と比べたときに効率的な範囲に収まっているかです。
