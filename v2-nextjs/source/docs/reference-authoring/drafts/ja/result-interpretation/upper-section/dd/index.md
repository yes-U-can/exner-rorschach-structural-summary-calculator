# [解釈/Upper Section] Dd

## 別名・検索語

- Dd
- unusual detail
- 稀な細部

## 中核的定義

`Dd` は、あまり用いられない珍しい細部に対して与えられた反応の頻度です。
解釈では、共有されにくい部分や目立ちにくい部分へ注意がどの程度向くかを示します。

## 解釈の要点

- ある程度の `Dd` は、探索範囲の広さや目立たない細部への感度を示すことがあります。
- `Dd` が高い場合は、独特な焦点、非慣習的な関心、周辺部への走査と並ぶことがあります。
- `Dd` が非常に低い場合は、より典型的で予測しやすい単位に留まりやすいことを示すことがあります。

## 注意点・鑑別

- `Dd` が高いことを、そのまま創造性と読むのは危険です。
- `Dd` が高く、`P` が低く、`Xu%` も高いなら、創造性よりも非慣習的知覚の方に重心が寄ることがあります。
- `Dd` が高く、同時に `Zd` が大きく正なら、創造性より過剰点検の可能性も考えます。
