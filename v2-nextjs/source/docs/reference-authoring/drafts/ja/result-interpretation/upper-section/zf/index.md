# [解釈/Upper Section] Zf

## 別名・検索語

- Zf
- Z frequency
- Z反応頻度

## 中核的定義

`Zf` は、`Z` 得点が付いた反応の頻度です。
刺激場の部分同士の関係を組織化しようとする試みが、どのくらいの回数現れたかを示します。

## 解釈の要点

- `Zf` はまず、組織化努力がどのくらいの頻度で現れるかを示します。
- 適度な `Zf` は、浅すぎず、場に絡みつきすぎない探索と両立することが多いです。
- 高い `Zf` は、関係づけや点検を頻繁に行う傾向を示すことがあります。
- 低い `Zf` は、単純な反応の多さや構造化努力の少なさと並ぶことがあります。

## 注意点・鑑別

- `Zf` は `ZSum` と `Zd` を合わせて見る方が安全です。
- 高い `Zf` だけで効率の良さを意味するわけではありません。
- 低い `Zf` だけで障害を意味するわけでもなく、短いプロトコルや防衛性でも下がりえます。
