# Document Name: [解釈/Upper Section] DQv/+

## Aliases / Search Terms

- DQv/+
- vague synthesized responses
- 曖昧な統合反応数

## Core Definition

`DQv/+` は、統合の試みはあるが、その基盤が曖昧な反応の頻度です。
統合努力は見えるものの、構造的な支えが十分に安定していないという混合的な信号です。

## Interpretation Points

- `DQv/+` は `DQv` よりは組織化努力が強いことを示します。
- ただし、知覚基盤は `DQ+` ほど明確ではありません。
- 少量なら、十分に安定しきらない複雑化の試みの副産物として現れることがあります。

## Cautions / Distinctions

- `DQv/+` が `DQv` と一緒に増えるなら、効率的統合より不明瞭な組織化の方へ重心が寄ります。
- 良い形式水準を保ち、曖昧さが過度でなければ、混乱ではなく複雑化への努力を示す場合もあります。
- `DQ+` と混同してはいけません。
