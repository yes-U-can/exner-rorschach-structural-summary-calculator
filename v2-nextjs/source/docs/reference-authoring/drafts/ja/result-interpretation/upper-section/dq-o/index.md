# Document Name: [解釈/Upper Section] DQo

## Aliases / Search Terms

- DQo
- ordinary developmental quality
- 通常DQ反応数

## Core Definition

`DQo` は、通常の developmental quality を持つ反応の頻度です。
多くの場合、プロトコルにおける平均的な組織化水準の基準線として働きます。

## Interpretation Points

- 多くのプロトコルでは、`DQo` が `DQ` 分布の中心部分を占めます。
- 釣り合いの取れた `DQo` は、一般的で安定した組織化水準と並ぶことがあります。
- `DQo` が非常に優勢なときは、かなり慣習的で、複雑さがあまり高くない様式を示す場合があります。

## Cautions / Distinctions

- `DQo` が高いこと自体は弱さではありません。
- 意味は `DQ+`、`DQv/+`、`DQv` との関係で変わります。
- `DQo` が支配的で `DQ+` がほとんどない場合、統合的複雑性は限られているかもしれません。
