# [Interpretation/Mediation] P

## 別名・検索語

- P
- Popular
- 通俗反応数

## 中核的定義

`P` は、プロトコルの中にどれだけ通俗反応が出ているかを示します。
媒介領域では、知覚が広く共有される反応様式にどれくらい近いかを見積もる助けになります。

## 解釈の要点

- `P` が低いと、通俗的な読みとの重なりが少ない、あるいは合意から距離がある可能性があります。
- `P` が高いと、一般的な反応に近づきやすい、または慣習的な形を出しやすい可能性があります。
- `X+%` や `XA%` と比べると、通俗的一致がより広い形態的妥当性と一致しているかが見えます。

## 注意点・鑑別

- `P` は適用する包括システムの基準に従ってコーディングし、解釈では文化的文脈も考慮します。
- 値が高いから即適応的、低いから即病理的とは言えません。
- どのカードで通俗反応が出ているか、あるいは出ていないかも一緒に見ると役立ちます。
