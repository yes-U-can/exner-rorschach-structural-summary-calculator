# [Interpretation/Affect] Afr

## 別名・検索語

- Afr
- Affective Ratio
- 情動比率

## 中核的定義

`Afr` はカード VIII、IX、X への反応数を、カード I から VII への反応数と比較する比率です。
解釈では、情動的または色彩的に強い刺激へ近づきやすいか、退きやすいかを見積もるために使います。

## 解釈の要点

- `Afr` が低いと、情動を帯びた状況へ入り込む準備性が低めかもしれません。
- `Afr` が高いと、情動的に目立つ刺激へより開かれている可能性があります。
- 解釈は `EB`、`Lambda`、情動表出全体の様式と一緒に読むと安定します。
- 最近のストレス負荷で、一時的に `Afr` が下がることもあります。

## 注意点・鑑別

- 総反応数が少ないと、この比率は大きくぶれやすくなります。
- `Afr` が高いから即座に健全、低いから即座に病的回避とは読みません。
- カードごとの生産性差が極端でないかも確認してから強い結論を出してください。
