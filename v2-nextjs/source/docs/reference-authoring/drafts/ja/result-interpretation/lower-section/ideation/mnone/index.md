# Document Name: [Interpretation/Ideation] Mnone

## Aliases / Search Terms

- Mnone
- 無形式M
- 形態支持の弱い人間運動

## Core Definition

`Mnone` は人間運動が示されていても、通常の `M` として支えるだけの形態が足りない反応を指します。
解釈では、人間表象や空想活動は存在するが、構造化が弱いことを示す指標として扱います。

## Interpretation Points

- `Mnone` が出ると、人間的な想像や心的活動はあるが、形が曖昧で組織化が弱い可能性があります。
- きれいに形づくられた `M` と同じ重みでは読まず、弱くまとまった人間表象として読みます。
- `M-`、`WSum6`、`SCON` と一緒に出る場合は、観念化の脆弱さへの注意が強まります。

## Cautions / Distinctions

- `Mnone` は `M-` と同じではありません。`Mnone` は形態支持が足りず、`M-` は minus 形態質をもつ人間運動です。
- 単独の `Mnone` だけで重大な思考障害と決めつけないでください。
- まず `M` のコーディングが正しいかを確認し、単なる採点ミスと混同しないことが大切です。
