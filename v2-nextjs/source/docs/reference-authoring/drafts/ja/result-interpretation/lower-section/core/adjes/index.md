# Document Name: [Interpretation/Core] AdjEs

## Aliases / Search Terms

- AdjEs
- Adjusted es
- adjusted experienced stimulation
- 補正es

## Core Definition

`AdjEs` は `es` を補正した値である。
`m` と `Y` に由来する負荷の一部を差し引いたあとでも、どれだけ負荷が残っているかを見積もるために使う。

## Interpretation Points

- `AdjEs` が高い場合、負荷はその場限りではなく、より持続的な性質をもっている可能性が高い。
- `es` が高くても `AdjEs` がはっきり下がるなら、現在の苦しさのかなりの部分は状況的ストレスに結びついている可能性がある。
- `AdjEs` は主に [`AdjD`](ref://result-interpretation/lower-section/core/AdjD) を計算し、意味づけるための補正変数として役立つ。
- それ自体が独立した診断ラベルになるわけではない。

## Cautions / Distinctions

- `AdjEs` は [`es`](ref://result-interpretation/lower-section/core/es) を置き換えるものではなく、両方を比べて読む必要がある。
- 高い値だけで、苦痛の原因をひとつに決めてはいけない。
- 実際の負荷が主に [`m`](ref://result-interpretation/lower-section/core/m) や [`SumY`](ref://result-interpretation/lower-section/core/SumY) によるものかどうかも、あわせて確認する必要がある。
