# [Interpretation/Core] AdjEs

## 別名・検索語

- AdjEs
- Adjusted es
- adjusted experienced stimulation
- 補正es

## 中核的定義

`AdjEs` は `es` を補正した値です。
`m` と `Y` に由来する負荷の一部を差し引いたあとでも、どれだけ負荷が残っているかを見積もるために使います。

## 解釈の要点

- `AdjEs` が高い場合、負荷はその場限りではなく、より持続的な性質をもっている可能性が高いです。
- `es` が高くても `AdjEs` がはっきり下がるなら、現在の苦しさのかなりの部分は状況的ストレスに結びついている可能性があります。
- `AdjEs` は主に [`AdjD`](ref://result-interpretation/lower-section/core/AdjD) を計算し、意味づけるための補正変数として役立ちます。
- それ自体が独立した診断ラベルになるわけではありません。

## 注意点・鑑別

- `AdjEs` は [`es`](ref://result-interpretation/lower-section/core/es) を置き換えるものではなく、両方を比べて読む必要があります。
- 高い値だけで、苦痛の原因をひとつに決めてはいけません。
- 実際の負荷が主に [`m`](ref://result-interpretation/lower-section/core/m) や [`SumY`](ref://result-interpretation/lower-section/core/SumY) によるものかどうかも、あわせて確認する必要があります。
