# [Interpretation/Ideation] Ma:Mp

## 別名・検索語

- Ma:Mp
- active:passive human movement
- human movement active:passive
- 人間運動 能動:受動

## 中核的定義

`Ma:Mp` は、`M` 反応の中で能動的人間運動と受動的人間運動の比を表します。
頭の中で表象される人間活動が、行う・働きかける側に寄るのか、待つ・受ける・作用される側に寄るのかをみる助けになります。

## 解釈の要点

- `Mp` が `Ma` を上回る場合、人間に関する思考表象は、受動、待機、受容の方向に寄りやすいです。
- `Ma` が優勢な場合、人間に関する思考表象は、主導、介入、行動化の方向に寄りやすいです。
- 比がつり合っていれば、人間表象は一方の極に強く引かれていないと考えやすいです。
- 意味は `M` の質とあわせて読むとより有用になります。

## 注意点・鑑別

- `M` 反応が非常に少ないと、この比は安定しません。
- `Mp` 優勢だけで脆弱さを断定してはいけず、`Ma` 優勢だけで望ましい適応を断定してはいけません。
- 思考の質は、[`M-`](ref://result-interpretation/lower-section/ideation/M_minus) や [`Mnone`](ref://result-interpretation/lower-section/ideation/Mnone) でも確認する必要があります。
