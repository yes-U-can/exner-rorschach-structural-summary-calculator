# Document Name: [Interpretation/Interpersonal] PER

## Aliases / Search Terms

- PER
- personalized answers
- personal frame
- 個人的枠組み

## Core Definition

`PER` は、個人的基準、個人的経験、私的な視点によって反応を正当化する傾向が、対人的水準でどれくらい現れるかを示す。
Interpretation では、他者や対人やり取りの読み方が、どの程度 self-referential な frame によって形づくられているかを示唆する。

## Interpretation Points

- `PER` が高いと、共有された手がかりよりも個人的参照で対人世界を解釈しやすいことがある。
- rigidity、self-assertion の必要、非常に私的な経験の検証の仕方と一緒に現れることがある。
- `Xu%` が高いときに一緒に出ると、対人理解がさらに idiosyncratic になりやすい。
- 低い値は、この偏りが少ないことを示すだけで、直ちに openness や flexibility を意味しない。

## Cautions / Distinctions

- `PER` は creativity と同じではなく、それだけで narcissism を意味しない。
- 単なる個人的意見ではなく、反応が personal frame で支えられる仕方を指す。
- coding score の `PER` と関連はあるが、interpretation 上は区別して読む。

## Limits of Isolated Interpretation

- `PER` だけで relational egocentricity を結論づけない。
- `a:p`、`HumanCont`、`Xu%`、`H ratio` と一緒に読む必要がある。
