# [Interpretation/Interpersonal] PER

## 別名・検索語

- PER
- personalized answers
- personal frame
- 個人的枠組み

## 中核的定義

`PER` は、個人的基準、個人的経験、私的な視点によって反応を正当化する傾向が、対人的水準でどれくらい現れるかを示します。
解釈では、他者や対人やり取りの読み方が、どの程度自己言及的な枠組みによって形づくられているかを示唆します。

## 解釈の要点

- `PER` が高いと、共有された手がかりよりも個人的参照で対人世界を解釈しやすいことがあります。
- 硬直性、自己主張の必要、非常に私的な経験の検証の仕方と一緒に現れることがあります。
- `Xu%` が高いときに一緒に出ると、対人理解がさらに個人的で特異になりやすいです。
- 低い値は、この偏りが少ないことを示すだけで、直ちに開放性や柔軟性を意味しません。

## 注意点・鑑別

- `PER` は創造性と同じではなく、それだけで自己愛を意味しません。
- 単なる個人的意見ではなく、反応が個人的枠組みで支えられる仕方を指します。
- コーディング上のスコアの `PER` と関連はあるが、解釈上は区別して読みます。

## 単独解釈の限界

- `PER` だけで対人関係における自己中心性を結論づけません。
- `a:p`、`HumanCont`、`Xu%`、`H ratio` と一緒に読む必要があります。

## 併せて確認する変数

- `a:p`
- `HumanCont`
- `Xu%`
- `H ratio`
- `COP`
