# Document Name: [Interpretation/Core] FM

## Aliases / Search Terms

- FM
- animal movement total
- total animal movement
- 動物運動総数

## Core Definition

`FM` は動物運動反応の総数である。
`core` では、まだ十分に整理されていない欲求緊張、放出圧、動機づけの負荷を示す左側成分として読まれやすい。

## Interpretation Points

- 中程度の `FM` は、欲求や内的圧力が行動を押していることを示しやすい。
- `FM` が高い場合、満たされていない欲求、衝動、放出要求からの圧力がより強いことを示しやすい。
- `FM` は [`eb`](ref://result-interpretation/lower-section/core/eb) の構成要素であり、[`m`](ref://result-interpretation/lower-section/core/m) と合わせて [`es`](ref://result-interpretation/lower-section/core/es) を高めることがある。
- 意味は [`m`](ref://result-interpretation/lower-section/core/m)、[`D`](ref://result-interpretation/lower-section/core/D)、[`AdjD`](ref://result-interpretation/lower-section/core/AdjD) と比較するとより明確になる。

## Cautions / Distinctions

- `FM` が高いだけで、衝動統制が明らかに崩れていると決めてはいけない。
- `FM` は [`M`](ref://scoring-input/determinants/M) と混同してはいけない。ここでは成熟した観念化よりも、より基礎的で統合されていない圧力を指しやすい。
- 臨床的な重みは、[`m`](ref://result-interpretation/lower-section/core/m)、[`es`](ref://result-interpretation/lower-section/core/es)、[`D`](ref://result-interpretation/lower-section/core/D)、[`AdjD`](ref://result-interpretation/lower-section/core/AdjD) とのバランスによって変わる。
