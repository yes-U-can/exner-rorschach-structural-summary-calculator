# [Interpretation/Core] FM

## 別名・検索語

- FM
- animal movement total
- total animal movement
- 動物運動総数

## 中核的定義

`FM` は動物運動反応の総数です。
`core` では、まだ十分に整理されていない欲求緊張、放出圧、動機づけの負荷を示す左側成分として読まれやすいです。

## 解釈の要点

- 中程度の `FM` は、欲求や内的圧力が行動を押していることを示しやすいです。
- `FM` が高い場合、満たされていない欲求、衝動、放出要求からの圧力がより強いことを示しやすいです。
- `FM` は [`eb`](ref://result-interpretation/lower-section/core/eb) の構成要素であり、[`m`](ref://result-interpretation/lower-section/core/m) と合わせて [`es`](ref://result-interpretation/lower-section/core/es) を高めることがあります。
- 意味は [`m`](ref://result-interpretation/lower-section/core/m)、[`D`](ref://result-interpretation/lower-section/core/D)、[`AdjD`](ref://result-interpretation/lower-section/core/AdjD) と比較するとより明確になります。

## 注意点・鑑別

- `FM` が高いだけで、衝動統制が明らかに崩れていると決めてはいけません。
- `FM` は [`M`](ref://scoring-input/determinants/M) と混同してはいけません。ここでは成熟した観念化よりも、より基礎的で統合されていない圧力を指しやすいです。
- 臨床的な重みは、[`m`](ref://result-interpretation/lower-section/core/m)、[`es`](ref://result-interpretation/lower-section/core/es)、[`D`](ref://result-interpretation/lower-section/core/D)、[`AdjD`](ref://result-interpretation/lower-section/core/AdjD) とのバランスによって変わります。
