# [Interpretation/Interpersonal] Isolation Index

## 別名・検索語

- ISO Index
- Isolation Index
- Isolate/R
- isolation ratio
- 孤立指数

## 中核的定義

`ISO_Index` は `(Bt + 2Cl + Ge + Ls + 2Na) / R` の比率です。
解釈では、記録が距離、社会的引きこもり、限定された対人参加をどれくらい中心に組織化しているかを見積もる助けになります。

## 解釈の要点

- 指数が `.24` や `.25` を超えると、社会的距離や対人的引きこもりの仮説をより慎重に検討する価値があります。
- `1/3` 近く、またはそれ以上なら、意味のある孤立への懸念がさらに高まります。
- `ISO_Index` が高くて `COP` が低いと、対人参加は乏しく相互性が弱い形に見えやすいです。
- `ISO_Index` が高い一方で `SumT_inter` や `Food` も上がるなら、親密さへの欲求と引きこもりが共存していることがあります。

## 注意点・鑑別

- 比率なので `R` が低いと過大に見積もられやすいです。
- 慢性的孤立の単独の根拠として読みません。
- `COP`、`HumanCont`、`CDI`、`DEPI` の文脈が必要です。

## 単独解釈の限界

- `ISO_Index` 単独では、孤立が防衛的なのか抑うつ的なのか状況依存的なのかを説明しません。
- 反応数と他の対人変数を合わせて読む必要があります。

## 併せて確認する変数

- `COP`
- `HumanCont`
- `Food`
- `CDI`
- `DEPI`
