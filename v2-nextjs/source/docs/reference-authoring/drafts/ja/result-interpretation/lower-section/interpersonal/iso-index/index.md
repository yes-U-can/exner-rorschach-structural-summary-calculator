# Document Name: [Interpretation/Interpersonal] Isolation Index

## Aliases / Search Terms

- ISO Index
- Isolation Index
- Isolate/R
- isolation ratio
- 孤立指数

## Core Definition

`ISO_Index` は `(Bt + 2Cl + Ge + Ls + 2Na) / R` の比率である。
Interpretation では、記録が distance、social withdrawal、限定された interpersonal participation をどれくらい中心に組織化しているかを見積もる助けになる。

## Interpretation Points

- 指数が `.24` や `.25` を超えると、社会的距離や対人的 withdrawal の仮説をより慎重に検討する価値がある。
- `1/3` 近く、またはそれ以上なら、意味のある isolation への懸念がさらに高まる。
- `ISO_Index` が高くて `COP` が低いと、対人参加は乏しく相互性が弱い形に見えやすい。
- `ISO_Index` が高い一方で `SumT_inter` や `Food` も上がるなら、親密さへの欲求と withdrawal が共存していることがある。

## Cautions / Distinctions

- 比率なので `R` が低いと inflated されやすい。
- chronic isolation の stand-alone proof として読まない。
- `COP`、`HumanCont`、`CDI`、`DEPI` の文脈が必要である。

## Limits of Isolated Interpretation

- `ISO_Index` 単独では、孤立が defensive なのか depressive なのか situational なのかを説明しない。
- response volume と他の対人変数を合わせて読む必要がある。
