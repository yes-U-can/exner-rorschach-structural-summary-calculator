# Document Name: [Interpretation/Affect] Pure C

## Aliases / Search Terms

- Pure C
- pure color
- 純粋色彩

## Core Definition

`Pure C` は、形態支持を伴わない色彩反応の頻度です。
解釈では、情動が構造や形にあまり支えられず、より直接的に表出される傾向を示す指標として使います。

## Interpretation Points

- `Pure C` が出ると、感情がより即時的に表れやすい場面があると考えます。
- `Pure C` が繰り返し出ると、情動が形態的組織化を上回って前面に出やすい可能性があります。
- 成人プロトコルでは基礎出現率が高くないため、少数でも注意して見ます。

## Cautions / Distinctions

- `Pure C` だけで病的衝動性を意味するわけではありません。
- `CF`、`FC`、`CP` と混同せず分けて読んでください。
- 有彩色反応の機会が少ないプロトコルでは、`Pure C` がないことをそのまま情動欠如とは読みません。
