# Document Name: [Interpretation/Ideation] WSum6

## Aliases / Search Terms

- WSum6
- Weighted Sum6
- weighted six special scores
- 加重Sum6

## Core Definition

`WSum6` は、思考関連の6種特殊得点に重みづけをして合計した値である。
認知的特異性がいくつ現れたかだけでなく、それぞれがどの程度重い臨床的意味を持つかも反映する。

## Interpretation Points

- `WSum6` が高いほど、認知的ゆがみや特異性の全体的な重みは大きい。
- `Sum6` が似ていても、より重い指標を含むプロトコルでは `WSum6` が高くなることがある。
- 成人プロトコルで `WSum6` が高い場合、思考の一貫性やまとまりの崩れの危険をより注意深く点検する必要がある。

## Cautions / Distinctions

- `WSum6` が高くても、どの特殊得点が含まれているかの確認を省いてはいけない。
- 上昇が `Lv2`、`CONTAM`、あるいは軽い指標の積み重なりのどれで起きているかを見ることが重要である。
- 単独指標ではなく、[`M-`](ref://result-interpretation/lower-section/ideation/M_minus)、[`PTI`](ref://result-interpretation/special-indices/PTI)、mediation、processing と一緒に読む必要がある。
