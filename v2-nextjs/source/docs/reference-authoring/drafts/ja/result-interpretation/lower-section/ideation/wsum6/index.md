# [Interpretation/Ideation] WSum6

## 別名・検索語

- WSum6
- Weighted Sum6
- weighted six special scores
- 加重Sum6

## 中核的定義

`WSum6` は、思考関連の6種特殊スコアに重みづけをして合計した値です。
認知的特異性がいくつ現れたかだけでなく、それぞれがどの程度重い臨床的意味を持つかも反映します。

## 解釈の要点

- `WSum6` が高いほど、認知的ゆがみや特異性の全体的な重みは大きいです。
- `Sum6` が似ていても、より重い指標を含むプロトコルでは `WSum6` が高くなることがあります。
- 成人プロトコルで `WSum6` が高い場合、思考の一貫性やまとまりの崩れの危険をより注意深く点検する必要があります。

## 注意点・鑑別

- `WSum6` が高くても、どの特殊スコアが含まれているかの確認を省いてはいけません。
- 上昇が `Lv2`、`CONTAM`、あるいは軽い指標の積み重なりのどれで起きているかを見ることが重要です。
- 単独指標ではなく、[`M-`](ref://result-interpretation/lower-section/ideation/M_minus)、[`PTI`](ref://result-interpretation/special-indices/PTI)、認知的媒介、情報処理と一緒に読む必要があります。
