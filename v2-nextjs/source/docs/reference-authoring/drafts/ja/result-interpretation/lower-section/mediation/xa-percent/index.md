# Document Name: [Interpretation/Mediation] XA%

## Aliases / Search Terms

- XA%
- Form Appropriate Extended
- 拡張適合率

## Core Definition

`XA%` は、形態が拡張された許容範囲の中に入る反応の全体的割合を示します。
`FQ+`、`FQo`、`FQu` を含むため、知覚的媒介の広い全体像をみる指標になります。

## Interpretation Points

- `XA%` が高いと、知覚は比較的共有可能で妥当な読み取りに近いことが多いです。
- これが下がると、歪み、不正確さ、知覚的不一致の可能性が高くなります。
- `WDA%` と比べることで、困難が全般的か、それともより複雑な場面に集中しているかが見えやすくなります。

## Cautions / Distinctions

- `XA%` が高くても、知覚の奇異さが完全にないとは限りません。`Xu%` が重要になることがあります。
- `XA%` が低いときは、`X-%` と一緒に読んでください。
- 反応数が少ない場合、この比率はぶれやすくなります。
