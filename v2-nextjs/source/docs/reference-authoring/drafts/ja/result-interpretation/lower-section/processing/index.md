# Document Name: [Interpretation/Lower Section/Processing] Processing

## Aliases / Search Terms

- Processing
- 情報処理
- 処理様式

## Core Definition

`Processing` は、被検者が知覚場をどのように走査し、組織化し、区切っているかを示す変数群です。
通常は `Zf`、`Zd`、`W:D:Dd`、`W:M`、`PSV`、`DQ+`、`DQv` を通して読みます。

## Interpretation Points

- この群は、どれくらい材料を見ているか、どれくらい複雑さを取り込んでいるか、そして場をどれくらい柔軟に組み立てているかを示します。
- `Zf` と `Zd` は、走査の範囲と効率を見積もる助けになります。
- `W:D:Dd` と `W:M` は、分節化の仕方と組織化水準を示します。
- `PSV`、`DQ+`、`DQv` は、反復、組織化の質、単純化の傾向を補足します。

## Cautions / Distinctions

- `Processing` は `Mediation` と同じではありません。広く見ていても、知覚が不正確なことはあります。
- `Processing` は `Ideation` とも同じではありません。知覚の分節化と概念的精緻化は別の水準です。
- 短いプロトコルでは、この群の比率は不安定になりやすいです。
