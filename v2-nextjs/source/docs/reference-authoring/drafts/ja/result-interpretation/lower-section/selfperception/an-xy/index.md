# Document Name: [Interpretation/Self Perception] An + Xy

## Aliases / Search Terms

- An+Xy
- anatomy plus x-ray
- 身体関心

## Core Definition

`An+Xy` は anatomy content と x-ray content の合計です。  
自己知覚では、身体、臓器、身体機能、身体の脆弱さへの関心を読む手がかりになります。

## Interpretation Points

- 高いときは、臓器、身体損傷、病気、身体的脆弱さへの注目が強いことがあります。
- 身体不安、somatic concern、最近の健康ストレスに関係することがあります。
- `MOR_self` も高いと、身体像がより傷ついたもの、弱いもの、脅かされるものとして経験されやすくなります。
- 反応ごとに、身体が現実的に扱われているか、恐れの対象か、象徴的かを区別して見ます。

## Cautions / Distinctions

- 医療経験、実際の疾患、最近の治療でも上がることがあります。
- `Xy` は毎回 `An` と同じ意味ではないので、反応文脈を確認します。
- 低値でも身体関心が完全にないとは言えません。

## Limits of Isolated Interpretation

- これは身体焦点の存在を示しますが、それだけで hypochondriacal か traumatic か situational かは決めません。
- `MOR_self`、`SumV_self`、`HVI` と一緒に読む必要があります。
