# Document Name: [Interpretation/Core] D

## Aliases / Search Terms

- D
- D score
- current stress tolerance
- 現在のストレス耐性

## Core Definition

`D` は `EA` から `es` を引いた差である。
現在の要求に対して、利用できる資源が足りているか、それとも今まさに過負荷になっているかをまとめて示す。

## Interpretation Points

- `D = 0` は、現在の要求と使える資源がほぼつり合っていることを示しやすい。
- `D > 0` は、今の場面では要求より資源のほうが上回り、すぐに圧倒される危険が比較的低いことを示しやすい。
- `D < 0` は、現在の要求が資源を上回っており、緊張、いらだち、欲求不満への耐性低下と結びつきやすい。
- `D` は状況の影響を受けやすいので、最近のストレスや直近の負荷によって動きやすい。

## Cautions / Distinctions

- `D = 0` だけで適応が良好と決めてはいけない。単純化された防衛的なプロトコルでも、見かけ上つり合って見えることがある。
- `D > 0` だけで安定した強さが十分あると断定してはいけない。
- `D < 0` は現在の過負荷を示すが、それが一時的なものか、より持続的なものかをみるには [`AdjD`](ref://result-interpretation/lower-section/core/AdjD) との比較が必要である。
