# [Interpretation/Core] D

## 別名・検索語

- D
- D score
- current stress tolerance
- 現在のストレス耐性

## 中核的定義

`D` は生の差そのものではありません。まず `EA - es` を求め、その生の差を尺度化された D スコアへ変換します。生の差が `-2.5` 以上 `+2.5` 以下なら `D = 0` であり、その範囲を超えると、さらに `2.5` 点離れるごとに符号を保ったまま D が1段階変化します。極端なプロトコルでは、通常印刷される `-5` から `+5` の表を超えても同じ規則を続けます。

現在の要求に対して、利用できる資源が足りているか、それとも今まさに過負荷になっているかをまとめて示します。

## 解釈の要点

- `D = 0` は、現在の要求と使える資源がほぼつり合っていることを示しやすいです。
- `D > 0` は、今の場面では要求より資源のほうが上回り、すぐに圧倒される危険が比較的低いことを示しやすいです。
- `D < 0` は、現在の要求が資源を上回っており、緊張、いらだち、欲求不満への耐性低下と結びつきやすいです。
- `D` は状況の影響を受けやすいので、最近のストレスや直近の負荷によって動きやすいです。

## 注意点・鑑別

- `D = 0` だけで適応が良好と決めてはいけません。単純化された防衛的なプロトコルでも、見かけ上つり合って見えることがあります。
- `D > 0` だけで安定した強さが十分あると断定してはいけません。
- `D < 0` は現在の過負荷を示すが、それが一時的なものか、より持続的なものかをみるには [`AdjD`](ref://result-interpretation/lower-section/core/AdjD) との比較が必要です。
