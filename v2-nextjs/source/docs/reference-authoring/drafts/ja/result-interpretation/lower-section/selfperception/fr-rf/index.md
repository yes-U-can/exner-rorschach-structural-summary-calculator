# [Interpretation/Self Perception] Fr + rF

## 別名・検索語

- Fr+rF
- reflection responses
- 自己像を映す反応

## 中核的定義

`Fr+rF` は、プロトコルにどれくらい反射反応が出ているかを示します。
自己像を特別に輝かせたい、確認したい、誇張したい動きがどれくらいあるかを読む手がかりです。

## 解釈の要点

- `Fr+rF > 0` なら、自己重要感や自己像の補強欲求を見直す価値があります。
- 穏やかな自信というより、再確認や承認の必要に結びつくことがあります。
- `3r+(2)/R` も高いと、自己注目の比重はさらに強くなります。
- `SumV` と一緒に高いと、自己高揚と自己批判が不安定に同居している可能性があります。

## 注意点・鑑別

- 成人では反射反応は多くないので、1つでも丁寧に見ます。
- ただし 1 反応だけで人格全体を決めません。
- 反応がないからといって、自己愛的な傷つきや承認欲求が必ず低いとは限りません。

## 単独解釈の限界

- `3r+(2)/R`、`SumV`、`FD`、`MOR` と統合して読む必要があります。
- この変数は自己像の一方向を示すだけで、安定性全体をそれだけで決めません。

## 併せて確認する変数

- `3r+(2)/R`
- `SumV_self`
- `FD`
- `Fr`
- `rF`
