# Document Name: [Interpretation/Affect] CP

## Aliases / Search Terms

- CP
- color projection
- 色彩投影

## Core Definition

`Affect` における `CP` は、本来そこにないはずの色や情動的な色合いを知覚場へ付与している状態を読むための指標です。
解釈では、情動が通常とは違う形で知覚に強く入り込んでいる可能性を見るときに使います。

## Interpretation Points

- `CP` が出ると、比較的中性的な状況に感情を強く載せやすい傾向があるかもしれません。
- 出現率が低いので、1件でも臨床的には注意して見る価値があります。
- 陰影 determinant や、緊張を示す他の指標と一緒に出るかを確認すると読みやすくなります。

## Cautions / Distinctions

- `CP` は `Pure C`、`CF`、`C` と同じではありません。
- 解釈の前に、special score としての採点が正しいか確認してください。
- まれな所見なので重みはありますが、1件だけで広い結論を出すことは避けてください。
