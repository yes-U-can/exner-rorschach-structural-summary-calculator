# [Interpretation/Processing] W:M

## 別名・検索語

- W:M
- W_M
- processing W:M
- 処理 W:M
- Aspirational Ratio

## 中核的定義

`W:M` は、`W` 反応数と `M` 反応数の関係を示します。
情報処理では、広い取り方と観念的資源が現実的な比例に保たれているかをみるために使います。

## 解釈の要点

- `W` 側が強すぎると、目標や整理の広がりが利用できる観念的資源を上回ることがあります。
- よりつり合った比率なら、広い探索と実際の概念的資源が比較的合っていると読みやすいです。
- `AdjD` や `Ma:Mp` と一緒にみると、実際の運用可能性が分かりやすくなります。

## 注意点・鑑別

- `W:M` は成熟度を単独で決める指標ではありません。
- `M` 反応が非常に少ない記録では、比率が誇張されやすいです。
- 視野の広さと、実際の適応的柔軟性は同じではありません。

## 単独解釈の限界

- `W:M` は広がりと資源の関係を示す手がかりであって、全体機能の総括ではありません。
- `W:D:Dd`、`M`、`Ma:Mp`、`AdjD`、`3r+(2)/R` と合わせて読む必要があります。

## 併せて確認する変数

- `W:D:Dd`
- `M`
- `Ma:Mp`
- `AdjD`
- `3r+(2)/R`
