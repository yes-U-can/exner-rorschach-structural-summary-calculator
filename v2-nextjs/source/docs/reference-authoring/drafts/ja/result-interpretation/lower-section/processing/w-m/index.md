# Document Name: [Interpretation/Processing] W:M

## Aliases / Search Terms

- W:M
- W_M
- processing W:M
- 処理 W:M
- Aspirational Ratio

## Core Definition

`W:M` は、`W` 反応数と `M` 反応数の関係を示す。
Processing では、広い取り方と観念的資源が現実的な比例に保たれているかをみるために使う。

## Interpretation Points

- `W` 側が強すぎると、目標や整理の広がりが利用できる観念的資源を上回ることがある。
- よりつり合った比率なら、広い探索と実際の conceptual resource が比較的合っていると読みやすい。
- `AdjD` や `Ma:Mp` と一緒にみると、実際の運用可能性が分かりやすくなる。

## Cautions / Distinctions

- `W:M` は成熟度を単独で決める指標ではない。
- `M` 反応が非常に少ない記録では、比率が誇張されやすい。
- 視野の広さと、実際の適応的 flexibility は同じではない。

## Limits of Isolated Interpretation

- `W:M` は breadth と resource の関係を示す手がかりであって、全体機能の総括ではない。
- `W:D:Dd`、`M`、`Ma:Mp`、`AdjD`、`3r+(2)/R` と合わせて読む必要がある。
