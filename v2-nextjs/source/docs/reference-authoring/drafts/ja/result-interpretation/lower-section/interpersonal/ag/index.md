# [Interpretation/Interpersonal] AG

## 別名・検索語

- AG
- aggressive movement
- 攻撃的イメージ

## 中核的定義

`AG` は、対人場面が攻撃的、敵対的、対立的な形でどれくらい表象されるかを示します。
解釈では、対人関係性が葛藤や脅威を軸に組織化されやすいかを読むために使います。

## 解釈の要点

- `AG` が高いと、対人接触が緊張的、攻撃的、対立的に想像されやすいことがあります。
- `AG` が認められなくても葛藤が全くないとは言えませんが、露骨に攻撃的な場面の重みは下がります。
- `AG` は対人表象の仕方を示すのであって、暴力行動の直接証拠ではありません。
- `COP` と比べると、記録が主として協力的か、両価的か、葛藤的かを見やすいです。

## 注意点・鑑別

- `AG` をそのまま行動上のリスクに翻訳しません。
- `MOR`、`S_aff`、`PHR` と一緒に出ると意味が変わることがあります。
- 攻撃性が防衛として出ているのか、期待として出ているのか、一般的な想像上の情緒的調子なのかを見分ける必要があります。

## 単独解釈の限界

- `AG` は対人スタイルの一部にすぎません。
- `COP`、`ISO_Index`、`SumT_inter`、`MOR`、`PHR` と一緒に読む必要があります。

## 併せて確認する変数

- `COP`
- `PHR`
- `S`
- `MOR`
- `ISO_Index`
