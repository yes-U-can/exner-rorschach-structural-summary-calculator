# Document Name: [Interpretation/Interpersonal] AG

## Aliases / Search Terms

- AG
- aggressive movement
- 攻撃的イメージ

## Core Definition

`AG` は、対人場面が攻撃的、敵対的、対立的な形でどれくらい表象されるかを示す。
Interpretation では、relatedness が conflict や threat を軸に組織化されやすいかを読むために使う。

## Interpretation Points

- `AG` が高いと、対人接触が緊張的、攻撃的、対立的に想像されやすいことがある。
- `AG` が absent でも conflict が全くないとは言えないが、露骨に攻撃的な場面の重みは下がる。
- `AG` は対人表象の仕方を示すのであって、暴力行動の直接証拠ではない。
- `COP` と比べると、記録が主として協力的か、ambivalent か、conflictual かを見やすい。

## Cautions / Distinctions

- `AG` をそのまま behavioral risk に翻訳しない。
- `MOR`、`S_aff`、`PHR` と一緒に出ると意味が変わりうる。
- 攻撃性が防衛として出ているのか、期待として出ているのか、一般的な imagined tone なのかを見分ける必要がある。

## Limits of Isolated Interpretation

- `AG` は対人スタイルの一部にすぎない。
- `COP`、`ISO_Index`、`SumT_inter`、`MOR`、`PHR` と一緒に読む必要がある。
