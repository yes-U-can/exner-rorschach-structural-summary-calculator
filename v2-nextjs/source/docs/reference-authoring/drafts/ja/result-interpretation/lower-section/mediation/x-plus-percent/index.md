# [Interpretation/Mediation] X+%

## 別名・検索語

- X+%
- Conventional Form
- 慣習的形態

## 中核的定義

`X+%` は、形態が慣習的で明確に共有しやすい反応の割合を示します。
媒介が一般的で合意されやすい読み取りにどれだけ沿っているかをまとめる指標です。

## 解釈の要点

- `X+%` が高いと、知覚は比較的安定し、合意的な翻訳に近いことが多いです。
- これが下がると、慣習的読みへの依存が減り、珍しい、または精度の低い翻訳に重みが移る可能性があります。
- `P` と一緒に読むと、社会的に共有された基準点がまだ利用できているかが見えます。

## 注意点・鑑別

- `X+%` は `XA%` と同じではありません。`XA%` はより広い妥当範囲を含みます。
- `X+%` が低いだけで、ただちに重い歪みとは言えません。`Xu%` と `X-%` も一緒に確認してください。
- 短いプロトコルでは、少数の反応で比率が動きやすくなります。
