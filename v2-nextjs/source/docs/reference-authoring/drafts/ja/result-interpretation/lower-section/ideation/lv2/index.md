# [Interpretation/Ideation] Lv2

## 別名・検索語

- Lv2
- Level 2 special scores
- level 2 markers
- レベル2特殊スコア

## 中核的定義

`Lv2` は、プロトコル内にレベル 2 の特殊スコアがいくつ現れるかを示します。
解釈では、より重い認知的特異性が思考の中でどの程度の重みをもつかを見るために使います。

## 解釈の要点

- `Lv2 = 0` なら、特殊スコア群の中にレベル 2 マーカーは出ていません。
- 出現が1回だけなら、過大評価する前に文脈を慎重に確認する必要があります。
- レベル 2 の出現が複数あると、よりはっきりした思考のまとまりの弱さやゆがみの可能性が高まります。

## 注意点・鑑別

- `Lv2` は単純な頻度よりも、重症度の重みを強く反映します。
- これだけで思考障害を結論づけることはできず、[`WSum6`](ref://result-interpretation/lower-section/ideation/WSum6)、[`M-`](ref://result-interpretation/lower-section/ideation/M_minus)、[`PTI`](ref://result-interpretation/special-indices/PTI)、認知的媒介変数と合わせて読む必要があります。
- [`CONTAM`](ref://scoring-input/special-score/CONTAM) もレベル 2 に含まれるが、構造は `DV2`、`DR2`、`INCOM2`、`FABCOM2` と同じではありません。
