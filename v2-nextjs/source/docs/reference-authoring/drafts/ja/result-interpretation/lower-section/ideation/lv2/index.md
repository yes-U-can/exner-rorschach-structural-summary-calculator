# Document Name: [Interpretation/Ideation] Lv2

## Aliases / Search Terms

- Lv2
- Level 2 special scores
- level 2 markers
- レベル2特殊得点

## Core Definition

`Lv2` は、プロトコル内に level 2 の特殊得点がいくつ現れるかを示す。
解釈では、より重い認知的特異性が思考の中でどの程度の重みをもつかを見るために使う。

## Interpretation Points

- `Lv2 = 0` なら、特殊得点群の中に level 2 マーカーは出ていない。
- 出現が1回だけなら、過大評価する前に文脈を慎重に確認する必要がある。
- level 2 の出現が複数あると、よりはっきりした思考のまとまりの弱さやゆがみの可能性が高まる。

## Cautions / Distinctions

- `Lv2` は単純な頻度よりも、重症度の重みを強く反映する。
- これだけで思考障害を結論づけることはできず、[`WSum6`](ref://result-interpretation/lower-section/ideation/WSum6)、[`M-`](ref://result-interpretation/lower-section/ideation/M_minus)、[`PTI`](ref://result-interpretation/special-indices/PTI)、mediation 変数と合わせて読む必要がある。
- [`CONTAM`](ref://scoring-input/special-score/CONTAM) も level 2 に含まれるが、構造は `DV2`、`DR2`、`INCOM2`、`FABCOM2` と同じではない。
