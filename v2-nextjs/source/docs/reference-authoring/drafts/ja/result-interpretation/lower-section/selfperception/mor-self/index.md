# [Interpretation/Self Perception] MOR

## 別名・検索語

- MOR
- morbid self-image
- 傷ついた自己像

## 中核的定義

`Self Perception` における `MOR` は、自己像が傷ついたもの、欠陥のあるもの、消耗したもの、価値の乏しいものとして経験される度合いを示します。
`Ideation` の `MOR` が思考全体の調子を示すのに対し、ここではその調子が自己像にどれだけ向いているかを見ます。

## 解釈の要点

- 高いときは、欠陥感、損傷感、個人的な痛みが自己像に強く結びついている可能性があります。
- `SumV_self` と一緒に高いと、否定的自己像に強い自己批判が伴いやすいです。
- `An_Xy` も高いと、身体像がより壊れやすいもの、脅かされるものとして感じられやすくなります。
- 思考の `MOR` も高いと、否定的なトーンが広く、しかも個人的に深くなっている可能性があります。

## 注意点・鑑別

- 高い `MOR_self` だけで臨床的うつ病と断定しません。
- 実際の病気、外傷、喪失の時期でも反応的に上がることがあります。
- 0 だからといって、健康で安定した自己像を保証するわけではありません。

## 単独解釈の限界

- `MOR_self` は自己像の否定的なトーンを示しますが、それをどう防衛し、どう調整しているかまでは単独で決めません。
- `SumV_self`、`FD`、`Fr+rF`、`DEPI` と一緒に読む必要があります。
