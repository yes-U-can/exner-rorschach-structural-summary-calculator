# Document Name: [Interpretation/Self Perception] FD

## Aliases / Search Terms

- FD
- form dimension
- 内省

## Core Definition

`FD` は、経験から一歩引いて自分を観察する傾向を示します。  
`Self Perception` では、意識的な自己観察や内省の手がかりとして読みます。

## Interpretation Points

- `FD > 0` なら、ある程度の inward observation があることが多いです。
- その観察は適応的なこともあれば、批判、罪悪感、落ち込みを帯びることもあります。
- `SumV` や `MOR_self` と一緒に高いと、内省はより重く、痛みを伴いやすくなります。
- 自己中心性指数が高いときは、自己への注意がさらに固定されることがあります。

## Cautions / Distinctions

- 高い `FD` は自動的に insight の良さを意味しません。
- `FD = 0` でも内省が完全にないとは言えません。
- determinant の `FD` と content code の `Fd` は区別して扱います。

## Limits of Isolated Interpretation

- `FD` は自己観察の存在を示しますが、その質まで単独では決めません。
- `SumV`、`MOR_self`、`3r+(2)/R` と一緒に読む必要があります。
