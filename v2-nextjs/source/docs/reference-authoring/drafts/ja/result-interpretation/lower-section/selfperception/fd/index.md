# [Interpretation/Self Perception] FD

## 別名・検索語

- FD
- form dimension
- 内省

## 中核的定義

`FD` は、経験から一歩引いて自分を観察する傾向を示します。
`Self Perception` では、意識的な自己観察や内省の手がかりとして読みます。

## 解釈の要点

- `FD > 0` なら、ある程度の内向き観察があることが多いです。
- その観察は適応的なこともあれば、批判、罪悪感、落ち込みを帯びることもあります。
- `SumV` や `MOR_self` と一緒に高いと、内省はより重く、痛みを伴いやすくなります。
- 自己中心性指数が高いときは、自己への注意がさらに固定されることがあります。

## 注意点・鑑別

- 高い `FD` は自動的に洞察の良さを意味しません。
- `FD = 0` でも内省が完全にないとは言えません。
- 決定因の `FD` と内容コードの `Fd` は区別して扱います。

## 単独解釈の限界

- `FD` は自己観察の存在を示しますが、その質まで単独では決めません。
- `SumV`、`MOR_self`、`3r+(2)/R` と一緒に読む必要があります。

## 併せて確認する変数

- `3r+(2)/R`
- `SumV_self`
- `MOR_self`
- `FD coding`
- `Fd content`
