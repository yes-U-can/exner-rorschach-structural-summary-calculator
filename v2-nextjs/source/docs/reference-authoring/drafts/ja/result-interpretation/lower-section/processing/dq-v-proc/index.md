# Document Name: [Interpretation/Processing] DQv

## Aliases / Search Terms

- DQv
- Vague Developmental Quality
- processing DQv
- 処理 DQv

## Core Definition

`DQv` は、processing の中にあいまいで簡略化された、統合の弱い組織化がどれくらい現れているかを示す。
Interpretation では、組織化の質がより弱く脆い形へ傾いていないかをみる指標として使う。

## Interpretation Points

- `DQv` が増えると、processing がより単純で統合の弱い形になりやすい。
- `DQv` と `DQv/+` が `DQ+` より重くなると、組織化の質はより fragile に見えやすい。
- 意味づけは `Zd`、`PSV`、全体の developmental quality 分布と比べたときに明確になる。

## Cautions / Distinctions

- `DQv` は `DQv/+` と一緒に読み、同じものとして扱わない。
- 高い値だけでは、なぜ組織化が弱いのかまでは説明できない。
- それだけで自動的に重い clinical severity と結びつけない。

## Limits of Isolated Interpretation

- `DQv` は processing 上の弱点を示すが、それだけで認知スタイル全体を説明するわけではない。
- `DQ+`、`PSV`、`X-%`、`WSum6` と合わせて読む必要がある。
