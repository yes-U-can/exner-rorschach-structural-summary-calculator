# Document Name: [Interpretation/Processing] Zf

## Aliases / Search Terms

- Zf
- processing Zf
- 処理 Zf
- Z frequency
- Z頻度

## Core Definition

`Zf` は、刺激野を探索し整理しようとする試みがどれくらい頻繁に現れるかを示す。
Processing では、効率そのものではなく、組織化の試行回数として扱う。

## Interpretation Points

- `Zf` が高いと、刺激を見渡して整理しようとする試みが多いことが多い。
- `Zf` が低いと、能動的な探索や構造化の試みが少ない可能性がある。
- ただし頻度と効率は同じではないため、`Zd` と一緒に読む必要がある。

## Cautions / Distinctions

- `Zf` が高いことは、そのまま良い processing を意味しない。
- `Zf` が低い値でも、反応数が少ないプロトコルでは自然に見られうる。
- 解釈では `W:D:Dd` も合わせて確認し、探索の広がり方を見たほうがよい。
