# [Interpretation/Processing] Zf

## 別名・検索語

- Zf
- processing Zf
- 処理 Zf
- Z frequency
- Z頻度

## 中核的定義

`Zf` は、刺激野を探索し整理しようとする試みがどれくらい頻繁に現れるかを示します。
情報処理では、効率そのものではなく、組織化の試行回数として扱います。

## 解釈の要点

- `Zf` が高いと、刺激を見渡して整理しようとする試みが多いことが多いです。
- `Zf` が低いと、能動的な探索や構造化の試みが少ない可能性があります。
- ただし頻度と効率は同じではないため、`Zd` と一緒に読む必要があります。

## 注意点・鑑別

- `Zf` が高いことは、そのまま良い情報処理を意味しません。
- `Zf` が低い値でも、反応数が少ないプロトコルでは自然に見られることがあります。
- 解釈では `W:D:Dd` も合わせて確認し、探索の広がり方を見たほうがよいです。
