# [Interpretation/Processing] Zd

## 別名・検索語

- Zd
- processing Zd
- 処理 Zd
- processing efficiency

## 中核的定義

`Zd` は、実際の組織化水準が期待される水準と比べてどうかを示します。
情報処理では、探索と整理が均衡しているか、過剰か、不足しているかをみる指標として使います。

## 解釈の要点

- 値が期待範囲に近いと、探索と整理の量が比較的つり合っていることを示しやすいです。
- 強い正の `Zd` は、必要以上に取り込みすぎる過剰取り込みを示すことがあります。
- 強い負の `Zd` は、十分に取り込まず早く通り過ぎる過少取り込みを示すことがあります。
- `PSV` と一緒にみると、 硬直性による非効率と、単純な知覚的節約を区別しやすいです。

## 注意点・鑑別

- 正の `Zd` が自動的に「良い」という意味にはなりません。
- 負の `Zd` だけで重い障害を意味すると決めつけてはいけません。
- 解釈では `Zf`、`PSV`、`W:D:Dd` を必ず文脈として足す必要があります。
