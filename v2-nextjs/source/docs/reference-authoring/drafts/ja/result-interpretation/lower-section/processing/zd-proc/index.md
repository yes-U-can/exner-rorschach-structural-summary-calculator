# Document Name: [Interpretation/Processing] Zd

## Aliases / Search Terms

- Zd
- processing Zd
- 処理 Zd
- processing efficiency

## Core Definition

`Zd` は、実際の組織化水準が期待される水準と比べてどうかを示す。
Processing では、探索と整理が均衡しているか、過剰か、不足しているかをみる指標として使う。

## Interpretation Points

- 値が期待範囲に近いと、探索と整理の量が比較的つり合っていることを示しやすい。
- 強い正の `Zd` は、必要以上に取り込みすぎる overincorporation を示すことがある。
- 強い負の `Zd` は、十分に取り込まず早く通り過ぎる underincorporation を示すことがある。
- `PSV` と一緒にみると、 rigidity による非効率と、単純な知覚的 economy を区別しやすい。

## Cautions / Distinctions

- 正の `Zd` が自動的に「良い」という意味にはならない。
- 負の `Zd` だけで重い障害を意味すると決めつけてはいけない。
- 解釈では `Zf`、`PSV`、`W:D:Dd` を必ず文脈として足す必要がある。
