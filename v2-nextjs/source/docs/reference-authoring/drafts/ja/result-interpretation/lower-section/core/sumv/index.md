# [Interpretation/Core] SumV

## 別名・検索語

- SumV
- vista total
- total vista
- Vista総数

## 中核的定義

`SumV` はビスタ決定因の合計です。
`core` では、自己批判、自己への不満、自己評価に伴う苦しさが、現在の緊張にどの程度関わっているかを示しやすいです。

## 解釈の要点

- `SumV` が高い場合、自己批判、後悔、自分自身への不満が強まりやすいです。
- `core` では現在の負荷の一部として読み、自己概念との関係は [`SumV_self`](ref://result-interpretation/lower-section/selfPerception/SumV_self) でより具体的に検討します。
- `SumV` はほかの負荷指標と重なると [`es`](ref://result-interpretation/lower-section/core/es) を高め、適応余力を下げることがあります。
- 自己評価や不快感に関わる他の指標と比べると意味がより明確になります。

## 注意点・鑑別

- `SumV` が高いだけで、自己嫌悪や固定的性格特性をそのまま断定してはいけません。
- `SumV = 0` でも、別の形で自己批判が表れている可能性は残ります。
- `core` の現在負荷としての読みと、`self-perception` のより構造的な読みを区別することが重要です。
