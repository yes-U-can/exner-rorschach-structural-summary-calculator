# Document Name: [Interpretation/Affect] S

## Aliases / Search Terms

- S
- 白地S
- white space affect

## Core Definition

`Affect` における `S` は、白地を用いた反応のうち、情動的な意味をもつものの頻度を指します。
反対性、自己主張、緊張、あるいは刺激場から距離を取ろうとする傾向の手がかりとして見ます。

## Interpretation Points

- `S` が高いと、反対性、否定的構え、防衛的自己主張から反応しやすい可能性があります。
- プロトコルによっては、怒り、内的圧迫感、距離を取りたい必要性を反映することもあります。
- 白地反応のすべてが敵意ではなく、探索的な知覚様式や探りながらの反応を示す場合もあります。

## Cautions / Distinctions

- `upper section` の `S` や、`mediation` の `S-%` と混同しないでください。
- 件数が少ない場合、それだけで安定した反対性スタイルと決めつけるのは危険です。
- `AG`、`CP`、より広い緊張指標と一緒に出ているかを確認すると解釈が安定します。
