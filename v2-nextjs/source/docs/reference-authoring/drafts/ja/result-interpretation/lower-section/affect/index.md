# [Interpretation/Lower Section/Affect] Affect

## 別名・検索語

- affect
- 情動
- 感情様式

## 中核的定義

`Affect` は、プロトコルの中で情動がどのように体験され、調整され、表出されるかを読むための領域です。
通常は `FC:CF+C`、`Pure C`、`SumC / WSumC`、`Afr`、`S`、`Blends:R`、`CP` をまとめて見ます。

## 解釈の要点

- `FC:CF+C` は、感情が形態で調整されやすいか、それとも直接出やすいかを見ます。
- `Pure C`、`SumC / WSumC`、`CP` は、情動表出の強さと出方をみる助けになります。
- `Afr` は情動を帯びた刺激へ近づきやすいか、避けやすいかをみる比率です。
- `S` と `Blends:R` は、反対性や混合感情、複雑さを補足します。

## 注意点・鑑別

- 情動指標は `Core`、`Mediation`、総反応数と切り離して単独で読まないでください。
- どれか一つが少し高いだけで、すぐに情動統制不全と判断しないことが大切です。
- `R` が少ないプロトコルでは、情動比率の安定性が下がります。
