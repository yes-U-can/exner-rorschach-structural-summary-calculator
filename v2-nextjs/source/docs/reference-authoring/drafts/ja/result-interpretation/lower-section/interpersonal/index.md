# Document Name: [Interpretation/Lower Section/Interpersonal] Interpersonal

## Aliases / Search Terms

- interpersonal
- 対人関係
- relational style

## Core Definition

`Interpersonal` は、他者との接触をどのように想像し、求め、恐れ、組織化しているかを読むための変数群をまとめた cluster である。
この領域では主に `COP`、`AG`、`a:p`、`Food`、`SumT`、human content、対人孤立をみる。

## Interpretation Points

- `COP` と `AG` は、対人場面がより協力的に想像されるか、より衝突的に想像されるかを示す。
- `SumT_inter` と `Food` は、親密さ、支え、依存への欲求を見積もる助けになる。
- `HumanCont` と `PureH` は、記録の中にどれくらい人間表象が現れているかを示す。
- `PER` と `ISO_Index` は、自己言及、距離、社会的孤立についての情報を補う。

## Cautions / Distinctions

- 対人変数を単独で読まない。
- human content が高いだけでは、関係が健康とは言えない。
- 対人スタイルの意味は affect、mediation、ideation によって変わる。

## Limits of Isolated Interpretation

- この cluster は relatedness の期待や組織化の方向を示すが、対人生活の全説明ではない。
- `Core`、`Affect`、`Ideation`、`Self Perception` と統合して読む必要がある。
