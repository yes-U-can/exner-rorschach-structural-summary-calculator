# [Interpretation/Lower Section/Interpersonal] Interpersonal

## 別名・検索語

- interpersonal
- 対人関係
- relational style

## 中核的定義

`Interpersonal` は、他者との接触をどのように想像し、求め、恐れ、組織化しているかを読むための変数群です。
この領域では主に `COP`、`AG`、`a:p`、`Food`、`SumT`、人間内容、対人孤立をみます。

## 解釈の要点

- `COP` と `AG` は、対人場面がより協力的に想像されるか、より衝突的に想像されるかを示します。
- `SumT_inter` と `Food` は、親密さ、支え、依存への欲求を見積もる助けになります。
- `HumanCont` と `PureH` は、記録の中にどれくらい人間表象が現れているかを示します。
- `PER` と `ISO_Index` は、自己言及、距離、社会的孤立についての情報を補います。

## 注意点・鑑別

- 対人変数を単独で読みません。
- 人間内容が高いだけでは、関係が健康とは言えません。
- 対人スタイルの意味は感情、認知的媒介、思考によって変わります。

## 単独解釈の限界

- この変数群は対人関係性の期待や組織化の方向を示すが、対人生活の全説明ではありません。
- `Core`、`Affect`、`Ideation`、`Self Perception` と統合して読む必要があります。

## 併せて確認する変数

- `SumT`
- `a:p`
- `H ratio`
- `CDI`
- `DEPI`
