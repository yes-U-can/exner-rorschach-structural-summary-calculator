# [Interpretation/Interpersonal] SumT

## 別名・検索語

- SumT
- interpersonal SumT
- texture total
- need for closeness
- 対人 SumT

## 中核的定義

`SumT_inter` は、`SumT` を対人的水準で読んだものです。
解釈では、親密さ、接触、支えてもらう感覚への欲求が対人関係性の中にどれくらい現れているかをみます。

## 解釈の要点

- `SumT_inter` が高いと、情緒的近接や保護的接触への欲求がより強いことがあります。
- これは対人技能よりも、親密さを必要とする度合いを示します。
- `COP` と一緒に出ると、その近接欲求が相互的な関係の中に表象されやすいです。
- `ISO_Index` と組み合わさると、親密さへの欲求と距離・孤立が同時に存在する形が見えやすいです。
- `Food` と一緒なら、より依存的で切望を帯びた色合いになることがあります。

## 注意点・鑑別

- 高い値がそのまま良い関係を意味するわけではありません。
- 0 だからといって親密さへの欲求が全くないとは言えません。陰影の使い方全体にも左右されます。
- `Core` の `SumT` と、対人的に読む `SumT_inter` は区別する必要があります。

## 単独解釈の限界

- この変数は、対人的文脈の中で読んで初めてはっきりします。
- `Food`、`COP`、`HumanCont`、`ISO_Index` と合わせて読む必要があります。
