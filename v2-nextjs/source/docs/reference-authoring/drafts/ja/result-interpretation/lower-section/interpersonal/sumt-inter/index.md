# Document Name: [Interpretation/Interpersonal] SumT

- interpersonal SumT

- 対人 SumT

## Core Definition

`SumT_inter` は、`SumT` を対人的水準で読んだものである。
Interpretation では、親密さ、接触、支えてもらう感覚への欲求が relatedness の中にどれくらい現れているかをみる。

## Interpretation Points

- `SumT_inter` が高いと、情緒的近接や保護的接触への欲求がより強いことがある。
- これは relational skill よりも、親密さを必要とする度合いを示す。
- `COP` と一緒に出ると、その近接欲求が相互的な関係の中に表象されやすい。
- `ISO_Index` と組み合わさると、親密さへの欲求と距離・孤立が同時に存在する形が見えやすい。
- `Food` と一緒なら、より dependent で longing な色合いを帯びることがある。

## Cautions / Distinctions

- 高い値がそのまま良い関係を意味するわけではない。
- 0 だからといって親密さへの欲求が全くないとは言えない。shading の使い方全体にも左右される。
- `Core` の `SumT` と、対人的に読む `SumT_inter` は区別する必要がある。

## Limits of Isolated Interpretation

- この変数は、対人的文脈の中で読んで初めてはっきりする。
- `Food`、`COP`、`HumanCont`、`ISO_Index` と合わせて読む必要がある。
