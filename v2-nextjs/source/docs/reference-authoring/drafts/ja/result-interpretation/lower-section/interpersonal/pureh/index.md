# Document Name: [Interpretation/Interpersonal] Pure H

## Aliases / Search Terms

- Pure H
- pure human
- whole human content
- 完全な人間表象

## Core Definition

`PureH` は、人間内容が完全で通常の形、つまり `H` としてどれくらい現れるかを示す。
対人的水準では、相手のイメージが part-object や partial figure、fantasy human ではなく、全体的な人物として組織化されやすいかをみる助けになる。

## Interpretation Points

- `PureH` が比較的高いと、人をより全体的で統合された像として表しやすいことが多い。
- `HumanCont` は低くないのに `PureH` が低いなら、人間表象がより部分的、理想化的、あるいは歪みを帯びていないかを確認したい。
- `PureH < 2` のときは、対人スタイルと human content の質をより慎重に見直す根拠になりやすい。
- この変数は human representation の structural quality を示すのであって、親切さや共感性そのものではない。

## Cautions / Distinctions

- `PureH` と `HumanCont` は同じではない。前者は complete human content、後者は total human content を見る。
- 高くても `PHR`、`AG`、`MOR` が高いなら、成熟した relatedness を保証しない。
- 低いからといって、それだけで人への拒否を意味するわけではない。

## Limits of Isolated Interpretation

- `PureH` は `HumanCont`、`GHR`、`PHR`、`H ratio` と比べて読む必要がある。
- human response 自体が非常に少ない記録では意味づけが変わる。
