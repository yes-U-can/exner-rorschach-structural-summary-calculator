# [Interpretation/Interpersonal] Pure H

## 別名・検索語

- Pure H
- pure human
- whole human content
- 完全な人間表象

## 中核的定義

`PureH` は、人間内容が完全で通常の形、つまり `H` としてどれくらい現れるかを示します。
対人的水準では、相手のイメージが部分対象や部分的人物像、空想的人間ではなく、全体的な人物として組織化されやすいかをみる助けになります。

## 解釈の要点

- `PureH` が比較的高いと、人をより全体的で統合された像として表しやすいことが多いです。
- `HumanCont` は低くないのに `PureH` が低いなら、人間表象がより部分的、理想化的、あるいは歪みを帯びていないかを確認する必要があります。
- `PureH < 2` のときは、対人スタイルと人間内容の質をより慎重に見直す根拠になりやすいです。
- この変数は人間表象の構造的質を示すのであって、親切さや共感性そのものではありません。

## 注意点・鑑別

- `PureH` と `HumanCont` は同じではありません。前者は完全な人間内容、後者は人間内容の総数を見ます。
- 高くても `PHR`、`AG`、`MOR` が高いなら、成熟した対人関係性を保証しません。
- 低いからといって、それだけで人への拒否を意味するわけではありません。

## 単独解釈の限界

- `PureH` は `HumanCont`、`GHR`、`PHR`、`H ratio` と比べて読む必要があります。
- 人間反応自体が非常に少ない記録では意味づけが変わります。

## 併せて確認する変数

- `HumanCont`
- `GHR`
- `PHR`
- `H ratio`
- `CDI`
