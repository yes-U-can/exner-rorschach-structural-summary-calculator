# [Interpretation/Affect] FC:CF+C

## 別名・検索語

- FC:CF+C
- FC_CF_C
- 情動比
- 形態色彩比

## 中核的定義

`FC:CF+C` は `FC : (CF + C + Cn)` と表示し、各出現を同じ重みで数えます。有彩色反応の中で、形態と色彩がどう組み合わさっているかを示す比率です。
解釈では、感情が形態で調整されやすいか、それとも直接放出されやすいかを見積もるのに使います。

## 解釈の要点

- `FC` が優勢なら、感情表出は比較的統制され、形に支えられていることが多いです。
- `CF+C` が優勢なら、感情がより直接的、即時的、あるいは調整の少ない形で出やすくなります。
- 両側が比較的つり合うときは、統制と直接表出が混在していると考えます。
- この比率は情動の量そのものより、情動の出し方の様式を読む指標です。

## 注意点・鑑別

- 色彩反応が少ないプロトコルでは、この比率は安定しにくくなります。
- `CF+C` 側が強いだけで、すぐに病理や衝動性と決めつけないでください。
- `Pure C` は、より直接的な情動放出を示すことがあり、この比率だけでは十分に表れません。
- `Cn` は表示比率の右辺にだけ含めます。`WSumC`、別規則である `S-CON` の比較 `CF+C > FC`、色彩・陰影ブレンド数からは除外します。
