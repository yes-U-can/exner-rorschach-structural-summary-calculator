# Document Name: [Interpretation/Interpersonal] Food

## Aliases / Search Terms

- Food
- food content
- dependency imagery
- 食物内容

## Core Definition

`Food` は、対人的水準で読んだときに food content がどれくらい現れるかを示す。
Interpretation では、care、support、dependency への欲求を示す補助的な指標として使う。

## Interpretation Points

- 成人記録で `Food` が出るときは、支えられたい欲求や relational gratification の必要を見直す価値がある。
- `SumT_inter` と一緒に出ると、親密さを求める方向がより強く見えやすい。
- これは literal な空腹よりも、「養われたい」「世話されたい」という象徴的な手がかりとして使われることが多い。

## Cautions / Distinctions

- 子どものプロトコルでは、成人とは重みが違いうる。
- `Food = 1` だけで intense dependency を結論づけない。
- coding 上の `Fd` と、対人的 deprivation の interpretation を自動的に同一視しない。

## Limits of Isolated Interpretation

- `Food` は補助変数である。
- `SumT_inter`、`COP`、`ISO_Index`、`CDI` と一緒に読む必要がある。
