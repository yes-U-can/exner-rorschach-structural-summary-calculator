# [Interpretation/Interpersonal] Food

## 別名・検索語

- Food
- food content
- dependency imagery
- 食物内容

## 中核的定義

`Food` は、対人的水準で読んだときに食物内容がどれくらい現れるかを示します。
解釈では、ケア、支え、依存への欲求を示す補助的な指標として使います。

## 解釈の要点

- 成人記録で `Food` が出るときは、支えられたい欲求や対人関係上の充足の必要を見直す価値があります。
- `SumT_inter` と一緒に出ると、親密さを求める方向がより強く見えやすいです。
- これは文字どおりの空腹よりも、「養われたい」「世話されたい」という象徴的な手がかりとして使われることが多いです。

## 注意点・鑑別

- 子どものプロトコルでは、成人とは重みが異なることがあります。
- `Food = 1` だけで強い依存性を結論づけません。
- コーディング上の `Fd` と、対人的欠乏の解釈を自動的に同一視しません。

## 単独解釈の限界

- `Food` は補助変数です。
- `SumT_inter`、`COP`、`ISO_Index`、`CDI` と一緒に読む必要があります。

## 併せて確認する変数

- `SumT_inter`
- `SumT`
- `COP`
- `ISO_Index`
- `CDI`
