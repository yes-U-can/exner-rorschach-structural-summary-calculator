# Document Name: [Interpretation/Ideation] MOR

## Aliases / Search Terms

- MOR
- morbid ideation
- morbid content tone
- 病的内容トーン

## Core Definition

`MOR` は、損傷、喪失、欠陥、破綻、失敗、劣化の主題が思考の中にどれだけ現れるかを示す。
ideation 領域では、思考内容がどの程度、否定的で傷ついたイメージに染められているかを見る助けになる。

## Interpretation Points

- `MOR` が高い場合、思考が損傷、喪失、失敗、劣化の主題により頻繁に向かいやすい。
- より悲観的な認知トーンや、否定的結果を予期しやすい構えと結びつくことがある。
- こうした主題が人間内容や自己関連内容に出るときは、読みは self-perception 領域に近づく。
- 他の否定的感情指標や思考の質指標と統合すると意味がより明確になる。

## Cautions / Distinctions

- `MOR` が高いだけで、抑うつ、トラウマ、特定の障害をそのまま確定してはいけない。
- 単発の不快イメージと、思考全体にわたる持続的な病的トーンは区別する必要がある。
- 低い値でも、自己批判や絶望感がまったくないとは限らない。
