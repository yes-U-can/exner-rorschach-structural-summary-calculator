# [Interpretation/Ideation] MOR

## 別名・検索語

- MOR
- morbid ideation
- morbid content tone
- 病的内容トーン

## 中核的定義

`MOR` は、損傷、喪失、欠陥、破綻、失敗、劣化の主題が思考の中にどれだけ現れるかを示します。
思考領域では、思考内容がどの程度、否定的で傷ついたイメージに染められているかを見る助けになります。

## 解釈の要点

- `MOR` が高い場合、思考が損傷、喪失、失敗、劣化の主題により頻繁に向かいやすいです。
- より悲観的な認知トーンや、否定的結果を予期しやすい構えと結びつくことがあります。
- こうした主題が人間内容や自己関連内容に出るときは、読みは自己知覚領域に近づきます。
- 他の否定的感情指標や思考の質指標と統合すると意味がより明確になります。

## 注意点・鑑別

- `MOR` が高いだけで、抑うつ、トラウマ、特定の障害をそのまま確定してはいけません。
- 単発の不快イメージと、思考全体にわたる持続的な病的トーンは区別する必要があります。
- 低い値でも、自己批判や絶望感がまったくないとは限りません。
