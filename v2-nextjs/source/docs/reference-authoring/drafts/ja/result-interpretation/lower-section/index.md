## Core Definition

`Lower Section` は、構造要約の下段で資源、負荷、思考、感情、媒介、処理、対人傾向、自己知覚を機能的なまとまりとして読む領域です。
`Upper Section` が最初の見取り図なら、`Lower Section` は臨床的な理解を具体化する中心部分です。

## Interpretation Points

- 通常の読み順は `core -> ideation -> affect -> mediation -> processing -> interpersonal -> selfPerception` です。
- 単独の数値よりも、複数の指標が互いを支えるか修正するかが重要です。
- [`Upper Section`](ref://result-interpretation/upper-section) で見えた初期シグナルを、この領域で確認したり意味づけし直したりします。

## Cautions / Distinctions

- `Lower Section` の各クラスターは互いに影響し合うため、孤立して読むと過度な一般化が起きやすくなります。
- 反応数、防衛性、文化的背景、検査場面のストレスは各変数の重みを変えます。
