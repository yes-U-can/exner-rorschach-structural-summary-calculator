# Document Name: [Interpretation/Processing] DQ+

## Aliases / Search Terms

- DQ+
- Developmental Quality Plus
- processing DQ+
- 処理 DQ+

## Core Definition

`DQ+` は、processing の中にどれだけ統合された複雑な組織化が現れているかを示す。
Interpretation では、知覚内容を複雑で分節化された形にまとめる力のサインとして使う。

## Interpretation Points

- `DQ+` が十分に見られるときは、複雑な材料を統合的に整理する力が比較的保たれていることが多い。
- `DQ+` が続き、`DQv` が強すぎない場合は、processing がより elaborated で sophisticated に見えやすい。
- 意味づけは `Zd`、`PSV`、全体の developmental quality 分布と比べたときに明確になる。

## Cautions / Distinctions

- `DQ+` が高いからといって、自動的に適応が優れているとは言えない。
- `DQ+` が高くても `DQv` も高いなら、組織化は uneven なままのことがある。
- `DQ+` と `v/+` は同じ coding family に属するが、interpretation 上の役割は同じではない。

## Limits of Isolated Interpretation

- `DQ+` は processing の洗練さを示す一手がかりであって、全機能の総括ではない。
- `DQv`、`Zd`、`PSV`、`W:M` と一緒に読む必要がある。
