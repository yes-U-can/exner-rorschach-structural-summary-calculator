# Document Name: [Interpretation/Self Perception] Egocentricity Index

## Aliases / Search Terms

- 3r+(2)/R
- Egocentricity Index
- 自己中心性指数

## Core Definition

`3r+(2)/R` は `[3 x (Fr + rF) + Sum(2)] / R` で求める自己中心性指数です。reflection determinant 1個は pair response 3個分として重みづけされ、名称中の pair 記号 `(2)` があっても乗数は2ではありません。
自己への関心、自己重要感、自己価値の占める比重を読む手がかりになります。

## Interpretation Points

- 高いときは、自己注目や自己確認の必要が強くなっている可能性があります。
- 極端に高いときは、落ち着いた自信というより、自己価値を守ったり補強したりする必要が目立つことがあります。
- 低いときは、自己主張や自己価値への投資が弱い可能性があります。
- これは自己関心の指標であって、健康な自尊感情をそのまま示すものではありません。

## Cautions / Distinctions

- 成人では `.33` 未満や `.44` 超を丁寧に見ることが多いですが、この指数だけで決めません。
- `R` が少ないと比率が極端に見えやすいです。
- 高値でも自動的に narcissism と断定しません。

## Limits of Isolated Interpretation

- `Fr+rF`、`FD`、`SumV`、`MOR`、`H ratio` と一緒に読む必要があります。
- pair response や total response が少ないときは意味づけが不安定になります。
