# [Interpretation/Self Perception] Egocentricity Index

## 別名・検索語

- 3r+(2)/R
- Egocentricity Index
- 自己中心性指数

## 中核的定義

`3r+(2)/R` は `[3 x (Fr + rF) + Sum(2)] / R` で求める自己中心性指標です。反射決定因1個はペア反応3個分として重みづけされ、名称中のペア反応記号 `(2)` があっても乗数は2ではありません。
自己への関心、自己重要感、自己価値の占める比重を読む手がかりになります。

## 解釈の要点

- 高いときは、自己注目や自己確認の必要が強くなっている可能性があります。
- 極端に高いときは、落ち着いた自信というより、自己価値を守ったり補強したりする必要が目立つことがあります。
- 低いときは、自己主張や自己価値への投資が弱い可能性があります。
- これは自己関心の指標であって、健康な自尊感情をそのまま示すものではありません。

## 注意点・鑑別

- 成人では `.33` 未満や `.44` 超を丁寧に見ることが多いですが、この指数だけで決めません。
- `R` が少ないと比率が極端に見えやすいです。
- 高値でも自動的に自己愛と断定しません。

## 単独解釈の限界

- `Fr+rF`、`FD`、`SumV`、`MOR`、`H ratio` と一緒に読む必要があります。
- ペア反応や総反応数が少ないときは意味づけが不安定になります。

## 併せて確認する変数

- `Fr+rF`
- `FD`
- `H_ratio`
- `R`
- `pair`
