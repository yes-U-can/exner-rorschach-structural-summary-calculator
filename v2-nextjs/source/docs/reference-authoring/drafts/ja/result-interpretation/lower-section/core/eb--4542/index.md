# [Interpretation/Core] EB

## 別名・検索語

- EB
- Erlebnistypus
- Experience Balance
- experiential balance
- 体験型

## 中核的定義

`EB` は `M : WSumC` の比率です。
問題に向き合うとき、思考的・内省的な精緻化により頼るのか、感情調整や情動反応により頼るのかという優勢方向を示します。

## 解釈の要点

- `M` 側が優勢なら、通常はより `introversive` な傾向として読みます。
- `WSumC` 側が優勢なら、通常はより `extratensive` な傾向として読みます。
- 明確だが極端ではない方向がある場合は、好みの対処方向を持ちながらも一定の柔軟さが残っていると考えます。
- はっきりした方向が出ない場合は `ambitent` と読み、対処方向の一貫性が低い可能性を示します。

## 注意点・鑑別

- 強い方向性そのものが、直ちに健康性や病理を意味するわけではありません。
- 差がかなり大きいときは [`EBPer`](ref://result-interpretation/lower-section/core/EBPer) も確認します。
- 短いプロトコル、高い [`Lambda`](ref://result-interpretation/lower-section/core/Lambda)、低い [`EA`](ref://result-interpretation/lower-section/core/EA) の場合、`EB` は実際以上に硬く見えたり空疎に見えたりすることがあります。
