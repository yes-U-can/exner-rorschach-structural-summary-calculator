# Document Name: [Interpretation/Special Indices] Obsessive Style Index

## Aliases / Search Terms

- OBS
- Obsessive Style Index
- 強迫的スタイル指数

## Core Definition

`OBS` は、過度の正確さ、control、几帳面さ、rigid checking をまとめる constellation です。  
formal diagnosis ではなく、強迫的スタイルの方向を検出するために使います。

## Interpretation Points

- 陽性なら、overcontrol、detail への過度の注意、exactness への強い要求が出やすいです。
- そのスタイルは秩序や正確さを助ける一方で、processing を硬くしやすいです。
- `Zf`、`Zd`、`Dd`、`P`、`FQ+` がまとまるときに重みが増します。
- これは obsessive-compulsive disorder そのものではなく、style の方向です。

## Cautions / Distinctions

- `OBS` は yes/no 判定で、細かい度合いを測るものではありません。
- OCD diagnosis と混同しません。
- control が求められる文化的・職業的文脈でも上がり得ます。

## Limits of Isolated Interpretation

- style を理解するには、必ず base variable を見直します。
- `Zf`、`Zd`、`Dd`、`P`、`FQ+` と一緒に読みます。
