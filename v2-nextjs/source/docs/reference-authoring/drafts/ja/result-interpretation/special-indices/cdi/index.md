# Document Name: [Interpretation/Special Indices] CDI

## Aliases / Search Terms

- CDI
- Coping Deficit Index
- 対処欠損指数

## Core Definition

`CDI` は、coping resource と interpersonal effectiveness の脆弱さをまとめた composite index です。  
要求、緊張、対人圧力に対して、自分がうまく対処できないと感じやすいかを見る手がかりです。

## Interpretation Points

- 陽性なら、依存、対人 inefficiency、要求への耐性の低さを見直します。
- `Food`、`SumT_inter`、`ISO_Index`、`PureH`、`EA` と一緒に出ると意味が強まります。
- helplessness、support need、stress 下での柔軟性低下を示すことがあります。
- `DEPI` も陽性なら、脆弱さに depressive tone が混ざっている可能性があります。

## Cautions / Distinctions

- `CDI` は depression index ではありません。
- 固定した immature trait の証拠として扱いません。
- 年齢、社会的支援、現在の要求水準で意味は変わります。

## Limits of Isolated Interpretation

- resource 不足のリスクを示しますが、その原因までは単独で決めません。
- `Food`、`SumT_inter`、`ISO_Index`、`PureH`、`EA` に戻って読みます。
