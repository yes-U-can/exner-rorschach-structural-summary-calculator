# Document Name: [Interpretation/Special Indices] PTI

## Aliases / Search Terms

- PTI
- Perceptual-Thinking Index
- 知覚-思考指数

## Core Definition

`PTI` は、知覚と思考の解体や歪みを示す徴候をまとめた composite index です。  
現実検討の歪み、ideational slippage、perceptual precision の低下を優先的に見直すべきかを知らせます。

## Interpretation Points

- `PTI` が陽性なら、まず mediation と ideation を詳しく見ます。
- `WSum6`、`Lv2`、`M-`、`X-%`、`XA%` が複数方向から崩れているときに意味が強くなります。
- これは thought disturbance の可能性を示すのであって、自動診断ではありません。
- protocol の別領域も同じ方向を示すほど重みが増します。

## Cautions / Distinctions

- 陽性でも psychosis をそれだけで確定しません。
- 陰性でも disturbed thinking の可能性を完全には消しません。
- 強い stress、intoxication、situational disorganization、反応数の少なさで読みが難しくなることがあります。

## Limits of Isolated Interpretation

- `PTI` は actuarial signal なので、必ず base variable に戻ります。
- `WSum6`、`Lv2`、`M-`、`X-%`、`XA%` を確認してから強い結論に進みます。
