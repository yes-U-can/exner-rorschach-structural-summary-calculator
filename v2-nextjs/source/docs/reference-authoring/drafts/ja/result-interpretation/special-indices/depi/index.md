# Document Name: [Interpretation/Special Indices] DEPI

## Aliases / Search Terms

- DEPI
- Depression Index
- 抑うつ指数

## Core Definition

`DEPI` は、protocol の中で depressive feature を示す徴候がどれだけまとまっているかを示す composite index です。  
affect、負荷、否定的自己像、emotion withdrawal を統合して見ます。

## Interpretation Points

- 陽性なら、落胆、hopelessness、guilt、活力低下を丁寧に見直します。
- `SumV`、`SumY`、`MOR_self`、変化した `Afr` と一緒に出ると意味が強くなります。
- probable depressive pattern を示し得ますが、それだけで major depressive episode を確定しません。
- `CDI` も陽性なら、 helplessness の色合いがさらに強まることがあります。

## Cautions / Distinctions

- 陽性でも formal diagnosis ではありません。
- 陰性でも depression を除外しません。
- grief、life context、recent stress で constellation は変わります。

## Limits of Isolated Interpretation

- base variable に戻ってから臨床結論を固定します。
- `Affect`、`SumV`、`SumY`、`MOR_self`、`CDI` と統合します。
