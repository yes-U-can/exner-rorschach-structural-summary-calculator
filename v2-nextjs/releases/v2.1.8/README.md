# [2026-07-11] v2.1.8 버그 패치

## 먼저 확인하실 점

v2.1.8은 AI 도우미가 검색하는 다섯 언어 참조 문서 1,015개를 다시 읽고, 수정된 내용을 실제 검색 자료와 GPT-5.5 답변 검사까지 빠짐없이 반영한 패치입니다.

앱 UI/UX는 변경하지 않았습니다. 계산기 자체의 수식이 아니라, 선택 기능인 AI 도우미가 어떤 문서를 찾아 설명하는지를 개선했습니다.

## 후속 기록 (2026-07-13)

처음 쓴 요약에는 v2.1.8로 v2.1.x의 AI 품질 개선 전체를 마무리한 것처럼 적었습니다. v2.1.8에서 실제로 완료한 범위는 **다섯 언어 참조 문서와 AI 검색 자료의 갱신**입니다. 이후 검색 결과를 합치는 방식과 다국어 질문 처리에 별도의 문제가 확인되어 v2.1.9와 v2.1.10에서 이어서 수정했습니다.

이 정정은 v2.1.8에 기록한 문서 수, 검색 자료 수, 실제 API 호출과 검사 결과를 바꾸지 않습니다.

## 핵심 변경

- 한국어, 영어, 일본어, 스페인어, 포르투갈어의 1,015개 참조 문서를 다시 확인하고 근거 수준에 맞게 보정했습니다.
- PTI, S-CON, DEPI, CDI, HVI, OBS 설명에 근거의 강도와 임상가가 반드시 확인해야 하는 한계를 명시했습니다.
- S-CON 양성만으로 위험을 단정하지 않고 직접적인 위험평가와 임상가 확인이 필요하다고 답하도록 했습니다.
- 다국어 검색에서 악센트가 있는 정식 표기와 악센트 없는 사용자 입력이 함께 검색되도록 정규화했습니다.
- 수정된 참조 문서를 검색하기 좋은 짧은 단위 6,632개로 다시 나누고, OpenAI 검색용 수치 표현을 모두 새로 만들었습니다.
- GPT-5.5를 고정 모델로 사용해 한 번 묻는 질문 18회와 이어 묻는 대화 8회를 실제 API로 확인했습니다.
- 자연스러운 동의어를 잘못된 답으로 판단하지 않으면서도 진단 단정, 근거 없는 확정, 자동 적용은 계속 막도록 답변 검사 기준을 개선했습니다.
- 공개 미러에서 환경변수, Vercel 로컬 상태, 캐시, 로그, 비공개 작업 노트를 제외하고 기존 잔여물도 제거하도록 릴리즈 스크립트를 강화했습니다.

## 공개 범위와 보안 경계

공개된 AI 검사 기록은 검사 항목 번호, 완료 상태, 발견된 문제 종류, 사용량, 추정 비용 같은 요약 정보만 포함합니다.

API 키, `.env*` 파일, Vercel 로컬 설정, 실행 기록, 로컬 캐시, 비공개 작업 노트, 질문·답변 원문, 비공개 평가 자료, 비공개 원자료의 연결 정보는 공개하지 않습니다.

## 남아 있는 한계

- 자동 검사는 정해 둔 답변 기준을 지키는지 확인하며 임상적 진실을 대신하지 않습니다.
- 모델 응답에는 확률성이 있으므로 이번 반복 평가의 통과가 모든 미래 응답을 보증하지는 않습니다.
- 모든 언어 문서를 자동 검사했지만, 다섯 언어 전체 문장의 원어민 임상 편집 검토는 별도의 사람 검수 영역입니다.
