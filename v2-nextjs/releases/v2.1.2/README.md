# [2026-06-28] v2.1.2 버그 패치

v2.1.2는 코딩 도우미와 해석 도우미가 필요한 만큼 답하고, 중간에 끊기지 않으며, 임상가의 판단 범위를 넘지 않도록 답변 규칙과 검사 방법을 정리한 패치입니다. API 키 입력 창의 정렬을 조금 다듬은 것 외에는 기존 화면 구성을 유지했습니다.

## 핵심 요약

- 코딩 도우미와 해석 도우미의 기본 지시, 답변 길이, 답변 완료 여부, 글자가 순서대로 표시되는 방식을 한곳에서 관리하도록 정리했습니다.
- 코딩 도우미가 후보 코드와 임상가 검토 경계를 더 분명히 말하도록 보강했습니다. 행 자동 입력이나 자동 적용처럼 오해될 수 있는 답변은 막고, 근거가 부족하면 관찰 정보를 더 요청하도록 했습니다.
- 해석 도우미가 단일 지표만으로 진단, 치료, 법적 판단을 확정하지 않도록 답변 한계를 강화했습니다. 구조요약 값이 부족한 넓은 질문에는 없는 지표를 지어내지 않고 확인 순서를 안내하도록 했습니다.
- 실제 OpenAI API를 호출해 답변이 끝까지 작성되는지 확인하고, 최대 출력량에 도달해 중간에 끊긴 경우를 기록하도록 했습니다.
- AI가 질문과 관련된 코딩 또는 해석 참조 문서를 실제로 찾았는지 확인하는 자동 검사를 추가했습니다.
- BYOK 시작 모달의 모델명과 API 키 입력칸 정렬을 다듬어, OpenAI 모델 표시와 키 입력 흐름이 한 줄에서 더 자연스럽게 보이도록 조정했습니다.
- 푸터 버전, i18n 버전 문자열, 버전 아카이브를 v2.1.2로 갱신했습니다.

## 실제 API 호출 결과

- 개선 과정에서 문제를 발견한 검사 결과: 27개

## 공개 범위

공개 소스에는 앱 실행과 검토에 필요한 코드, 자동 검사, 참조 문서, AI 검사 보고서와 원문을 제외한 요약 기록을 포함했습니다. 운영 환경변수, API 키, Vercel 로컬 설정, 로컬 캐시, `node_modules`, AI 답변 원문, 비공개 작업 노트는 포함하지 않았습니다.
