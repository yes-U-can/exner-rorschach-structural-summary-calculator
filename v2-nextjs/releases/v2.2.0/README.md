# [2026-07-14] v2.2.0 마이너 릴리즈

> **현재 계산식 안내:** 아래의 UI/UX 변경 기록은 그대로 유효합니다. D/AdjD의 극단값, EBPer 표시 조건, GHR/PHR 판정 순서, Lower Section `FC:CF+C`의 Cn 누락, WDA%와 Afr의 0분모 처리는 v2.2.1에서 수정했습니다. v2.2.2에서는 Cn이 들어가는 계산과 들어가지 않는 계산을 각각 확인하고, 형태질(FQ)이 비어 있는 미완성 행의 GHR/PHR 분류를 보완했습니다. 계산기는 연령을 입력받지 않으며, 연령 제한이 있는 해석을 요청한 경우에만 AI 해석 대화에서 필요한 정보를 질문합니다. **현재 계산 기준은 [v2.2.2 문서](../v2.2.2/)를 확인해 주세요.**

## 먼저 확인하실 점

v2.2.0은 계산기를 중심에 둔 채 데스크톱 화면의 주요 메뉴를 왼쪽 사이드바로 모으고, 해석 도우미를 일반적인 AI 대화 화면에 가깝게 다시 구성한 v2.2.x 첫 릴리즈입니다. AI 답변 평가는 대화 원문을 저장하지 않고 미리 정한 이유만 선택하도록 만들었습니다.

계산식과 AI 답변 범위도 함께 점검했습니다. 이 릴리즈에서 확인한 계산 항목 가운데 추가 교정이 필요했던 항목은 v2.2.1과 v2.2.2에서 수정했습니다. AI 도우미는 Exner 종합체계 밖의 질문이나 시스템 지시를 빼내려는 요청에 답변 범위를 넓히지 않도록 제한하고, 한국어·일본어·영어 질문으로 실제 GPT-5.5 호출을 확인했습니다.

## 핵심 요약

### 공통 화면과 사이드바

- 기존 상단·하단 메뉴를 데스크톱의 고정 아이콘 막대, 본문 위에 열리는 사이드바, 모바일용 메뉴로 통합했습니다.
- 사이드바는 본문을 밀지 않고, 열린 상태에서 다른 페이지로 이동해도 유지되며, 언어·테마·AI 세션 제어를 한곳에 모읍니다.
- 로그인 전에는 AI 세션 시작, 로그인 후에는 AI 세션 종료만 표시하고, 종료 전에 확인 창을 거칩니다.
- 기존 계산기 단축키와 사이드바 단축키, 채점 화면 확대·축소·이동 방법을 모은 다국어 단축키 안내를 추가했습니다.
- 계산기, 해석 도우미, 참조 문서, 서비스 소개, 이용약관, 개인정보처리방침, 버전 아카이브의 배경과 라이트·다크 모드 색상을 통일했습니다.
- 서비스·약관·개인정보 페이지의 불필요한 외곽 테두리와 페이지 전체 복사 버튼을 제거했습니다.

### BYOK와 해석 도우미

- BYOK(Bring Your Own Key)는 사용자가 본인의 OpenAI API 키를 직접 연결해 선택형 AI 기능을 사용하는 방식입니다.
- OpenAI API 키와 `OpenAI GPT-5.5` 표시를 같은 줄에 배치하고, 중복 제목과 자동 모델 선택 안내를 제거해 세션 창을 간결하게 정리했습니다.
- API 키 비저장, 24시간 암호화 쿠키 보관, 세션 종료 시 삭제 안내를 중앙에 배치하고, 잘못된 키 안내가 나타나도 창 크기가 흔들리지 않게 했습니다.
- 해석 도우미를 외곽 카드 없는 전체 작업공간으로 바꾸고, 긴 대화는 페이지 전체가 아니라 대화 영역 안에서만 스크롤되게 했습니다.
- AI 답변을 따라가는 자동 스크롤은 사람이 위로 이동하면 멈추며, 필요할 때 최신 답변으로 다시 이동할 수 있습니다.
- 답변 중지, 메시지 복사, 이전 질문 위치 찾기, 반투명 입력창, 구조요약 붙여넣기 상태를 추가했습니다.
- 사용자와 AI 메시지에 복사 버튼을 제공하고, AI 메시지에는 좋아요·싫어요와 미리 정한 이유 선택 창을 추가했습니다.
- 질문·답변 원문과 자유 서술은 피드백 데이터베이스에 저장하지 않습니다. 도움이 됐는지, 선택한 이유, 사용 언어, 모델, 답변 완료 여부, 대략적인 길이처럼 정해 둔 최소 정보만 별도 데이터베이스에 180일 보관합니다.

### 채점 화면

- 행 이동 손잡이가 표 경계에 걸친 채 잘리지 않도록 표시 순서를 수정했습니다.
- 일반 클릭, `Shift + 클릭`, `Ctrl/Command + 클릭`을 이용한 여러 행 선택과 행을 옮길 때의 미리보기를 안정화했습니다.
- 값이 있는 선택 상자, 빈 선택 상자, 사용할 수 없는 선택 상자가 같은 크기와 정렬을 유지하고 명암으로만 구분되도록 했습니다.
- 일반적인 데스크톱 화면에서는 표가 한 화면에 맞고, 화면이 좁을 때에는 채점 영역 안에서만 가로로 이동하도록 열 너비를 조정했습니다.
- `Alt + 마우스 스크롤`은 마우스 위치를 중심으로 채점 화면 전체를 40~125% 범위에서 확대·축소하고, `Ctrl + 드래그`는 확대된 화면을 이동합니다.
- 행 순서 안내는 표 바로 아래의 푸른 영역에 표시합니다.
- 하단 버튼은 왼쪽의 추가·삭제·안내와 중앙의 결과 계산·입력값 초기화로 정리했습니다.

### 참조 문서와 버전 아카이브

- 참조 문서 검색창이 검색 결과와 상세 문서에서도 사라지지 않고, 검색어도 다음 화면에 이어지도록 했습니다.
- 검색 결과가 없을 때의 안내, 문서 분류 버튼, 문서 복사 완료 안내를 다섯 언어로 맞췄습니다.
- 참조 문서 본문을 외곽 카드 없는 읽기 화면으로 바꾸고 검색창·분류·본문의 중심선을 맞췄습니다.
- 버전 2와 GAS 버전 1 기록을 제목을 눌러 펼치고 접는 형태로 정리하고, 펼칠 때 세로 스크롤바가 생겨도 화면이 좌우로 흔들리지 않게 했습니다.
- GAS 실행 안내는 제목 옆의 키보드로도 열 수 있는 정보 도움말로 옮겼습니다.

## v2.2.0에서 확인한 계산 항목

v2.2.0에서는 다음 일곱 항목을 확인해 수정했습니다. 이 표는 v2.2.0에서 실제로 바뀐 내용을 보존한 기록이며, 현재 계산식은 뒤이은 v2.2.1과 v2.2.2의 교정까지 반영합니다.

| 항목 | 수정 내용 |
| --- | --- |
| EBPer | `EA >= 4`, M과 WSumC 양수, 비율 `>= 2.5`일 때만 표시 |
| 능동·수동 운동 | `Ma-p`, `FMa-p`, `ma-p`를 능동과 수동 양쪽에 각각 합산 |
| `3r+(2)/R` | 반사반응 `Fr+rF` 가중치를 2에서 표준식의 3으로 수정 |
| HVI | Zd 보조조건의 경계를 `> 3.0`에서 `> 3.5`로 수정 |
| ZEst | 마지막 유효 경계 `Zf=50`이 `173`을 반환하도록 수정 |
| D/AdjD | 음의 구간에서 JavaScript `-0`이 표시되지 않도록 정규화 |
| Lambda | 모든 반응이 순수 F일 때 0이 아니라 `∞`로 처리 |

계산 결과를 비교할 때에는 다음 네 종류의 자료를 사용했습니다.

- [Essentials of Rorschach Assessment의 sample computerized score reports](https://elmirmohammedmemorypsy.com/wp-content/uploads/2021/04/essentials-of-rorschach-assessment.pdf)
- [Engelman et al., “Why am I so stuck?”](https://www.therapeuticassessment.com/docs/Engelman_et_al_2016_copy.pdf)
- [Tibon Czopp et al. amnestic syndrome 사례](https://pubmed.ncbi.nlm.nih.gov/23985019/)와 [정오표](https://www.tandfonline.com/doi/pdf/10.1080/13554794.2014.910345)
- 초기 v1 개발에 참고한 [공개 배포 2019 Excel workbook](https://blog.naver.com/jin_k84/221539279596)의 실제 수식. workbook 내부 제작 표기는 `[Scoring Program] _by. Ju-Ri`이며, 실명을 추정하거나 원본 파일을 재배포하지 않습니다.

출판된 표와 공개 점수열이 서로 달랐던 두 항목은 어느 한쪽에 억지로 맞추지 않고 차이를 검산 문서와 반복 검사 설명에 남겼습니다. 자세한 수식, v2 코드의 해당 부분, 대표 기대값과 한계는 [v2.2.0 구조요약 계산 로직 검산 기록](../../source/docs/ops/2026-07-15-v2.2.0-calculation-audit.md)에 기록했습니다.

국가별 집단 평균은 동일 점수열의 계산 정답지가 아닙니다. 문화권 차이는 계산 산식이 아니라 부호화·규준·해석 적용에 주로 영향을 주므로, 한국·일본·영어권 자료는 GPT-5.5 문화·연령 시나리오 평가에 별도로 사용했습니다.

## AI 경계와 실전 평가

- R-PAS 일반 해석, MMPI 일반 해석, 무관한 질문, 숨겨진 지시나 비밀정보를 요구하는 요청을 Exner 종합체계 도우미의 답변 범위 밖으로 분류합니다.

## 공개 범위와 보안 경계

공개 소스에는 공통 화면 구조, UI 변경, 계산 수정, 자동 검사, 다섯 언어 화면 문구, 공개 참조 문서, 질문·답변 원문을 제외한 AI 검사 요약과 기술 보고서를 포함합니다.

API 키, `.env*`, Vercel 로컬 설정, Neon 연결 정보, 질문·답변 원문, 대화 내용, 반응 메모, 구조요약 원문, 환자 정보, 비공개 자료의 출처 연결 정보, 제3자 Excel 원본 파일, 운영 DB 정보, 실행 기록과 캐시는 공개하지 않습니다.

GPT-5.5 고정 모델, 본인의 API 키를 연결하는 BYOK 방식, API 키와 대화 원문 비저장, 최종 판단을 사람이 맡는 HITL(Human-in-the-Loop) 원칙은 유지합니다. 계산과 AI 도우미의 자동 검증 통과는 임상적 타당성 인증이나 모든 미래 응답에 대한 보증을 뜻하지 않습니다.

## 남아 있는 한계

- 이번 계산 검산이 모든 가능한 반응 조합을 수학적으로 증명한 것은 아닙니다.
- 계산기는 연령을 직접 입력받지 않으므로 만 14세 이하 S-CON 해석 보류는 사용하시는 임상가가 확인해야 합니다.
- GPT-5.5 검사는 개발 과정에서 마련한 질문으로 수행했으며, 처음 보는 임상 자료를 이용한 별도의 성능 연구는 아닙니다.
- 실제 임상 유용성, 다국어 문장 품질, 안전성은 전문가의 사람 검토가 계속 필요합니다.
- v2.2.0은 desktop 중심의 첫 v2.2.x UX 릴리즈입니다. 모바일 전용 동선과 세부 최적화는 후속 v2.2.x에서 계속합니다.
