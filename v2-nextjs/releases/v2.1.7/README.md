# [2026-07-05] v2.1.7 버그 패치

## 먼저 확인하실 점

v2.1.7은 공개 README, 변경 기록, 릴리즈 노트, 버전 아카이브, AI 검사 문서를 일관된 방식으로 작성하기 위한 첫 문서 원칙을 마련한 패치입니다.

앱 UI/UX와 AI 답변 방식은 변경하지 않았습니다. 공개할 수 있는 검사 결과와 비공개로 남겨야 하는 API 키, 질문·답변 원문, 평가 자료를 구분하고, 한국어 공식 기록과 영어 개요의 역할을 정했습니다.

> **2026-07-17 문서 안내 보완:** 이후 패치노트에서는 계산 결과가 어떻게 달라졌는지, 어떤 프로토콜이 영향을 받을 수 있는지, 기존 결과를 다시 계산해야 하는지를 먼저 설명합니다. 확인에 사용한 근거와 개발 세부사항은 그 뒤에서 따로 확인할 수 있도록 정리했습니다.

## 후속 기록 (2026-07-13)

처음 쓴 요약에는 v2.1.7로 v2.1.x의 AI 품질 작업 전체를 마무리한 것처럼 적었습니다. v2.1.7에서 실제로 완료한 범위는 **공개 릴리즈 문서 작성 원칙**입니다. AI 참조 문서와 다국어 검색 개선은 v2.1.8-v2.1.10에서 이어서 다뤘습니다.
