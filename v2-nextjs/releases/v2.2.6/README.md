## 주요 수정사항

### 개요

다섯 언어 페이지를 검색엔진과 링크 미리보기가 읽을 때 일부 언어 정보가 화면과 다르게 전달되던 문제를 수정했습니다. 채점 화면에서는 `Alt+마우스 휠` 확대·축소가 일부 Windows 브라우저 입력에서 화면 이동으로 처리되던 문제도 함께 고쳤습니다.

채점표의 [Score]와 [G/PHR] 헤더에 마우스를 올렸을 때 나타나는 설명에서는 [Card], [Z], [Contents], [Determinants], [FQ], [Special Score]처럼 관련 항목 이름을 모두 대괄호로 구분했습니다.

이번 패치는 구조요약 계산식, 채점 입력값, 화면 배치, AI 도우미의 답변 규칙을 변경하지 않습니다. 기존 프로토콜을 다시 계산할 필요도 없습니다.

### 다섯 언어 검색 정보

언어 표시가 없는 기본 주소는 한국어 페이지입니다. 그러나 서비스 소개, 이용약관, 개인정보처리방침, 참조 문서, 버전 기록의 기본 주소에서는 화면과 본문이 한국어인데도 검색 제목과 대표 주소가 영어로 전달될 수 있었습니다. 이제 이 주소들은 화면, 검색 제목, 설명, 대표 주소가 모두 한국어로 일치합니다.

브라질 포르투갈어 페이지 주소는 기존 링크와 호환되도록 `?lang=pt`를 그대로 사용합니다. 검색엔진과 보조 기술에 전달하는 언어 표시는 브라질 포르투갈어를 뜻하는 `pt-BR`로 바로잡았습니다.

검색할 수 있는 각 페이지는 다음 정보를 같은 기준으로 제공합니다.

- 현재 페이지의 대표 주소
- 한국어, 영어, 일본어, 스페인어, 브라질 포르투갈어 대체 주소
- 언어를 정하지 않았을 때 사용할 한국어 기본 주소
- 검색 결과와 링크 공유에 사용하는 언어별 제목, 설명, 대표 이미지

기존 즐겨찾기와 외부 링크의 주소는 바뀌지 않습니다.

### 채점 화면 확대·축소

일부 Windows 브라우저에서는 `Alt`를 누른 상태가 마우스 휠 이벤트에 온전히 전달되지 않아, 채점 화면이 확대·축소되지 않고 이동하는 경우가 있었습니다. 이제 키보드의 `Alt` 상태를 함께 확인하고 확대·축소를 화면 이동보다 먼저 처리합니다.

`Ctrl+마우스 휠`로 브라우저 화면 전체를 확대·축소하는 기존 동작은 그대로 유지합니다. 채점 화면의 최소·최대 배율, 마우스 포인터를 중심으로 확대하는 방식, 이동 가능한 여백도 바꾸지 않았습니다.

## 테스트 및 검증

- 언어 표시가 없는 주요 공개 페이지가 모두 한국어 제목, 설명, 대표 주소를 내보내는지 확인했습니다.
- 다섯 언어의 모든 검색 대상 주소가 자기 자신을 대표 주소로 가리키고, 다른 네 언어와 한국어 기본 주소를 함께 안내하는지 검사했습니다.
- 브라질 포르투갈어의 페이지 언어, 대체 언어 링크, 사이트맵과 구조화 데이터가 `pt-BR`로 일치하는지 확인했습니다.
- AI 전용 경로와 API 경로가 검색 대상에서 제외되는 기존 설정을 다시 확인했습니다.
- `Alt`, Windows의 대체 입력 상태, `Ctrl`이 함께 전달되는 경우를 나누어 채점 화면 확대·축소와 브라우저 확대·축소가 서로 침범하지 않는지 반복 검사했습니다.
- 다섯 언어의 채점표 헤더 설명에서 관련 항목 이름이 같은 대괄호 표기를 사용하는지 검사했습니다.
- 전체 자동 검사에서는 82개 테스트 파일의 458개 검사가 통과했고, 실행 조건이 없는 7개는 건너뛰었습니다. 배포용 빌드, 코드 정적 검사, 다섯 언어 문구 감사, 비밀정보 및 의존성 감사도 통과했습니다.

자동 검사는 검색 메타데이터와 입력 이벤트의 처리 규칙을 확인합니다. 실제 검색 결과의 갱신 시점은 검색엔진의 재수집 주기에 따르며, 실제 키보드와 마우스의 조작감은 배포 화면에서도 함께 확인합니다.

## UI/UX, 개인정보와 계산 범위

- 새 화면이나 새 입력 항목을 추가하지 않았습니다.
- 화면에 보이는 메뉴와 배치는 변경하지 않았습니다.
- 채점표 헤더의 설명 문구만 같은 표기 방식으로 정리했습니다.
- 구조요약 계산식과 결과값은 변경하지 않았습니다.
- 새로운 개인정보를 수집하지 않습니다.
- 채점 자료와 OpenAI API 키를 서버 데이터베이스에 저장하지 않는 기존 원칙을 유지합니다.
- AI 코퍼스, 임베딩, 코딩·해석 도우미의 답변 규칙은 변경하지 않았습니다.

## 공개 범위와 보안 경계

공개 소스에는 언어별 검색 메타데이터, 대표 주소와 대체 언어 링크, 사이트맵, 채점 화면 확대·축소 처리와 관련 자동 검사를 포함합니다.

운영 환경변수, API 키, 비공개 배포 설정, 로컬 경로와 작업 기록은 공개하지 않습니다.
