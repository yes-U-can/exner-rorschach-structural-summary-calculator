# [2026-07-13] v2.1.9 버그 패치

## 먼저 확인하실 점

v2.1.9는 코딩 도우미와 해석 도우미가 질문에 맞는 참조 문서를 더 안정적으로 찾도록 검색 방식을 개선한 패치입니다. RAG는 답변 전에 관련 문서를 찾아 GPT-5.5에 함께 전달하는 방식입니다.

앱 UI/UX는 변경하지 않았습니다. 다섯 언어의 단어 검색과 의미 검색을 함께 사용하는 방식, 코딩 도우미가 어떤 행을 검색에 사용하는지, Google 검색 자료가 남지 않고 OpenAI 자료만 사용되는지를 점검했습니다.

## 핵심 요약

- 한 글자 자연어가 로샤 코드로 오인되던 문제를 줄이면서 `Card I`, `Content A`, 소문자 결정인 `m`처럼 문맥상 명시된 코드는 보존했습니다.
- 한국어 조사·어미와 일본어 단어 분절을 보강하고, `3r+(2)/R` 같은 복합 지표를 내부의 짧은 기호보다 우선하도록 수정했습니다.
- 단어가 정확히 겹치는 정도와 의미가 비슷한 정도를 직접 더하지 않고, 두 검색 결과의 순위를 안정적으로 합치는 RRF 방식을 사용했습니다.
- 의미가 너무 약한 결과를 제외하고 같은 참조 문서가 여러 번 나타나지 않도록 했습니다.
- 코딩 도우미는 현재 선택한 행과 사람이 명시적으로 함께 선택한 행만 검색 질문에 사용하도록 제한했습니다.
- 실제 API 키 세션, 구조요약 CSV 검사, 참조 문서 검색, GPT-5.5의 순차 답변, 임상가 최종 판단 범위를 한꺼번에 확인하는 실제 API 호출 검사를 추가했습니다.
- 검색 자료를 만든 시각의 표기를 바로잡고, OpenAI 이외의 검색 자료가 남아 있으면 릴리즈 전 검사를 통과하지 못하도록 했습니다.

## 어떻게 확인했나요?

- 다섯 언어의 준비된 참조 문서 검색 질문: 365/365 통과
- 25개 집중 검색 질문: 넓은 질문의 첫 결과 적중 10% → 100%, 이름을 지정한 질문의 상위 4개 안 적중 90% → 100%, 넓은 코딩 질문에 다른 종류 문서가 섞인 비율 70% → 0%
- GPT-5.5 최종 한 번 묻는 질문: 27/27 통과, 발견된 문제 0, 실행 실패 0
- GPT-5.5 이어 묻기 대화: 대화 4건에서 질문·답변 8회 모두 완료
- 실제 AI 대화 경로: 정상 응답, 답변 완료, 발견된 문제 0
- 기존 Google 검색 자료 6,597개 → 0개, OpenAI 검색 자료 6,632개 보존
- 예상하지 않은 제공자 자료 0, 이전 내용이 남은 자료 0, 다섯 언어 모두 준비
- 전체 계산·기능 자동 검사: 45개 파일, 194개 통과, 의도적으로 제외한 검사 5개
- 배포용 페이지 생성: 221/221 통과
- 저장된 AI 검사 기록에 질문·답변 원문과 API 키가 없음을 확인
- 외부 프로그램 의존성에서 알려진 취약점 0

검토 중 스페인어 3건과 포르투갈어 2건에서 검색 결과가 다시 나빠지는 문제가 실제로 발견됐습니다. 실패한 질문을 그대로 재현하는 예시를 먼저 추가하고 검색 규칙을 고친 뒤, 다섯 언어 질문 365개를 처음부터 다시 확인했습니다.

수정 전후 검색 결과, 검색 방식, 실제 비용과 남은 한계는 [v2.1.9 참조 문서 검색 개선 보고서](../../source/docs/ai-evals/2026-07-13-v2.1.9-rag-retrieval-hardening-report.md)에 기록했습니다.

## 공개 범위와 보안 경계

공개 AI 검사 기록에는 검사 항목 번호, 언어, 참조 문서 경로, 검색 순위, 종합 점수, 완료 상태, 발견된 문제 종류, 사용량, 답변 길이, 추정 비용 같은 요약 정보만 포함합니다.

API 키, `.env*`, 질문·답변 원문, 대화 내용, 비공개 평가 자료, 환자 정보, 비공개 원자료와 그 연결 정보, Vercel 로컬 설정, 실행 기록과 캐시는 공개하지 않습니다.

이번 패치는 검색과 평가 방식을 개선하지만 AI가 최종 코드를 자동 확정하거나, 진단을 내리거나, 임상가의 판단을 대체하도록 허용하지 않습니다. 본인의 API 키를 연결해 사용하는 BYOK(Bring Your Own Key) 방식과, 최종 판단을 사람이 맡는 HITL(Human-in-the-Loop) 원칙은 그대로 유지합니다.

## 남아 있는 한계

- 25개 집중 질문은 검색 방식을 고치는 데 사용했으므로, 수정 과정과 완전히 분리해 처음 공개한 별도 시험 자료는 아닙니다.
- 365개 준비 질문은 같은 문제가 다시 생기는 것을 찾는 데 유용하지만, 정해 둔 표현이 많아 실제 임상가 질문의 정확도를 그대로 추정하지는 못합니다.
- 자동 검사는 정해 둔 답변 기준을 확인하며 임상적 정확성을 증명하지 않습니다.
- GPT-5.5 응답에는 확률성이 있으므로 이번 통과가 모든 미래 응답을 보장하지 않습니다.
- 실제 임상 유용성, 다국어 문장 품질, 안전성은 전문가의 사람 검토가 계속 필요합니다.
