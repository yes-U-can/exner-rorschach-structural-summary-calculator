# v1 GAS Archive

[한국어](./README.md) | [English](./README.en.md) | [日本語](./README.ja.md) | [Español](./README.es.md) | [Português (Brasil)](./README.pt-BR.md)

This directory preserves the version 1 Google Apps Script releases of the Rorschach Structural Summary calculator.

- `current/`: source for `v1.4.1`, the final v1 deployment.
- `releases/`: patch notes and source code for each version.
- The `source/` directory in each release contains files that can be pasted directly into a GAS project.
