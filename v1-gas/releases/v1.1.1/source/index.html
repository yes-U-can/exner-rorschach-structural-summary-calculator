안녕하세요, 서울임상심리연구소에서 제공하는 간편 로샤 채점 도우미입니다. 이 메시지를 클릭하여 토글을 열고 이용법을 확인하세요!

【 작동 방법 】

  • ① Scoring 탭에서 채점 내용을 입력합니다.
  • ② Scoring 탭에 처음 들어가면, 저장된 내용이 없을 경우 테스트용 샘플 데이터를 불러올지 묻습니다. 새로 입력하려면 취소를 누르세요.
  • ③ 입력 내용은 브라우저에 자동 저장됩니다. 재접속 시 "이전에 자동 저장된 내용이 있습니다. 불러올까요?"라는 메시지가 뜨며, 확인을 누르면 이어서 작업할 수 있습니다.
  • ④ 채점 내용을 하나도 입력하지 않고 Calculate Results을 누르면 "Error: No responses entered." 메시지가 나타납니다.
  • ⑤ 채점 완료 후 Calculate Results을 누르면 결과 화면으로 이동합니다. 이전 화면으로 돌아가려면 Go Back을 누르세요.

【 Scoring 탭에 점수 입력 시 유의사항 】

  • ① [Response] 항목은 피검자의 언어 반응을 메모하는 공간이며, 계산에 영향을 주지 않으므로 비워둬도 괜찮습니다.
  • ② [Determinants], [Contents], [Special Score]에 여러 코드를 입력할 경우, 왼쪽 칸부터 순서대로 채워주세요.
  • ③ [Score]와 [G/PHR] 항목은 자동으로 계산되므로 직접 입력할 필요가 없습니다. 단, [Score]는 [Card]와 [Z] 유형을 선택해야 값이 자동 계산됩니다.

【 더 자세한 정보는 아래를 참고하세요 】

  • ① 프로그램 이용 안내 챗봇 (Google 계정이 있어야 이용 가능하며, Gemini로 연결됩니다.)
  • ② 네이버 블로그: 개선사항 및 버그 제보
  • ③ 프로그램 패치 노트

【 구조 요약지 인쇄 】

  • ③ 가장 마지막 페이지에는 [Response]에 입력한 메모들이 출력되므로, 불필요하다면 프린트 설정에서 출력 범위를 지정하세요.

  • ⑤ 세부 설정에서 시트당 페이지 수를 (2)로 변경하면 Upper/Lower Section 탭이 1매 안에 모두 들어옵니다.

  • ③ 데이터 분석 시 유용하게 사용하세요.
